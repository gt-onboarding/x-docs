<div id="educators">
  ## Educadores
</div>

Leve a API para a sala de aula para ensinar aos alunos fundamentos de programação e
habilidades valiosas de processamento de dados.

**Recursos:**

Você é educador e quer usar a API do X no seu curso? Este
[guia para ensinar com a API do X](https://dev.to/suhemparack/a-guide-to-teaching-with-the-twitter-api-v2-3n08)
ajudará você a entender a API do X, por que ela é usada por educadores, que tipo de
disciplinas estão sendo ministradas com a API e tudo o que você precisa saber para
começar.

**Destaque:**

Os currículos abaixo foram desenvolvidos e aplicados usando a API do X v2:

- [Social Media Data Mining and Analysis](https://canvas.uw.edu/courses/1434897)
  por Dra. Tanu Mitra

- [Data Science & Society](https://dssoc.github.io/) por Dr. Chris Bail

- [Introduction to Cultural Analytics & Python](https://melaniewalsh.github.io/Intro-Cultural-Analytics/welcome.html)
  por Dra. Melanie Walsh

- [Oxford Spring School in Advanced Research Methods, Analysing X Data](https://github.com/cjbarrie/oxspsch)
  por Dr. Chris Barrie
