<div id="requesting-additional-app-level-permissions">
  ### Solicitação de permissões adicionais no nível do App
</div>

Se seu aplicativo estiver limitado a quaisquer das permissões de nível de app abaixo e você desejar acessar endpoints adicionais, entre em contato com seu representante do X para solicitar **Acesso Padrão**.

Desenvolvedores que solicitaram acesso à API de Anúncios do X antes de julho de 2023 podem ter solicitado quaisquer das seguintes permissões de nível de app, que agora estão incluídas no nível de Acesso Padrão:

- **Analytics** (somente leitura): O usuário tem acesso aos endpoints de [Analytics](https://developer.x.com/en/docs/twitter-ads-api/analytics/overview) com acesso [somente leitura](https://developer.x.com/en/docs/apps/app-permissions). O usuário não pode acessar os endpoints de Gerenciamento de Campanhas, Criativos ou Públicos personalizados, mas poderá extrair métricas ao incluir entidades de criativos nos parâmetros da solicitação dos endpoints de Analytics. Para obter a granularidade de media_id ou media_name, o aplicativo do usuário precisaria ter acesso de escrita à API de Anúncios do X.

- **Gerenciamento de Campanhas e Criativos** (leitura e escrita): O usuário tem acesso aos endpoints de [Gerenciamento de Campanhas](https://developer.x.com/en/docs/twitter-ads-api/campaign-management/overview) e de [Criativos](https://developer.x.com/en/docs/twitter-ads-api/creatives/overview) com acesso de [leitura e escrita](https://developer.x.com/en/docs/apps/app-permissions). O usuário também tem acesso aos endpoints de Analytics.

- **Públicos personalizados** (leitura e escrita): O usuário tem acesso aos endpoints de [Públicos personalizados](https://developer.x.com/en/docs/twitter-ads-api/audiences/overview) com acesso de [leitura e escrita](https://developer.x.com/en/docs/apps/app-permissions). O usuário também tem acesso aos endpoints de Gerenciamento de Campanhas, Criativos e Analytics.

- **Conversões** (leitura e escrita): O usuário tem acesso aos endpoints de [Conversões em dispositivos móveis](https://developer.x.com/en/docs/twitter-ads-api/measurement/mobile-conversions/overview) e de [Conversões na Web](https://developer.x.com/en/docs/twitter-ads-api/measurement/web-conversions/overview) com acesso de [leitura e escrita](https://developer.x.com/en/docs/apps/app-permissions). O usuário pode ou não ter acesso a outros endpoints da API de Anúncios do X. 

<div id="requesting-an-increase-to-user-oauth-token-limits">
  ### Solicitando aumento no limite de tokens OAuth de usuário
</div>

Desenvolvedores que solicitaram acesso à API de Anúncios antes de julho de 2023 podem ter um limite de 5 tokens OAuth de usuário no aplicativo. Se você precisar aumentar o limite de tokens do seu App de desenvolvedor, entre em contato com seu representante do X.

<Note>**Observação:** Não aumentamos os [limites de taxa](https://developer.x.com/en/docs/twitter-ads-api/rate-limiting) de nenhum endpoint da API de Anúncios. Consulte nossas [práticas recomendadas](https://developer.x.com/en/docs/twitter-ads-api/rate-limiting) para saber como permanecer dentro desses limites.</Note>
