<div id="community-tools-and-libraries">
  ## Ferramentas e bibliotecas da comunidade
</div>

Estas são algumas das muitas bibliotecas mantidas pela comunidade que cobrem a API de Anúncios do X, em várias linguagens de programação e plataformas. As bibliotecas listadas aqui devem dar suporte à maioria dos recursos da API de Anúncios do X, salvo indicação em contrário — confirme os detalhes com os autores.

Se você desenvolveu uma biblioteca que seja compatível com as APIs do X, conte para nós nos [fóruns da Comunidade de Desenvolvedores](https://devcommunity.x.com/c/libraries-and-sdks/63).
