title: Currency
---

O tipo de moeda é identificado pelo padrão [ISO-4217](http://en.wikipedia.org/wiki/ISO_4217). É uma sequência de três letras, como `USD` ou `EUR`. O valor de uma moeda é representado em micros. Para USD, US$ 5,50 é codificado como 5,50\*1e6, ou 5.500.000. Para representar um “valor inteiro”, é necessário multiplicar o micro local por 1e6 (1\_000\_000) para todas as moedas.

A API de Anúncios do X reconhece as moedas oficiais de todos os países onde o produto X SMB Ads está disponível. Para a lista completa desses países, clique [aqui](https://business.x.com/en/help/overview/about-eligibility-for-twitter-ads.html).

A maioria das moedas tem “precisão” de duas casas (como USD: US$ 1,23).

- `AED`, `VEF`, `PKR`, `IDR`, `HKD`, `QAR`, `PHP`, `NZD`, `INR`, `EUR`, `ARS`, `THB`, `MXN`, `EGP`, `DZD`, `SAR`, `LBP`, `COP`, `TRY`, `SGD`, `MAD`, `AUD`, `USD`, `MYR`, `CAD`, `BRL`, `GBP`.

As seguintes moedas não usam casas decimais:

- `VND`, `KRW`, `CLP`, `JPY`

E estas usam três casas decimais:

- `KWD`, `IQD`, `BHD`, `TND`

O uso de moeda é feito por meio de [funding instruments](/pt-BR/x-ads-api/introduction).
