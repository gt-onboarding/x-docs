<div id="basic-authentication">
  ## Autenticação básica
</div>

Muitas APIs empresariais do X exigem o uso de autenticação básica HTTP. Para fazer uma solicitação bem-sucedida a uma API que requer autenticação básica, você deve enviar, em cada solicitação, um cabeçalho de autorização com uma combinação válida de endereço de e-mail e senha. Essa combinação é a mesma que você usa para acessar o [console de API enterprise](https://console.gnip.com/) e pode ser editada nesse próprio console.

Ao construir uma solicitação usando autenticação básica, certifique-se de adicionar o cabeçalho HTTP Authorization: Basic com as credenciais codificadas, via HTTPS.

No exemplo de solicitação cURL a seguir, substitua `<email_address>` e `<password>` pelas suas credenciais antes de enviar a solicitação:

```bash
curl -v --compressed -u<email_address>:<password>
    "https://gnip-api.x.com/search/30day/accounts/<account-name>/prod/counts.json?query=from%3Axdevelopers"
```
