<div id="comparing-x-apis-manage-posts-endpoints">
  ## Comparando os endpoints de gerenciamento de Posts da API do X
</div>

Os endpoints v2 de gerenciamento de Posts substituirão os endpoints padrão v1.1 [POST statuses/update](https://developer.x.com/en/docs/twitter-api/v1/tweets/post-and-engage/api-reference/post-statuses-update) e [POST statuses/destroy/:id](https://developer.x.com/en/docs/twitter-api/v1/tweets/post-and-engage/api-reference/post-statuses-destroy-id). Se você tem código, apps ou ferramentas que usam a versão v1.1 dos endpoints de gerenciamento de Posts e está avaliando migrar para o endpoint mais recente da API do X v2, este conjunto de guias é para você.

As tabelas a seguir comparam os endpoints padrão v1.1 e os endpoints de gerenciamento de Posts da API do X v2:

<div id="create-a-post">
  ### Criar um post
</div>

| Descrição | v1.1 padrão | API do X v2 |
| :--- | :--- | :--- |
| Métodos HTTP compatíveis | POST | POST |
| Domínio do host | https://api.x.com | https://api.x.com |
| Caminho do endpoint | /1.1/statuses/update.json | /2/tweets |
| [Autenticação](/pt-BR/resources/fundamentals/authentication) | Contexto de usuário OAuth 1.0a | Contexto de usuário OAuth 1.0a<br /><br />OAuth 2.0 com PKCE (Código de Autorização) |
| Limites padrão de [taxa de requisições](/pt-BR/x-api/fundamentals/rate-limits) | Nenhum<br /><br />300 solicitações por janela de 3 horas por usuário, por app. Compartilhado com o endpoint POST Retweets v1.1. | 200 solicitações por 15 min por usuário<br /><br />300 solicitações por janela de 3 horas por usuário, por app. Compartilhado com o endpoint POST Retweets v2. |
| Requer o uso de credenciais de um [App de desenvolvedor](/pt-BR/resources/fundamentals/developer-apps) associado a um [Projeto](/pt-BR/resources/fundamentals/projects) |     | ✔   |

<div id="delete-a-post">
  ### Excluir um post
</div>

| Descrição | v1.1 padrão | API do X v2 |
| :--- | :--- | :--- |
| Métodos HTTP compatíveis | POST | DELETE |
| Domínio do host | https://api.x.com | https://api.x.com |
| Caminho do endpoint | /1.1/statuses/destroy/:id.json | /2/tweets/:id |
| [Autenticação](/pt-BR/resources/fundamentals/authentication) | Contexto de usuário OAuth 1.0a | Contexto de usuário OAuth 1.0a<br /><br />OAuth 2.0 com PKCE |
| [Limites de taxa](/pt-BR/x-api/fundamentals/rate-limits) padrão da solicitação | Nenhum | 50 solicitações por 15 min por usuário |
| Exige o uso de credenciais de um [App de desenvolvedor](/pt-BR/resources/fundamentals/developer-apps) associado a um [Projeto](/pt-BR/resources/fundamentals/projects) |     | ✔   |
