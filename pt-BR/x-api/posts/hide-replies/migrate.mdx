  ## Comparando os endpoints de ocultar respostas da API do X
</div>

O endpoint de ocultar respostas da v2 está substituindo o endpoint de ocultar respostas do Labs. Se você tem código, apps ou ferramentas que usam a versão do Labs desse endpoint e está considerando migrar para o endpoint mais recente da API do X v2, este guia é para você.

Para usar a nova API do X v2 (incluindo o endpoint de ocultar respostas), você precisará [aderir ao novo portal de desenvolvedores](https://developer.x.com/en/portal/opt-in), criar um [Projeto](/pt-BR/resources/fundamentals/projects) e adicionar um App a esse Projeto. Em seguida, você pode usar as credenciais associadas a esse App para fazer solicitações ao endpoint de ocultar respostas. Adicionar o mesmo App que está inscrito no endpoint de ocultar respostas do Labs v2 manterá seus usuários autenticados.

A tabela a seguir compara as diferenças entre o Labs e o endpoint mais recente da API do X v2:
 

| **Descrição** | **Labs v2** | **API do X v2** |
| :--- | :--- | :--- |
| Domínio do host | https://api.x.com | https://api.x.com |
| Caminho do endpoint | /labs/2/tweets/:id/hidden | /2/tweets/:id/hidden |
| Autenticação | Contexto de usuário OAuth 1.0a | Contexto de usuário OAuth 1.0a |
| Métodos HTTP compatíveis | PUT | PUT |
| Limites padrão de requisições | 10 solicitações por 15 minutos (compartilhado entre todos os usuários autenticados) | 50 solicitações por 15 minutos (para cada usuário autenticado) |
| Pode ocultar respostas | ✔︎  | ✔︎  |
| Pode desocultar uma resposta anteriormente oculta | ✔︎  | ✔︎  |
| Pode ocultar ou desocultar respostas várias vezes | ✔︎  | ✔︎  |
| Requer o uso de credenciais de um [App de desenvolvedor](/pt-BR/resources/fundamentals/developer-apps) associado a um [Projeto](/pt-BR/resources/fundamentals/projects) |     | ✔   |
