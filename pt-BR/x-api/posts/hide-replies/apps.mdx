<div id="apps-using-the-hide-replies-endpoint">
  ## Apps que usam o endpoint de ocultar respostas
</div>

Aqui está uma coleção de apps de destaque criados por desenvolvedores que integraram o endpoint de ocultar respostas. Ao combinar suas tecnologias com a API do X, eles criaram experiências úteis para ajudar todos os usuários a aprimorar a conversa pública.

![Clarabridge](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/clarabridge.jpg.twimg.1920.jpg)

**Clarabridge**

A Clarabridge integra a ocultação de respostas para que as empresas mantenham seu feed do X limpo e agradável de ler. As respostas podem ser ocultadas manualmente, e agentes também podem usar a análise de texto da Clarabridge para detectar automaticamente palavrões e outras formas de linguagem que não estejam alinhadas com a política da marca.

[Ir para o site ▸](https://www.clarabridge.com/ "Ir para o site ▸")

[Clarabridge no X](https://x.com/Clarabridge "Clarabridge no X")

![Perspective API template app](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/perspective-logo.png.twimg.1920.png)

**App modelo da Perspective API**

Perspective é uma inteligência artificial treinada pela Jigsaw (uma unidade da Alphabet) para detectar comentários tóxicos. Este app detecta as respostas recebidas por uma pessoa e oculta automaticamente uma resposta com base na “pontuação” que indica o grau de confiança da Perspective de que um comentário é semelhante a comentários tóxicos que ela já viu. Este é um app modelo de código aberto desenvolvido pelo X. Incentivamos você a personalizá-lo para criar ferramentas para seus usuários.

[Usar app ▸](https://quintessential-yoke.glitch.me "Usar app ▸")

[Jigsaw no X](https://x.com/jigsaw "Jigsaw no X")

![Reshuffle](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 150 150%22%3E%3C/svg%3E)

**Reshuffle**

Reshuffle é uma plataforma para conectar aplicativos de negócios com scripts flexíveis. Eles criaram um script de integração para detectar e ocultar respostas que incluam as palavras-chave que você definir. Para experimentar, certifique-se de ter um app válido habilitado para Ocultar respostas e, em seguida, siga as instruções no repositório.

[Ir para o app ▸](https://github.com/reshufflehq/reshuffle "Ir para o app ▸")

[Reshuffle no X](https://x.com/reshufflehq "Reshuffle no X")

![Ocultar respostas indesejadas](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 396 397%22%3E%3C/svg%3E)

**Ocultar respostas indesejadas**

Dara Oladosu, o desenvolvedor por trás do popular app QuotedReplies, criou um app que oculta automaticamente respostas que atendem a alguns dos critérios que ele determinou serem mais propensos a indicar comportamento abusivo, incluindo respostas que contêm certas palavras-chave que ele já silenciou.

[Ir para o app ▸](https://hideunwantedreplies.com/ "Ir para o app ▸")

[Dara Oladosu no X](https://x.com/dara_tobi "Dara Oladosu no X")

![Pandaflow](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 94 94%22%3E%3C/svg%3E)

**Pandaflow**

Pandaflow é uma plataforma no-code para automatizar fluxos de trabalho. Eles integraram o endpoint para que os usuários possam ocultar respostas programaticamente com base em um fluxo personalizado que definirem.
