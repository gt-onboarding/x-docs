Este endpoint permite ocultar ou reexibir respostas de forma programática com base em critérios que você definir. Assim como na experiência principal do X, as respostas ficarão ocultas da conversa principal, mas continuarão visíveis em uma página separada. Você pode usar o endpoint para criar apps que ajudem as pessoas a ocultar respostas que considerem sem valor.

O endpoint de ocultar respostas usa [OAuth 1.0a User Context](/pt-BR/resources/fundamentals/authentication#oauth-1-0a-2) ou [OAuth 2.0 Authorization Code with PKCE](pt-BR/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) para autenticação. Se bem-sucedido, o endpoint oculta uma única resposta publicada em uma conversa de Post iniciada por um usuário autenticado. Cada conversa permite ocultar até 725 Posts.

<Note>
  **Configuração da conta**

  Para acessar esses endpoints, você precisará de:

  - Uma [conta de desenvolvedor](https://developer.x.com/en/portal/petition/essential/basic-info) aprovada.
  - Autenticar-se usando as chaves e tokens de um [App de desenvolvedor](/pt-BR/resources/fundamentals/developer-apps) que esteja dentro de um [Projeto](/pt-BR/resources/fundamentals/projects).

  Saiba mais sobre como obter acesso aos endpoints da API do X v2 em nosso [guia de primeiros passos](/pt-BR/x-api/getting-started/getting-access).
</Note>
