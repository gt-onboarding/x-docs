<div id="comparing-x-apis-retweets-endpoints">
  ## Comparando os endpoints de Retweets da API do X
</div>

**Consulta de Retweets**

O endpoint de consulta de Retweets da v2 substituirá os endpoints padrão [v1.1 GET statuses/retweets/:id](https://developer.x.com/en/docs/twitter-api/v1/tweets/post-and-engage/api-reference/get-statuses-retweets-id) e [v1.1 GET statuses/retweets/:ids](https://developer.x.com/en/docs/twitter-api/v1/tweets/post-and-engage/api-reference/get-statuses-retweeters-ids).

As tabelas a seguir comparam os endpoints de Retweets do padrão v1.1 e da API do X v2:

| Descrição | Padrão v1.1 | API do X v2 |
| :--- | :--- | :--- |
| Métodos HTTP compatíveis | GET | GET |
| Domínio do host | https://api.x.com | https://api.x.com |
| Caminho do endpoint | /1.1/retweeters/id.json<br /><br />`/1.1/retweets/ids.json` | /2/users/:id/retweeted\_by |
| [Autenticação](/pt-BR/resources/fundamentals/authentication) | OAuth 1.0a (contexto do usuário) | OAuth 2.0 Bearer Token<br /><br />OAuth 1.0a (contexto do usuário) |
| Limites padrão de [taxa de solicitação](/pt-BR/resources/fundamentals/rate-limits) | 75 solicitações por 15 min | 75 solicitações por 15 min (por App)<br /><br />75 solicitações por 15 min (por usuário) |
| Formato de dados | Formato padrão v1.1 | [Formato da API do X v2](/pt-BR/x-api/fundamentals/data-dictionary) (determinado pelos parâmetros de solicitação fields e expansions; não é compatível com os formatos v1.1)<br /><br />Para saber mais sobre como migrar do formato padrão v1.1 para o formato da API do X v2, visite nosso [guia de migração de formatos de dados](/pt-BR/x-api/migrate/data-format-migration). |
| Requer o uso de credenciais de um [App de desenvolvedor](/pt-BR/resources/fundamentals/developer-apps) associado a um [Projeto](/pt-BR/resources/fundamentals/projects) |     | ✔️  |

<div id="manage-retweets">
  ### Gerenciar Retweets
</div>

As tabelas a seguir comparam o endpoint de desfazer Retweet do v1.1 padrão e da API do X v2:

**Retweetar um Post**

| Descrição | v1.1 padrão | API do X v2 |
| :--- | :--- | :--- |
| Métodos HTTP compatíveis | POST | POST |
| Domínio do host | https://api.x.com | https://api.x.com |
| Caminho do endpoint | /1.1/statuses/retweet/:id.json | /2/users/:id/retweets |
| [Autenticação](/pt-BR/resources/fundamentals/authentication) | Contexto de Usuário OAuth 1.0a | Contexto de Usuário OAuth 1.0a |
| [Limites de taxa](/pt-BR/resources/fundamentals/rate-limits) padrão da solicitação | Nenhum<br /><br />300 solicitações por janela de 3 horas (por usuário, por app). Compartilhado com o endpoint POST Tweet | 50 solicitações por 15 min (por usuário)<br /><br />300 solicitações por janela de 3 horas (por usuário, por app). Compartilhado com o endpoint POST Tweet para gerenciamento de Posts. |
| Requer o uso de credenciais de um [App de desenvolvedor](/pt-BR/resources/fundamentals/developer-apps) associado a um [Projeto](/pt-BR/resources/fundamentals/projects) |     | ✔️  |
