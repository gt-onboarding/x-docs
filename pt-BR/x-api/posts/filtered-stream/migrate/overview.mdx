<div id="comparing-x-apis-filtered-stream-endpoints">
  ## Comparando os endpoints de fluxo filtrado da API do X
</div>

O conjunto de endpoints de fluxo filtrado da v2 está substituindo o [endpoint standard v1.1 statuses/filter](https://developer.x.com/en/docs/x-api/v1/tweets/filter-realtime/api-reference/post-statuses-filter) e a [PowerTrack API](/pt-BR/x-api/enterprise-gnip-2.0/powertrack-api). Se você tem código, apps ou ferramentas que usam uma versão mais antiga do endpoint de fluxo filtrado e está considerando migrar para o endpoint mais recente da API do X v2, esta comparação pode ajudar você a começar.

Consulte nossos guias de migração mais detalhados:

[Migrando do Standard v1.1 para a API do X v2](pt-BR/x-api/posts/filtered-stream#standard-v1-1-compared-to-x-api-v2)

[Migrando da PowerTrack API para a API do X v2](/pt-BR/x-api/enterprise-gnip-2.0/powertrack-api#powertrack-operators)

A tabela a seguir compara os endpoints de fluxo filtrado oferecidos pelo X:
