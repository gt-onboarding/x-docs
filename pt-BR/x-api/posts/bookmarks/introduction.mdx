<div id="manage-bookmarks">
  ### Gerenciar Favoritos
</div>

Temos dois métodos disponíveis para gerenciar Favoritos: POST e DELETE. O método POST permite criar Favoritos. Da mesma forma, o método DELETE permite excluir um Favorito específico.

Há um limite de 50 solicitações por usuário a cada 15 minutos para os métodos POST e DELETE.

Como você fará solicitações em nome de um usuário com os endpoints de gerenciamento de Favoritos, é necessário autenticar gerando um token de acesso de usuário com OAuth 2.0. Você pode usar o [fluxo de autorização Authorization Code com PKCE](/pt-BR/resources/fundamentals/authentication/oauth-2-0/user-access-token) para isso. Para usar este endpoint, você deve informar os escopos `tweet.read`, `users.read` e `bookmark.write`.

<div id="bookmarks-lookup">
  ### Consulta de favoritos
</div>

O endpoint de consulta de favoritos tem um único método disponível: GET. Esse método permite recuperar os seus favoritos ou os de uma conta autenticada. Tokens de paginação são fornecidos para navegar por conjuntos grandes de resultados deste endpoint.

Há um limite por usuário de 180 solicitações a cada janela de 15 minutos para o método GET. Com este endpoint, você receberá até 800 dos seus posts mais recentes marcados como favorito.

Como as solicitações com os endpoints de consulta de favoritos são feitas em nome de um usuário, é necessário autenticar gerando um token de acesso de usuário com OAuth 2.0. Você pode usar o [fluxo de concessão Authorization Code com PKCE](/pt-BR/resources/fundamentals/authentication/oauth-2-0/user-access-token) para isso. Para usar este endpoint, você deve informar os escopos `tweet.read`, `users.read` e `bookmark.read`.

**Configuração da conta**

Para acessar esses endpoints, você precisará de:

- Uma [conta de desenvolvedor](https://developer.x.com/en/portal/petition/essential/basic-info) aprovada.
- Autenticar usando as chaves e tokens de um [App de desenvolvedor](/pt-BR/resources/fundamentals/developer-apps) que esteja dentro de um [Projeto](/pt-BR/resources/fundamentals/projects).

Saiba mais sobre como obter acesso aos endpoints da API do X v2 em nosso [guia de primeiros passos](/pt-BR/x-api/getting-started/getting-access).
