title: "Sobre a API do X"
---

A API do X pode ser usada para recuperar e analisar dados do X de forma programática, além de desenvolver para a conversa no X.

Ao longo dos anos, a API do X evoluiu com a adição de novos níveis de acesso, permitindo que desenvolvedores escalem seu acesso para aprimorar e pesquisar a conversa pública.

Recentemente, lançamos a API do X v2. A API do X v2 inclui uma base moderna, recursos novos e avançados e uma integração rápida ao [Acesso Básico](https://developer.x.com/en/portal/products/basic).

As três abas a seguir explicam as diferentes versões e níveis de acesso da API do X, o que há de novo na v2 e quais recursos do X você pode recuperar, criar, excluir e ajustar usando a API.

  <Tab title="What's new with v2">
    ## O que há de novo na v2

    A API do X v2 representa a maior atualização da API do X desde 2012. Ela traz um conjunto de recursos novos e avançados, além de acesso rápido e gratuito à API.

    Alguns dos recursos disponíveis na v2 incluem:

    ### Novos endpoints

    Lançamos um conjunto de endpoints totalmente novos para a API do X v2. Você pode ver a lista completa de endpoints da v2, incluindo os que são novos, em nosso guia do mapa de endpoints da API do X.

    [Visite o mapa de endpoints da API do X >](/pt-BR/x-api/migrate/x-api-endpoint-map)

    ### Novos objetos de dados, mais detalhados

    Modernizamos nossos objetos de dados com uma série de melhorias que permitem navegar e analisar os dados com mais facilidade.

    [Visite o dicionário de dados >](/pt-BR/x-api/fundamentals/data-dictionary)

    ### Novos parâmetros para ajudar você a recuperar apenas os objetos e campos desejados

    Adicionamos os parâmetros fields e expansions aos nossos endpoints de dados, permitindo solicitar objetos e campos relacionados além daqueles retornados por padrão.

    [Saiba como usar fields e expansions >](/pt-BR/x-api/fundamentals/data-dictionary#how-to-use-fields-and-expansions)

    ### Métricas avançadas

    Entenda com mais facilidade o desempenho de Posts, usuários, mídias e enquetes diretamente no seu payload solicitando métricas públicas e privadas, incluindo impressões, visualizações de vídeo, perfil de usuário e cliques em URL, algumas delas separadas nos contextos orgânico e promovido.

    [Saiba mais sobre métricas >](/pt-BR/x-api/fundamentals/metrics)

    ### Filtre e identifique quais Posts contêm diferentes tópicos

    Ao usar search Posts ou o Fluxo filtrado, agora você pode filtrar por tópico usando nossos operadores de entidade e de contexto. Também incluímos esses tópicos no payload do Post para facilitar a análise.

    [Saiba mais sobre anotações de Post >](/pt-BR/x-api/fundamentals/post-annotations)

    ### Filtre e identifique quais Posts pertencem a um encadeamento de respostas

    Facilite a identificação de um Post como parte de um encadeamento de conversa ao usar search Posts, o Fluxo filtrado e Post lookup. Também adicionamos a capacidade de determinar se as configurações de resposta da conversa foram definidas para um Post com o campo do Post `reply_settings`.

    [Saiba mais sobre rastreamento de conversas >](/pt-BR/x-api/fundamentals/conversation-id)

    ### E muito mais...

    - Filtro de spam com alta confiabilidade
    - URLs encurtadas são totalmente expandidas para facilitar a análise de URLs
    - Objetos de resposta JSON simplificados, com remoção de campos obsoletos e modernização de rótulos
    - Funcionalidade de recuperação e redundância para nossos endpoints de streaming
    - Retorno de 100% dos Posts públicos e disponíveis que correspondem às consultas de busca
    - “Regras” de streaming para que você possa fazer alterações sem derrubar conexões
    - Linguagem de consulta mais expressiva para o Fluxo filtrado e para busca
    - Especificação OpenAPI para criar novas bibliotecas e rastrear alterações com mais transparência
    - Suporte da API a novos recursos e endpoints com mais rapidez, à medida que nossa plataforma evolui para atender às necessidades de desenvolvedores, pesquisadores, empresas e pessoas que usam o X
  </Tab>

  <Tab title="X API resources">
    ## Recursos da plataforma da API do X

    No design de APIs, um recurso é uma entidade com dados associados, relacionamentos com outros recursos e um conjunto de métodos que operam sobre ela. Por exemplo, um Post é um recurso que você pode criar, excluir ou recuperar usando diversas ferramentas, como pesquisá-los no histórico ou obtê-los em tempo real.

    A API do X permite criar, excluir, receber ou ajustar diversos recursos na plataforma, incluindo os seguintes:

    | Resource            | Description                                                                                                                                                                             |
    | :------------------ | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
    | **Posts**           | Acesse milhões de Posts para entender a conversa pública ou crie os seus para participar dela. Disponibilidade atual:  - API do X v2  - Enterprise  - Standard v1.1                    |
    | **Users**           | Gerencie ou pesquise usuários do X para analisar redes, entender seu público ou promover relacionamentos positivos online. Disponibilidade atual:  - API do X v2  - Enterprise  - Standard v1.1 |
    | **Spaces**          | Pesquise e encontre X Spaces e seus participantes para ajudar as pessoas a descobrir conversas de áudio interessantes e relevantes. Disponibilidade atual:  - API do X v2              |
    | **Direct Messages** | Envie e receba Direct Messages para tratar demandas de clientes, enviar mensagens de boas-vindas ou criar interações humanas positivas. Disponibilidade atual:  - Standard v1.1         |
    | **Lists**           | Crie e gerencie listas de contas para acompanhar especialistas do setor, vozes influentes ou organizar quem você segue. Disponibilidade atual:  - API do X v2  - Standard v1.1         |
    | **Trends**          | Identifique tendências geográficas para mapear movimentos do setor, descobrir assuntos em alta ou se antecipar à última moda. Disponibilidade atual:  - Standard v1.1                  |
    | **Media**           | Envie objetos de mídia para compartilhar sua criatividade, criar experiências interativas ou desenvolver ferramentas de acessibilidade. Disponibilidade atual:  - Standard v1.1         |
    | **Places**          | Pesquise lugares para entender o que está acontecendo na sua vizinhança e ao redor do mundo. Disponibilidade atual:  - Standard v1.1                                                    |
