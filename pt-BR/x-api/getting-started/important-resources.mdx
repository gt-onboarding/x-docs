<div id="learn-about-whats-possible">
  ## Saiba o que é possível
</div>

Nossos casos de uso, soluções e páginas de produto acessíveis pela navegação superior são uma ótima maneira de obter uma visão geral de alto nível do que é possível na plataforma.

Também criamos vários [tutoriais](/pt-BR/tutorials) que ajudam você a identificar como usar uma ou várias ferramentas da plataforma para atender a uma variedade de casos de uso.

No nível mais detalhado da nossa documentação estão nossos endpoints e a documentação de ferramentas, que incluem conteúdo introdutório com uma breve visão geral da funcionalidade de cada ferramenta e de seus casos de uso típicos. Por exemplo, aqui está nossa seção de endpoint da API do X v2 para [Buscar posts](/pt-BR/x-api/posts/search/introduction).

<div id="integrate-an-endpoint-into-your-system">
  ### Integrar um endpoint ao seu sistema
</div>

Para tornar sua integração com nossos endpoints mais fluida, criamos um conjunto de ferramentas, bibliotecas e guias de integração. A maneira mais fácil de integrar é usar nossas [ferramentas e bibliotecas](/pt-BR/x-api/tools-and-libraries/overview), que tratam automaticamente de funcionalidades complexas, como [autenticação OAuth 1.0a em contexto de usuário](/pt-BR/resources/fundamentals/authentication#oauth-1-0a-2), paginação e [tratamento de limites de taxa](/pt-BR/resources/fundamentals/rate-limits).

Se preferir criar uma solução do zero, nossos guias de integração de endpoints descrevem a funcionalidade e as melhores práticas para integrar nossos endpoints ao seu sistema. Esses guias podem ser encontrados na documentação dos diferentes endpoints e ferramentas, listados como “integrate” ou simplesmente “guides”. Por exemplo, aqui está um guia de integração sobre [como criar uma regra para o Fluxo filtrado](/pt-BR/x-api/posts/filtered-stream/integrate/build-a-rule).

<div id="troubleshoot-an-issue">
  ### Solucionar um problema
</div>

Se você estiver enfrentando um erro ou tiver uma dúvida, acesse nosso [hub de suporte](https://developer.x.com/en/support/twitter-api). Lá, você encontrará dicas de solução de problemas, respostas para perguntas frequentes e outros recursos úteis.

<div id="keep-up-to-date-with-the-latest-releases">
  ### Mantenha-se atualizado com as versões mais recentes
</div>

Fazemos atualizações frequentes na plataforma para disponibilizar novas funcionalidades do X via API. Além disso, buscamos lançar pelo menos uma nova versão principal por ano, o que pode incluir alterações incompatíveis com versões anteriores.

Para receber as novidades mais recentes, assine a [categoria Anúncios do nosso fórum](https://devcommunity.x.com/c/announcements/22), siga [@API](https://x.com/api) e [@XDevelopers](https://x.com/XDevelopers) no X e ative as notificações, e confira nossos recursos na página [mantenha-se informado](https://developer.x.com/en/stay-informed).

Além disso, criamos um novo [hub de migração](/pt-BR/x-api/migrate/overview) para orientar você no processo de atualização para a versão mais recente da API.
