Os endpoints de consulta por ID retornam dados que você pode usar para criar experiências que exibam informações de Spaces aos seus usuários ou para retornar detalhes sobre um ou mais Spaces. É possível consultar um ou mais Spaces na mesma solicitação.

Você também pode consultar Spaces pelo ID do criador. Este endpoint é útil para consultar Spaces de usuários em uma lista ou para verificar se vários hosts recorrentes estão ao vivo agora ou têm um Space programado. É possível consultar até 100 usuários em uma única solicitação.

**Configuração da conta**

Para acessar esses endpoints, você precisará de:

- Uma [conta de desenvolvedor](https://developer.x.com/en/portal/petition/essential/basic-info) aprovada.
- Autenticar usando as chaves e tokens de um [App de desenvolvedor](/pt-BR/resources/fundamentals/developer-apps) que está dentro de um [Projeto](/pt-BR/resources/fundamentals/projects).

Saiba mais sobre como obter acesso aos endpoints da API do X v2 em nosso [guia de primeiros passos](/pt-BR/x-api/getting-started/getting-access).
