<div id="community-tools-and-libraries-for-v2">
  ## Ferramentas e bibliotecas da comunidade para v2
</div>

As bibliotecas listadas aqui foram criadas por membros da comunidade de desenvolvedores. Observe que elas podem estar em diferentes estágios de cobertura da API.

Se você criou sua própria biblioteca da API do X ou uma ferramenta útil, [conte para nós](https://devcommunity.x.com/c/libraries-and-sdks/63), e nós a adicionaremos a esta lista para ajudar outras pessoas a encontrá-la. Também temos alguns [selos de versão](https://twbadges.glitch.me/) que você pode usar nos seus próprios arquivos README.

Procurando inspiração? Você pode navegar e pesquisar nos tópicos [X](https://github.com/topics/twitter?o=desc&s=updated) e [X-api-v2](https://github.com/topics/twitter-api-v2?o=desc&s=updated) no GitHub para encontrar exemplos de código úteis de outros desenvolvedores.

**Ir para:** [C# / .NET](#csharp), [Dart / Flutter](#dart), [Go](#go), [Java](#java), [JavaScript (Node.js) / TypeScript](#nodejs), [Kotlin](#kotlin), [PHP](#php), [PowerShell](#powershell), [Python](#python), [R](#r), [Ruby](#ruby), [Rust](#rust), [Swift](#swift)

<div id="community-tools-and-libraries">
  ## Ferramentas e bibliotecas da comunidade
</div>

Estas são algumas das muitas bibliotecas mantidas pela comunidade que abrangem a API do X em várias linguagens de programação e plataformas. Observe que esses recursos podem não ter sido testados pela equipe do Twitter.

As bibliotecas listadas aqui devem implementar a maioria dos recursos da API Padrão **v1.1**, a menos que indicado de outra forma — verifique com os autores para obter detalhes e suporte adicional.\
Se você desenvolveu uma biblioteca que oferece suporte à API do X **v2**, conte para nós nos [fóruns da comunidade](https://devcommunity.x.com/c/libraries-and-sdks/63) para possível inclusão nesta página. Você também pode usar os fóruns para relatar quaisquer alterações nestas listagens.

Se estiver faltando uma biblioteca ou ferramenta para sua linguagem de programação favorita, avise-nos pela [plataforma de feedback](https://twitterdevfeedback.uservoice.com/), onde você também pode votar em ideias ou se inspirar para criar e enviar algo novo.
