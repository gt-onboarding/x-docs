A tabela a seguir relaciona os endpoints da API do X v2 aos respectivos endpoints das versões padrão v1.1 e enterprise. Para saber mais sobre cada uma dessas versões e níveis, visite nosso [guia de introdução à API do X](/pt-BR/x-api/getting-started/about-x-api).

Você notará que ainda temos vários itens marcados como **\[Em breve]**. Se vir algo nesta tabela marcado como **\[Substituição em avaliação]** ou **\[Sem substituição planejada]** e desejar que lancemos uma versão v2 desse endpoint, avise-nos por meio do nosso [fórum da comunidade](https://devcommunity.x.com/) ou do seu gerente de conta enterprise.
