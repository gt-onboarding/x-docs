A Usage API na API do X v2 permite que desenvolvedores recuperem programaticamente o uso do seu projeto. Com esse endpoint, é possível acompanhar e monitorar a quantidade de Posts obtidos em um determinado ciclo de cobrança.

Os desenvolvedores podem usar o endpoint GET para obter o uso diário do projeto nos últimos 90 dias. O uso também pode ser agregado por app cliente conectado ao seu projeto.

Há um limite de taxa por app de 50 solicitações a cada 15 minutos para esse endpoint GET.

<div id="getting-started">
  ### Introdução
</div>

Para usar este endpoint, você precisa de um \[bearer token]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token) do [portal do desenvolvedor](https://developer.x.com/en/portal/dashboard). Com o bearer token em mãos, você pode chamar a API de uso conforme mostrado abaixo:

```
      curl 'https://api.x.com/2/usage/tweets' --header 'Authorization: Bearer XXXXX'
```

Se a solicitação for bem-sucedida, você verá a resposta JSON conforme abaixo:

```
{
    "data": {
        "cap_reset_day": 10,
        "project_cap": "5000000000",
        "project_id": "1369785403853424",
        "project_usage": "43435"
    }
}
```

<Note>
  **Configuração da conta**

  Para acessar esses endpoints, você precisará de:

  - Uma [conta de desenvolvedor](https://developer.x.com/en/portal/petition/essential/basic-info) aprovada.
  - Autenticar-se usando as chaves e tokens de um [developer App](/pt-BR/resources/fundamentals/developer-apps) que está localizado dentro de um [Project](/pt-BR/resources/fundamentals/projects). 

  Saiba mais sobre como obter acesso aos endpoints da API do X v2 em nosso [getting started guide](/pt-BR/x-api/getting-started/getting-access).
</Note>
