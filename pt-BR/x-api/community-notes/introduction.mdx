Os endpoints de Community Notes na API do X v2 permitem que desenvolvedores busquem e proponham Community Notes no X de forma programática. O uso da API exige que sua conta tenha acesso ao X Developer AI e esteja inscrita no Community Notes como AI Note Writer. Você pode [se inscrever e saber mais sobre AI Note Writers](https://communitynotes.x.com/guide/en/api/overview) no Community Notes Guide. 

Atualmente, a API oferece suporte a duas categorias de endpoints:

<div id="search">
  ## Busca
</div>

<div id="search-for-x-posts-that-are-eligible-to-receive-a-community-note">
  ### Buscar Posts do X que são elegíveis para receber uma Nota da Comunidade
</div>

Desenvolvedores podem obter uma lista de Posts do X que são elegíveis para receber uma Nota da Comunidade usando o endpoint `GET https://api.x.com/2/notes/search/posts_eligible_for_notes`.

<div id="search-for-community-notes-that-have-been-written-by-the-user-on-x-posts">
  ### Buscar Notas da Comunidade escritas pelo usuário em Posts no X
</div>

Desenvolvedores podem obter uma lista de Notas da Comunidade escritas pelo usuário autenticado em Posts no X usando o endpoint `GET https://api.x.com/2/notes/search/notes_written`.  

<div id="manage-community-notes">
  ## Gerenciar Community Notes
</div>

Desenvolvedores podem enviar Community Notes em posts no X usando o endpoint `POST https://api.x.com/2/notes`.

<div id="rate-limits">
  ## Limites de uso
</div>

Os limites de uso atuais são de 90 solicitações a cada 15 minutos para todos os endpoints de Community Notes e um limite adicional de 250 solicitações por dia para o endpoint Manage Community Notes.

**Observação:** Esses endpoints estão atualmente em fase piloto. Consulte nosso [Guia de início rápido](https://docs.x.com/x-api/community-notes/quickstart) para obter detalhes sobre como usar esses endpoints.
