Disponível para desenvolvedores com acesso Pro ou superior

O endpoint Users Search oferece uma interface de busca simples, baseada em relevância, para contas públicas de usuários no X. Experimente pesquisar por interesse, nome completo, nome da empresa, localização ou outros critérios.

Este endpoint atualmente oferece suporte à [User Auth](/pt-BR/resources/fundamentals/authentication). Por padrão, você recebe 100 usuários por requisição e pode solicitar até 1000 usuários por requisição. O limite é de 10.000 requisições a cada 24 horas (além de 300 requisições a cada 15 minutos).

<Note>
  **Configuração da conta**

  Para acessar esses endpoints, você precisará de:

  - Uma [conta de desenvolvedor](https://developer.x.com/en/portal/petition/essential/basic-info) aprovada.
  - Autenticar usando as chaves e os tokens de um [App de desenvolvedor](/pt-BR/resources/fundamentals/developer-apps) que esteja dentro de um [Projeto](/pt-BR/resources/fundamentals/projects).

  Saiba mais sobre como obter acesso aos endpoints da API do X v2 em nosso [guia de primeiros passos](/pt-BR/x-api/getting-started/getting-access).
</Note>
