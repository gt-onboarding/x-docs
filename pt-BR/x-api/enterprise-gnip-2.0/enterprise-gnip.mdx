Nossas APIs empresariais oferecem o mais alto nível de acesso e confiabilidade para quem depende dos dados do X. Ideais para quando você ultrapassa o nível premium e precisa de acesso mais estável, pacotes sob medida ou contratos de longo prazo. O acesso à API empresarial inclui gerentes de conta dedicados e suporte técnico personalizado.

Se você identificou que as soluções Basic e Pro não são suficientes para atender às suas necessidades, você pode solicitar acesso aos produtos empresariais no link a seguir:

<div id="available-endpoints">
  ## Endpoints disponíveis
</div>

| Em tempo real | [PowerTrack API](/pt-BR/x-api/enterprise-gnip-2.0/powertrack-api) | Fornece cobertura completa de posts públicos à medida que acontecem, com filtragem por uma ampla variedade de operadores e regras. |
| :--- | :--- | :--- |
| [Decahose API](/pt-BR/x-api/enterprise-gnip-2.0/fundamentals/decahose-api) | Transmita uma amostra aleatória de 10% de todos os posts públicos e seus likes. |
| :--- | :--- | :--- |
| [Account Activity API](/pt-BR/x-api/enterprise-gnip-2.0/fundamentals/account-activity) | Assine usuários e receba diversas atividades, incluindo posts, Mensagens Diretas, likes, follows e mais. |
| :--- | :--- | :--- |
| Histórico | [Search API](/pt-BR/x-api/enterprise-gnip-2.0/fundamentals/search-api) | Pesquise nos últimos 30 dias ou em todo o arquivo de posts públicos usando ferramentas avançadas de filtragem. |
| [Historical PowerTrack API](/pt-BR/x-api/enterprise-gnip-2.0/powertrack-api) | Filtre todo o arquivo de posts públicos usando um processo econômico de acesso em lote. |
| Insights | [Engagement API](/pt-BR/x-api/enterprise-gnip-2.0/fundamentals/engagement-api) | Meça o volume de impressões e engajamentos de posts públicos. |
| Conformidade | [Compliance Firehose API](/pt-BR/x-api/enterprise-gnip-2.0/fundamentals/firehouse) | Transmita todos os eventos de conformidade. |
| Uso | [Usage API](/pt-BR/x-api/enterprise-gnip-2.0/fundamentals/usage) | Fornece acesso programático ao consumo de atividade nos produtos enterprise. |

  ## Conceitos importantes
</div>

<div id="gnip-console">
  ### Gnip Console
</div>

Você pode gerenciar seus produtos para empresas e o acesso à conta por meio do [Gnip Console](/pt-BR/x-api/enterprise-gnip-2.0/fundamentals/overview). 

<div id="enrichments">
  ### Enriquecimentos
</div>

Adicionamos metadados ao payload de resposta das APIs premium para desenvolvedores que utilizam um plano pago. Saiba mais sobre URLs expandidas, dados de enquetes e geolocalização de perfil acessando nossa [seção de enriquecimentos](/pt-BR/x-api/enterprise-gnip-2.0/fundamentals/data-enrichments).
 

<div id="advanced-filtering-tools">
  ### Ferramentas avançadas de filtragem
</div>

A Premium Search API oferece um conjunto avançado de operadores que não estão disponíveis no endpoint de pesquisa padrão. Muitos desses operadores estão disponíveis para usuários do sandbox gratuito, mas você terá um conjunto mais robusto se estiver em um nível pago. Saiba mais na nossa [seção de regras e filtros](/pt-BR/x-api/enterprise-gnip-2.0/fundamentals/rules-filtering#getting-started-with-enterprise-rules-and-queries).
