<div id="overview">
  ## Visão geral
</div>

Todos os dias, milhares de desenvolvedores fazem solicitações à API do X. Para ajudar a gerenciar o grande volume dessas solicitações, definimos limites para a quantidade de solicitações que podem ser feitas. Esses limites nos ajudam a fornecer a API confiável e escalável da qual nossa comunidade de desenvolvedores depende.

O número máximo de solicitações permitidas é definido com base em um intervalo de tempo, um período ou janela de tempo específica. Se um endpoint tiver um limite de taxa de 900 solicitações/15 minutos, será permitido fazer até 900 solicitações em qualquer intervalo de 15 minutos.
