<div id="comparing-x-apis-list-lookup-endpoints">
  ## Comparando os endpoints de consulta de listas da API do X
</div>

O grupo de endpoints de consulta de listas da v2 substituirá os endpoints padrão da v1.1 [GET lists/show](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-show) e [GET lists/ownership](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-ownerships). Se você tem código, apps ou ferramentas que usam uma dessas versões dos endpoints de consulta de listas e está considerando migrar para o endpoint mais recente da API do X v2, este conjunto de guias é para você.

As tabelas a seguir comparam os endpoints de listas da v1.1 padrão e da API do X v2:

**Consulta de lista por ID**

|     |     |     |
| :--- | :--- | :--- |
| Descrição | v1.1 padrão | API do X v2 |
| Métodos HTTP compatíveis | `GET` | `GET` |
| Domínio do host | `https://api.x.com` | `https://api.x.com` |
| Caminho do endpoint | `/1.1/lists/show.json` | `/2/lists/:id` |
| [Autenticação](/pt-BR/resources/fundamentals/authentication) | OAuth 1.0a contexto do usuário<br /><br />Somente app | OAuth 1.0a contexto do usuário<br /><br />OAuth 2.0 Authorization Code com PKCE<br /><br />Somente app |
| [Limites de taxa](/pt-BR/resources/fundamentals/rate-limits) padrão de requisição | 75 solicitações por 15 min com OAuth 1.0a<br /><br />75 solicitações por 15 min com OAuth 2.0<br /><br />75 solicitações por 15 min com somente app | 75 solicitações por 15 min com OAuth 1.0a<br /><br />75 solicitações por 15 min com OAuth 2.0<br /><br />75 solicitações por 15 min com somente app |

**Consulta de listas de propriedade do usuário**

|     |     |     |
| :--- | :--- | :--- |
| Descrição | v1.1 padrão | API do X v2 |
| Métodos HTTP compatíveis | `GET` | `GET` |
| Domínio do host | `https://api.x.com` | `https://api.x.com` |
| Caminho do endpoint | `/1.1/lists/ownerships.json` | `/2/users/:id/owned_lists` |
| [Autenticação](/pt-BR/resources/fundamentals/authentication) | OAuth 1.0a contexto do usuário<br /><br />Somente app | OAuth 1.0a contexto do usuário<br /><br />OAuth 2.0 Authorization Code com PKCE<br /><br />Somente app |
| [Limites de taxa](/pt-BR/resources/fundamentals/rate-limits) padrão de requisição | 15 solicitações por 15 min com OAuth 1.0a<br /><br />15 solicitações por 15 min com somente app | 15 solicitações por 15 min com OAuth 1.0a<br /><br />15 solicitações por 15 min com OAuth 2.0<br /><br />15 solicitações por 15 min com somente app |

Para acessar os endpoints da API do X v2, você deve [criar uma conta de desenvolvedor](https://developer.x.com/en/portal/petition/essential/basic-info). Ao autenticar, use as chaves e tokens de um [App de desenvolvedor](/pt-BR/resources/fundamentals/developer-apps) que esteja dentro de um [Projeto](/pt-BR/resources/fundamentals/projects).

Saiba mais sobre como obter acesso aos endpoints da API do X v2 em nossa página de [primeiros passos](/pt-BR/x-api/getting-started/getting-access).
