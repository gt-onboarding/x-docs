Há um limite de 75 solicitações a cada 15 minutos ao consultar uma Lista específica por ID e um limite de 15 solicitações a cada 15 minutos ao consultar Listas pertencentes ao usuário.

Você pode usar [OAuth 1.0a User Context](/pt-BR/resources/fundamentals/authentication), \[App only]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token), ou [OAuth 2.0 Authorization Code com PKCE](pt-BR/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) para autenticar suas solicitações a esses endpoints.

<Note>
  **Configuração da conta**

  Para acessar esses endpoints, você precisará de:

  - Uma [conta de desenvolvedor](https://developer.x.com/en/portal/petition/essential/basic-info) aprovada.
  - Autenticar-se usando as chaves e tokens de um [developer App](/pt-BR/resources/fundamentals/developer-apps) que esteja dentro de um [Projeto](/pt-BR/resources/fundamentals/projects).

  Saiba mais sobre como obter acesso aos endpoints da API do X v2 em nosso [guia de primeiros passos](/pt-BR/x-api/getting-started/getting-access).
</Note>
