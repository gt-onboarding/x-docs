[Lists do X](https://help.x.com/en/using-twitter/twitter-lists) permitem que os usuários personalizem, organizem e priorizem os posts que veem na sua timeline. Com os endpoints de Lists, você pode criar soluções que permitem às pessoas curar e organizar posts com base em preferências, interesses, grupos ou tópicos.

Como você está fazendo solicitações em nome de um usuário com os endpoints de gerenciamento de Lists, deve autenticar esses endpoints com [OAuth 1.0a User Context](/pt-BR/resources/fundamentals/authentication) ou [OAuth 2.0 Authorization Code with PKCE](pt-BR/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) e usar os tokens de acesso do usuário associados a um usuário que autorizou seu app, os quais podem ser gerados usando o [fluxo OAuth de 3 etapas](/pt-BR/resources/fundamentals/authentication#obtaining-access-tokens-using-3-legged-oauth-flow) (OAuth 1.0a) ou o [fluxo de concessão Authorization Code with PKCE](/pt-BR/resources/fundamentals/authentication#how-to-connect-to-endpoints-using-oauth-2-0-authorization-code-flow-with-pkce) (OAuth 2.0).

<div id="manage-lists">
  ### Gerenciar Listas
</div>

Os endpoints de gerenciamento de Listas permitem criar, excluir e atualizar Listas em nome de um usuário autenticado. Para esses endpoints, há três métodos disponíveis: POST, DELETE e PUT. O método POST permite criar uma Lista, o método DELETE permite excluir uma Lista e o método PUT permite atualizar os metadados de uma Lista. Há um limite de taxa por usuário de 300 solicitações a cada 15 minutos para os três endpoints.

Observe que você pode criar até 1000 Listas por conta.

**Configuração da conta**

Para acessar esses endpoints, você precisará de:

- Uma [conta de desenvolvedor](https://developer.x.com/en/portal/petition/essential/basic-info) aprovada.
- Autenticar-se usando as chaves e tokens de um [App de desenvolvedor](/pt-BR/resources/fundamentals/developer-apps) que esteja dentro de um [Projeto](/pt-BR/resources/fundamentals/projects).

Saiba mais sobre como obter acesso aos endpoints da API do X v2 em nosso [guia de primeiros passos](/pt-BR/x-api/getting-started/getting-access).
