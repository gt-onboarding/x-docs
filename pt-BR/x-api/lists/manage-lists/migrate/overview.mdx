<div id="comparing-x-apis-lists-endpoints">
  ## Comparando os endpoints de Listas da API do X
</div>

Os endpoints de gerenciamento de Listas da v2 eventualmente substituirão [POST lists/create](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-create), [POST lists/destroy](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-destroy) e [POST lists/update](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-update). Se você tem código, apps ou ferramentas que usam uma versão mais antiga desses endpoints e está considerando migrar para a nova API do X v2, este guia é para você.

As tabelas a seguir comparam os endpoints de Listas do padrão v1.1 e da API do X v2:

**Criar uma Lista**

| Descrição | Padrão v1.1 | API do X v2 |
| :--- | :--- | :--- |
| Métodos HTTP compatíveis | POST | POST |
| Domínio do host | https://api.x.com | https://api.x.com |
| Caminho do endpoint | /1.1/lists/create.json | /2/lists |
| [Autenticação](/pt-BR/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context |
| [Limites de taxa](/pt-BR/resources/fundamentals/rate-limits) padrão de requisições | Nenhum | 300 requisições por 15 min (por usuário) |

**Excluir uma Lista**

| Descrição | Padrão v1.1 | API do X v2 |
| :--- | :--- | :--- |
| Métodos HTTP compatíveis | POST | DELETE |
| Domínio do host | https://api.x.com | https://api.x.com |
| Caminho do endpoint | /1.1/lists/destroy.json | /2/lists/:id |
| [Autenticação](/pt-BR/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context |
| [Limites de taxa](/pt-BR/resources/fundamentals/rate-limits) padrão de requisições | Nenhum | 300 requisições por 15 min (por usuário) |

**Atualizar uma Lista**

| Descrição | Padrão v1.1 | API do X v2 |
| :--- | :--- | :--- |
| Métodos HTTP compatíveis | POST | PUT |
| Domínio do host | https://api.x.com | https://api.x.com |
| Caminho do endpoint | /1.1/lists/update.json | /2/lists/:id |
| [Autenticação](/pt-BR/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context |
| [Limites de taxa](/pt-BR/resources/fundamentals/rate-limits) padrão de requisições | Nenhum | 300 requisições por 15 min (por usuário) |

Para acessar os endpoints da API do X v2, você deve [se inscrever para uma conta de desenvolvedor](https://developer.x.com/en/portal/petition/essential/basic-info). Ao autenticar, use chaves e tokens de um [App de desenvolvedor](/pt-BR/resources/fundamentals/developer-apps) que esteja dentro de um [Projeto](/pt-BR/resources/fundamentals/projects).

Saiba mais sobre como obter acesso aos endpoints da API do X v2 na nossa página de [primeiros passos](/pt-BR/x-api/getting-started/getting-access).
