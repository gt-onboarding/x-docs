<div id="steps-to-build-a-manage-pinned-list-request">
  ### Etapas para criar uma solicitação de gerenciamento de Lista fixada
</div>

**Primeiro passo: escolha a coleção de endpoints de Lista no Postman**

Depois de carregar a coleção da API do X v2 no Postman, navegue até a pasta “List”, selecione a pasta “Pinned Lists” e, em seguida, escolha "Pin a List".
 

**Segundo passo: especifique a Lista a ser fixada**

Os endpoints de gerenciamento de Lista fixada exigem dois IDs: um para o usuário (o usuário autenticado que vai fixar uma Lista) e o da Lista de destino (a Lista a ser fixada). O ID do usuário deve corresponder ao ID do usuário autenticado, o que significa que você deve passar os tokens de acesso associados ao ID do usuário ao autenticar sua solicitação. Neste caso, você pode especificar o ID pertencente ao seu usuário. Você pode encontrar seu ID de duas maneiras:

1. Usando o endpoint [users lookup](/pt-BR/x-api/users/lookup/introduction) por nome de usuário, você pode informar um nome de usuário e receber o campo id.
2. No seu token de acesso, você verá que a parte numérica é o seu ID de usuário.

O ID da Lista de destino pode ser qualquer lista válida. No Postman, navegue até a aba "Params" e insira o ID do usuário na coluna "Value" da variável de caminho id. Navegue até a aba “Body” e informe o ID da Lista que você deseja fixar como valor do parâmetro list\_id. Certifique-se de não incluir espaços antes ou depois de qualquer ID.

|     |     |     |
| :--- | :--- | :--- |
| **Key** | **Value** | **Parameter type** |
| `id` | O ID do usuário | path |
| list\_id | O ID da Lista de destino a ser fixada | body |

Agora você deve ver uma URL semelhante ao lado do botão "Send":

```
      https://api.x.com/2/users/2244994945/pinned_lists
```

Step three: Make your request and review your response

Depois que tudo estiver configurado, clique no botão "Send" e você receberá uma resposta semelhante ao exemplo a seguir:

```
{
  "data": {
    "pinned": true
  }
}
```

Você fixou a Lista de destino com sucesso se o objeto de resposta retornado contiver o valor true para a chave pinned. 

Para desafixar uma Lista, selecione a solicitação "Unpin List", também encontrada na pasta “Lists” da coleção da API do X v2 carregada no Postman. Este endpoint requer o ID do usuário autenticado e o ID da Lista a ser desafixada. Na aba “Params”, insira o ID do usuário como valor da coluna id e o ID da Lista a ser desafixada como valor da coluna list\_id. 

Em uma solicitação de exclusão bem-sucedida, você receberá uma resposta semelhante ao exemplo a seguir:

```
{
  "data": {
    "pinned": false
  }
}
```
