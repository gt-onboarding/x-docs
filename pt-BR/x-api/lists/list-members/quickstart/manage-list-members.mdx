<div id="manage-list-members-quick-start-guide">
  ### Guia de início rápido para gerenciar membros de listas
</div>

<Note>
  **Atenção:** Este guia pressupõe que você já concluiu os pré-requisitos da [visão geral de início rápido](/pt-BR/x-api/lists/list-members#getting-started-with-the-list-members-endpoint-group).
</Note>

<div id="steps-to-build-a-manage-list-member-request">
  ### Etapas para criar uma solicitação para gerenciar membros de uma Lista
</div>

<div id="step-one-choose-the-list-endpoint-collection-from-postman">
  #### Etapa 1: escolha a coleção de endpoints de List no Postman
</div>

Depois de carregar a coleção da API do X v2 no Postman, navegue até a pasta "List", selecione a pasta "List members" e, em seguida, escolha "Add a member".
 

<div id="step-two-specify-the-user-to-add">
  #### Etapa dois: especifique o usuário a ser adicionado
</div>

Os endpoints de gerenciamento de membros de Lista exigem dois IDs: o ID da Lista e o ID do usuário a ser adicionado. Você pode encontrar o ID do usuário usando o [users lookup](/pt-BR/x-api/users/lookup/introduction) e passando um nome de usuário para receber o campo id.

O ID de destino pode ser qualquer ID de usuário válido. No Postman, vá até a guia "Params" e insira seu ID na coluna "Value" da variável de caminho id. Depois, vá até a guia “Body” e informe o ID do usuário que você deseja adicionar como valor do parâmetro user\_id. Certifique-se de não incluir espaços antes ou depois de nenhum ID.

|     |     |     |
| :--- | :--- | :--- |
| **Key** | **Value** | **Parameter type** |
| `id` | O ID da Lista | path |
| user\_id | O ID do usuário de destino que você deseja adicionar como membro | body |

Agora você deve ver uma URL semelhante ao lado do botão "Send":

`https://api.x.com/2/lists/1441162269824405510/members`

Etapa três: faça sua solicitação e revise sua resposta

Depois que tudo estiver configurado, clique no botão "Send" e você receberá uma resposta semelhante ao seguinte exemplo:

```
{
  "data": {
    "is_member": true
  }
}
```

Se o objeto de resposta retornado contiver o valor true para a chave is\_member, você adicionou o usuário como membro da Lista com sucesso.

Para remover um membro de uma Lista, selecione a solicitação “Remove a member”, também localizada na pasta “Lists” da coleção da API do X v2 carregada no Postman. Esse endpoint requer o ID da Lista e o ID do usuário do membro que você deseja remover. Na guia “Params”, insira o ID da Lista como valor da coluna id e o ID do usuário a ser removido como valor da coluna user\_id.

Em uma solicitação de exclusão bem-sucedida, você receberá uma resposta semelhante ao seguinte exemplo:

```
{
  "data": {
    "is_member": false
  }
}
```
