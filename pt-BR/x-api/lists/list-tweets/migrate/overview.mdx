  ## Comparando os endpoints de busca de posts de List da API do X
</div>

O endpoint de busca de posts de List na versão v2 substituirá o [GET lists/statuses](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-statuses) padrão da v1.1. Se você tem código, apps ou ferramentas que usam essa versão do endpoint e está considerando migrar para o endpoint mais recente da API do X v2, este conjunto de guias é para você.

As tabelas a seguir comparam os endpoints de List padrão da v1.1 e da API do X v2:

**Busca de List por ID**

|     |     |     |
| :--- | :--- | :--- |
| Descrição | Padrão v1.1 | API do X v2 |
| Métodos HTTP compatíveis | `GET` | `GET` |
| Domínio do host | `https://api.x.com` | `https://api.x.com` |
| Caminho do endpoint | `/1.1/lists/statuses.json` | `/2/lists/:id/tweets` |
| [Autenticação](/pt-BR/resources/fundamentals/authentication) | OAuth 1.0a (User Context)<br /><br />Somente app | OAuth 1.0a (User Context)<br /><br />OAuth 2.0 Authorization Code com PKCE<br /><br />Somente app |
| Limites de taxa [rate limits](/pt-BR/resources/fundamentals/rate-limits) padrão | 900 solicitações por 15 min com OAuth 1.0a<br /><br />900 solicitações por 15 min com somente app | 900 solicitações por 15 min com OAuth 1.0a<br /><br />900 solicitações por 15 min com OAuth 2.0<br /><br />900 solicitações por 15 min com somente app |

Para acessar os endpoints da API do X v2, você deve [se inscrever para obter uma conta de desenvolvedor](https://developer.x.com/en/portal/petition/essential/basic-info). Ao autenticar, use as chaves e os tokens de um [App de desenvolvedor](/pt-BR/resources/fundamentals/developer-apps) dentro de um [Projeto](/pt-BR/resources/fundamentals/projects).

Saiba mais sobre como obter acesso aos endpoints da API do X v2 na nossa página de [introdução](/pt-BR/x-api/getting-started/getting-access).
