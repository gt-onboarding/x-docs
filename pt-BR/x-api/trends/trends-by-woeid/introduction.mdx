O endpoint de consulta de Tendências permite que desenvolvedores obtenham as Tendências de um local, especificado usando o where-on-earth ID (WOEID).

<Note>
  **Observação**: WOEID é um identificador legado criado pelo Yahoo e foi descontinuado. A API do X usa o valor numérico para identificar locais de tendências por cidade e país. Consulte nossa [postagem de blog legada](https://blog.x.com/engineering/en_us/a/2010/woeids-in-twitters-trends.html) ou [dados arquivados](https://archive.org/details/geoplanet_data_7.10.0.zip0.)
</Note>

O `tweet_count` das últimas 24 horas também é retornado para muitas tendências, se disponível.

Este endpoint oferece suporte à autenticação app-auth e tem um limite de 75 solicitações por janela de 15 minutos.

<div id="getting-started">
  ### Introdução
</div>

Para usar este endpoint, você precisa de um \[bearer token]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token) do [portal do desenvolvedor](https://developer.x.com/en/portal/dashboard). Depois de obter o bearer token, você pode chamar a API de uso conforme mostrado abaixo:

```
      curl 'https://api.x.com/2/trends/by/woeid/26062' --header 'Authorization: Bearer XXXXX'
```

Se a solicitação for bem-sucedida, você verá a resposta JSON como mostrado abaixo:

**Configuração da conta**

Para acessar esses endpoints, você precisará de:

- Uma [conta de desenvolvedor](https://developer.x.com/en/portal/petition/essential/basic-info) aprovada.
- Autenticar-se usando as chaves e tokens de um [developer App](/pt-BR/resources/fundamentals/developer-apps) que esteja dentro de um [Projeto](/pt-BR/resources/fundamentals/projects). 

Saiba mais sobre como obter acesso aos endpoints da API do X v2 em nosso [getting started guide](/pt-BR/x-api/getting-started/getting-access).
