O endpoint Personalized Trends permite obter tendências personalizadas para uma conta de usuário autenticada.

<Note>
  **Observação:** As tendências personalizadas só serão retornadas por este endpoint para contas de usuário assinantes do X Premium.
</Note>

<Note>
  **Configuração da conta**

  Para acessar este endpoint, você precisará de:

  - Uma [conta de desenvolvedor](https://developer.x.com/en/portal/petition/essential/basic-info) aprovada.
  - Autenticar usando as chaves e tokens de um [App de desenvolvedor](/pt-BR/resources/fundamentals/developer-apps) que esteja dentro de um [Projeto](/pt-BR/resources/fundamentals/projects).

  Saiba mais sobre como obter acesso aos endpoints da API do X v2 em nosso [guia de primeiros passos](/pt-BR/x-api/getting-started/getting-access).
</Note>

<div id="getting-started">
  ### Primeiros passos
</div>

<div id="authentication">
  #### Autenticação
</div>

Você pode autenticar este endpoint com [OAuth 1.0a em contexto de usuário](/pt-BR/resources/fundamentals/authentication) ou [OAuth 2.0 com PKCE](pt-BR/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2).
 

<div id="making-a-request">
  #### Fazendo uma solicitação
</div>

Você pode chamar o endpoint Personalized Trends conforme mostrado abaixo:

`curl 'https://api.x.com/2/users/personalized_trends' --header 'Authorization: ••••••'`

Se a solicitação for bem-sucedida, você verá a resposta JSON conforme abaixo:

```
{
    "data": [
        {
            "category": "Bitcoin",
            "post_count": "42K posts",
            "trend_name": "MicroStrategy's $42 Billion Bitcoin Investment",
            "trending_since": "5 hours ago"
        },
        {
            "category": "Music",
            "post_count": "2K posts",
            "trend_name": "Drake's 'No Face' Music Video Release",
            "trending_since": "Trending now"
        },
        {
            "category": "Weather",
            "post_count": "1.2K posts",
            "trend_name": "Super Typhoon Leon Impacts Taiwan and Batanes",
            "trending_since": "Trending now"
        }
    ]
}
```

**Observação:** Se o usuário autenticado da solicitação não for assinante do X Premium, receberá a seguinte resposta:

```
{
    "data": [
        {
            "category": "Unknown",
            "post_count": "Unknown",
            "trending_since": ""
        }
    ]
}
```
