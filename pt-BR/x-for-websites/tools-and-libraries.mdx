  ## Recursos oficiais
</div>

As ferramentas abaixo podem ajudar você a começar com o X para Websites.

<CardGroup cols={3}>
<Card
  title="publish.x.com"
  icon="x-twitter"
  href="https://publish.x.com/"
>
Crie seus próprios posts incorporados, timelines ou botões.
</Card>
<Card
  title="Twitter for WordPress"
  icon="x-twitter"
  href="https://github.com/twitter/wordpress"

Um plugin oficial para otimizar seu site WordPress para o X, incluindo embeds de posts e X Cards.
</Card>

  title="twemoji"

  href="https://twemoji.x.com/"

O conjunto de emojis gratuito e de código aberto do X, incluindo uma biblioteca JavaScript para suporte multiplataforma.
</Card>
</CardGroup>
