<div id="ways-to-build-for-people-on-x">
  ## X 上の人々のために構築する方法
</div>

私たちはプラットフォームを開放し、公共の会話に前向きな影響を与え、X 上の人々に役立つ新しいツールやプロダクトの構築を促進します。特に、次の分野での新たなイノベーションと開発を奨励します。

<CardGroup>
  <Card title="健全性と安全性のために会話をモデレートする" icon="bullhorn" iconType="solid">
    コンテンツモデレーションツールやその他のソリューションを構築し、公共の会話の健全性と安全性の向上に貢献してください。人々が X 上の会話に安心して意欲的に参加できるよう支援します。人々が自らのインタラクションを制御できる包括的なツールの選択肢を提供する、補完的・付加的な技術を構築しましょう。

    [**詳しく見る**](/ja/x-api/what-to-build#moderate-conversations-for-health-and-safety) <Icon icon="arrow-right" />
  </Card>

  <Card title="創造と自己表現を可能にする" icon="bullhorn" iconType="solid">
    リーチを広げ、ブランドを体現し、情熱を共有する人々とつながりたいクリエイター、ブランド、個人向けのツールを構築しましょう。あるいは、クロスポスト、ゲーミフィケーション、役立つボットなど、X での関わり方を広げましょう。

    [**詳しく見る**](/ja/x-api/what-to-build#enable-creation-and-personal-expression) <Icon icon="arrow-right" />
  </Card>

  <Card title="「いま起きていること」を測定・分析する" icon="bullhorn" iconType="solid">
    公共の会話や自分のコンテンツからの測定・分析・インサイトの抽出を支援します。オーディエンスをより深く理解し、コンテンツのパフォーマンスと影響を測定し、より多くの情報に基づいた意思決定を行えるようにしましょう。

    [**詳しく見る**](/ja/x-api/what-to-build#measure-and-analyze-whats-happening) <Icon icon="arrow-right" />
  </Card>

  <Card title="コミュニティ体験を向上させる" icon="bullhorn" iconType="solid">
    X は誰にでも開かれており、X 上の多様なコミュニティの固有のニーズにより良く応えるために、皆さんの専門性の活用を奨励し、支援します。新興グローバル市場向けにローカライズされたソリューションを構築する場合でも、ゲームやファイナンスなどプラットフォーム上で成長するコミュニティの体験を強化する場合でも、誰もが X に自分の居場所を見つけられるようにしたいと考えています。

    [**詳しく見る**](/ja/x-api/what-to-build#improve-community-experiences) <Icon icon="arrow-right" />
  </Card>

  <Card title="コンテンツをキュレートしてレコメンドする" icon="bullhorn" iconType="solid">
    人々が新しいコンテンツを発見したり、新たなつながりを作るのを助けるアプリ、ボット、その他のツールを構築しましょう。あるいは、関心の高いコンテンツをより多く目にできるよう、厳選されカスタマイズ可能な体験を構築しましょう。

    [**詳しく見る**](/ja/x-api/what-to-build#curate-and-recommend-content) <Icon icon="arrow-right" />
  </Card>

  <Card title="より大きな公益に貢献する" icon="bullhorn" iconType="solid">
    画期的な研究を行い、公衆衛生や気候危機に関する知見の進展に寄与し、非商用の取り組みで公益に資する構築を行いましょう。

    [**詳しく見る**](/ja/x-api/what-to-build#impact-the-greater-good) <Icon icon="arrow-right" />
  </Card>
</CardGroup>
