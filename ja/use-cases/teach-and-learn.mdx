<div id="educators">
  ## 教育者
</div>

API を授業に取り入れ、学生にプログラミングの基礎と実践的なデータ処理スキルを身につけさせましょう。

**リソース:**

授業で X API の活用を検討中の教育者の方へ。
[Teaching with the X API ガイド](https://dev.to/suhemparack/a-guide-to-teaching-with-the-twitter-api-v2-3n08)
は、X API の概要、教育現場での活用理由、API を用いた授業の種類、そして導入にあたって知っておくべきポイントを網羅的に解説します。

**スポットライト:**

以下のカリキュラムは X API v2 を用いて構築・実施されています:

- [Social Media Data Mining and Analysis](https://canvas.uw.edu/courses/1434897)
  指導: Dr. Tanu Mitra

- [Data Science & Society](https://dssoc.github.io/) 指導: Dr. Chris Bail

- [Introduction to Cultural Analytics & Python](https://melaniewalsh.github.io/Intro-Cultural-Analytics/welcome.html)
  指導: Dr. Melanie Walsh

- [Oxford Spring School in Advanced Research Methods, Analysing X Data](https://github.com/cjbarrie/oxspsch)
  指導: Dr. Chris Barrie
