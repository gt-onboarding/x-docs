X API v2 は、X のグローバルな会話（投稿、ユーザー、スペース、DM、リスト、トレンド、メディアなど）にプログラムからアクセスできるようにします。強力でモダンなエンドポイントを使って、アプリを構築し、データを分析し、新たなエクスペリエンスを創出しましょう。

</div>

<CardGroup cols="3">
  <Card title="豊富なデータオブジェクト" icon="database" href="/ja/x-api/fundamentals/data-dictionary">
    投稿、ユーザー、メディアなどの詳細で構造化されたデータに、柔軟なフィールド選択でアクセスできます。
  </Card>

  <Card title="高度なメトリクス" icon="chart-pie" href="/ja/x-api/fundamentals/metrics">
    エンゲージメント分析、リアルタイムメトリクス、データに関する詳細なインサイトを取得できます。
  </Card>

  <Card title="リアルタイムストリーム" icon="bolt" href="/ja/x-api/posts/filtered-stream/introduction">
    フィルタリングまたはサンプリングされたストリームを購読し、ライブ更新とリアルタイム処理を行えます。
  </Card>
</CardGroup>
