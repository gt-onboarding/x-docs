リストの投稿ルックアップには、指定したリストから投稿を取得するためのエンドポイントが1つ用意されています。このエンドポイントを使うと、ユーザーがタイムラインで目にする投稿をカスタマイズ、整理、優先度付けできるソリューションを構築できます。

このエンドポイントには15分あたり900リクエストのレート制限があり、レスポンスでは指定したリストの最新800件の投稿を取得できます。

このエンドポイントへのリクエスト認証には、[OAuth 1.0a User Context](/ja/resources/fundamentals/authentication)、\[App only]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)、または [OAuth 2.0 Authorization Code with PKCE](ja/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) を使用できます。

**アカウントのセットアップ**

これらのエンドポイントにアクセスするには、次が必要です。

- 承認済みの[開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)
- [プロジェクト](/ja/resources/fundamentals/projects) 内にある[開発者アプリ](/ja/resources/fundamentals/developer-apps)のキーとトークンを用いた認証

X API v2 エンドポイントへのアクセスについての詳細は、[はじめにガイド](/ja/x-api/getting-started/getting-access)をご覧ください。
