Pro アクセス以上の開発者が利用可能

Users Search エンドポイントは、X 上の公開ユーザーアカウントに対する、関連性に基づくシンプルな検索インターフェースを提供します。トピックの関心、氏名、会社名、場所などの条件で検索してみてください。

このエンドポイントは現在、[User Auth](/ja/resources/fundamentals/authentication) をサポートしています。既定ではリクエストあたり 100 ユーザーが返され、最大でリクエストあたり 1,000 ユーザーを指定できます。24 時間あたり 10,000 リクエストの上限があります（これに加えて 15 分あたり 300 リクエスト）。

<Note>
  **アカウントの準備**

  これらのエンドポイントにアクセスするには、以下が必要です。

  - 承認済みの[開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)。
  - [プロジェクト](/ja/resources/fundamentals/projects) 内にある[開発者アプリ](/ja/resources/fundamentals/developer-apps)のキーとトークンを使用して認証すること。

  X API v2 エンドポイントへのアクセス方法の詳細は、[はじめにガイド](/ja/x-api/getting-started/getting-access)をご覧ください。
</Note>
