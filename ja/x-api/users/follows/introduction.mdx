フォロー関係の参照・作成・削除を行うために、follows lookup と manage follows の2種類のエンドポイントグループを提供しています。

<div id="follows-lookup">
  ### フォローのルックアップ
</div>

Only available to those with [Enterprise access](/ja/x-api/getting-started/about-x-api)

フォローのルックアップエンドポイントを使用すると、ユーザー間の関係（ネットワーク分析と呼ばれることもあります）を探索・分析できます。具体的には、指定したユーザーがフォローしている相手、または指定したユーザーをフォローしている相手を表す[user objects](/ja/x-api/fundamentals/data-dictionary#user)を返す、2 つの REST エンドポイントがあります。

You can authenticate this endpoint with either [OAuth 1.0a User Context](/ja/resources/fundamentals/authentication), \[App only]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token), or [OAuth 2.0 Authorization Code with PKCE](ja/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2). You can request up to 1,000 users per request, and pagination tokens will be provided for paging through large sets of results.
 

<div id="manage-follows">
  ### フォローの管理
</div>

「フォローの管理」エンドポイントを使用すると、ユーザーのフォロー／フォロー解除が行えます。

ユーザーに代わってリクエストを行うため、これらのエンドポイントは [OAuth 1.0a User Context](/ja/resources/fundamentals/authentication) または [OAuth 2.0 Authorization Code with PKCE](ja/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) で認証し、対象ユーザーに紐づくユーザーアクセストークンを使用する必要があります。ユーザーアクセストークンは、[3-legged OAuth flow](/ja/resources/fundamentals/authentication#obtaining-access-tokens-using-3-legged-oauth-flow)（OAuth 1.0a）または [Authorization Code with PKCE grant flow](/ja/resources/fundamentals/authentication#how-to-connect-to-endpoints-using-oauth-2-0-authorization-code-flow-with-pkce)（OAuth 2.0）で生成できます。

認証済みの各ユーザーごとに、1日あたり最大400件のフォロー操作が可能で、認証済みの全ユーザー合計では、アプリごとに1日あたり1,000件の操作に制限されます。たとえば、認証済みユーザーが5人いる場合、うち2人のユーザーで各400件（ユーザーごとの上限）、合計800件のフォローができ、残りの200件（アプリごとの上限）を残り3人のユーザーで分配する必要があります。この制限はフォロー解除エンドポイントには適用されず、フォロー解除には別途、アプリごとに1日あたり500件の上限があります。

<Note>
  **アカウント設定**

  これらのエンドポイントにアクセスするには、以下が必要です:

  - 承認済みの [developer account](https://developer.x.com/en/portal/petition/essential/basic-info)
  - [Project](/ja/resources/fundamentals/projects) 内にある [developer App](/ja/resources/fundamentals/developer-apps) のキーとトークンを使用した認証
</Note>

X API v2 エンドポイントへのアクセス方法の詳細は、[はじめにガイド](/ja/x-api/getting-started/getting-access)をご覧ください。
