Trends ルックアップエンドポイントは、where-on-earth ID（WOEID）を使用して指定したロケーションのトレンドを取得できます。

<Note>
  **注記**: WOEID は Yahoo によって作成されたレガシー識別子で、現在は非推奨です。X API は、都市や国のトレンドのロケーションを識別するために数値を使用します。レガシーの[ブログ記事](https://blog.x.com/engineering/en_us/a/2010/woeids-in-twitters-trends.html)や[アーカイブデータ](https://archive.org/details/geoplanet_data_7.10.0.zip0.)を参照してください。
</Note>

多くのトレンドでは、利用可能な場合、直近24時間の `tweet_count` も返されます。

このエンドポイントは app-auth 認証をサポートし、15分あたり75件のレート制限があります。

<div id="getting-started">
  ### はじめに
</div>

このエンドポイントを使用するには、[開発者ポータル](https://developer.x.com/en/portal/dashboard)から\[ベアラートークン]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)が必要です。ベアラートークンを取得したら、以下に示すようにAPIを呼び出すことができます：

```
      curl 'https://api.x.com/2/trends/by/woeid/26062' --header 'Authorization: Bearer XXXXX'
```

**アカウント設定**

これらのエンドポイントにアクセスするには、以下が必要です：

- 承認された[開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)
- [プロジェクト](/ja/resources/fundamentals/projects)内に配置された[開発者アプリ](/ja/resources/fundamentals/developer-apps)のキーとトークンを使用した認証

X API v2エンドポイントへのアクセス取得について詳しくは、[はじめにガイド](/ja/x-api/getting-started/getting-access)をご覧ください。
