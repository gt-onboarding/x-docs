投稿は X における主要なリソースの一つです。最も単純な形では、最大 280 文字を含み、アカウント設定に応じて公開または非公開で投稿できます。また、メディア、場所、投票、URL など、さまざまなオブジェクトを投稿に添付できます。さらに、ほとんどの投稿は作成後 30 分間は編集可能です。

投稿には HTTP、選択、配信のさまざまな方法がありますが、この REST エンドポイント群は、[投稿 ID](/ja/resources/fundamentals/x-ids)で指定した単一または複数の投稿を返すことに特化しています。これらのエンドポイントは、投稿の最新情報の取得、利用可能性の確認、編集履歴の参照が可能です。また、[コンプライアンスイベント](/ja/x-api/enterprise-gnip-2.0/fundamentals/firehouse)の管理にも不可欠です。

投稿ルックアップエンドポイントは、編集済み投稿のメタデータを提供します。2022 年 9 月 29 日以降に作成されたすべての投稿には、編集されていない場合でも編集メタデータが含まれます。各編集で新しい投稿 ID が発行され、投稿の編集履歴は元の投稿から始まる投稿 ID の配列で表されます。

このエンドポイントは常に、最新の編集とその編集履歴を返します。30 分の編集ウィンドウが終了した後に収集された投稿は、最終版を表します。編集投稿のメタデータの詳細は、[Edit Posts の基本](/ja/x-api/fundamentals/edit-posts)をご参照ください。

これらのエンドポイントは GET HTTP メソッドを使用し、投稿本文、作成タイムスタンプ、ハッシュタグ・メンション・写真のリストおよびメタデータなどのフィールドを含む、1 件または複数の[Posts オブジェクト](/ja/x-api/fundamentals/data-dictionary)を返します。

---

<Note>
  ### アカウントのセットアップ

  これらのエンドポイントにアクセスするには、以下が必要です。

  - 承認済みの[開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)。
  - [プロジェクト](/ja/resources/fundamentals/projects)内の[開発者アプリ](/ja/resources/fundamentals/developer-apps)で取得したキーとトークンによる認証。

  X API v2 エンドポイントへのアクセス方法の詳細は、[はじめにガイド](/ja/x-api/getting-started/getting-access)をご覧ください。
</Note>
