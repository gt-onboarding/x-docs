このエンドポイントでは、定義した条件に基づいて返信をプログラムから非表示／再表示できます。メインの X の体験と同様に、返信はメインの会話からは見えなくなりますが、別ページで引き続き閲覧できます。このエンドポイントを使って、価値がないと感じる返信を非表示にするためのアプリを作成できます。

hide replies エンドポイントは [OAuth 1.0a User Context](/ja/resources/fundamentals/authentication#oauth-1-0a-2) または [OAuth 2.0 Authorization Code with PKCE](ja/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) 認証を使用します。成功すると、このエンドポイントは、認証済みユーザーが開始した投稿の会話内に公開された単一の返信を非表示にします。各会話では最大 725 件まで投稿を非表示にできます。

<Note>
  **アカウント設定**

  これらのエンドポイントにアクセスするには、次が必要です。

  - 承認済みの[開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)。
  - [プロジェクト](/ja/resources/fundamentals/projects) 内にある [developer App](/ja/resources/fundamentals/developer-apps) のキーとトークンを使用して認証すること。 

  X API v2 エンドポイントへのアクセス方法の詳細は、[はじめにガイド](/ja/x-api/getting-started/getting-access)をご覧ください。
</Note>
