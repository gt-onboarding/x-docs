<div id="apps-using-the-hide-replies-endpoint">
  ## 返信非表示エンドポイントを利用するアプリ
</div>

返信非表示エンドポイントを統合した開発者が構築した注目アプリのまとめです。自社の技術をX APIと組み合わせることで、公共の会話の質を高める有用な体験を生み出しています。

![Clarabridge](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/clarabridge.jpg.twimg.1920.jpg)

**Clarabridge**

Clarabridgeは返信非表示機能を取り入れ、企業がXのフィードを読みやすく魅力的な状態に保てるようにします。返信は手動で非表示にでき、担当者はClarabridgeのテキスト分析を使って、不適切表現やブランドポリシーに反する言語を自動検出することもできます。

[ウェブサイトへ ▸](https://www.clarabridge.com/ "Go to website ▸")

[X上のClarabridge](https://x.com/Clarabridge "Clarabridge on X")

![Perspective API template app](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/perspective-logo.png.twimg.1920.png)

**Perspective API テンプレートアプリ**

Perspectiveは、Alphabet内の組織であるJigsawが有害なコメントを検出するために訓練した人工知能です。このアプリは受信した返信を判定し、過去に確認した有害コメントに類似している可能性をPerspectiveがどの程度高いとみなすかを示す「スコア」に基づいて、返信を自動的に非表示にします。これはXが開発したオープンソースのテンプレートアプリです。ユーザー向けツールの構築にあたって自由にカスタマイズしてください。

[アプリを使う ▸](https://quintessential-yoke.glitch.me "Use app ▸")

[X上のJigsaw](https://x.com/jigsaw "Jigsaw on X")

![Reshuffle](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 150 150%22%3E%3C/svg%3E)

**Reshuffle**

Reshuffleは、柔軟なスクリプトでビジネスアプリケーション同士を接続するプラットフォームです。指定したキーワードを含む返信を検出して非表示にする統合用スクリプトを作成しました。試す際は、返信非表示が有効なアプリを用意し、リポジトリの手順に従ってください。

[アプリへ ▸](https://github.com/reshufflehq/reshuffle "Go to app ▸")

[X上のReshuffle](https://x.com/reshufflehq "Reshuffle on X")

![Hide unwanted replies](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 396 397%22%3E%3C/svg%3E)

**Hide unwanted replies**

人気アプリQuotedRepliesの開発者Dara Oladosuは、過去にミュートした特定のキーワードを含む返信など、虐待的な行為を示す可能性が高いと判断した基準を満たす返信を自動的に非表示にするアプリを構築しました。

[アプリへ ▸](https://hideunwantedreplies.com/ "Go to app ▸")

[X上のDara Oladosu](https://x.com/dara_tobi "Dara Oladosu on X")

![Pandaflow](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 94 94%22%3E%3C/svg%3E)

**Pandaflow**

Pandaflowは、ワークフロー自動化のためのノーコードプラットフォームです。エンドポイントを統合し、ユーザーが定義したカスタムフローに基づいて返信をプログラムで非表示にできるようにしています。

[アプリへ ▸](https://pandaflow.io/ "Go to app ▸")

[X上のPandaflow](https://x.com/pandaflowio "Pandaflow on X")
