<div id="standard-v11-compared-to-x-api-v2">
  ## Standard v1.1 と X API v2 の比較
</div>

標準版 v1.1 の [POST statuses/update](https://developer.x.com/en/docs/twitter-api/v1/tweets/post-and-engage/api-reference/post-statuses-update) および [POST statuses/destroy/:id](https://developer.x.com/en/docs/twitter-api/v1/tweets/post-and-engage/api-reference/post-statuses-destroy-id) エンドポイントを利用している場合、本ガイドは、標準版と X API v2 の投稿管理エンドポイントにおける共通点と相違点の理解を支援することを目的としています。

- **共通点**
  - 認証
- **相違点**
  - エンドポイント URL

  - アプリとプロジェクトの要件

  - リクエストパラメータ

<div id="similarities">
  ### 共通点
</div>

**認証**

標準 v1.1 と X API v2 の両方で、投稿（[POST statuses/update](https://developer.x.com/en/docs/twitter-api/v1/tweets/post-and-engage/api-reference/post-statuses-update) および [POST statuses/destroy/:id](https://developer.x.com/en/docs/twitter-api/v1/tweets/post-and-engage/api-reference/post-statuses-destroy-id)）エンドポイントは [OAuth 1.0a User Context](https://developer.x.com/content/developer-twitter/resources/fundamentals/authentication) を使用します。したがって、以前に標準 v1.1 のいずれかのエンドポイントを使用していた場合、X API v2 版へ移行しても同じ認証方法を引き続き使用できます。 

<div id="differences">
  ### 相違点
</div>

**エンドポイント URL**

- Standard v1.1 エンドポイント:
  - https://api.x.com/1.1/statuses/update.json
    （投稿を作成）
  - `https://api.x.com/1.1/statuses/destroy/:id.json`
    （投稿を削除）
- X API v2 エンドポイント:
  - https://api.x.com/2/tweets
    （投稿を作成）
  - https://api.x.com/2/tweets/:id
    （指定した投稿を削除）

<div id="app-and-project-requirements">
  ### アプリとプロジェクトの要件
</div>

X API v2 のエンドポイントでは、リクエストを認証する際に、[developer App](/ja/resources/fundamentals/developer-apps) の資格情報を使用し、そのアプリが[Project](/ja/resources/fundamentals/developer-apps)に紐づいている必要があります。X API v1.1 のすべてのエンドポイントでは、スタンドアロンのアプリ、またはプロジェクトに紐づいたアプリの資格情報を使用できます。

<div id="request-parameters">
  ### リクエストパラメータ
</div>

v1.1 では、標準のリクエストで 2 つのクエリパラメータ（user\_id または screen\_name）を受け付けていました。X API v2 では、DELETE エンドポイントは数値の Post ID のみを受け付け、これはエンドポイントパスの一部として渡す必要があります。

POST エンドポイントでは、追加パラメータをリクエストの JSON ボディに含めて渡します。利用可能なパラメータの詳細は [API リファレンスガイド](/ja/x-api/posts/manage-tweets/introduction)をご覧ください。
