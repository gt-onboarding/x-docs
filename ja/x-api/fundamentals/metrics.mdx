title: "メトリクス"
---

<div id="overview">
  ## 概要
</div>

metrics フィールドを使うと、開発者は投稿およびメディアオブジェクトの公開・非公開エンゲージメントメトリクスにアクセスできます。公開メトリクスは開発者アカウントを持つ人なら誰でも利用でき、非公開メトリクスは所有者／承認済みアカウント（定義は後述）からのみアクセス可能です。メトリクスには、リクエストで指定した各投稿について、インプレッション、リポスト、引用、いいね、返信、動画再生数、動画再生の四分位、URL およびプロフィールリンクのクリック数の合計が含まれます。投稿が[広告](https://ads.x.com/)として配信された場合、オーガニックとプロモーションそれぞれのコンテキストで獲得したメトリクスの内訳を表示することもできます。

公開メトリクスは [App-only Token](/ja/resources/fundamentals/authentication#oauth-2-0) 認証でリクエストできます。非公開メトリクスは所有者／承認済みの投稿に対してのみリクエスト可能で、開発者は [OAuth 2.0](/ja/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-3) または [OAuth 1.0a](/ja/resources/fundamentals/authentication#oauth-1-0a-2) のユーザーコンテキスト認可で認証する必要があります。

非公開・オーガニック・プロモーションの各メトリクスは、過去30日以内に作成された投稿に対してのみ利用可能です。

<div id="terminology">
  ### 用語
</div>

- **承認済みアカウント**: あなたの[X developer app](/ja/resources/fundamentals/developer-apps)に、そのアカウントへのアクセスを付与して認可した X アカウント（どの[アプリ権限レベル](/ja/resources/fundamentals/developer-apps#app-permissions)でも投稿メトリクスへアクセス可能）。
- **所有アカウント**: あなたの[X developer app](/ja/resources/fundamentals/developer-apps)にリンクされた X アカウント。
- **公開メトリクス**: [X](https://x.com/home)上で誰でも参照できる合計値（例: いいね数、リポスト数）。
- **非公開メトリクス**: [X](https://x.com/home)上で一般公開されていない合計値（例: インプレッション数、動画視聴クォータイル）。利用には[OAuth 2.0](/ja/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-3)または[OAuth 1.0a](/ja/resources/fundamentals/authentication#oauth-1-0a-2)のユーザーコンテキスト認証が必要。
- **オーガニックメトリクス**: オーガニックなコンテキスト（通常どおりに投稿・閲覧された場合）に紐づく公開・非公開メトリクスのグループ。利用には[OAuth 2.0](/ja/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-3)または[OAuth 1.0a](/ja/resources/fundamentals/authentication#oauth-1-0a-2)のユーザーコンテキスト認証が必要。
- **プロモーションメトリクス**: プロモーションコンテキスト（広告キャンペーンの一部として投稿または配信・閲覧された場合）に紐づく公開・非公開メトリクスのグループ。利用には[OAuth 2.0](/ja/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-3)または[OAuth 1.0a](/ja/resources/fundamentals/authentication#oauth-1-0a-2)のユーザーコンテキスト認証が必要であり、かつ対象の投稿が広告としてプロモーションされている必要があります。

<div id="available-metrics">
  ## 利用可能なメトリクス
</div>

| Metric               | API Representations                                                                                                                                                                                                                                     | Description                                                                                                                                                                                                                                                                                                                                                                                |
|:----------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Impressions**      | `data.non_public_metrics.impression_count`, `data.organic_metrics.impression_count`, `data.promoted_metrics.impression_count`                                                                                                                           | 投稿が表示された回数（ユーザー単位のユニーク数ではありません）。投稿の一部でも画面に表示されればビューとしてカウントされます。OAuth 1.0a ユーザーコンテキスト認証が必要です。                                                                                                               |
| **Retweets**         | `data.public_metrics.retweet_count`, `data.organic_metrics.retweet_count`, `data.promoted_metrics.retweet_count`                                                                                                                                       | 投稿がリツイートされた回数。引用投稿（「コメント付きリツイート」）は含みません。X クライアントに表示される「リツイートとコメント」の合計を取得するには、`retweet_count` と `quote_count` を合算してください。                                           |
| **Quote Tweets**     | `data.public_metrics.quote_count`                                                                                                                                                                                                                      | 投稿が新しいコメント（メッセージ）付きでリツイートされた回数。有料コンテキストでの引用投稿は存在しないため、すべての引用投稿はオーガニックです。                                                                                                                                           |
| **Likes**            | `data.public_metrics.like_count`, `data.organic_metrics.like_count`, `data.promoted_metrics.like_count`                                                                                                                                                 | 投稿が「いいね」された回数。`public_metrics` フィールドは、公開表示との整合性を保つため、オーガニックと有料の両方のコンテキストからの「いいね」の合計を返します。                                                                                                   |
| **Replies**          | `data.public_metrics.reply_count`, `data.organic_metrics.reply_count`, `data.promoted_metrics.reply_count`                                                                                                                                              | 投稿に返信された回数。`public_metrics` フィールドは、オーガニックと有料の両方のコンテキストからの返信の合計を返します。                                                                                                                                                   |
| **URL Link Clicks**  | `data.non_public_metrics.url_link_clicks`, `data.organic_metrics.url_link_clicks`, `data.promoted_metrics.url_link_clicks`                                                                                                                             | ユーザーが投稿内の URL リンクまたは URL プレビューカードをクリックした回数。OAuth 1.0a ユーザーコンテキスト認証が必要です。                                                                                                                                                                  |
| **User Profile Clicks** | `data.non_public_metrics.user_profile_clicks`, `data.organic_metrics.user_profile_clicks`, `data.promoted_metrics.user_profile_clicks`                                                                                                             | 投稿内の該当箇所（表示名、ユーザー名、プロフィール画像）をユーザーがクリックした回数。OAuth 1.0a ユーザーコンテキスト認証が必要です。                                                                                                                                               |
| **Video views**      | `includes.media.public_metrics.view_count`, `includes.media.organic_metrics.view_count`, `includes.media.promoted_metrics.view_count`                                                                                                                  | 投稿に含まれる動画が視聴された回数。これは、当該動画が含まれるすべての投稿を横断して集計された動画視聴回数です。メディアのエクスパンション `expansions=attachment.media_keys` が必要です。                                                                |
| **Video view quartiles** | `includes.media.non_public_metrics.playback_0_count`, `includes.media.non_public_metrics.playback_25_count`, `includes.media.non_public_metrics.playback_50_count`, `includes.media.non_public_metrics.playback_75_count`, `includes.media.non_public_metrics.playback_100_count` | 動画の各四分位点まで再生したユーザー数。OAuth 1.0a ユーザーコンテキスト認証およびメディアのエクスパンション `expansions=attachment.media_keys` が必要です。                                                                                                 |

<div id="requesting-metrics">
  ## メトリクスの取得リクエスト
</div>

<div id="public-metrics">
  ### 公開メトリクス
</div>

次のリクエストでは、以下のフィールドと拡張を指定して、投稿および添付動画の公開メトリクスを取得します。必ず `$BEARER_TOKEN` を自身で生成したベアラートークンに置き換えてください。

- `tweet.fields=public_metrics`
- `expansions=attachments.media_keys&media.fields=public_metrics`

<div id="sample-request">
  #### サンプルリクエスト
</div>

```bash
curl 'https://api.x.com/2/tweets?ids=1204084171334832128&tweet.fields=public_metrics&expansions=attachments.media_keys&media.fields=public_metrics' --header 'Authorization: Bearer $BEARER_TOKEN'
```

<div id="private-metrics-non-public-organic-metrics">
  ### 非公開メトリクス（非公開・オーガニックメトリクス）
</div>

次のリクエストでは、投稿と添付動画について、オーガニックメトリクスの詳細を含む非公開メトリクスを取得します。これらのフィールドは非公開で X 上で一般には表示されないため、リクエストには [OAuth 2.0](/ja/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-3) または [OAuth 1.0a](/ja/resources/fundamentals/authentication#oauth-1-0a-2) のユーザーコンテキスト認証が必要です。以下で必要となる OAuth 1.0a 署名の生成方法は、当社の[ガイド](/ja/resources/fundamentals/authentication#creating-a-signature)を参照してください。

- `tweet.fields=non_public_metrics,organic_metrics`
- `expansions=attachments.media_keys&media.fields=non_public_metrics,organic_metrics`

<div id="sample-request">
  #### リクエスト例
</div>

```bash
curl 'https://api.x.com/2/tweets/1204084171334832128?tweet.fields=non_public_metrics,organic_metrics&media.fields=non_public_metrics,organic_metrics&expansions=attachments.media_keys' --header 'authorization: OAuth oauth_consumer_key="CONSUMER_API_KEY", oauth_nonce="OAUTH_NONCE", oauth_signature="OAUTH_SIGNATURE", oauth_signature_method="HMAC-SHA1", oauth_timestamp="OAUTH_TIMESTAMP", oauth_token="ACCESS_TOKEN", oauth_version="1.0"'
```
