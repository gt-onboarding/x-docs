---
title: "X API について"
---

X API は、X のデータをプログラムで取得・分析したり、X 上の会話を支えるプロダクトを構築したりするために利用できます。

長年にわたり、X API は開発者が公開の会話を拡充・研究できるよう、スケールに応じた複数のアクセスレベルを追加して発展してきました。

最近、X API v2 をリリースしました。X API v2 には、モダンな基盤、新機能および高度な機能、そして [Basic access](https://developer.x.com/en/portal/products/basic) への迅速なオンボーディングが含まれています。

以下の 3 つのタブでは、X API の各バージョンとアクセスレベル、v2 の新機能、そして API を使用して取得・作成・削除・調整できる X リソースについて説明します。

  <Tab title="What's new with v2">
    ## v2 の新機能

    X API v2 は、2012 年以降で最大の X API のアップグレードです。これにより、多数の新機能や高度な機能に加え、API への高速かつ無料のアクセスが可能になります。

    v2 で利用可能な機能には、次のようなものがあります。

    ### 新しいエンドポイント

    X API v2 に対して、まったく新しい一連のエンドポイントをリリースしました。新規エンドポイントを含む v2 エンドポイントの全リストは、X API エンドポイントマップガイドでご確認ください。

    [Visit the X API endpoint map >](/ja/x-api/migrate/x-api-endpoint-map)

    ### 新しく、より詳細なデータオブジェクト

    データオブジェクトを最新化し、さまざまな改良を加えました。これにより、データの探索や解析がさらに容易になります。

    [Visit the data dictionary >](/ja/x-api/fundamentals/data-dictionary)

    ### 取得したいオブジェクトとフィールドだけを取り出すための新しいパラメータ

    データエンドポイントに `fields` と `expansions` のパラメータを追加しました。これにより、デフォルトで返されるフィールドを超えて、関連オブジェクトやフィールドをリクエストできます。

    [Learn how to use fields and expansions >](/ja/x-api/fundamentals/data-dictionary#how-to-use-fields-and-expansions)

    ### 高度なメトリクス

    インプレッション、動画再生数、ユーザープロフィール、URL クリックなどの公開・非公開メトリクスをリクエストすることで、ペイロード内から直接、投稿、ユーザー、メディア、投票のパフォーマンスを容易に把握できます。これらの一部はオーガニックとプロモーションの文脈で区別して提供されます。

    [Learn more about metrics >](/ja/x-api/fundamentals/metrics)

    ### トピック別に、異なるトピックを含む投稿のフィルタリングと特定

    検索やフィルタードストリームを使用する際、エンティティおよびコンテキストオペレーターを用いてトピックでフィルタリングできるようになりました。分析を支援するため、これらのトピックは投稿のペイロード内にも含めています。

    [Learn more about Post annotations >](/ja/x-api/fundamentals/post-annotations)

    ### 返信スレッドに属する投稿のフィルタリングと特定

    検索、フィルタードストリーム、投稿ルックアップを使用する際、会話スレッドの一部としての投稿をより容易に特定できます。さらに、投稿フィールド `reply_settings` により、その投稿に会話の返信設定が適用されているかどうかを判定できるようになりました。

    [Learn more about conversation tracking >](/ja/x-api/fundamentals/conversation-id)

    ### そのほか多数の改善

    - 高精度なスパムフィルタリング
    - 短縮 URL を完全に展開し、URL 分析を容易に
    - 非推奨フィールドの削除とラベルの最新化によるシンプルな JSON レスポンスオブジェクト
    - ストリーミングエンドポイントのリカバリーと冗長化
    - 検索クエリで一致する公開かつ利用可能な投稿を 100% 返却
    - 接続を切断せずに変更可能なストリーミングの「ルール」
    - フィルタードストリームと検索のための、より表現力の高いクエリ言語
    - 新しいライブラリの構築と変更の透明性の高い追跡のための OpenAPI 仕様
    - プラットフォームの進化に合わせ、開発者、研究者、企業、X を利用する人々のニーズに応える新機能やエンドポイントへの API サポートを、より迅速に提供
  </Tab>

  <Tab title="X API resources">
    ## X API プラットフォームのリソース

    API 設計の文脈では、リソースとは、関連データや他のリソースとの関係性を持ち、それに対して操作可能なメソッド群が定義されたエンティティを指します。たとえば、Post はリソースであり、過去データの検索やリアルタイム取得など、さまざまなツールを用いて作成・削除・取得できます。

    X API は、プラットフォーム上の多様なリソースの作成・削除・受信・調整にアクセスを提供しており、以下を含みます。

    | Resource            | Description                                                                                                                                                                             |
    | :------------------ | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
    | **Posts**           | 何百万もの Post にアクセスしてパブリックな会話を把握したり、自分の投稿を作成して会話に参加できます。 現在の提供状況:  - X API v2  - Enterprise  - Standard v1.1 |
    | **Users**           | X のユーザーを管理・照会してネットワークを分析し、オーディエンスを理解し、健全なオンライン関係を育むことができます。 現在の提供状況:  - X API v2  - Enterprise  - Standard v1.1     |
    | **Spaces**          | X Spaces とその参加者を照会・検索して、人々が興味深く関連性の高い音声会話を見つけられるよう支援します。 現在の提供状況:  - X API v2                                    |
    | **Direct Messages** | Direct Messages を送受信して、顧客課題のトリアージ、ウェルカムメッセージの送信、前向きなコミュニケーションを実現します。 現在の提供状況:  - Standard v1.1                        |
    | **Lists**           | アカウントのリストを作成・管理して、業界の有識者や影響力のある声を追跡したり、フォロー対象を整理したりできます。 現在の提供状況:  - X API v2  - Standard v1.1               |
    | **Trends**          | 地理的トレンドをいち早く特定して、業界動向を把握し、注目トピックを発見し、最新の流行を先取りできます。 現在の提供状況:  - Standard v1.1                           |
    | **Media**           | メディアオブジェクトをアップロードして、クリエイティブを共有し、インタラクティブな体験を作成したり、アクセシビリティツールを構築できます。 現在の提供状況:  - Standard v1.1                               |
    | **Places**          | 場所を検索して、身近な地域や世界中で起きていることを把握できます。 現在の提供状況:  - Standard v1.1                                                     |
