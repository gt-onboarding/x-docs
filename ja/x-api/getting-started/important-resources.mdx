<div id="learn-about-whats-possible">
  ## できることを知る
</div>

トップナビゲーションからアクセスできるユースケース、ソリューション、製品ページは、プラットフォームで可能なことを俯瞰するのに最適です。

また、さまざまなユースケースに対応するために、1つまたは複数のプラットフォームツールの使い方を理解するのに役立つ[チュートリアル](/ja/tutorials)も複数用意しています。

ドキュメントの最下層にはエンドポイントおよびツールのドキュメントがあり、いずれもそのツールの機能と典型的なユースケースを簡潔に説明する導入コンテンツを含んでいます。たとえば、こちらは[X API v2 の Search Posts](/ja/x-api/posts/search/introduction)エンドポイントセクションです。

<div id="integrate-an-endpoint-into-your-system">
  ### エンドポイントをシステムに統合する
</div>

エンドポイントとの統合を円滑に進められるよう、各種ツール、ライブラリ、統合ガイドを用意しています。最も簡単な方法は、[ツールとライブラリ](/ja/x-api/tools-and-libraries/overview)を利用することです。これらは、[OAuth 1.0a User Context 認証](/ja/resources/fundamentals/authentication#oauth-1-0a-2)、ページネーション、[レート制限のハンドリング](/ja/resources/fundamentals/rate-limits)などの複雑な処理を自動で行います。

ゼロから実装したい場合は、エンドポイント統合ガイドで、エンドポイントをシステムに組み込む際の機能とベストプラクティスを解説しています。これらのガイドは、各エンドポイントやツールのドキュメントに「integrate」または単に「guides」として掲載されています。例えば、[フィルタードストリームのルールを作成する方法](/ja/x-api/posts/filtered-stream/integrate/build-a-rule)に関する統合ガイドがあります。

<div id="troubleshoot-an-issue">
  ### 問題のトラブルシューティング
</div>

エラーが発生したりご不明点がある場合は、[サポートハブ](https://developer.x.com/en/support/twitter-api)をご確認ください。トラブルシューティングのヒント、よくある質問への回答、その他の有用なリソースをご覧いただけます。

<div id="keep-up-to-date-with-the-latest-releases">
  ### 最新リリースを常に把握する
</div>

私たちは、API を通じて新しい X の機能を利用可能にするため、プラットフォームを頻繁にアップデートしています。さらに、少なくとも年に一度はメジャーバージョンを更新し、破壊的変更が含まれる場合があります。

最新情報を受け取るには、[フォーラムの Announcements カテゴリ](https://devcommunity.x.com/c/announcements/22)を購読し、X で [@API](https://x.com/api) と [@XDevelopers](https://x.com/XDevelopers) をフォローして通知をオンにし、[最新情報](https://developer.x.com/en/stay-informed)ページで各種リソースをご確認ください。

あわせて、最新の API バージョンへの移行を案内する新しい[移行ハブ](/ja/x-api/migrate/overview)も用意しています。
