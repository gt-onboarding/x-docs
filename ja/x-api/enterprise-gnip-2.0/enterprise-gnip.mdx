エンタープライズ向け API は、X のデータに依存する皆さまに、最高水準のアクセスと信頼性を提供します。プレミアムを超えて拡張し、より高い可用性のアクセスやカスタムパッケージ、長期契約が必要な場合に最適です。エンタープライズ API へのアクセスには、専任のアカウントマネージャーと個別の技術サポートが含まれます。

Basic および Pro のソリューションではニーズを満たせない場合は、以下のリンクからエンタープライズ製品へのアクセスを申請してください。

  アクセスを申請
</Button>

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

| リアルタイム | [PowerTrack API](/ja/x-api/enterprise-gnip-2.0/powertrack-api) | 公開投稿を発生と同時に完全網羅で取得し、幅広いオペレーターとルールでフィルタリングできます。 |
| :--- | :--- | :--- |
| [Decahose API](/ja/x-api/enterprise-gnip-2.0/fundamentals/decahose-api) | すべての公開投稿の10%のランダムサンプルと、その「いいね」をストリーミング配信します。 |
| :--- | :--- | :--- |
| [Account Activity API](/ja/x-api/enterprise-gnip-2.0/fundamentals/account-activity) | ユーザーを購読し、投稿、ダイレクトメッセージ、いいね、フォローなど多様なアクティビティを受信します。 |
| :--- | :--- | :--- |
| 履歴 | [Search API](/ja/x-api/enterprise-gnip-2.0/fundamentals/search-api) | 高度なフィルタリングツールを使って、直近30日または公開投稿の全アーカイブを検索します。 |
| [Historical PowerTrack API](/ja/x-api/enterprise-gnip-2.0/powertrack-api) | 公開投稿の全アーカイブを、コスト効率の高いバッチアクセス方式でフィルタリングします。 |
| インサイト | [Engagement API](/ja/x-api/enterprise-gnip-2.0/fundamentals/engagement-api) | 公開投稿のインプレッション数とエンゲージメント数を計測します。 |
| コンプライアンス | [Compliance Firehose API](/ja/x-api/enterprise-gnip-2.0/fundamentals/firehouse) | すべてのコンプライアンスイベントをストリーミングします。 |
| 利用状況 | [Usage API](/ja/x-api/enterprise-gnip-2.0/fundamentals/usage) | 企業向け製品全体のアクティビティ消費状況にプログラムからアクセスできます。 |

<div id="important-concepts">
  ## 重要な概念
</div>

<div id="gnip-console">
  ### Gnip コンソール
</div>

[Gnip コンソール](/ja/x-api/enterprise-gnip-2.0/fundamentals/overview)から、エンタープライズ製品とアカウントへのアクセスを管理できます。 

<div id="enrichments">
  ### エンリッチメント
</div>

有料ティアをご利用の開発者向けに、プレミアム API のレスポンスペイロードにメタデータを追加しました。展開済み URL、投票データ、プロフィールの位置情報については、[エンリッチメントのセクション](/ja/x-api/enterprise-gnip-2.0/fundamentals/data-enrichments)をご参照ください。
 

<div id="advanced-filtering-tools">
  ### 高度なフィルタリングツール
</div>

Premium Search API では、標準の検索エンドポイントでは使えない高度なオペレーター群を提供しています。これらのオペレーターの多くは無料のサンドボックス環境でも利用できますが、有料ティアをご利用いただくと、より充実したオペレーターをお使いいただけます。詳しくは、[ルールとフィルターのセクション](/ja/x-api/enterprise-gnip-2.0/fundamentals/rules-filtering#getting-started-with-enterprise-rules-and-queries)をご覧ください。

<div id="explore-x-api-v2">
  ## X API v2 を探索する
</div>

v2 で利用可能な機能を確認したい方は、まずドキュメントで新機能をご覧ください。移行ハブでは、最終的に X API v2 へ移行するための各種リソースも提供しています。新しい v2 のエンドポイントセクションに掲載されている各エンドポイントの移行ガイドにもアクセスできます。

[詳細を見る](/ja/x-api/migrate/overview)

サポートが必要ですか？

サポートセクションをご確認いただくか、アカウントマネージャーにご連絡のうえ、サポートチームへの連絡方法をご確認ください。
