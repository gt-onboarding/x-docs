title: "レート制限（エンタープライズ）"
---

<div id="overview">
  ## 概要
</div>

毎日、何千人もの開発者が X API にリクエストを送信しています。こうした膨大なリクエスト量を適切に管理するため、送信可能なリクエスト数には上限を設けています。これらの上限により、開発者コミュニティが依拠する信頼性とスケーラビリティに優れた API を提供できます。

許可されるリクエストの最大数は、一定の時間間隔、すなわち特定の期間や時間ウィンドウに基づきます。たとえば、あるエンドポイントに 900 リクエスト/15 分のレート制限がある場合、任意の 15 分間に最大 900 件のリクエストが許可されます。

|     | 同時に見積もり可能なジョブは最大2件 |
|     | 同時に実行可能なジョブは最大2件 |
| Decahose API |     | 1分あたり10リクエスト |
| Streaming likes API |     | 1分あたり10リクエスト |
| Firehose API |     | 1分あたり10リクエスト |
| Account Activity API | POST account_activity/webhooks | 15分あたり15リクエスト |
