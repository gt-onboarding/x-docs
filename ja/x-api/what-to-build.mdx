<div id="introduction">
  ## はじめに
</div>

私たちの開発者コミュニティは、X が単独では成し得ない範囲を超えて不足を補い、課題を解決し、X 上で新たなイノベーションの芽を育むための独自のスキル、経験、視点を備えています。X をより良く、より健全にし、公共の対話を広げるツールやプロダクトの構築を推奨します。特に、次のカテゴリでのイノベーションを期待しています。

- [健全性と安全性のための会話のモデレーション](#moderate)
- [創作と自己表現を可能にする](#enable)
- [起きていることを測定・分析する](#measure)
- [コミュニティの体験を向上させる](#improve)
- [コンテンツをキュレーションし推奨する](#curate)
- [より大きな公益に資する](#impact)

<div id="moderate-conversations-for-health-and-safety">
  ### 健全性と安全のために会話をモデレートする
</div>

X 上の会話に、誰もが安心・安全に、そして前向きな気持ちで参加できるようにしたいと考えています。Post の返信設定、ブロック、ミュートといった、利用者が自身の体験をより細かく管理できるツールの提供に取り組んできました。とはいえ、取り組むべき課題は常に存在します。そこで、プラットフォームの健全性と安全性の向上に資する補完的・付加的なソリューションの開発を奨励します。以下のリソースを活用して、取り組みを始めてください。

<div id="enable-creation-and-personal-expression">
  ### 創造と自己表現を可能にする
</div>

X は、人々が自分の見解、アイデア、情熱を共有する場です。個人とブランド、創業者とアスリート、著名人と CEO が交差する場所でもあります。開発者は、X 上の人々がリーチを拡大し、新たな方法で自己表現し、志を同じくするコミュニティとつながれるようにする創造的なソリューションを構築できます。クリエイターでありイノベーターでもある開発者は、役立つボット、ゲーミフィケーション、クロスポストされたコンテンツなど、人々の X での体験を広げるものも構築できます。以下は、イノベーションの着想を得るためのいくつかのアイデアです。

<div id="improve-community-experiences">
  ### コミュニティ体験の向上
</div>

新たなコミュニティがX上で成長し集うに伴い、私たちは開発者の専門性を活かして、各コミュニティの固有のニーズに応えてほしいと考えています。Xはグローバルな会話の場であり、言語、タイムゾーン、文化的な違い、その他の地域特有のニーズにかかわらず、誰もが参加できるようにしたいと考えています。さらに、金融やゲームなどのトピックを軸に成長するコミュニティには、その体験やニーズに最適化されたツールが提供されることを目指しています。
