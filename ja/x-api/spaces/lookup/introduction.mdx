ID ルックアップ用エンドポイントは、ユーザーに Spaces の情報を表示するエクスペリエンスを構築したり、1 つ以上の Spaces の詳細を取得したりするために利用できるデータを返します。1 回のリクエストで複数の Spaces を照会できます。

作成者 ID で Spaces を照会することも可能です。このエンドポイントは、リスト内のユーザーが開催した Spaces を取得したい場合や、複数の定期的なホストが現在配信中か、今後の Space を予定しているかを判定したい場合に有用です。1 回のリクエストで最大 100 ユーザーまで照会できます。

**アカウントのセットアップ**

これらのエンドポイントにアクセスするには、次が必要です。

- 承認済みの[開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)
- [プロジェクト](/ja/resources/fundamentals/projects) 内の[開発者アプリ](/ja/resources/fundamentals/developer-apps)で発行されたキーとトークンを用いた認証

X API v2 の各エンドポイントへのアクセス方法については、[はじめにガイド](/ja/x-api/getting-started/getting-access)をご覧ください。
