このエンドポイントでは、Space をタイトルで検索でき、ステータスで結果を絞り込めます。タイトルに含まれるキーワード、メンションされたユーザー、またはハッシュタグに基づいて、ライブ配信中または今後予定の Space を見つけるのに役立ちます。

このエンドポイントは、1 つ以上のキーワードをクエリとして受け付けます。結果は Space のステータス（ライブまたは予定）でフィルタリングできます。既定では、指定したクエリに一致するライブおよび予定の Space を両方返します。

<Note>
  **アカウント設定**

  これらのエンドポイントにアクセスするには、以下が必要です。

  - 承認済みの [開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)。
  - [プロジェクト](/ja/resources/fundamentals/projects) 内の [開発者アプリ](/ja/resources/fundamentals/developer-apps) のキーとトークンを使用して認証すること。

  X API v2 のエンドポイントへのアクセス方法の詳細は、[はじめにガイド](/ja/x-api/getting-started/getting-access)をご覧ください。
</Note>
