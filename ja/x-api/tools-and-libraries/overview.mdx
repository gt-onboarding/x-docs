<div id="community-tools-and-libraries-for-v2">
  ## v2 向けのコミュニティ製ツールとライブラリ
</div>

ここに掲載しているライブラリは、開発者コミュニティのメンバーによって作成されたものです。API の対応範囲や成熟度はそれぞれ異なる場合があります。

独自の X API ライブラリや便利なツールを作成された方は、[こちらからご連絡ください](https://devcommunity.x.com/c/libraries-and-sdks/63)。このリストに追加し、他の方が見つけやすいようにします。README に使える[バージョンバッジ](https://twbadges.glitch.me/)も用意していますのでご活用ください。

インスピレーションをお探しですか？他の開発者による有用なコード例は、GitHub の [X](https://github.com/topics/twitter?o=desc&s=updated) および [X-api-v2](https://github.com/topics/twitter-api-v2?o=desc&s=updated) トピックで閲覧・検索できます。

**ジャンプ先:** [C# / .NET](#csharp), [Dart / Flutter](#dart), [Go](#go), [Java](#java), [JavaScript (Node.JS) / TypeScript](#nodejs), [Kotlin](#kotlin), [PHP](#php), [PowerShell](#powershell), [Python](#python), [R](#r), [Ruby](#ruby), [Rust](#rust), [Swift](#swift)

<div id="additional-official-resources">
  ## 追加の公式リソース
</div>

以下のツールも、X API を扱う際に有用です。\
さらにコードをお探しですか？[GitHub](https://github.com/xdevplatform) や [Glitch](https://glitch.com/@twitter) にサンプルがあります。

| ツール／ライブラリ         | 説明                                                                                                                                                                               |
| :-------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| twemoji               | Twitter の無償・オープンソースの絵文字セット。クロスプラットフォーム対応のための JavaScript ライブラリを含みます。                                                                       |
| twitter-text          | 投稿テキストの構文解析とトークン化を標準化するライブラリ群。Java、JavaScript、Objective-C、Ruby に対応。[投稿の文字数カウントについて詳しくはこちら。](#)                                  |
| OpenAPI specification | この仕様を用いて、[Postman](#) や [Insomnia](#) などのツールで **v2** API を試せます。                                                                                                  |
| twurl                 | OAuth 認証を含む Twitter API と対話するためのコマンドラインツール（CLI）。Ruby の実行環境が必要です。                                                                                   |
| Postman collection    | テストや開発のために X API のエンドポイントを扱える Postman コレクションをご利用ください。                                                                                              |

<div id="community-tools-and-libraries">
  ## コミュニティ製ツールとライブラリ
</div>

以下は、複数のプログラミング言語やプラットフォームにわたり X API をカバーする、コミュニティ支援のライブラリの一部です。これらのリソースは、Twitter チームによる検証を受けていない場合がありますのでご留意ください。

ここに掲載しているライブラリは、特に記載がない限り、Standard API **v1.1** の機能の大部分を実装しているはずです。詳細や追加サポートについては、各作者にご確認ください。\
X API **v2** をサポートするライブラリを作成された場合は、本ページへの掲載を検討するため、[コミュニティフォーラム](https://devcommunity.x.com/c/libraries-and-sdks/63)からお知らせください。フォーラムでは、掲載内容の更新情報の報告にもご利用いただけます。

ご利用のプログラミング言語向けのライブラリやツールが見当たらない場合は、[フィードバックプラットフォーム](https://twitterdevfeedback.uservoice.com/)からお知らせください。そこでアイデアに投票したり、新しいものの作成・投稿のきっかけを得ることもできます。
