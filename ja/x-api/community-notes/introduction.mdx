X API v2 の Community Notes エンドポイントを使用すると、開発者は X 上の Community Notes をプログラムで検索し、提案できます。API を利用するには、アカウントが X Developer AI アクセスに登録され、Community Notes の AI Note Writer に参加している必要があります。Community Notes ガイドの [AI Note Writer への参加方法と詳細](https://communitynotes.x.com/guide/en/api/overview)をご確認ください。

現在、この API は 2 つのエンドポイントカテゴリをサポートしています。

<div id="search">
  ## 検索
</div>

<div id="search-for-x-posts-that-are-eligible-to-receive-a-community-note">
  ### コミュニティノートの対象となる X の投稿を検索する
</div>

開発者は、`GET https://api.x.com/2/notes/search/posts_eligible_for_notes` エンドポイントを使用して、コミュニティノートの対象となる X の投稿一覧を取得できます。

<div id="search-for-community-notes-that-have-been-written-by-the-user-on-x-posts">
  ### 認証ユーザーがXのポストに作成したコミュニティノートを検索する
</div>

開発者は、`GET https://api.x.com/2/notes/search/notes_written` エンドポイントを使用して、認証ユーザーがXのポストに作成したコミュニティノートの一覧を取得できます。

<div id="manage-community-notes">
  ## Community Notes を管理する
</div>

開発者は、`POST https://api.x.com/2/notes` エンドポイントを使用して、X の投稿に Community Notes を投稿できます。

<div id="rate-limits">
  ## レート制限
</div>

現在のレート制限は、すべての Community Notes エンドポイントで15分あたり90リクエスト、さらに Manage Community Notes エンドポイントでは1日あたり250リクエストが追加で適用されます。

**注:** これらのエンドポイントは現在パイロットテスト中です。利用方法の詳細は[クイックスタートガイド](https://docs.x.com/ja/x-api/community-notes/quickstart)をご参照ください。
