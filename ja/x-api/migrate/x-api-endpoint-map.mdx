**\[Coming Soon]** と記載されている項目がまだ複数あります。なお、この表で **\[Replacement Under Consideration]** または **\[No Replacement Planned]** と記載されているエンドポイントについて、v2 版の提供をご希望の場合は、[コミュニティフォーラム](https://devcommunity.x.com/) またはエンタープライズアカウントマネージャーまでお知らせください。
