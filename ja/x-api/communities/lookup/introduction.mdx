**はじめに**

<div id="authentication">
  ### 認証
</div>

このエンドポイントは、[OAuth 1.0a ユーザーコンテキスト](/ja/resources/fundamentals/authentication)、
\[OAuth 2.0 アプリのみ]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)、
または[OAuth 2.0 認可コード（PKCE）](ja/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2)で認証できます。
 

<div id="making-a-request">
  ### リクエストの送信
</div>

以下のように ID（ここでは `1758747817642700922`）を指定して、Communities Lookup エンドポイントを呼び出せます。

```
      curl --location 'https://api.x.com/2/communities/1758747817642700922' --header 'Authorization: ••••••'
```

リクエストが成功すると、次のような JSON レスポンスが返されます。

```
{
    "data": {
        "access": "Public",
        "description": "Welcome to the Anime Community! Where anime fans gather to share their favorite shows and discuss everything anime-related.",
        "join_policy": "Open",
        "name": "Anime Community"
    }
}
```

<div id="community-fields">
  ### コミュニティフィールド
</div>

コミュニティに関連付けられた追加データは、Community Fields を指定することで取得できます。

利用可能な Community Fields は次のとおりです。

| フィールド値 | 型 | 説明 |     |
| :--- | :--- | :--- | :--- |
| created\_at | date (ISO 8601) | コミュニティの作成日時。 |     |
| id  | string | コミュニティの一意の識別子。 |     |
| name | string | コミュニティ名。 |     |
| description | string | 提供されている場合のコミュニティの説明文。 |     |
| access | string | コミュニティのアクセスレベル。  <br />  <br />次のいずれか：<br /><br />\* `Public`<br />\* `Closed` |     |
| join\_policy | string | コミュニティの参加方針。  <br />  <br />次のいずれか：<br /><br />\* `Open`<br />\* `RestrictedJoinRequestsDisabled`<br />\* `RestrictedJoinRequestsRequireAdminApproval`<br />\* `RestrictedJoinRequestsRequireModeratorApproval`<br />\* `SuperFollowRequired` |     |
| member\_count | integer | コミュニティに参加しているメンバー数。 |     |

<div id="example-request">
  #### リクエスト例
</div>

以下のように、追加のコミュニティフィールドを指定して Communities Lookup エンドポイントを呼び出せます（ID `1758747817642700922` を使用）:

```
curl --location 'https://api.x.com/2/communities/1758747817642700922?community.fields=access,created_at,description,id,join_policy,member_count,name' --header 'Authorization: ••••••'
```

**レスポンス例**

リクエストが成功すると、次のような JSON レスポンスが返されます:

```
{
    "data": {
        "description": "Welcome to the Anime Community! Where anime fans gather to share their favorite shows and discuss everything anime-related.",
        "access": "Public",
        "member_count": 40369,
        "created_at": "2024-02-17T06:58:50.000Z",
        "join_policy": "Open",
        "name": "Anime Community",
        "id": "Q29tbXVuaXR5OjE3NTg3NDc4MTc2NDI3MDA5MjI="
    }
}
```
