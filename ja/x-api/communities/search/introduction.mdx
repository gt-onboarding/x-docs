このエンドポイントでは、キーワードに基づいてコミュニティを検索・発見できます。

このエンドポイントは、クエリとして1つ以上のキーワードを受け付けます。既定では、指定したクエリに一致する公開コミュニティと非公開（クローズド）コミュニティの両方を返します。

<Note>
  **アカウントのセットアップ**

  これらのエンドポイントにアクセスするには、以下が必要です。

  - 承認済みの[開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)
  - [プロジェクト](/ja/resources/fundamentals/projects)内にある[開発者アプリ](/ja/resources/fundamentals/developer-apps)のキーとトークンを用いた認証

  X API v2 エンドポイントへのアクセス方法の詳細は、[はじめに](/ja/x-api/getting-started/getting-access)をご覧ください。
</Note>

**はじめに**

<div id="authentication">
  ### 認証
</div>

このエンドポイントは、[OAuth 1.0a User Context](/ja/resources/fundamentals/authentication)、\[OAuth 2.0 App-Only]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token) または [OAuth 2.0 Authorization Code with PKCE](ja/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2)で認証できます。
 

<div id="making-a-request">
  ### リクエストの送信
</div>

以下のように Search Communities エンドポイントを呼び出します。

```
      curl --location 'https://api.x.com/2/communities/search?query=anime' --header 'Authorization: ••••••'
```

リクエストが成功すると、次のような JSON レスポンスが返されます。


<div id="community-fields">
  ### コミュニティフィールド
</div>

Community Fields を指定すると、コミュニティに紐づく追加データを取得できます。

利用可能な Community Fields は次のとおりです。

| フィールド値 | 型 | 説明 |     |
| :--- | :--- | :--- | :--- |
| created\_at | date (ISO 8601) | コミュニティの作成日時。 |     |
| id  | string | コミュニティの一意の識別子。 |     |
| name | string | コミュニティ名。 |     |
| description | string | コミュニティの説明文（存在する場合）。 |     |
| access | string | コミュニティのアクセスレベル。  <br />  <br />次のいずれか：<br /><br />\* `Public`<br />\* `Closed` |     |
| join\_policy | string | コミュニティの参加ポリシー。  <br />  <br />次のいずれか：<br /><br />\* `Open`<br />\* `RestrictedJoinRequestsDisabled`<br />\* `RestrictedJoinRequestsRequireAdminApproval`<br />\* `RestrictedJoinRequestsRequireModeratorApproval`<br />\* `SuperFollowRequired` |     |
| member\_count | integer | コミュニティの参加メンバー数。 |     |

<div id="example-request">
  #### リクエスト例
</div>

以下のように、追加のコミュニティフィールドを指定して Search Communities エンドポイントを呼び出せます:

```
      curl --location 'https://api.x.com/2/communities/search?query=anime&community.fields=access,created_at,description,id,join_policy,member_count,name' --header 'Authorization: ••••••'
```
