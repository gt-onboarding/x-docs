<Note>
**要点（tl;dr）は次のとおりです**

成功への歩み：Red Cloud Financial Services は、ある鉱業会社の新たなグリーンテクノロジーの市場展開を支援するにあたり、X のアナリティクスに基づいて構築された Audiense のターゲティングツールを活用し、データを掘り下げて新たなオーディエンスへとリーチしました。
</Note>

<div id="situation">
  ## 状況
</div>

Red Cloud Financial Services は、上場鉱業会社のクライアントが開発した特許出願中の新技術について、まったく新しいオーディエンスに認知を高める支援を求められました。 

同社は、この新製品で成果を生み、新規地域へ展開しながら、新たな顧客にリーチできる効果的なキャンペーンの進め方を見つける必要がありました。 

この依頼には 2 つの主要な課題がありました。 

1. **新しいオーディエンスにリーチする**

    既存の鉱業顧客に注力するのではなく、クライアントの新技術はグリーンイノベーションや社会的責任に関心のある、より幅広い層に訴求しました。これは Red Cloud にとって未踏の領域で、これまでのキャンペーンとはまったく性質が異なるものでした。

1. **具体的な ROI を提示する**
    
    クライアントは有料キャンペーンに慣れていませんでした。認知目的のキャンペーンでの ROI を Red Cloud が明確に示し、一次的な知見が限られるオーディエンスで結果を出すという期待は、ハードルの高い課題でした。 
    
    Red Cloud は、既存顧客にリーチしつつ見込みの新規顧客をリサーチするために Audiense を活用しました。Audiense の X マーケティング製品は、強力なオーディエンスターゲティングツールを用いて、効果的な X キャンペーンを提供することを得意としています。

<div id="solution">
  ## 解決策
</div>

Red Cloud はまず期待値の調整から着手しました。Audiense のインサイトを活用してキャンペーンのパフォーマンス予測を作成し、キャンペーンの最終目標と関連する KPI についてクライアントの期待値を適切に設定しました。 

Audiense の支援を受け、Red Cloud は鉱業関連の企業ニュースや株式情報の収集に向けて X 上でのリサーチに注力しました。Audiense のプラットフォームにより、キャンペーンと、動画再生数やウェブサイトトラフィックの増加に表れる関心の高まりとの間の潜在的な相関を容易に見いだせました。

3 段階のアプローチ:

1. まったく新しいオーディエンスを構築する。

    クライアントの方針に基づき、Red Cloud Financial はキャンペーンの重点対象となる 2 つの主要オーディエンスを特定しました。第 1 のセグメントは鉱業と投資、第 2 はクリーンテック、電気自動車、ESG、責任投資です。X のデータは、これらのオーディエンスを将来のキャンペーンと整合させるうえでも Red Cloud を後押ししました。

1. 新しいオーディエンスをセグメント化する。 

    新しいオーディエンスは幅広い業界をまたいでいましたが、多くの共通項がありました。Audiense は、Red Cloud Financial がこれらの新セグメントを深掘りし、インフルエンサーやコンテンツ消費の傾向などを特定するのに役立ちました。これらのインサイトは、相互フォローを見込んでフォローすべきアカウントの選定にも寄与し、Red Cloud とクライアントのオーディエンス拡大につながりました。

1. どれが最も価値が高いかを見極める。 
    
    どれが最も価値が高いかを見極める。Audiense の[Targeting Pack 機能](https://help.audiense.com/knowledge/what-is-a-targeting-pack)を使用し、Red Cloud は高精度にターゲティングした有料の X 広告キャンペーンを実施する一方、蓄積データを活用して X セグメント向けのオーガニックキャンペーンも展開しました。Targeting Pack から得られたインサイトを基に、Audiense と Red Cloud は 2 つのオーディエンスそれぞれに向けた 2 本の独自動画を作成。どの動画が一貫して最高の成果を上げているかを特定し、狙いをぶらさずにターゲティングを維持できました。

<div id="impact">
  ## 影響
</div>

Audiense のプラットフォームと X 上でのリサーチにより、Red Cloud は適切なオーディエンスを特定・セグメント化し、活用してフォロワーを拡大するとともに、オーガニックおよび有料コンテンツ双方の全体的なソーシャルメディア戦略の策定に役立てることができました。

Red Cloud Financial はクライアントの期待を上回りました。実際、キャンペーンの成果が非常に好調だったため、クライアントは次のキャンペーンに進み、規模を拡大する決定を下しました。 

- 複数のキャンペーン合計で 350 万超のインプレッションと 280 万超の総ビューを獲得
- クライアントの特許出願中の新技術に関する会話量が増加
- 新技術に関するプロモーション記事およびプレスリリース経由での 7,000 人の新規訪問者を含め、ウェブサイトへのリファラルトラフィックが増加

これは、X におけるオーディエンスアナリティクスとターゲティングが、新たなオーディエンスの開拓、マーケティングキャンペーンの精緻化、そして実際の成果の創出にどのように寄与するかを示す好例です。クライアントのブランド構築と事業成長を後押しする絶好の機会となり得ます。
