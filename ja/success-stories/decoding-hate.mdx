<div id="situation">
  ## 状況
</div>

COVID-19 の流行後、アジア系アメリカ人および太平洋諸島系（AAPI）に対するヘイト事案が増加しました。いくつかはニュースで報じられましたが、全国的な報告窓口である [Stop AAPI Hate](https://stopaapihate.org/) には、リサーチ、アドボカシー、基盤強化を通じてこうした事案に対応・対処している同団体に、何千もの個人から報告が寄せられていました。この時期に [The Asian American Foundation (TAAF)](https://www.taaf.org/) が設立されました。同財団は、これらの事案が標的となった人々にどのような個人的影響を及ぼしているかに新たな光を当てるべく、X を活用することを決めたチームの牽引役となりました。

<div id="solution">
  ## ソリューション
</div>

2名のエンジニアからなる小規模チームは、インシデントデータの取得元としてXを選定しました。理由は2点あります。第一に、ニュース、人々の考えや意見、日常の出来事、コミュニティ内の出来事など、豊富で多様な情報が含まれているためです。これにより、Xは多様な人々からなるユニークで親密性の高いサンプルを提供します。加えて、TAAFの開発者がデータにアクセスでき、かつその収集・分析に高度なツールを備えた真にオープンなAPIを持つ唯一のプラットフォームでもあります。

X API v2 の search Tweets と Tweets lookup エンドポイントを用いて、チームは投稿のステータスを検索し、人々がヘイトインシデントを報告したタイミング（さらに使用されるハッシュタグ）を特定しました。これにより、1023文字の検索クエリを設計し、数十億件の投稿をチームが扱える規模まで絞り込むことに成功しました。

次に、投稿が実際にヘイトインシデントに関するものかを検証する作業に取り組みました。チームは自然言語処理を用いたツールを開発し、数千件の投稿からプロジェクトに関連するものをふるい分けました。学習済み機械学習モデルを適用した後、精度をさらに高めるために人手で検証を行いました。

2021年、TAAFはStop AAPI Hate連合と共同で[Documenting Anti-AAPI Hate Codebook](https://stopaapihate.org/wp-content/uploads/2021/12/SAH-x-TAAF-Codebook-FINAL-2021-08-19.pdf)を執筆しました。このリソースには、コミュニティ主導のデータ収集に関する標準案と実践が含まれており、X上で共有されたインシデントの分類に役立ちました。

次の課題は、このデータを人々が容易に理解できる形で提示することでした。TAAFはデータ可視化ツール[Decoding Hate](https://decodinghate.taaf.org/)を作成し、膨大なデータをインタラクティブに表示しました。データビジュアライゼーションスタジオの協力を得て、注釈付きのXデータを活用し、数千件の投稿を、AAPIコミュニティ内で実際に起きていた事象の実態を明らかにする洞察に富んだストーリーへと変換しました。

<div id="impact">
  ## 影響
</div>

Decoding Hate プロジェクトは、これまで報告されていなかった数千件のヘイト関連事案の検出に貢献し、現在も継続的に成果を上げています。本プロジェクトは、AAPI コミュニティ内で起きている個々の体験への認知を広げるための継続的な手段となりました。

次のフェーズでは、TAAF は Stop AAPI Hate と連携し、X 上でヘイト事案を報告した人に対して人による個別対応を試験導入します。この期間中、Stop AAPI Hate は、自身が経験したヘイトについて投稿している多くの人々に積極的に働きかけ、[Stop AAPI Hate Reporting Center](https://stopaapihate.org/) を通じた報告を促します。より多くの人が Stop AAPI Hate に報告するほど、同団体とそのパートナーは発生しているヘイト事案の性質をより深く把握し、最も効果的な政策的解決策を提言できるようになります。 

AAPI コミュニティで起きているヘイト関連事案を可視化するだけでなく、本プロジェクトは X の API の価値も浮き彫りにしています。本ケースは、適切なデータと適切なツールを活用することで、開発者が公益のために X を用いたイノベーションを実現できることを示しています。
