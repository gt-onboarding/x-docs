<Note>
**要点（tl;dr）**

AI駆動のインサイトで組織の意思決定を支援する企業である PublicSonar との協業により、デン・ハーグ市は道路のモビリティと安全性を向上させました。ソーシャルインサイトを活用することで、PublicSonar は追加情報を提供し、デン・ハーグの交通管制センターが変化する道路状況に迅速かつ効果的に対応できるよう支援しています。
</Note>

<div id="challenge">
  ## 課題
</div>

大陸や時差を越えて、交通は日常の中で予測しづらく混沌とした存在です。世界中の人々が直面する課題であり、多くの生活に影響を及ぼしています。道路の主導権を取り戻す取り組みは各地で進められていますが、その中でもハーグ市ほど真剣にこの問題に取り組んでいる自治体は多くありません。

ハーグの交通管制センターは、CCTVや地下センサーといったテクノロジーやインテリジェンスツールに投資し、大都市で不可避的に発生する交通関連インシデントに対して、迅速かつ安全な対応と解決を提供することを目指しました。とはいえ、課題が一つ残っていました。最適な意思決定のために、必要な情報をリアルタイムでどう入手するか、という点です。

先進的な技術ツールキットを備えていたにもかかわらず、ハーグの交通管制センターには依然として重要な情報がいくつか欠けていました。そこで彼らは、そのギャップを埋めるべくPublicSonarに目を向けました。PublicSonarは、AI駆動のインサイトを用いて、組織の情報に基づく意思決定を支援する企業です。

<div id="solution">
  ## 解決策
</div>

移動の円滑化と安全の確保における最大の課題は、現場から情報を収集できるかどうかです。幸いにも、PublicSonar と提携することで、ハーグ市は交通管制に欠けていた最後のピース、すなわちソーシャルメディアを見いだしました。

そこで X の出番です。X 上のリアルタイムな会話は、PublicSonar が情報の空白を埋めるための洞察を得るうえで極めて重要な役割を果たしました。

PublicSonar にとって、X データの提供形態と X の API がもたらす安定性・サポートは業務の要です。キーワードを用いて交通関連の情報を絞り込み、ハーグ市が活用できる見やすいオンラインダッシュボードを構築しました。彼らの強みは、交通管制センターを過負荷にする過剰な偽陽性を避けつつ、重要な情報を取りこぼさない最適なバランスを取る点にあります。

<div id="results">
  ## 結果
</div>

ソーシャルメディアのモニタリングを通じて、PublicSonar はハーグ市に対し、計画イベント（例：コンサート）、日常的な状況（例：海岸の混雑日）、および事案（例：交通事故）で適切な判断を下せるインサイトを提供しています。

最終的に、テクノロジーとソーシャルからのインサイトが相互に補完し合うことで全体像がより明確になり、ソーシャルメディアは道路上の「多くの目」として機能します。これらの情報の流れにより、交通管制センターは常に状況を把握し続け、ハーグ市により安全で円滑な走行環境を提供できます。
