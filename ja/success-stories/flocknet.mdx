<Note>
**要点（tl;dr）**

開発者兼クリエイターの Aaron Ng は、X の API を活用して、X 上の自分のネットワークを横断検索できる検索エンジン「Flocknet」を構築しました。Flocknet はネットワーク拡大、人材採用、投資リードの獲得に活用されています。個人用途で始まったプロジェクトでしたが、現在では X 上の数百万件に及ぶ連絡先の管理を支援しています。
</Note>

<div id="challenge">
  ## 課題
</div>

Aaron Ng はニューヨークで開かれたデザインカンファレンスに参加した際、同業界の人々とつながり、実際に会いたいと考えていました。ところが、X 上で親しいデザイナー仲間とでさえつながることが、想像以上に難しいと気づいたのです。コミュニティの多く、特に業界の友人たちは X を使っているのに、町に誰が来ているのかを知る術はないのでしょうか？誰が近くにいるのか分からないまま、どうやって業界のつながりと再会できるのでしょうか？

Aaron はこの課題をすぐにチャンスだと捉えました。こうして、彼の起業家精神から Flocknet が生まれました。 

<div id="solution">
  ## 解決策
</div>

わずか週末のうちに、Aaron は、相手の投稿を見なくても、自身の X ネットワーク内の人々、友人、他のプロフェッショナルを検索・並べ替え・キャッチアップできるプロダクトを構築しました。彼はこのプロジェクトに X の API を選び、「質問できる相手がいること、そして［X の］チームが気にかけてくれているとわかることが、体験をずっと良いものにしてくれる」と語っています。さらに「私が作り続ける動機となったコミュニティの多くは X にいた——それが X が私にとって非常に重要な場所になった理由であり、私の使い方を高めるためのツールを作りたかった理由です」と述べました。  

自身の創造性、推進力、経験を組み合わせて、Flocknet が誕生しました——X ネットワーク内の人々を見つけるための検索エンジン兼コンタクト管理ツールです。X のフォロー系エンドポイントと Python を活用し、Flocknet はユーザーがフォロワーを職種、勤務先、所在地、その他多数の属性で検索できるようにします。

当初は個人的な利用を想定していましたが、Flocknet は大きな関心を集め、共有すべきだと確信するに至りました。そこで彼はすぐに取り組みを拡大し、Flocknet を現在の姿へとスケールさせました。

<div id="impact">
  ## 影響
</div>

わずか数か月で、Flocknet はユーザーに代わって数百万件規模のコンタクトを管理するようになりました。テクノロジー業界のプロフェッショナルに特に重宝されており、採用から投資まで、幅広い用途で活用されています。実際、Aaron 自身も Flocknet を通じて人材を採用しています。

ただし、Aaron の挑戦はまだ続きます。寄せられる新機能リクエストに耳を傾けながら、製品を継続的に拡張する方法を探り続けています。

最終的に、Aaron は X をメタプラットフォームとして活用し、コミュニティの拠点が主に X 上にある人の体験を高める方法を見出しました。彼はこう語ります。「とにかくたくさん作る。自分にも他の人にも役立つものを作るのが大好きなんだ」。そして、まさにそれを実現しました。
