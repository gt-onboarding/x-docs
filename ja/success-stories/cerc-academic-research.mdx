**要点（tl;dr）**

Ryerson University の Canada Excellence Research Chair（CERC）「移民と統合」プログラムのリサーチフェローである Stein Monteiro 氏が、農民抗議運動の最中におけるインド系ディアスポラとインドの関係に関する研究をどのように Twitter データで前進させたかをご紹介します。

Monteiro 氏にとって、インドの農民抗議運動は興味深い問いを投げかけました。すなわち、インドで大きな出来事が起きたとき、インド系ディアスポラはどのように行動するのか。Twitter の API の助けを借り、彼はその答えを明らかにしました。詳しくはこちらをご覧ください。
</Note>

<div id="challenge">
  ## 課題
</div>

ライヤソン大学の Canada Excellence Research Chair（CERC）「移民と統合」研究フェローである Stein Monteiro にとって、仕事は統計、文化、そして時事の交差点にあります。学術界で成功するための土台となったのは彼の経済学の専門知識でしたが、移民研究への情熱をかき立てたのは彼自身の経験でした。

インドで生まれ育った Stein は、高校時代をドバイで過ごし、その後カナダで学部課程を履修しました。彼にとって、異国で「新参者」である感覚はごく馴染み深いものでした。新しい社会制度、気候、生活様式に向き合う経験は初めてではありません。新しい文化や環境に適応するプロセスは、移民の統合の歩みに対する彼の関心に火をつけました。経済学の研究を進めても、こうした個人的な体験の魅力は色あせず、彼は両者を結び付けたいと考えるようになりました。経済学の博士号を取得するにあたり、彼は学位論文のテーマを移民研究の文献に絞りました。

この学位論文に取り組む中で、Stein は独学でコーディングを身につけました。のちに Humber College で統計担当として働いた際には、その学びを実務に活かすことができました。コーディングは自動化を可能にし、データの整理と集計によって業務を効率化しました。

作業プロセスをコード化することで、Stein は問題をより容易に特定できるようになり、自身の作業／思考プロセスを他者に段階的に説明できるようになりました。これは、問題の捉え方や解決策の構築にまったく新しい道を開くものでした。彼はこのスキルセットの無限の可能性に次第に気づき、この知見を Canada Excellence Research Chair（CERC）に持ち込みました。 

CERC 在籍中、Stein の関心を引いたのは、ソーシャルメディア上で目の当たりにしたオンライン・アクティビズム、とりわけ（2020年8月に導入された3つの物議を醸す法律をきっかけとする）インドの農家による抗議運動でした。彼は、これらの抗議と自身の研究との関連性を見出しました。これは重要な瞬間でした。リアルタイムで起きていることが、移民およびディアスポラ全体の経験についての情報を明らかにし得るからです。

議論が沸騰する中で、彼はオンライン・アクティビズムが、カナダ、英国、オーストラリア、そして湾岸諸国に住むインド系ディアスポラの人々の活動にどのような影響を与えているのかを考えました。ソーシャルネットワークは、移民をどのようにインドでの議論につないでいるのか。 

いよいよ本格的に取り組む時が来ました。 

<div id="solution">
  ## 解決策
</div>

インド系ディアスポラと農民抗議運動との関係をより深く理解するため、Stein はオンライン上の議論に耳を傾ける必要があると考え、Twitter に照準を定めました。

Stein によれば、「これらのローカルな出来事が国際的なオーディエンスに届いたのは、Twitter が国際的な現象であるからというだけでなく、普通の人々がオンラインで互いに語り合い、ハッシュタグだけでつながることができたからです」。彼にとって Twitter は、「インドで起きている農民抗議に、インド人でなくても、あるいはインド人でも世界の別の場所にいても参加できる」場でした。

Stein は、抗議者、反抗議のレトリック、政府寄りの主体、そして現場で実際に起きていることの生の体験が交錯する場として Twitter を捉えました。そして、そのダイナミクスの中に身を投じたいと考えました。 

やり取りを精読しながら、Stein はいくつかの点に焦点を絞る必要がありました。インドで誰かがオンラインに何かを投稿したとき、そのツイートはどのように世界へ波及するのか。情報はネットワークやハッシュタグなどを通じてどのように流れるのか。そして、ディアスポラはインドから流れてくるツイートにどう反応しているのか。 

Stein は Twitter の API を活用して洞察を深めました。同 API は Academic Research のプロダクトトラックを導入し、さらなる分析のために、大量のツイートや投稿者情報に加えて、過去データへのアクセスを可能にしました。 

Stein にとって、API の強みは長期間にわたるデータへのアクセス性であり、会話の時間軸—誰が何をリツイートし、誰が誰とやり取りしているのか—を構築できる点にありました。 

こうした情報を手にして、彼は目の当たりにした事実に驚かされました。インド系ディアスポラがインドの状況に反応する活動は非常に活発だったのです。Stein は「遅れての反応というより、ほとんど即時でした。つまり、1 日のうちに、インドで投稿されたもののリツイートが見られ、続いて世界各地でもリツイートが起きていました」と語ります。

データは明白でした。ソーシャルメディアは、移民やディアスポラが母国とつながり続け、重要な議論に積極的な役割を果たすことを可能にしています。

<div id="impact">
  ## 影響
</div>

Twitter のオープン API を活用して、Stein は移民というテーマをめぐる幅広い議論への洞察につながる、時宜を得た出来事を調査することができました。データへのアクセスと、そのデータを絞り込み・並べ替えるための機能は、このプロジェクトの成否を左右する要素でした。 

さらに重要なのは、Stein が学術研究全体におけるツールとしての Twitter の意義を見いだしたことです。彼にとって研究の大きなハードルの一つは、データを世界中から収集する必要がある点です。幸い、Twitter の国際的なプレゼンスにより、世界各地で多額の費用をかけて調査やフォーカスグループを実施しなくても、グローバルな情報へアクセスできます。 

Stein は次のように述べています。「Twitter は、研究者や学術コミュニティが自らの研究目的でデータにアクセスし、新たな方法論を開発できるだけでなく、それを費用対効果の高い形で実現できるようにしてくれます。」彼は、オンライン上の会話が有用だったのは「非常に生のままで、研究者が介入しない、人々の間で行われる偏りのないやり取りだったからだ」と考えています。 

Stein の取り組みは、Twitter が今なお世界中の人々をつなぐためのツールであり続けていること、そして Twitter の API がそのつながりから生まれる大規模なインサイトの収集に寄与しうることを示しました。
