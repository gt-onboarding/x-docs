<div id="connect-to-the-x-ads-platform">
  ## X 広告プラットフォームに接続する
</div>

X 広告 API は、世界中の広告主のニーズに応えるソリューションを構築できるよう、開発者を X の広告プラットフォームに接続します。

  <Card title="Campaign management" icon="list-check" href="/ja/x-ads-api/campaign-management">X 上のユーザーと効果的にエンゲージするために、広告キャンペーンをプログラムで作成・スケジュール・管理できます。</Card>
