---
title: 通貨
---

通貨の種類は [ISO-4217](http://en.wikipedia.org/wiki/ISO_4217) によって識別されます。`USD` や `EUR` のような3文字の文字列です。通貨の値はマイクロ単位で表します。USD の場合、$5.50 は 5.50\*1e6、すなわち 5,500,000 としてエンコードされます。「整数値」を表すには、すべての通貨でローカルのマイクロ値に 1e6（1\_000\_000）を掛けてください。

X 広告 API は、X の SMB Ads 製品が提供されているすべての国の公式通貨を認識します。該当国の一覧は[こちら](https://business.x.com/en/help/overview/about-eligibility-for-twitter-ads.html)をご覧ください。

ほとんどの通貨の「精度」は 2 です（例: USD の $1.23）。

- `AED`, `VEF`, `PKR`, `IDR`, `HKD`, `QAR`, `PHP`, `NZD`, `INR`, `EUR`, `ARS`, `THB`, `MXN`, `EGP`, `DZD`, `SAR`, `LBP`, `COP`, `TRY`, `SGD`, `MAD`, `AUD`, `USD`, `MYR`, `CAD`, `BRL`, `GBP`.

次の通貨は小数を使用しません:

- `VND`, `KRW`, `CLP`, `JPY`

次の通貨は小数点第3位まで使用します:

- `KWD`, `IQD`, `BHD`, `TND`

通貨は[資金手段](/ja/x-ads-api/introduction)を通じて使用します。
