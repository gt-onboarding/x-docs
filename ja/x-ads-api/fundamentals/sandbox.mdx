---
title: サンドボックス
---

サンドボックスは、X 広告 API 実装のテストと構築を行うための環境です。

サンドボックスで作成したキャンペーンはユーザーに配信されず、Funding Instruments などのオブジェクトに対するテストレコードを動的に作成できます。

サンドボックスの利用は、適切なホスト名 ads-api-sandbox.x.com にリクエストするだけと簡単ですが、サンドボックスにのみ存在する固有の機能も備えています。

新しい広告主アカウントを作成するには、まず [POST accounts](/ja/x-ads-api/campaign-management#post-accounts) にリクエストを送信します。このレスポンスには、以降のリクエストで使用できる account\_id が含まれます。これらのエンドポイントにより、利用者は複数の広告主アカウントを作成・テストできます。

<div id="sandbox-only-endpoint-list">
  ### サンドボックス専用エンドポイント一覧:
</div>

- [POST accounts](/ja/x-ads-api/campaign-management#post-accounts) サンドボックスでアカウントを作成
- [POST accounts/:account\_id/features](/ja/x-ads-api/campaign-management#post-accounts-account-id-features) サンドボックスのアカウントにアカウント機能を追加
- [POST accounts/:account\_id/funding\_instruments](/ja/x-ads-api/campaign-management#post-accounts-account-id-funding-instruments) サンドボックスのアカウントに資金手段を追加
- [DELETE accounts/:account\_id](/ja/x-ads-api/campaign-management#delete-accounts-account-id) サンドボックスのアカウントを削除
- [DELETE accounts/:account\_id/features](/ja/x-ads-api/campaign-management#delete-accounts-account-id-features) サンドボックスのアカウントからアカウント機能を削除
- [DELETE accounts/:account\_id/funding\_instruments/:funding\_instrument\_id](/ja/x-ads-api/campaign-management#delete-accounts-account-id-funding-instruments-funding-instrument-id) サンドボックスのアカウントから資金手段を削除
