---
title: レート制限
icon: gauge-simple-high
---

毎日、何千人もの開発者が X 開発者プラットフォームにリクエストを送信しています。こうした大量のリクエストを適切に管理するため、送信できるリクエスト数には制限を設けています。これらの制限により、開発者コミュニティが依拠する信頼性とスケーラビリティの高い API を提供できます。

各 API ではレート制限の適用方法が異なります。プラットフォーム間の違いの詳細は、各 API セクション内の該当するレート制限ページをご確認ください。

- [X API v2](/ja/x-api/fundamentals/rate-limits)
- [X API: Enterprise](/ja/x-api/enterprise-gnip-2.0/fundamentals/rate-limits)
- [X 広告 API](/ja/x-ads-api/fundamentals/rate-limiting)
