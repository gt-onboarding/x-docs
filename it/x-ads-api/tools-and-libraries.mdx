<div id="community-tools-and-libraries">
  ## Strumenti e librerie della community
</div>

Di seguito alcune delle numerose librerie supportate dalla community che coprono l’API X Ads, in diversi linguaggi di programmazione e piattaforme. Le librerie elencate qui dovrebbero supportare la maggior parte delle funzionalità dell’API X Ads, salvo diversa indicazione: verifica i dettagli con gli autori.

Se hai creato una libreria che supporta le API X, faccelo sapere tramite i nostri [forum della Developer Community](https://devcommunity.x.com/c/libraries-and-sdks/63).
