Noterai che ci sono ancora diversi elementi contrassegnati come **\[In arrivo]**. Se trovi in questa tabella elementi contrassegnati come **\[Sostituzione in valutazione]** o **\[Nessuna sostituzione prevista]** e desideri che pubblichiamo una versione v2 di quell’endpoint, faccelo sapere tramite il nostro [forum della community](https://devcommunity.x.com/) o tramite il tuo account manager enterprise.
