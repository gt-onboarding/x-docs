L'endpoint Personalized Trends consente di ottenere trend personalizzati per un account utente autenticato.

<Note>
  **Nota:** I trend personalizzati vengono restituiti dall'endpoint solo per gli account utente abbonati a X Premium.
</Note>

<Note>
  **Configurazione dell'account**

  Per accedere a questo endpoint, ti serviranno:

  - Un [account sviluppatore](https://developer.x.com/en/portal/petition/essential/basic-info) approvato.
  - L'autenticazione utilizzando le chiavi e i token di una [App sviluppatore](/it/resources/fundamentals/developer-apps) che si trova all'interno di un [Progetto](/it/resources/fundamentals/projects).

  Scopri di più su come ottenere l'accesso agli endpoint dell'API X v2 nella nostra [guida introduttiva](/it/x-api/getting-started/getting-access).
</Note>

<div id="getting-started">
  ### Guida introduttiva
</div>

<div id="authentication">
  #### Autenticazione
</div>

Puoi autenticare questo endpoint utilizzando [OAuth 1.0a in contesto utente](/it/resources/fundamentals/authentication) oppure [OAuth 2.0 Authorization Code con PKCE](it/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2).
 

<div id="making-a-request">
  #### Effettuare una richiesta
</div>

Puoi chiamare l’endpoint Personalized Trends come mostrato di seguito:

`curl 'https://api.x.com/2/users/personalized_trends' --header 'Authorization: ••••••'`

Se la richiesta va a buon fine, vedrai la risposta JSON come riportato di seguito:

```
{
    "data": [
        {
            "category": "Bitcoin",
            "post_count": "42K posts",
            "trend_name": "MicroStrategy's $42 Billion Bitcoin Investment",
            "trending_since": "5 hours ago"
        },
        {
            "category": "Music",
            "post_count": "2K posts",
            "trend_name": "Drake's 'No Face' Music Video Release",
            "trending_since": "Trending now"
        },
        {
            "category": "Weather",
            "post_count": "1.2K posts",
            "trend_name": "Super Typhoon Leon Impacts Taiwan and Batanes",
            "trending_since": "Trending now"
        }
    ]
}
```

**Nota:** Se l’utente autenticato della richiesta non è abbonato a X Premium, riceverà invece la seguente risposta:

```
{
    "data": [
        {
            "category": "Unknown",
            "post_count": "Unknown",
            "trending_since": ""
        }
    ]
}
```
