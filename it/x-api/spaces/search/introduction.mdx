Questo endpoint consente di cercare gli Spaces per titolo e di filtrare i risultati per stato. È utile per scoprire Spaces in diretta o imminenti in base a parole chiave, utenti menzionati o hashtag presenti nel titolo.

L’endpoint accetta una o più parole chiave come query. I risultati possono essere filtrati in base allo stato di uno Space (in diretta o pianificato). Per impostazione predefinita, una richiesta restituisce sia gli Spaces in diretta sia quelli pianificati che corrispondono alla query specificata.

<Note>
  **Configurazione dell’account**

  Per accedere a questi endpoint, ti serviranno:

  - Un [account sviluppatore](https://developer.x.com/en/portal/petition/essential/basic-info) approvato.
  - L’autenticazione utilizzando le chiavi e i token di una [App per sviluppatori](/it/resources/fundamentals/developer-apps) all’interno di un [Progetto](/it/resources/fundamentals/projects). 

  Scopri di più su come ottenere l’accesso agli endpoint dell’API X v2 nella nostra [guida introduttiva](/it/x-api/getting-started/getting-access).
</Note>
