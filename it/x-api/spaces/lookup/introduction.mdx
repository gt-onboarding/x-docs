Gli endpoint di ricerca per ID restituiscono dati che puoi utilizzare per creare esperienze che mostrino informazioni sugli Spaces ai tuoi utenti o per ottenere dettagli su uno o più Spaces. È possibile cercare uno o più Spaces all’interno della stessa richiesta.

Puoi anche cercare gli Spaces in base all’ID del creatore. Questo endpoint è utile per cercare gli Spaces degli utenti presenti in un elenco o per determinare se più host ricorrenti sono attualmente in diretta o hanno uno Space programmato a breve. È possibile cercare fino a 100 utenti in una singola richiesta.

**Configurazione dell’account**

Per accedere a questi endpoint, ti serviranno:

- Un [account sviluppatore](https://developer.x.com/en/portal/petition/essential/basic-info) approvato.
- L’autenticazione tramite le chiavi e i token di un’[App sviluppatore](/it/resources/fundamentals/developer-apps) che si trova all’interno di un [Progetto](/it/resources/fundamentals/projects).

Scopri di più su come ottenere l’accesso agli endpoint dell’API X v2 nella nostra [guida introduttiva](/it/x-api/getting-started/getting-access).
