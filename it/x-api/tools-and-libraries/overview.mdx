<div id="community-tools-and-libraries-for-v2">
  ## Strumenti e librerie della community per v2
</div>

Le librerie elencate qui sono state create da membri della community di sviluppatori. Nota che potrebbero trovarsi a diversi livelli di copertura dell’API.

Se hai creato una libreria per l’API X o uno strumento utile, [faccelo sapere](https://devcommunity.x.com/c/libraries-and-sdks/63) e lo aggiungeremo a questo elenco per aiutare altri a trovarlo. Abbiamo anche alcuni [badge di versione](https://twbadges.glitch.me/) che puoi utilizzare nei tuoi file README.

Cerchi ispirazione? Puoi sfogliare e cercare tra gli argomenti [X](https://github.com/topics/twitter?o=desc&s=updated) e [X-api-v2](https://github.com/topics/twitter-api-v2?o=desc&s=updated) su GitHub per trovare esempi di codice utili da altri sviluppatori.

**Vai a:** [C# / .NET](#csharp), [Dart / Flutter](#dart), [Go](#go), [Java](#java), [JavaScript (Node.JS) / TypeScript](#nodejs), [Kotlin](#kotlin), [PHP](#php), [PowerShell](#powershell), [Python](#python), [R](#r), [Ruby](#ruby), [Rust](#rust), [Swift](#swift)

<div id="community-tools-and-libraries">
  ## Strumenti e librerie della community
</div>

Queste sono alcune delle numerose librerie supportate dalla community che coprono l’API X in diversi linguaggi di programmazione e piattaforme. Tieni presente che queste risorse potrebbero non essere state tutte testate dal team di Twitter.

Le librerie elencate qui dovrebbero implementare la maggior parte delle funzionalità della Standard API **v1.1**, salvo diversa indicazione—verifica con gli autori per dettagli e supporto aggiuntivo.\
Se hai sviluppato una libreria che supporta l’API X **v2**, faccelo sapere tramite i nostri [forum della community](https://devcommunity.x.com/c/libraries-and-sdks/63) per una possibile aggiunta a questa pagina. Puoi anche usare i forum per segnalare eventuali aggiornamenti a questi elenchi.

Se ti manca una libreria o uno strumento per il tuo linguaggio di programmazione preferito, faccelo sapere tramite la [piattaforma di feedback](https://twitterdevfeedback.uservoice.com/), dove puoi anche votare le idee o trovare ispirazione per creare e inviare qualcosa di nuovo.
