<div id="manage-list-members-quick-start-guide">
  ### Guida rapida alla gestione dei membri di una Lista
</div>

<Note>
  **Nota bene:** Questa guida presuppone che tu abbia completato i prerequisiti nella [panoramica di avvio rapido](/it/x-api/lists/list-members#getting-started-with-the-list-members-endpoint-group).
</Note>

<div id="steps-to-build-a-manage-list-member-request">
  ### Passaggi per creare una richiesta per gestire i membri di una Lista
</div>

<div id="step-one-choose-the-list-endpoint-collection-from-postman">
  #### Passaggio uno: scegli la raccolta di endpoint List in Postman
</div>

Una volta caricata la raccolta API X v2 in Postman, vai alla cartella "List", seleziona la sottocartella "List members" e poi scegli "Add a member".
 

<div id="step-two-specify-the-user-to-add">
  #### Secondo passaggio: specificare l’utente da aggiungere
</div>

Gli endpoint per la gestione dei membri delle Liste richiedono due ID: l’ID della Lista e l’ID dell’utente da aggiungere. Puoi trovare l’ID dell’utente usando il [users lookup](/it/x-api/users/lookup/introduction) e passando un username per ottenere il campo id.

L’ID di destinazione può essere qualsiasi ID utente valido. In Postman, vai alla scheda "Params" e inserisci il tuo ID nella colonna "Value" della variabile di percorso id. Vai alla scheda "Body" e inserisci l’ID dell’utente che desideri aggiungere come valore del parametro user\_id. Assicurati di non includere spazi prima o dopo alcun ID.

|     |     |     |
| :--- | :--- | :--- |
| **Key** | **Value** | **Parameter type** |
| `id` | L’ID della Lista | path |
| user\_id | L’ID dell’utente di destinazione che desideri aggiungere come membro | body |

Ora dovresti vedere un URL simile accanto al pulsante "Send":

`https://api.x.com/2/lists/1441162269824405510/members`

Terzo passaggio: invia la richiesta e verifica la risposta

Una volta configurato tutto, fai clic sul pulsante "Send" e riceverai una risposta simile al seguente esempio:

```
{
  "data": {
    "is_member": true
  }
}
```

Se l’oggetto di risposta restituito contiene il valore true per la chiave is\_member, hai aggiunto correttamente l’utente come membro della Lista.

Per rimuovere un membro da una Lista, seleziona la richiesta "Remove a member" presente anche nella cartella "Lists" della raccolta API X v2 caricata in Postman. Questo endpoint richiede l’ID della Lista e l’ID utente del membro che desideri rimuovere. Nella scheda "Params", inserisci l’ID della Lista come valore per la colonna id e l’ID dell’utente da rimuovere come valore per la colonna user\_id.

Se la richiesta di eliminazione va a buon fine, riceverai una risposta simile al seguente esempio:

```
{
  "data": {
    "is_member": false
  }
}
```
