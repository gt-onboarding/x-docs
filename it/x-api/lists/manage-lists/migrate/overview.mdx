<div id="comparing-x-apis-lists-endpoints">
  ## Confronto degli endpoint Liste dell’API X
</div>

Gli endpoint v2 per la gestione delle Liste sostituiranno a breve [POST lists/create](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-create), [POST lists/destroy](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-destroy) e [POST lists/update](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-update). Se utilizzi codice, app o strumenti che si basano su una versione precedente di questi endpoint e stai valutando la migrazione alla nuova API X v2, questa guida fa per te.

Le tabelle seguenti confrontano gli endpoint Liste della v1.1 standard e dell’API X v2:

**Creare una Lista**

| Descrizione | v1.1 standard | API X v2 |
| :--- | :--- | :--- |
| Metodi HTTP supportati | POST | POST |
| Dominio host | https://api.x.com | https://api.x.com |
| Percorso endpoint | /1.1/lists/create.json | /2/lists |
| [Autenticazione](/it/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context |
| [Limiti di velocità](/it/resources/fundamentals/rate-limits) predefiniti per le richieste | Nessuno | 300 richieste ogni 15 min (per utente) |

**Eliminare una Lista**

| Descrizione | v1.1 standard | API X v2 |
| :--- | :--- | :--- |
| Metodi HTTP supportati | POST | DELETE |
| Dominio host | https://api.x.com | https://api.x.com |
| Percorso endpoint | /1.1/lists/destroy.json | /2/lists/:id |
| [Autenticazione](/it/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context |
| [Limiti di velocità](/it/resources/fundamentals/rate-limits) predefiniti per le richieste | Nessuno | 300 richieste ogni 15 min (per utente) |

**Aggiornare una Lista**

| Descrizione | v1.1 standard | API X v2 |
| :--- | :--- | :--- |
| Metodi HTTP supportati | POST | PUT |
| Dominio host | https://api.x.com | https://api.x.com |
| Percorso endpoint | /1.1/lists/update.json | /2/lists/:id |
| [Autenticazione](/it/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context |
| [Limiti di velocità](/it/resources/fundamentals/rate-limits) predefiniti per le richieste | Nessuno | 300 richieste ogni 15 min (per utente) |

Per accedere agli endpoint dell’API X v2, devi [registrarti per un account sviluppatore](https://developer.x.com/en/portal/petition/essential/basic-info). In fase di autenticazione, devi utilizzare le chiavi e i token di una [App sviluppatore](/it/resources/fundamentals/developer-apps) che si trova all’interno di un [Progetto](/it/resources/fundamentals/projects).

Per saperne di più su come ottenere l’accesso agli endpoint dell’API X v2, visita la nostra pagina [guida introduttiva](/it/x-api/getting-started/getting-access).
