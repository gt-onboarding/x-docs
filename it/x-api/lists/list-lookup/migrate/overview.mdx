<div id="comparing-x-apis-list-lookup-endpoints">
  ## Confronto degli endpoint di ricerca delle Liste dell’API X
</div>

Il gruppo di endpoint di ricerca delle Liste della v2 sostituirà gli endpoint standard v1.1 [GET lists/show](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-show) e [GET lists/ownership](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-ownerships). Se hai codice, app o strumenti che usano una di queste versioni degli endpoint di ricerca delle Liste e stai valutando la migrazione al più recente endpoint dell’API X v2, questa serie di guide fa per te.

Le tabelle seguenti confrontano gli endpoint Liste della v1.1 standard e dell’API X v2:

**Ricerca Lista per ID**

|     |     |     |
| :--- | :--- | :--- |
| Descrizione | Standard v1.1 | API X v2 |
| Metodi HTTP supportati | `GET` | `GET` |
| Dominio host | `https://api.x.com` | `https://api.x.com` |
| Percorso endpoint | `/1.1/lists/show.json` | `/2/lists/:id` |
| [Autenticazione](/it/resources/fundamentals/authentication) | OAuth 1.0a User Context<br /><br />Solo app | OAuth 1.0a User Context<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />Solo app |
| [Limiti di frequenza](/it/resources/fundamentals/rate-limits) predefiniti per le richieste | 75 richieste ogni 15 min con OAuth 1.0a<br /><br />75 richieste ogni 15 min con OAuth 2.0<br /><br />75 richieste ogni 15 min con solo app | 75 richieste ogni 15 min con OAuth 1.0a<br /><br />75 richieste ogni 15 min con OAuth 2.0<br /><br />75 richieste ogni 15 min con solo app |

**Ricerca delle Liste di proprietà dell’utente**

|     |     |     |
| :--- | :--- | :--- |
| Descrizione | Standard v1.1 | API X v2 |
| Metodi HTTP supportati | `GET` | `GET` |
| Dominio host | `https://api.x.com` | `https://api.x.com` |
| Percorso endpoint | `/1.1/lists/ownerships.json` | `/2/users/:id/owned_lists` |
| [Autenticazione](/it/resources/fundamentals/authentication) | OAuth 1.0a User Context<br /><br />Solo app | OAuth 1.0a User Context<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />Solo app |
| [Limiti di frequenza](/it/resources/fundamentals/rate-limits) predefiniti per le richieste | 15 richieste ogni 15 min con OAuth 1.0a<br /><br />15 richieste ogni 15 min con solo app | 15 richieste ogni 15 min con OAuth 1.0a<br /><br />15 richieste ogni 15 min con OAuth 2.0<br /><br />15 richieste ogni 15 min con solo app |

Per accedere agli endpoint dell’API X v2, devi [registrarti per un account sviluppatore](https://developer.x.com/en/portal/petition/essential/basic-info). In fase di autenticazione, devi usare chiavi e token di una [App sviluppatore](/it/resources/fundamentals/developer-apps) che si trova all’interno di un [Progetto](/it/resources/fundamentals/projects).

Scopri di più su come ottenere l’accesso agli endpoint dell’API X v2 nella nostra pagina [Guida introduttiva](/it/x-api/getting-started/getting-access).
