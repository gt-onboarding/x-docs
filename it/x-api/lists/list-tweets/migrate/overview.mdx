  ## Confronto tra gli endpoint di ricerca dei post delle Liste dell’API X
</div>

L’endpoint di ricerca dei post delle Liste v2 sostituirà l’endpoint standard v1.1 [GET lists/statuses](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-statuses). Se disponi di codice, app o strumenti che utilizzano questa versione dell’endpoint e stai valutando la migrazione al nuovo endpoint dell’API X v2, questa serie di guide è per te.

Le tabelle seguenti confrontano gli endpoint delle Liste dello standard v1.1 e dell’API X v2:

**Ricerca lista per ID**

|     |     |     |
| :--- | :--- | :--- |
| Descrizione | v1.1 standard | API X v2 |
| Metodi HTTP supportati | `GET` | `GET` |
| Dominio host | `https://api.x.com` | `https://api.x.com` |
| Percorso endpoint | `/1.1/lists/statuses.json` | `/2/lists/:id/tweets` |
| [Autenticazione](/it/resources/fundamentals/authentication) | OAuth 1.0a User Context<br /><br />Solo app | OAuth 1.0a User Context<br /><br />OAuth 2.0 Authorization Code con PKCE<br /><br />Solo app |
| [Limiti di velocità](/it/resources/fundamentals/rate-limits) predefiniti per le richieste | 900 richieste ogni 15 min con OAuth 1.0a<br /><br />900 richieste ogni 15 min con solo app | 900 richieste ogni 15 min con OAuth 1.0a<br /><br />900 richieste ogni 15 min con OAuth 2.0<br /><br />900 richieste ogni 15 min con solo app |

Per accedere agli endpoint dell’API X v2, devi [registrarti per un account sviluppatore](https://developer.x.com/en/portal/petition/essential/basic-info). In fase di autenticazione, devi utilizzare le chiavi e i token di una [app sviluppatore](/it/resources/fundamentals/developer-apps) che si trova all’interno di un [progetto](/it/resources/fundamentals/projects).

Scopri di più su come ottenere l’accesso agli endpoint dell’API X v2 nella nostra pagina [guida introduttiva](/it/x-api/getting-started/getting-access).
