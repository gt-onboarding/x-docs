L’endpoint di ricerca dei Post di una Lista consente di recuperare i Post da una Lista specifica. Con questo endpoint puoi creare soluzioni che permettono agli utenti di personalizzare, organizzare e dare priorità ai Post visualizzati nella loro timeline.

Per questo endpoint è previsto un rate limit di 900 richieste ogni 15 minuti e la risposta consente di interrogare gli ultimi 800 Post per una determinata Lista.

Puoi utilizzare [OAuth 1.0a User Context](/it/resources/fundamentals/authentication), \[App only]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token), oppure [OAuth 2.0 Authorization Code con PKCE](it/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) per autenticare le richieste a questo endpoint.

**Configurazione dell’account**

Per accedere a questi endpoint, avrai bisogno di:

- Un [account sviluppatore](https://developer.x.com/en/portal/petition/essential/basic-info) approvato.
- Autenticarti utilizzando le chiavi e i token di una [App sviluppatore](/it/resources/fundamentals/developer-apps) all’interno di un [Project](/it/resources/fundamentals/projects).

Scopri di più su come ottenere l’accesso agli endpoint dell’API X v2 nella nostra [guida introduttiva](/it/x-api/getting-started/getting-access).
