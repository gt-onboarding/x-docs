<div id="steps-to-build-a-manage-pinned-list-request">
  ### Passaggi per creare una richiesta di gestione delle Liste fissate
</div>

**Primo passaggio: scegli la raccolta List in Postman**

Una volta caricata in Postman la raccolta dell’API X v2, vai alla cartella “List”, seleziona la sottocartella “Pinned Lists”, quindi scegli “Pin a List”.
 

**Secondo passaggio: specifica la List da fissare**

Gli endpoint per gestire le Liste fissate richiedono due ID: uno per l’utente (l’utente autenticato che fissa una List) e uno per la List di destinazione (la List da fissare). L’ID dell’utente deve corrispondere all’ID dell’utente che effettua l’autenticazione, il che significa che devi passare gli Access Token associati all’ID utente quando autentichi la richiesta. In questo caso puoi specificare l’ID del tuo utente. Puoi trovare il tuo ID in due modi:

1. Utilizzando l’endpoint [users lookup](/it/x-api/users/lookup/introduction) per username, puoi passare uno username e ricevere il campo id.
2. Guardando il tuo Access Token, noterai che la parte numerica è il tuo ID utente.

L’ID della List di destinazione può essere qualsiasi List valida. In Postman, vai alla scheda “Params” e inserisci l’ID utente nella colonna “Value” della variabile di percorso id. Vai alla scheda “Body” e inserisci l’ID della List che desideri fissare come valore per il parametro list\_id. Assicurati di non includere spazi prima o dopo nessun ID.

|     |     |     |
| :--- | :--- | :--- |
| **Key** | **Value** | **Parameter type** |
| `id` | L’ID utente | path |
| list\_id | L’ID della List di destinazione da fissare | body |

Dovresti ora vedere un URL simile accanto al pulsante “Send”:

```
      https://api.x.com/2/users/2244994945/pinned_lists
```

Step three: Make your request and review your response

Una volta impostato tutto, premi il pulsante “Send” e riceverai una risposta simile al seguente esempio:

```
{
  "data": {
    "pinned": true
  }
}
```

Hai fissato correttamente la List di destinazione se l’oggetto di risposta contiene il valore true per la chiave pinned. 

Per rimuovere la fissazione di una List, seleziona la richiesta “Unpin List” che si trova anche nella cartella “Lists” della raccolta API X v2 caricata in Postman. Questo endpoint richiede l’ID utente autenticato e l’ID della List da defissare. Nella scheda “Params”, inserisci l’ID utente come valore per la colonna id e l’ID della List da defissare come valore per la colonna list\_id. 

In caso di richiesta di eliminazione riuscita, riceverai una risposta simile al seguente esempio:

```
{
  "data": {
    "pinned": false
  }
}
```
