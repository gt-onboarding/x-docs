Le nostre API Enterprise offrono il massimo livello di accesso e affidabilità per chi fa affidamento sui dati di X. Ideali quando si supera il livello Premium e si necessita di accesso più affidabile, pacchetti personalizzati o contratti a lungo termine. L’accesso alle API Enterprise include account manager dedicati e supporto tecnico personalizzato.

Se hai riscontrato che le soluzioni Basic e Pro non sono sufficienti per le tue esigenze, puoi richiedere l’accesso ai prodotti Enterprise al seguente link:

  Richiedi l’accesso
</Button>

  ## Endpoint disponibili

| :--- | :--- | :--- |
| [Decahose API](/it/x-api/enterprise-gnip-2.0/fundamentals/decahose-api) | Trasmette un campione casuale del 10% di tutti i Post pubblici e dei relativi like. |
| :--- | :--- | :--- |
| [Account Activity API](/it/x-api/enterprise-gnip-2.0/fundamentals/account-activity) | Consenti la sottoscrizione agli utenti e ricevi varie attività, tra cui Post, Messaggi Diretti, like, follow e altro. |
| :--- | :--- | :--- |
| Storico | [Search API](/it/x-api/enterprise-gnip-2.0/fundamentals/search-api) | Esegui ricerche sugli ultimi 30 giorni o sull’archivio completo dei Post pubblici usando strumenti di filtraggio avanzati. |

<div id="gnip-console">

</div>

Puoi gestire i prodotti enterprise e l’accesso al tuo account tramite la [Gnip Console](/it/x-api/enterprise-gnip-2.0/fundamentals/overview). 

<div id="enrichments">
  ### Arricchimenti
</div>

Abbiamo aggiunto metadati al payload di risposta delle API premium per gli sviluppatori che utilizzano un piano a pagamento. Per saperne di più su URL espansi, dati dei sondaggi e geolocalizzazione del profilo, visita la nostra [sezione sugli arricchimenti](/it/x-api/enterprise-gnip-2.0/fundamentals/data-enrichments).
 

<div id="advanced-filtering-tools">
  ### Strumenti avanzati di filtro
</div>

La Premium Search API offre un set avanzato di operatori non disponibili con l’endpoint di ricerca standard. Molti di questi operatori sono accessibili anche agli utenti della sandbox gratuita, ma con un piano a pagamento avrai a disposizione un set di operatori più completo. Per saperne di più, visita la nostra [sezione su regole e filtri](/it/x-api/enterprise-gnip-2.0/fundamentals/rules-filtering#getting-started-with-enterprise-rules-and-queries).
