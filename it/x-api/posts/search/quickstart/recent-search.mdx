**Terzo passaggio: crea una query di ricerca**

Ogni ricerca recente richiede una singola [query di ricerca](/it/x-api/posts/search/integrate/build-a-query). | **Key** | **Value** | **Description** |

**Quarto passaggio: identifica e specifica quali campi desideri recuperare**

Per questo esercizio, richiederemo quattro diversi set di campi da oggetti differenti:

1. I campi predefiniti del Post object.

|     |     |     |
| :--- | :--- | :--- |

| `tweet.fields` | `created_at` | `tweets.created_at` |
| `expansions` | `author_id` | `includes.users.id`, `includes.users.name`, `includes.users.username` |
