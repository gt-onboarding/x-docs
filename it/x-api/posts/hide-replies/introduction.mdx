Questo endpoint consente di nascondere o ripristinare la visibilità delle risposte in modo programmatico utilizzando criteri da te definiti. Come nell’esperienza principale di X, le risposte verranno rimosse dalla conversazione principale ma resteranno visibili su una pagina separata. Puoi usare l’endpoint per creare app che aiutino le persone a nascondere le risposte che non ritengono utili.

L’endpoint per nascondere le risposte utilizza [OAuth 1.0a User Context](/it/resources/fundamentals/authentication#oauth-1-0a-2) o [OAuth 2.0 Authorization Code with PKCE](it/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) per l’autenticazione. In caso di esito positivo, l’endpoint nasconde una singola risposta pubblicata in una conversazione di Post avviata da un utente autenticato. Ogni conversazione supporta l’occultamento fino a 725 Post.

<Note>
  **Configurazione dell’account**

  Per accedere a questi endpoint, ti serviranno:

  - Un [account sviluppatore](https://developer.x.com/en/portal/petition/essential/basic-info) approvato.
  - Autenticarti utilizzando le chiavi e i token di una [App sviluppatore](/it/resources/fundamentals/developer-apps) che si trova all’interno di un [Project](/it/resources/fundamentals/projects).

  Scopri di più su come ottenere l’accesso agli endpoint dell’API X v2 nella nostra [guida introduttiva](/it/x-api/getting-started/getting-access).
</Note>
