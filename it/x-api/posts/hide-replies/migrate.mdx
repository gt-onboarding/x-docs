  ## Confronto tra gli endpoint di API X per nascondere le risposte
</div>

L’endpoint v2 per nascondere le risposte sta sostituendo l’endpoint Labs per nascondere le risposte. Se disponi di codice, app o strumenti che utilizzano la versione Labs di questo endpoint e stai valutando la migrazione al nuovo endpoint API X v2, questa guida fa per te.

Per utilizzare la nuova API X v2 (incluso l’endpoint per nascondere le risposte), dovrai [aderire al nuovo portale per sviluppatori](https://developer.x.com/en/portal/opt-in), creare un [Project](/it/resources/fundamentals/projects) e aggiungere un’App a quel Project. Potrai quindi usare le credenziali associate a tale App per effettuare richieste all’endpoint per nascondere le risposte. Aggiungere la stessa App registrata per l’endpoint Labs v2 per nascondere le risposte manterrà gli utenti autenticati.

La seguente tabella confronta le differenze tra Labs e il nuovo endpoint di API X v2:
 

| **Descrizione** | **Labs v2** | **API X v2** |
| :--- | :--- | :--- |
| Dominio host | https://api.x.com | https://api.x.com |
| Percorso endpoint | /labs/2/tweets/:id/hidden | /2/tweets/:id/hidden |
| Autenticazione | OAuth 1.0a in contesto utente | OAuth 1.0a in contesto utente |
| Metodi HTTP supportati | PUT | PUT |
| Limiti predefiniti di richiesta | 10 richieste ogni 15 minuti (condivise tra tutti gli utenti autenticati) | 50 richieste ogni 15 minuti (per ciascun utente autenticato) |
| Può nascondere risposte | ✔︎  | ✔︎  |
| Può rendere visibile una risposta precedentemente nascosta | ✔︎  | ✔︎  |
| Può nascondere o rendere visibili nuovamente le risposte più volte | ✔︎  | ✔︎  |
| Richiede l’uso di credenziali da una [developer App](/it/resources/fundamentals/developer-apps) associata a un [project](/it/resources/fundamentals/projects) |     | ✔   |
