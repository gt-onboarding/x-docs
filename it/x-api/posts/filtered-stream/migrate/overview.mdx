<div id="comparing-x-apis-filtered-stream-endpoints">
  ## Confronto tra gli endpoint del flusso filtrato dell’API X
</div>

Il gruppo di endpoint del flusso filtrato v2 sta sostituendo lo [standard v1.1 statuses/filter](https://developer.x.com/en/docs/x-api/v1/tweets/filter-realtime/api-reference/post-statuses-filter) e la [PowerTrack API](/it/x-api/enterprise-gnip-2.0/powertrack-api). Se disponi di codice, app o strumenti che utilizzano una versione precedente dell’endpoint del flusso filtrato e stai valutando la migrazione al nuovo endpoint dell’API X v2, questo confronto può aiutarti a iniziare.

Consulta le nostre guide di migrazione più approfondite:

[Migrazione da Standard v1.1 a API X v2](it/x-api/posts/filtered-stream#standard-v1-1-compared-to-x-api-v2)

[Migrazione da PowerTrack API ad API X v2](/it/x-api/enterprise-gnip-2.0/powertrack-api#powertrack-operators)
