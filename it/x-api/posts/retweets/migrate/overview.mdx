<div id="comparing-x-apis-retweets-endpoints">
  ## Confronto degli endpoint Retweet dell’API X
</div>

**Lookup dei Retweet**

L’endpoint di lookup dei Retweet della v2 sostituirà gli endpoint standard [v1.1 GET statuses/retweets/:id](https://developer.x.com/en/docs/twitter-api/v1/tweets/post-and-engage/api-reference/get-statuses-retweets-id) e [v1.1 GET statuses/retweets/:ids](https://developer.x.com/en/docs/twitter-api/v1/tweets/post-and-engage/api-reference/get-statuses-retweeters-ids).

Le tabelle seguenti confrontano gli endpoint Retweet della v1.1 standard e dell’API X v2:

| Descrizione | v1.1 standard | API X v2 |
| :--- | :--- | :--- |
| Metodi HTTP supportati | GET | GET |
| Dominio host | https://api.x.com | https://api.x.com |
| Percorso endpoint | /1.1/retweeters/id.json<br /><br />`/1.1/retweets/ids.json` | /2/users/:id/retweeted\_by |
| [Autenticazione](/it/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 2.0 Bearer Token<br /><br />OAuth 1.0a User Context |
| [Limiti di frequenza](/it/resources/fundamentals/rate-limits) predefiniti per le richieste | 75 richieste ogni 15 min | 75 richieste ogni 15 min (per App)<br /><br />75 richieste ogni 15 min (per utente) |
| Formato dei dati | Formato v1.1 standard | [Formato API X v2](/it/x-api/fundamentals/data-dictionary) (determinato dai parametri di richiesta fields ed expansions; non retrocompatibile con i formati v1.1)<br /><br />Per ulteriori informazioni sulla migrazione dal formato v1.1 standard al formato API X v2, consulta la nostra [guida alla migrazione dei formati dei dati](/it/x-api/migrate/data-format-migration). |
| Richiede l’uso di credenziali di una [App sviluppatore](/it/resources/fundamentals/developer-apps) associata a un [Progetto](/it/resources/fundamentals/projects) |     | ✔️  |

<div id="manage-retweets">
  ### Gestione dei Retweet
</div>

Le seguenti tabelle confrontano l’endpoint di annullamento del Retweet delle versioni standard v1.1 e API X v2:

**Retwitta un post**

| Descrizione | Standard v1.1 | API X v2 |
| :--- | :--- | :--- |
| Metodi HTTP supportati | POST | POST |
| Dominio host | https://api.x.com | https://api.x.com |
| Percorso endpoint | /1.1/statuses/retweet/:id.json | /2/users/:id/retweets |
| [Autenticazione](/it/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context |
| [Limiti di richiesta](/it/resources/fundamentals/rate-limits) predefiniti | Nessuno<br /><br />300 richieste per finestra di 3 ore (per utente, per app). Condiviso con l’endpoint POST Tweet | 50 richieste per 15 min (per utente)<br /><br />300 richieste per finestra di 3 ore (per utente, per app). Condiviso con l’endpoint POST Tweet per la gestione dei post. |
| Richiede l’uso di credenziali da un’[App per sviluppatori](/it/resources/fundamentals/developer-apps) associata a un [Progetto](/it/resources/fundamentals/projects) |     | ✔️  |
