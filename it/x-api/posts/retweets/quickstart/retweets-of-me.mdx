La consultazione “Repost dei miei Post” consente di vedere quali tuoi Post sono stati ricondivisi da altri.

Questo fornisce agli utenti indicazioni su quali dei loro Post vengono condivisi e risuonano con il pubblico, aiutando a favorire interazioni più mirate e ad apportare modifiche alla strategia dei contenuti.

<div id="authentication">
  #### Autenticazione
</div>

Poiché stai effettuando richieste per conto di un utente, devi autenticare questi endpoint con [OAuth 1.0a User Context](/it/resources/fundamentals/authentication) oppure con [OAuth 2.0 Authorization Code con PKCE](/it/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) e utilizzare i token di accesso associati all’utente per conto del quale stai effettuando la richiesta. Puoi generare questo token di accesso utente utilizzando il [flusso OAuth a 3 parti](https://developer.x.com/resources/fundamentals/authentication/obtaining-user-access-tokens) (OAuth 1.0a) oppure il [flusso Authorization Code con PKCE](/it/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-3) (OAuth 2.0).

<div id="making-a-request">
  #### Effettuare una richiesta
</div>

Una volta che un utente ha effettuato l’autenticazione con la tua app, puoi chiamare l’API Retweets of Me per conto dell’utente come mostrato di seguito:

```bash
      curl 'https://api.x.com/2/users/reposts_of_me' --header 'Authorization: ••••••'
```
