L'API X può essere usata per recuperare e analizzare in modo programmatico i dati di X, oltre che per sviluppare soluzioni per la conversazione su X.

Nel corso degli anni, l'API X si è evoluta aggiungendo ulteriori livelli di accesso, consentendo agli sviluppatori di scalare l’accesso per approfondire e studiare la conversazione pubblica.

Di recente abbiamo rilasciato l’API X v2. L’API X v2 offre una base moderna, funzionalità nuove e avanzate e un onboarding rapido al [Basic access](https://developer.x.com/en/portal/products/basic).

Le tre schede seguenti illustrano le diverse versioni e i livelli di accesso dell’API X, le novità della v2 e quali risorse di X è possibile recuperare, creare, eliminare e modificare tramite l’API.

  <Tab title="What's new with v2">
    ## Novità della v2

    L'API X v2 rappresenta il più grande aggiornamento dell'API X dal 2012. Introduce numerose funzionalità nuove e avanzate, oltre a un accesso rapido e gratuito all'API.

    Alcune delle funzionalità disponibili con la v2 includono:

    ### Nuovi endpoint

    Abbiamo rilasciato una serie di endpoint completamente nuovi per l'API X v2. Puoi consultare l'elenco completo degli endpoint v2, inclusi quelli appena introdotti, nella nostra guida alla mappa degli endpoint dell'API X.

    [Visita la mappa degli endpoint dell'API X >](/it/x-api/migrate/x-api-endpoint-map)

    ### Nuovi oggetti dati, più dettagliati

    Abbiamo modernizzato i nostri oggetti dati con una serie di miglioramenti che ti permetteranno di navigare e analizzare i dati più facilmente.

    [Visita il dizionario dei dati >](/it/x-api/fundamentals/data-dictionary)

    ### Nuovi parametri per aiutarti a recuperare solo gli oggetti e i campi che desideri

    Abbiamo aggiunto i parametri `fields` ed `expansions` ai nostri endpoint dati, che consentono di richiedere oggetti e campi correlati oltre a quelli restituiti per impostazione predefinita.

    [Scopri come usare fields ed expansions >](/it/x-api/fundamentals/data-dictionary#how-to-use-fields-and-expansions)

    ### Metriche avanzate

    Comprendi più facilmente le prestazioni di Post, utenti, media e sondaggi direttamente nel payload richiedendo sia metriche pubbliche sia private, incluse impression, visualizzazioni video, clic su profilo utente e su URL, alcune delle quali sono distinte tra organico e sponsorizzato.

    [Scopri di più sulle metriche >](/it/x-api/fundamentals/metrics)

    ### Filtra e identifica quali Post contengono argomenti diversi

    Quando utilizzi la ricerca dei Post o il flusso filtrato, ora puoi filtrare per argomento usando i nostri operatori `entity` e `context`. Abbiamo anche incluso questi argomenti nel payload del Post per facilitare l'analisi.

    [Scopri di più sulle annotazioni dei Post >](/it/x-api/fundamentals/post-annotations)

    ### Filtra e identifica quali Post appartengono a un thread di risposte

    Rendi più semplice identificare un Post come parte di un thread di conversazione quando utilizzi la ricerca dei Post, il flusso filtrato e il lookup dei Post. Abbiamo anche aggiunto la possibilità di determinare se sono state impostate le impostazioni di risposta per un Post con il campo del Post `reply_settings`.

    [Scopri di più sul tracciamento delle conversazioni >](/it/x-api/fundamentals/conversation-id)

    ### E molto altro ancora...

    - Filtraggio antispam ad alta affidabilità
    - Gli URL accorciati vengono completamente espansi per semplificare l’analisi degli URL
    - Oggetti di risposta JSON semplificati, con rimozione dei campi deprecati e modernizzazione delle etichette
    - Funzionalità di ripristino e ridondanza per i nostri endpoint di streaming
    - Restituzione del 100% dei Post pubblici corrispondenti e disponibili nelle query di ricerca
    - “Regole” di streaming per apportare modifiche senza interrompere le connessioni
    - Linguaggio di query più espressivo per flusso filtrato e ricerca
    - Specifica OpenAPI per creare nuove librerie e tracciare le modifiche in modo più trasparente
    - Supporto dell’API per nuove funzionalità ed endpoint in tempi più rapidi, man mano che la nostra piattaforma evolve per soddisfare le esigenze di sviluppatori, ricercatori, aziende e utenti di X
  </Tab>

  <Tab title="X API resources">
    ## Risorse della piattaforma API X

    Nel design delle API, una risorsa è un’entità con dati associati, relazioni con altre risorse e un insieme di metodi che operano su di essa. Ad esempio, un Post è una risorsa che puoi creare, eliminare o recuperare utilizzando vari strumenti, come cercarlo nello storico o ottenerlo in tempo reale.

    L’API X consente di creare, eliminare, ricevere o modificare diverse risorse sulla piattaforma, tra cui le seguenti:

    | Risorsa             | Descrizione                                                                                                                                                                             |
    | :------------------ | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
    | **Posts**           | Accedi a milioni di Post per comprendere la conversazione pubblica o crea i tuoi per partecipare alla conversazione.  Disponibilità attuale:  - API X v2  - Enterprise  - Standard v1.1 |
    | **Users**           | Gestisci o cerca utenti X per analizzare le reti, comprendere il tuo pubblico o favorire relazioni positive online.  Disponibilità attuale:  - API X v2  - Enterprise  - Standard v1.1     |
    | **Spaces**          | Cerca e consulta gli X Spaces e i loro partecipanti per aiutare le persone a trovare conversazioni audio interessanti e pertinenti.  Disponibilità attuale:  - API X v2                                    |
    | **Direct Messages** | Invia e ricevi Direct Messages per gestire i problemi dei clienti, inviare messaggi di benvenuto o favorire interazioni umane positive.  Disponibilità attuale:  - Standard v1.1                        |
    | **Lists**           | Crea e gestisci elenchi di account per tenere d’occhio gli esperti del settore, le voci influenti o organizzare chi segui.  Disponibilità attuale:  - API X v2  - Standard v1.1               |
    | **Trends**          | Individua per primo le tendenze geografiche per cogliere i movimenti del settore, scoprire argomenti caldi o anticipare l’ultima moda.  Disponibilità attuale:  - Standard v1.1                           |
    | **Media**           | Carica oggetti multimediali per condividere la tua creatività, creare esperienze interattive o sviluppare strumenti di accessibilità.  Disponibilità attuale:  - Standard v1.1                               |
    | **Places**          | Cerca luoghi per capire cosa succede nel tuo quartiere e nel resto del mondo.  Disponibilità attuale:  - Standard v1.1                                                     |
  </Tab>
