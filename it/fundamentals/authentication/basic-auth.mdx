<div id="basic-authentication">
  ## Autenticazione di base
</div>

Molte API enterprise di X richiedono l’uso dell’autenticazione HTTP Basic. Per effettuare correttamente una richiesta a un’API che richiede l’autenticazione Basic, devi includere per ogni richiesta un’intestazione di autorizzazione con una combinazione valida di indirizzo email e password. La combinazione email/password è la stessa che utilizzi per accedere alla [console delle API enterprise](https://console.gnip.com/) e può essere modificata direttamente da lì.

Quando crei una richiesta utilizzando l’autenticazione Basic, assicurati di aggiungere l’intestazione HTTP Authorization: Basic con le credenziali codificate su HTTPS.

Nel seguente esempio di richiesta cURL, sostituisci `<email_address>` e `<password>` con le tue credenziali prima di inviare la richiesta:

```bash
curl -v --compressed -u<email_address>:<password>
    "https://gnip-api.x.com/search/30day/accounts/<account-name>/prod/counts.json?query=from%3Axdevelopers"
```

- [Decahose stream API](http://localhost:3000/x-api/enterprise-gnip-2.0/fundamentals/decahose-api) enterprise
- [30-Day Search API](/it/x-api/enterprise-gnip-2.0/fundamentals/search-api) enterprise
- [Full-Archive Search API](/it/x-api/enterprise-gnip-2.0/fundamentals/search-api) enterprise
