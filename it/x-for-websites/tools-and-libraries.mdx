<div id="official-resources">
  ## Risorse ufficiali
</div>

Gli strumenti seguenti possono aiutarti a iniziare con X per i siti web.

<CardGroup cols={3}>
<Card
  title="publish.x.com"
  icon="x-twitter"
  href="https://publish.x.com/"
>
Crea un post incorporato, una timeline o un pulsante personalizzati.
</Card>
<Card
  title="Twitter for WordPress"
  icon="x-twitter"
  href="https://github.com/twitter/wordpress"
>
Plugin ufficiale per ottimizzare il tuo sito WordPress per X, con incorporamento di post e X Cards.
</Card>
<Card
  title="twemoji"
  icon="x-twitter"
  href="https://twemoji.x.com/"
>
Set di emoji gratuito e open source di X, che include una libreria JavaScript per il supporto multipiattaforma.
</Card>
</CardGroup>
