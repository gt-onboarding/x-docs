<div id="basic-authentication">
  ## 基本身份验证
</div>

许多 X 的企业级 API 需要使用 HTTP 基本身份验证。要成功向需要基本身份验证的 API 发出请求，您必须在每个请求中通过授权标头传递有效的电子邮箱地址与密码组合。该邮箱和密码与您用于访问 [enterprise API console](https://console.gnip.com/) 的凭据相同，并可在该控制台中进行修改。

使用基本身份验证构建请求时，请确保通过 HTTPS 添加带有编码凭据的 Authentication: Basic HTTP 标头。

在以下 cURL 请求示例中，您需要在发送请求之前将 `<email_address>` 和 `<password>` 替换为您的凭据：

```bash
curl -v --compressed -u<email_address>:<password>
    "https://gnip-api.x.com/search/30day/accounts/<account-name>/prod/counts.json?query=from%3Axdevelopers"
```

**需要基本身份验证的 API：**

- [PowerTrack API](/zh/x-api/enterprise-gnip-2.0/powertrack-api) enterprise
- [Decahose stream API](http://localhost:3000/x-api/enterprise-gnip-2.0/fundamentals/decahose-api) enterprise
- [30-Day Search API](/zh/x-api/enterprise-gnip-2.0/fundamentals/search-api) enterprise
- [Full-Archive Search API](/zh/x-api/enterprise-gnip-2.0/fundamentals/search-api) enterprise
- [Usage API](/zh/x-api/enterprise-gnip-2.0/fundamentals/usage) enterprise
