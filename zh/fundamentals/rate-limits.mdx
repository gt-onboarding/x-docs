---
title: 速率限制
icon: gauge-simple-high
---

每天有成千上万的开发者向 X 开发者平台发起请求。为管理如此庞大的请求量，我们对可发起的请求数量设置了限制。这些限制帮助我们为开发者社区提供可靠且可扩展的 API。

我们的各类 API 在速率限制的使用方式上各不相同。要了解不同平台之间的差异，请查阅各 API 版块中的相应速率限制页面：

- [X API v2](/zh/x-api/fundamentals/rate-limits)
- [X API：Enterprise](/zh/x-api/enterprise-gnip-2.0/fundamentals/rate-limits)
- [X Ads API](/zh/x-ads-api/fundamentals/rate-limiting)
