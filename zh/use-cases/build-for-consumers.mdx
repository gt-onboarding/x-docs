<div id="ways-to-build-for-people-on-x">
  ## 面向 X 用户的构建途径
</div>

我们正开放平台，鼓励你打造对公共对话产生积极影响、服务于 X 用户的新工具和产品。具体而言，我们希望在以下领域推动创新与开发：

<CardGroup>
  <Card title="为健康与安全进行对话治理" icon="bullhorn" iconType="solid">
    通过构建内容审核工具和其他解决方案，帮助提升公共对话的健康与安全，让人们在 X 上参与交流时感到安心并乐于参与。构建互补或增强型技术，为人们提供更完善的工具选择，以便自主掌控他们的互动体验。

    [**了解更多**](/zh/x-api/what-to-build#moderate-conversations-for-health-and-safety) <Icon icon="arrow-right" />
  </Card>

  <Card title="赋能创作与个性表达" icon="bullhorn" iconType="solid">
    为创作者、品牌和个体构建工具，帮助他们扩大触达、表达品牌并与志同道合的人建立连接。或拓展人们在 X 上的互动方式，例如跨平台发布、游戏化，或实用型机器人。

    [**了解更多**](/zh/x-api/what-to-build#enable-creation-and-personal-expression) <Icon icon="arrow-right" />
  </Card>

  <Card title="衡量与分析“正在发生的事”" icon="bullhorn" iconType="solid">
    帮助人们对公共对话或自身内容进行衡量、分析与洞察。更好地了解你的受众，评估内容表现与影响，做出更明智的决策，等等。

    [**了解更多**](/zh/x-api/what-to-build#measure-and-analyze-whats-happening) <Icon icon="arrow-right" />
  </Card>

  <Card title="优化社区体验" icon="bullhorn" iconType="solid">
    X 面向所有人，我们希望鼓励并赋能你运用专业能力，更好地服务 X 上各类社区的独特需求。无论你是在为新兴全球市场构建本地化解决方案，还是为平台上不断壮大的社区（如游戏、金融）优化体验，我们希望每个人都能在 X 上找到归属。

    [**了解更多**](/zh/x-api/what-to-build#improve-community-experiences) <Icon icon="arrow-right" />
  </Card>

  <Card title="策划与推荐内容" icon="bullhorn" iconType="solid">
    构建应用、机器人或其他工具，帮助人们发现新内容或建立新的连接。或打造经过策划且可自定义的体验，让人们看到更多他们最关心的内容。

    [**了解更多**](/zh/x-api/what-to-build#curate-and-recommend-content) <Icon icon="arrow-right" />
  </Card>

  <Card title="造福社会公益" icon="bullhorn" iconType="solid">
    开展突破性研究，推动对公共健康与气候危机的认知提升，并以非商业方式为社会公益而构建。

    [**了解更多**](/zh/x-api/what-to-build#impact-the-greater-good) <Icon icon="arrow-right" />
  </Card>
</CardGroup>
