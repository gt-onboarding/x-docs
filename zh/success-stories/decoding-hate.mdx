<div id="situation">
  ## 情况
</div>

新冠疫情暴发后，针对亚裔美国人和太平洋岛民（AAPI）的仇恨事件有所增加。尽管其中一些被媒体报道，仍有成千上万的人向全国举报中心 [Stop AAPI Hate](https://stopaapihate.org/) 提交报告。该机构通过研究、倡导和能力建设来回应并应对仇恨事件。与此同时，[The Asian American Foundation（TAAF）](https://www.taaf.org/) 成立。该组织成为项目团队的推动力量，团队决定利用 X 提供一个全新的窗口，展示这些事件如何在个人层面影响受害者。

<div id="solution">
  ## 解决方案
</div>

由两名工程师组成的小团队将 X 作为其事件数据来源。他们选择该平台有两个原因：首先，它包含丰富多样的信息，例如新闻、人的观点、想法、日常生活事件以及社区中发生的事情。正因如此，X 提供了各类人群独特且贴近的样本。X 也是唯一真正开放 API 的平台，TAAF 的开发者可以据此访问数据，并使用完善的工具进行数据收集与分析。 

利用 X API v2 的 search Tweets 和 Tweets lookup 端点，团队能够搜索帖子并识别出人们何时报告仇恨事件（甚至还能看到他们使用了哪些话题标签）。这帮助团队设计出一个长达 1023 个字符的搜索查询，把数十亿条帖子缩小到团队可处理的规模。 

接下来是验证这些帖子是否确实与仇恨事件相关。团队开发了一个自然语言处理工具，帮助从成千上万条帖子中筛选出与项目相关的内容。在应用其训练的机器学习模型后，这些帖子再由人工复核，以进一步提高准确性。 

2021 年，TAAF 还与 Stop AAPI Hate 联盟共同撰写了 [Documenting Anti-AAPI Hate Codebook](https://stopaapihate.org/wp-content/uploads/2021/12/SAH-x-TAAF-Codebook-FINAL-2021-08-19.pdf)。该资源包含面向社区的数据收集标准与实践草案，帮助团队对人们在 X 上分享的事件进行分类。 

在此基础上，下一个挑战是以便于理解的方式呈现这些数据。TAAF 创建了数据可视化工具 [Decoding Hate](https://decodinghate.taaf.org/)，以交互方式展示其海量数据。在一家数据可视化工作室的帮助下，他们将标注后的 X 数据转化为由数千条帖子构成的洞察性故事，揭示 AAPI 社群中正在发生事件的真相。

<div id="impact">
  ## 影响
</div>

Decoding Hate 项目帮助检测了数以千计此前未被报告的仇恨相关事件，并且至今仍在持续发挥作用。该项目已成为持续提升公众对 AAPI 社群中个体经历认知的工具。

在下一阶段的工作中，TAAF 将与 Stop AAPI Hate 合作，试点由真人对在 X 上报告仇恨事件的个人进行一对一回应。在此阶段，Stop AAPI Hate 将积极与许多发布帖子、讨论其所经历仇恨的用户互动，并鼓励他们通过 [Stop AAPI Hate Reporting Center](https://stopaapihate.org/) 报告事件。向 Stop AAPI Hate 报告的人越多，该联盟及其合作伙伴就能越好地理解正在发生的仇恨事件的性质，并为最有效的政策解决方案进行倡导。 

除了揭示 AAPI 社群中正在发生的仇恨相关事件外，该项目也彰显了 X API 的价值。本案例表明，通过利用合适的数据与合适的工具，开发者可以使用 X 为公益进行创新。
