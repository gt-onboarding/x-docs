<Note>
**简而言之**

通往成功之路：当 Red Cloud Financial Services 受托帮助一家矿业公司推广其全新的绿色技术时，他们借助基于 X 分析打造的 Audiense 定向工具深入挖掘数据，触达全新受众。
</Note>

<div id="situation">
  ## 情况
</div>

Red Cloud Financial Services 受托帮助其客户（一家上市矿业公司）在全新受众中提升其一项正在申请专利的新技术的知名度。

为在新市场落地并有效开展能够触达这些新客户的营销活动，该公司必须找到一条能够为这款新产品带来实际成果的路径。

这一需求带来了两个关键挑战：

1. **触达新受众**

    不同于以往专注于既有矿业客户，这项新技术更吸引关注绿色创新与社会责任的更广泛人群。对 Red Cloud 而言，这是未曾涉足的领域，与其此前的活动截然不同。

1. **提供明确的 ROI**
    
    该客户并不常做付费投放。要求 Red Cloud 为以提升认知为目标的活动清晰界定 ROI，并且在其几乎缺乏第一手经验的受众中产生成效，是一项艰巨任务。
    
    Red Cloud 因此求助于 Audiense，以便在研究潜在新客户的同时触达现有客户。Audiense 的 X 营销产品擅长借助其强大的受众定向工具，投放卓有成效的 X 活动。

<div id="solution">
  ## 解决方案
</div>

Red Cloud 先行设定了预期。借助 Audiense 的洞察生成活动表现预测，Red Cloud 得以就活动的最终目标及相关 KPI 与客户达成一致。 

在 Audiense 的支持下，Red Cloud 将研究重点放在 X 上，挖掘公司新闻与相关股票信息。Audiense 的平台便于发现活动与兴趣提升之间的潜在关联，这种提升体现在视频观看量上升和网站流量增长。

三步法：

1. 构建全新受众。

    在客户的指导下，Red Cloud Financial 确定了两类核心受众以聚焦投放：一类为采矿与投资，另一类为清洁科技、电动汽车、ESG 与责任投资。X 的数据进一步帮助 Red Cloud 将这些受众与后续活动对齐。

1. 细分新受众。 

    尽管新受众横跨多个行业，但它们之间存在诸多共性。Audiense 帮助 Red Cloud Financial 深入挖掘这些新细分，识别关键影响者、内容消费习惯等。这些洞察也帮助他们确定应关注哪些账号，以期获得互关，从而扩大 Red Cloud 及其客户的受众群体。

1. 评估价值高低。 
    
    评估价值高低。通过使用 Audiense 的[Targeting Pack 功能](https://help.audiense.com/knowledge/what-is-a-targeting-pack)，Red Cloud 开展了高度定向的 X 付费广告活动，同时利用为 X 细分积累的数据开展自然投放。借助 Targeting Pack 产出的洞察，Audiense 与 Red Cloud 为两类受众分别制作了两支独立的视频，从而识别出持续表现最佳的视频，并持续优化定向。

<div id="impact">
  ## 影响
</div>

Audiense 的平台及其基于 X 的研究帮助 Red Cloud 找到并细分合适的受众，借此促进粉丝增长，并为其有机与付费内容的整体社交媒体策略提供依据。

Red Cloud Financial 超出了客户预期。事实上，活动成效十分显著，客户决定继续推进并加大力度开展另一轮活动。

- 联合投放的活动获得了 350 万+ 次曝光和 280 万+ 次总观看
- 与客户处于专利申请中的新技术相关的讨论量上升
- 网站引荐流量提升，其中来自关于其新技术的推广文章与新闻稿的新增访客达 7,000 人

这充分说明，借助 X 上的受众分析与定向可以帮助你触达新受众、优化营销活动并取得实效。这也是助力塑造客户品牌、推动业务增长的良机。
