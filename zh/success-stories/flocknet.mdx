<Note>
**要点速览**

以下是开发者兼创作者 Aaron Ng 如何使用 X 的 API 构建 Flocknet——一款面向您在 X 上人脉网络的搜索引擎。人们正在使用 Flocknet 来拓展人脉、招聘人才和获取投资线索。Flocknet 最初为他个人使用而构建，如今正帮助用户在 X 上管理数百万条联系人关系。
</Note>

<div id="challenge">
  ## 挑战
</div>

当 Aaron Ng 在纽约参加一场设计大会时，他希望能与业内人士建立联系并见面。但他很快意识到，即便是在 X 上与自己的设计师朋友互动，也比想象中更难实现。他的圈子大多活跃在 X 上，尤其是业内的朋友。难道没有办法让他看到哪些朋友会来这座城市吗？在不知道谁会在附近的情况下，他又该如何与业内人脉叙旧？

Aaron 立刻将这一挑战视为机遇。也正因他的创业精神，Flocknet 应运而生。 

<div id="solution">
  ## 解决方案
</div>

仅用一个周末，Aaron 就做出了一个产品，即使不看他们的帖子，也能在他的 X 网络中搜索、排序并了解人脉、朋友和其他专业人士的动态。他选择在这个项目中使用 X 的 API，“知道有可以请教的人，而且[知道 X 的]团队很重视这件事，会让体验好很多。”他还补充说：“促使我不断构建的那部分社区大多在 X 上——这也是它为何对我如此重要，以及我为什么想做些工具来提升我的使用体验。”

凭借他的创造力、驱动力和经验，Flocknet 应运而生——一款用于在你的 X 网络中寻找人脉的搜索引擎和联系人管理工具。借助 X 的关注关系端点和 Python，Flocknet 允许用户按职务、任职机构、所在地以及众多其他属性来搜索其粉丝/关注者。

虽然最初只是为他个人使用而打造，Flocknet 却引发了广泛的公众兴趣，让他意识到应该向更多人开放。因此，他迅速转而将精力放在扩展 Flocknet，将其打造成如今的样子。

<div id="impact">
  ## 影响
</div>

在短短几个月内，Flocknet 就已代表其用户管理了数百万个联系人。技术从业者认为它格外实用，应用场景从招聘到投资，无所不包。事实上，连 Aaron 都通过 Flocknet 招募过人才！

但 Aaron 并未就此止步。围绕收到的功能需求，他持续探索扩展和打磨产品的路径。

最终，Aaron 找到了一种方法，将 X 打造成一个元平台，进而优化那些主要在 X 上构建社区的用户体验。他说：“我就是会做很多东西。我热衷于做对自己有用、也对他人有用的东西。”而他确实做到了。
