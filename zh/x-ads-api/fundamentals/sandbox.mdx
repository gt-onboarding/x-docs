---
title: Sandbox
---

Sandbox 是用于测试和构建你的 X 广告 API 实现的环境。

通过 Sandbox 创建的广告系列不会向用户投放，同时可以为诸如资金工具（Funding Instruments）等对象动态创建测试记录。

使用 Sandbox 只需调用 ads-api-sandbox.x.com 的相应主机名，但它还包含一些仅在 Sandbox 中提供的独有功能。

先向 [POST accounts](/zh/x-ads-api/campaign-management#post-accounts) 发起请求以创建新的广告主账号。该请求的响应会包含一个 account\_id，可用于后续请求。这些端点使使用者能够创建多个广告主账号并进行测试。

<div id="sandbox-only-endpoint-list">
  ### 仅限沙盒的端点列表：
</div>

- [POST accounts](/zh/x-ads-api/campaign-management#post-accounts) 在沙盒中创建账户
- [POST accounts/:account\_id/features](/zh/x-ads-api/campaign-management#post-accounts-account-id-features) 在沙盒中为账户添加功能
- [POST accounts/:account\_id/funding\_instruments](/zh/x-ads-api/campaign-management#post-accounts-account-id-funding-instruments) 在沙盒中为账户添加资金工具
- [DELETE accounts/:account\_id](/zh/x-ads-api/campaign-management#delete-accounts-account-id) 在沙盒中删除账户
- [DELETE accounts/:account\_id/features](/zh/x-ads-api/campaign-management#delete-accounts-account-id-features) 在沙盒中从账户移除功能
- [DELETE accounts/:account\_id/funding\_instruments/:funding\_instrument\_id](/zh/x-ads-api/campaign-management#delete-accounts-account-id-funding-instruments-funding-instrument-id) 在沙盒中从账户移除资金工具
