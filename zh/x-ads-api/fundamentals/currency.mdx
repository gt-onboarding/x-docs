---
title: 货币
---

货币类型遵循 [ISO-4217](http://en.wikipedia.org/wiki/ISO_4217) 标识，即由三个字母组成的字符串，如 `USD` 或 `EUR`。货币的数值以微单位（micros）表示。以 USD 为例，$5.50 编码为 5.50\*1e6，即 5,500,000。若要表示“整数金额”，所有货币均需将其本地微单位乘以 1e6（1\_000\_000）。

X 广告 API 认可 X SMB Ads 产品可用国家的官方货币。有关这些国家的完整列表，请点击[此处](https://business.x.com/en/help/overview/about-eligibility-for-twitter-ads.html)。

大多数货币的小数精度为两位（如 USD：$1.23）。

- `AED`, `VEF`, `PKR`, `IDR`, `HKD`, `QAR`, `PHP`, `NZD`, `INR`, `EUR`, `ARS`, `THB`, `MXN`, `EGP`, `DZD`, `SAR`, `LBP`, `COP`, `TRY`, `SGD`, `MAD`, `AUD`, `USD`, `MYR`, `CAD`, `BRL`, `GBP`。

以下货币不使用小数：

- `VND`, `KRW`, `CLP`, `JPY`

以下货币使用三位小数：

- `KWD`, `IQD`, `BHD`, `TND`

货币的使用通过[资金工具](/zh/x-ads-api/introduction)进行。
