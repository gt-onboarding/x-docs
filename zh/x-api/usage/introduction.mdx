X API v2 中的 Usage API 允许开发者以编程方式获取其项目用量。通过此端点，开发者可以在特定计费周期内跟踪并监控已拉取的帖子数量。

开发者可以使用 GET 端点获取过去最多 90 天的项目按日用量。用量也可按连接到你项目的各个客户端应用进行汇总。

此 GET 端点的应用级速率限制为每 15 分钟最多 50 次请求。

<div id="getting-started">
  ### 开始使用
</div>

要使用此端点，您需要从[开发者门户](https://developer.x.com/en/portal/dashboard)获取\[bearer token]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)。获得 bearer token 后，您可以按如下方式调用使用情况 API：

```
      curl 'https://api.x.com/2/usage/tweets' --header 'Authorization: Bearer XXXXX'
```

如果请求成功，您将看到如下所示的 JSON 响应：

```
{
    "data": {
        "cap_reset_day": 10,
        "project_cap": "5000000000",
        "project_id": "1369785403853424",
        "project_usage": "43435"
    }
}
```

<Note>
  **账户设置**

  要访问这些端点，您需要：

  - 一个已获批准的[开发者账户](https://developer.x.com/en/portal/petition/essential/basic-info)。
  - 使用位于[项目](/zh/resources/fundamentals/projects)内的[开发者应用](/zh/resources/fundamentals/developer-apps)的密钥和令牌进行身份验证。

  在我们的[入门指南](/zh/x-api/getting-started/getting-access)中了解更多关于获取 X API v2 端点访问权限的信息。
</Note>
