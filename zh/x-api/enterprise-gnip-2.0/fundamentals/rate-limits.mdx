<div id="overview">
  ## 概览
</div>

每天都有成千上万的开发者向 X API 发起请求。为管理如此庞大的请求量，我们对可发起的请求次数设定了限制。这些限制有助于我们为开发者社区提供可靠且可扩展的 API。

允许的最大请求数基于时间间隔，即某个特定的时间段或时间窗口。如果某个端点的速率限制为每 15 分钟 900 次请求，那么在任意 15 分钟内最多允许 900 次请求。
