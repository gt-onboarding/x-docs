我们的企业级 API 为依赖 X 数据的用户提供最高水准的访问能力与可靠性。非常适合当你需要在高级版之上继续扩展、获取更稳定的访问、自定义套餐，或签订更长期合同时使用。企业级 API 访问配备专属客户经理与个性化技术支持。

如果你确认 Basic 和 Pro 方案无法满足你的需求，可通过以下链接申请企业级产品的访问权限：

<div id="available-endpoints">
  ## 可用端点
</div>

| 实时 | [PowerTrack API](/zh/x-api/enterprise-gnip-2.0/powertrack-api) | 提供对实时公开帖子的完整覆盖，可使用广泛的运算符和规则进行过滤。 |
| :--- | :--- | :--- |
| [Decahose API](/zh/x-api/enterprise-gnip-2.0/fundamentals/decahose-api) | 流式传输所有公开帖子及其点赞的随机 10% 样本。 |
| :--- | :--- | :--- |
| [Account Activity API](/zh/x-api/enterprise-gnip-2.0/fundamentals/account-activity) | 订阅用户并接收多种活动事件，包括帖子、私信、点赞、关注等。 |
| :--- | :--- | :--- |
| 历史 | [Search API](/zh/x-api/enterprise-gnip-2.0/fundamentals/search-api) | 使用高级过滤工具搜索过去 30 天或公开帖子的全档案。 |
| [Historical PowerTrack API](/zh/x-api/enterprise-gnip-2.0/powertrack-api) | 通过经济高效的批量访问流程筛选公开帖子的全档案。 |
| 洞察 | [Engagement API](/zh/x-api/enterprise-gnip-2.0/fundamentals/engagement-api) | 衡量公开帖子的展示与互动量。 |
| 合规 | [Compliance Firehose API](/zh/x-api/enterprise-gnip-2.0/fundamentals/firehouse) | 流式传输所有合规事件。 |
| 使用情况 | [Usage API](/zh/x-api/enterprise-gnip-2.0/fundamentals/usage) | 提供对企业产品中活动使用情况的程序化访问。 |

<div id="important-concepts">
  ## 关键概念
</div>

<div id="gnip-console">
  ### Gnip 控制台
</div>

您可以通过 [Gnip 控制台](/zh/x-api/enterprise-gnip-2.0/fundamentals/overview)管理企业级产品和帐户访问权限。 

<div id="enrichments">
  ### 增强
</div>

我们在高级 API 的响应载荷中新增了元数据，供使用付费等级的开发者使用。请访问我们的[增强说明](/zh/x-api/enterprise-gnip-2.0/fundamentals/data-enrichments)，了解扩展 URL、投票数据和个人资料地理信息等内容。
 

<div id="advanced-filtering-tools">
  ### 高级过滤工具
</div>

Premium Search API 提供一组标准搜索端点不支持的高级运算符。许多运算符对免费沙盒用户同样可用，但使用付费层级将获得更全面、更强大的运算符集。请访问我们的[规则与过滤部分](/zh/x-api/enterprise-gnip-2.0/fundamentals/rules-filtering#getting-started-with-enterprise-rules-and-queries)了解更多信息。
