此端点允许你按标题搜索 Spaces，并按状态筛选结果。该端点可帮助你根据标题中的关键词、提及的用户或主题标签，发现正在直播或即将开始的 Spaces。

该端点接受一个或多个关键词作为查询。结果可按 Space 的状态（直播或已安排）进行筛选。默认情况下，请求将返回同时匹配指定查询的直播和已安排的 Spaces。

<Note>
  **账户设置**

  要访问这些端点，你需要：

  - 一个已批准的[开发者账户](https://developer.x.com/en/portal/petition/essential/basic-info)。
  - 使用[开发者 App](/zh/resources/fundamentals/developer-apps)中的密钥与令牌进行身份验证，该 App 位于某个[项目](/zh/resources/fundamentals/projects)内。 

  在我们的[入门指南](/zh/x-api/getting-started/getting-access)中了解更多关于访问 X API v2 端点的信息。
</Note>
