<div id="blocks-lookup">
  ### 屏蔽查询
</div>

blocks lookup 的 GET 端点可让你查看以授权用户身份已屏蔽的账号。此端点的速率限制为每位用户每 15 分钟 15 次请求。

由于你通过 blocks lookup 请求私密信息，且代表具备管理屏蔽权限的用户进行操作，你必须使用 [OAuth 1.0a 用户上下文](/zh/resources/fundamentals/authentication) 或 [OAuth 2.0（含 PKCE）的授权码模式](zh/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2)对这些端点进行身份验证，并使用与你的应用已授权用户关联的用户访问令牌。该令牌可通过 [三方（3-legged）OAuth 流程](/zh/resources/fundamentals/authentication#obtaining-access-tokens-using-3-legged-oauth-flow)（OAuth 1.0a）或 [含 PKCE 的授权码授予流程](/zh/resources/fundamentals/authentication#how-to-connect-to-endpoints-using-oauth-2-0-authorization-code-flow-with-pkce)（OAuth 2.0）生成。

<Note>
  **账户设置**

  要访问这些端点，你需要：

  - 一个已获批准的 [开发者账户](https://developer.x.com/en/portal/petition/essential/basic-info)。
  - 使用位于某个 [项目](/zh/resources/fundamentals/projects) 中的 [开发者应用](/zh/resources/fundamentals/developer-apps) 的密钥与令牌进行身份验证。

  在我们的[入门指南](/zh/x-api/getting-started/getting-access)中了解如何获取对 X API v2 端点的访问权限。
</Note>
