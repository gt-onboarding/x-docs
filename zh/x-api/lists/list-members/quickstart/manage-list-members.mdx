<div id="manage-list-members-quick-start-guide">
  ### 列表成员管理快速入门指南
</div>

<Note>
  **请注意：** 本指南假设你已完成[快速入门概览](/zh/x-api/lists/list-members#getting-started-with-the-list-members-endpoint-group)中的先决条件。
</Note>

<div id="steps-to-build-a-manage-list-member-request">
  ### 构建和管理列表成员请求的步骤
</div>

<div id="step-one-choose-the-list-endpoint-collection-from-postman">
  #### 第一步：在 Postman 中选择 List 端点集合
</div>

在你已将 X API v2 集合加载到 Postman 后，进入“List”文件夹，选择其中的“List members”文件夹，然后选择“Add a member”。
 

<div id="step-two-specify-the-user-to-add">
  #### 步骤二：指定要添加的用户
</div>

管理列表成员的端点需要两个 ID：列表的 ID 和要添加的用户 ID。你可以使用 [users lookup](/zh/x-api/users/lookup/introduction) 查找用户的 ID，传入用户名以获取 id 字段。

目标 ID 可以是任意有效的用户 ID。在 Postman 中，进入 "Params" 选项卡，在 id 路径变量的 "Value" 列中输入你的 ID。然后进入 “Body” 选项卡，将你希望添加的用户 ID 填入 user\_id 参数的值。请务必不要在任何 ID 的前后包含空格。

|     |     |     |
| :--- | :--- | :--- |
| **Key** | **Value** | **Parameter type** |
| `id` | 列表 ID | path |
| user\_id | 你希望添加为成员的目标用户 ID | body |

现在你应该会在 "Send" 按钮旁看到类似的 URL：

`https://api.x.com/2/lists/1441162269824405510/members`

步骤三：发起请求并查看响应

设置完成后，点击 "Send" 按钮，你将收到类似于以下示例的响应：

```
{
  "data": {
    "is_member": true
  }
}
```

如果返回的响应对象中键 is\_member 的值为 true，则表示你已成功将该用户添加为列表成员。

要从列表中移除成员，请在 Postman 中已加载的 X API v2 集合的 “Lists” 文件夹中选择 “Remove a member” 请求。此端点需要列表 ID 和要移除成员的用户 ID。在 “Params” 选项卡中，将列表 ID 填入 id 列的值，将待移除用户的 ID 填入 user\_id 列的值。

删除请求成功后，你将收到类似于以下示例的响应：

```
{
  "data": {
    "is_member": false
  }
}
```
