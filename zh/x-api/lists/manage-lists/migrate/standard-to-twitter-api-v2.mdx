<div id="manage-lists-standard-v11-compared-to-x-api-v2">
  ### 管理列表：标准版 v1.1 与 X API v2 对比
</div>

如果您一直在使用标准版 v1.1 的 [POST lists/create](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-create)、[POST lists/destroy](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-destroy) 和 [POST lists/update](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-update) 端点，本指南旨在帮助您了解标准版 v1.1 与 X API v2 在管理列表端点上的异同。

- **相同点**
  - 身份验证
- **不同点**
  - 端点 URL
  - 应用与项目要求
  - HTTP 方法
  - 速率限制
  - 请求参数

<div id="similarities">
  #### 相似之处
</div>

**身份验证**

两个端点版本均支持 [OAuth 1.0a 用户上下文](https://developer.x.com/content/developer-twitter/resources/fundamentals/authentication)。因此，如果您之前使用过标准 v1.1 的某个“管理列表”端点，在迁移到 X API v2 版本时，可以继续使用相同的身份验证方式。

<div id="differences">
  #### 差异
</div>

**端点 URL**

- Standard v1.1 端点：
  - POST https://api.x.com/1.1/lists/create.json
    （创建 List）
  - POST https://api.x.com/1.1/lists/destroy.json
    （删除 List）
  - POST https://api.x.com/1.1/lists/update.json
    （更新 List）
- X API v2 端点：
  - POST https://api.x.com/2/lists
    （创建 List）

  - DELETE https://api.x.com/2/lists/:id
    （删除 List）

  - PUT https://api.x.com/2/lists/:id
    （更新 List）

**速率限制**

| **Standard v1.1** | **X API v2** |
| :--- | :--- |
| /1.1/lists/create.json<br /><br />无 | /2/lists<br /><br />使用 OAuth 1.0a 用户上下文时，每 15 分钟窗口内最多 300 次请求 |
| /1.1/lists/destroy.json<br /><br />无 | /2/lists/:id<br /><br />使用 OAuth 1.0a 用户上下文时，每 15 分钟窗口内最多 300 次请求 |
| /1.1/lists/update.json<br /><br />无 | /2/lists/:id<br /><br />使用 OAuth 1.0a 用户上下文时，每 15 分钟窗口内最多 300 次请求 |

**应用与项目要求**

在对请求进行认证时，X API v2 端点要求使用与[开发者 App](/zh/resources/fundamentals/developer-apps)关联到[项目](/zh/resources/fundamentals/projects)的凭据。所有 X API v1.1 端点可使用独立 App 或与项目关联的 App 的凭据。

**请求参数**

以下 Standard v1.1 请求参数在 X API v2 中有对应项：

**创建 List**

| **Standard** | **X API v2** |
| :--- | :--- |
| name | name |
| mode | private |
| description | description |

**删除/更新 List**

| **Standard** | **X API v2** |
| :--- | :--- |
| owner\_screen\_name | 无对应项 |
| owner\_id | 无对应项 |
| list\_id | id  |
| slug | 无对应项 |

<Note>
  **请注意：** Standard v1.1 参数通过查询参数传递，而 X API v2 参数则通过正文参数（用于 POST 端点）或路径参数（用于 DELETE 和 PUT 端点）传递。
</Note>
