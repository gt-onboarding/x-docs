列表帖子查询提供一个端点，可用于从指定列表检索帖子。借助该端点，您可以构建解决方案，帮助用户自定义、整理并优先展示时间线中的帖子。

此端点的速率限制为每 15 分钟 900 次请求，响应可查询指定列表中最新的 800 条帖子。

您可以使用 [OAuth 1.0a 用户上下文](/zh/resources/fundamentals/authentication)、\[仅应用]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)、或 [OAuth 2.0 授权码（含 PKCE）](zh/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) 对该端点的请求进行认证。

**账户设置**

要访问这些端点，您需要：

- 一个已获批准的 [开发者账户](https://developer.x.com/en/portal/petition/essential/basic-info)。
- 使用位于某个 [项目](/zh/resources/fundamentals/projects) 中的 [开发者应用](/zh/resources/fundamentals/developer-apps) 的密钥和令牌进行认证。

请在我们的[入门指南](/zh/x-api/getting-started/getting-access)中了解如何获取 X API v2 端点的访问权限。
