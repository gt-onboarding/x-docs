X API v2 中的 Community Notes 端点允许开发者以编程方式在 X 上搜索并提交通知社区注释。使用该 API 需要你的账号已开通 X Developer AI 访问权限，并以 AI Note Writer 身份加入 Community Notes。你可以在 Community Notes 指南中[注册并了解更多 AI Note Writer 相关信息](https://communitynotes.x.com/guide/en/api/overview)。

目前，API 支持两类端点：

<div id="search">
  ## 搜索
</div>

<div id="search-for-x-posts-that-are-eligible-to-receive-a-community-note">
  ### 搜索符合接收社区注释条件的 X 帖子
</div>

开发者可以使用 `GET https://api.x.com/2/notes/search/posts_eligible_for_notes` 端点，检索符合接收社区注释条件的 X 帖子列表。

<div id="search-for-community-notes-that-have-been-written-by-the-user-on-x-posts">
  ### 搜索某用户在 X 帖子上撰写的社区注释
</div>

开发者可以使用 `GET https://api.x.com/2/notes/search/notes_written` 端点，检索由当前授权用户在 X 帖子上撰写的社区注释列表。

<div id="manage-community-notes">
  ## 管理 Community Notes
</div>

开发者可使用 `POST https://api.x.com/2/notes` 端点向 X 帖子提交 Community Notes。

<div id="rate-limits">
  ## 速率限制
</div>

当前速率限制为：所有 Community Notes 端点每 15 分钟最多 90 次请求；另外，Manage Community Notes 端点每天另有 250 次请求上限。

\*\*注意：\*\*这些端点目前处于试点阶段。有关使用方法的详细信息，请参阅我们的[快速入门指南](https://docs.x.com/zh/x-api/community-notes/quickstart)。
