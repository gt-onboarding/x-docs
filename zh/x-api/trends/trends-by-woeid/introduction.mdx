Trends 查询端点允许开发者根据 where-on-earth ID（WOEID）指定的位置获取 Trends。

<Note>
  **注意**：WOEID 是由 Yahoo 创建的遗留标识符，现已弃用。X API 使用该数值来标识城市和国家的趋势位置。请参考我们的遗留[博文](https://blog.x.com/engineering/en_us/a/2010/woeids-in-twitters-trends.html)或[存档数据](https://archive.org/details/geoplanet_data_7.10.0.zip0.)。
</Note>

如数据可用，许多趋势还会返回过去 24 小时的 `tweet_count`。

此端点支持应用级身份验证（app-auth），并在每个 15 分钟窗口内的速率限制为 75 次请求。

<div id="getting-started">
  ### 入门
</div>

要使用此端点，您需要一个\[bearer token]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token) from the [developer portal](https://developer.x.com/en/portal/dashboard)。获得 bearer token 后，您可以按如下所示调用用量 API：

```
      curl 'https://api.x.com/2/trends/by/woeid/26062' --header 'Authorization: Bearer XXXXX'
```

如果请求成功，您将看到如下所示的 JSON 响应：

**账户设置**

要访问这些端点，您需要：

- 一个已获批准的[开发者账户](https://developer.x.com/en/portal/petition/essential/basic-info)。
- 使用位于[项目](/zh/resources/fundamentals/projects)中的[开发者应用](/zh/resources/fundamentals/developer-apps)所提供的密钥和令牌进行身份验证。
