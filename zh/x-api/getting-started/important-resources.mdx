<div id="learn-about-whats-possible">
  ## 了解平台的可能性
</div>

从顶部导航可访问的用例、解决方案和产品页面，是快速获得平台能力宏观概览的绝佳途径。

我们还编写了多篇[教程](/zh/tutorials)，帮助你了解如何使用一种或多种平台工具来满足各种不同的用例需求。

在文档的最底层是端点与工具文档，均包含简介性内容，简要说明各工具的功能与典型用例。例如，这里是我们针对[搜索帖子](/zh/x-api/posts/search/introduction)的 X API v2 端点章节。

<div id="integrate-an-endpoint-into-your-system">
  ### 将端点集成到你的系统中
</div>

为使你与我们的端点集成更加顺畅，我们提供了一套工具、库和集成指南。最简便的方式是使用我们的[工具与库](/zh/x-api/tools-and-libraries/overview)，它们会自动处理诸如[OAuth 1.0a 用户上下文认证](/zh/resources/fundamentals/authentication#oauth-1-0a-2)、分页以及[速率限制处理](/zh/resources/fundamentals/rate-limits)等复杂功能。

如果你希望从零开始构建解决方案，我们的端点集成指南将介绍如何将我们的端点集成到你的系统中所需的功能与最佳实践。这些指南可在不同的端点与工具文档中找到，通常标记为“integrate”或“guides”。例如，这里有一篇关于[如何为过滤流构建规则](/zh/x-api/posts/filtered-stream/integrate/build-a-rule)的集成指南。

<div id="troubleshoot-an-issue">
  ### 排查问题
</div>

如遇错误或有疑问，请访问我们的[支持中心](https://developer.x.com/en/support/twitter-api)。在那里，您可以找到排障提示、常见问题解答以及其他有用资源。

<div id="keep-up-to-date-with-the-latest-releases">
  ### 及时了解最新版本
</div>

我们经常发布平台更新，通过 API 解锁新的 X 功能。除此之外，我们还计划每年至少发布一次主要版本升级，其中将包含一些不向后兼容的更改。

若要获取最新动态，请订阅我们的[论坛“公告”分类](https://devcommunity.x.com/c/announcements/22)，在 X 上关注 [@API](https://x.com/api) 和 [@XDevelopers](https://x.com/XDevelopers) 并开启通知，并通过我们的[信息更新](https://developer.x.com/en/stay-informed)页面查看各类资源。

此外，我们还推出了全新的[迁移中心](/zh/x-api/migrate/overview)，帮助你完成升级到最新 API 版本的流程。
