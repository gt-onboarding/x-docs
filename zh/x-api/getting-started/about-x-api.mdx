X API 可用于以编程方式检索和分析 X 数据，并用于围绕 X 上的对话进行构建。

多年来，X API 通过为开发者增加更多访问级别而不断演进，使其能够按需扩展访问，以增强并研究公开对话。

我们近期发布了 X API v2。X API v2 具备现代化的基础架构、全新与高级功能，并支持快速接入 [Basic access](https://developer.x.com/en/portal/products/basic)。

以下三个选项卡将介绍 X API 的不同版本与访问级别、v2 的新增内容，以及你可以通过该 API 检索、创建、删除和修改的 X 资源。

  <Tab title="What's new with v2">
    ## v2 的最新动态

    X API v2 是自 2012 年以来对 X API 最大的一次升级。它带来了大量全新和高级特性，并提供对 API 的快速且免费的访问。

    v2 提供的一些功能包括：

    ### 全新的端点

    我们为 X API v2 发布了一组全新端点。你可以在我们的 X API 端点映射指南中查看包含新增端点在内的 v2 端点完整列表。

    [访问 X API 端点映射 >](/zh/x-api/migrate/x-api-endpoint-map)

    ### 全新且更细化的数据对象

    我们对数据对象进行了现代化改造，引入了多项改进，帮助你更轻松地浏览和解析数据。

    [访问数据字典 >](/zh/x-api/fundamentals/data-dictionary)

    ### 新增参数，帮助你仅检索所需的对象和字段

    我们在数据端点中新增了 fields 和 expansions 参数，允许你请求默认返回之外的关联对象和字段。

    [了解如何使用 fields 和 expansions >](/zh/x-api/fundamentals/data-dictionary#how-to-use-fields-and-expansions)

    ### 高级指标

    通过在你的负载中直接请求包含展示次数、视频观看次数、用户资料访问和 URL 点击等公共与私有指标（其中一些区分自然与推广场景），可以更轻松地理解帖子、用户、媒体和投票的表现。

    [了解更多指标信息 >](/zh/x-api/fundamentals/metrics)

    ### 按主题筛选并识别帖子

    在使用搜索帖子或过滤流时，你现在可以使用我们的实体与上下文运算符按主题进行筛选。我们也在帖子负载中提供了这些主题，便于分析。

    [了解更多帖子注释 >](/zh/x-api/fundamentals/post-annotations)

    ### 筛选并识别属于同一回复线程的帖子

    在使用搜索帖子、过滤流和帖子查询时，更容易识别某个帖子是否属于会话线程的一部分。我们还新增了通过帖子字段 `reply_settings` 判断该帖子是否设置了会话回复权限的能力。

    [了解更多会话跟踪 >](/zh/x-api/fundamentals/conversation-id)

    ### 以及更多……

    - 高置信度垃圾信息过滤
    - 短链接将被完全解包，便于 URL 分析
    - 通过移除弃用字段并现代化标签来简化 JSON 响应对象
    - 为流式端点提供恢复和冗余功能
    - 在搜索查询中返回 100% 符合条件的公开且可用的帖子
    - 流式“规则”，可在不断开连接的情况下进行更改
    - 为过滤流和搜索提供更具表现力的查询语言
    - OpenAPI 规范，用于构建新库并更透明地跟踪变更
    - 随着平台演进以满足开发者、研究人员、企业和 X 用户的需求，更快速地为新功能和端点提供 API 支持
  </Tab>

  <Tab title="X API resources">
    ## X API 平台资源

    在 API 设计领域，“资源”指一种实体，具有相关数据、与其他资源的关系，以及一组可对其执行的操作方法。例如，帖子是一种资源，你可以通过多种工具创建、删除或检索它，例如进行全量历史搜索或实时获取。

    X API 提供在平台上创建、删除、获取或调整多种资源的能力，包括以下内容：

    | Resource            | Description                                                                                                                                                                             |
    | :------------------ | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
    | **Posts**           | 访问海量帖子以洞察公共讨论，或创建你自己的帖子参与对话。当前可用性：  - X API v2  - Enterprise  - Standard v1.1 |
    | **Users**           | 管理或查询 X 用户，以分析关系网络、了解你的受众，或促进积极的线上互动。当前可用性：  - X API v2  - Enterprise  - Standard v1.1     |
    | **Spaces**          | 查询与搜索 X Spaces 及其参与者，帮助人们发现有趣且相关的音频对话。当前可用性：  - X API v2                                    |
    | **Direct Messages** | 发送与接收私信，用于分流客服问题、发送欢迎消息，或营造积极的人际互动。当前可用性：  - Standard v1.1                        |
    | **Lists**           | 策划并管理账号列表，以追踪行业专家与重要声音，或组织你的关注对象。当前可用性：  - X API v2  - Standard v1.1               |
    | **Trends**          | 优先识别地域趋势，以定位行业动向、发现热门话题，或抢先把握最新潮流。当前可用性：  - Standard v1.1                           |
    | **Media**           | 上传媒体对象以分享创意、打造交互体验，或构建无障碍工具。当前可用性：  - Standard v1.1                               |
    | **Places**          | 搜索地点以了解你所在社区及全球正在发生的事情。当前可用性：  - Standard v1.1                                                     |
