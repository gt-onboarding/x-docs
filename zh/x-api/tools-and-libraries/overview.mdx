<div id="community-tools-and-libraries-for-v2">
  ## 面向 v2 的社区工具与库
</div>

此处列出的库由开发者社区成员构建。请注意，它们的 API 覆盖范围可能处于不同阶段。

如果你构建了自己的 X API 库或实用工具，请[告知我们](https://devcommunity.x.com/c/libraries-and-sdks/63)，我们会将其添加到此列表，便于他人发现。我们还有一些可借用的[版本徽章](https://twbadges.glitch.me/)，可用于你的 README 文件。

在寻找灵感？你可以在 GitHub 上浏览并搜索 [X](https://github.com/topics/twitter?o=desc&s=updated) 和 [X-api-v2](https://github.com/topics/twitter-api-v2?o=desc&s=updated) 主题，查找其他开发者提供的实用代码示例。

**跳转至：** [C# / .NET](#csharp)、[Dart / Flutter](#dart)、[Go](#go)、[Java](#java)、[JavaScript (Node.JS) / TypeScript](#nodejs)、[Kotlin](#kotlin)、[PHP](#php)、[PowerShell](#powershell)、[Python](#python)、[R](#r)、[Ruby](#ruby)、[Rust](#rust)、[Swift](#swift)

<div id="c-net">
  ### C# / .NET

- [**Tweetinvi**](https://github.com/linvi/tweetinvi) 一款直观的 X C# 库
- [**TwitterSharp**](https://github.com/Xwilarg/TwitterSharp) 面向 X API v2 的 C# 封装

- [**gotwtr**](https://github.com/sivchari/gotwtr) X API 库

<div id="java">
  ### java
</div>

<div id="javascript-nodejs-typescript">

</div>

- [**node-twitter-api-v2**](https://github.com/PLhery/node-twitter-api-v2) 强类型、功能完备、轻量且多功能的 Node.js X API 客户端

- [**twitter-v2**](https://github.com/HunterLarco/twitter-v2) X API 的异步客户端库
- [**tweet-json-to-html**](https://github.com/wdl/tweet-json-to-html) 将 X API v2 帖子 JSON 对象转换为 HTML

<div id="community-tools-and-libraries">
  ## 社区工具与库
</div>

以下是部分由社区维护、覆盖多种编程语言和平台的 X API 库。请注意，这些资源未必都经过 Twitter 团队的测试。

除非另有说明，此处列出的库应实现标准 API **v1.1** 的大多数功能；有关详情与额外支持，请联系各库作者。\
如果你构建了支持 X API **v2** 的库，请通过我们的[社区论坛](https://devcommunity.x.com/c/libraries-and-sdks/63)告知，我们可能会将其添加到本页面。你也可以在论坛报告此列表的任何变更。

如果你常用的编程语言缺少相应的库或工具，请通过[反馈平台](https://twitterdevfeedback.uservoice.com/)告诉我们；你也可以在该平台为想法投票，或获取灵感以构建并提交新的项目。
