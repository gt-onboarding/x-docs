“转发我的帖子”查询可让你查看自己的哪些帖子被他人分享。

这有助于用户了解哪些帖子正在被分享并与其受众产生共鸣，从而促进更有针对性的互动并推动内容策略的调整。

<div id="authentication">
  #### 认证
</div>

由于你将代表某位用户发起请求，必须使用 [OAuth 1.0a 用户上下文](/zh/resources/fundamentals/authentication) 或 [OAuth 2.0（含 PKCE）授权码](/zh/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) 对这些端点进行身份验证，并使用与该用户关联的用户访问令牌。你可以通过 [三方 OAuth 流程](https://developer.x.com/resources/fundamentals/authentication/obtaining-user-access-tokens)（OAuth 1.0a）或 [含 PKCE 的授权码授予流程](/zh/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-3)（OAuth 2.0）生成该用户访问令牌。

<div id="making-a-request">
  #### 发起请求
</div>

用户通过你的应用完成身份验证后，你即可按如下方式代表该用户调用 Retweets of Me API：

```bash
      curl 'https://api.x.com/2/users/reposts_of_me' --header 'Authorization: ••••••'
```
