- 1% 抽样流。
- 10% 抽样流，通常称为“Decahose”。

这些被称为“高体量流”，因为它们都会传输大量数据。即便是 1% 的流，每秒也可能产生数十条帖子。借助这些流，你可以识别并跟踪趋势、监测总体情绪、监控全球事件等。

这些流式端点通过持久的 HTTP GET 连接传送[帖子对象](/zh/x-api/fundamentals/data-dictionary#tweet)，并使用 [OAuth 2.0 App-Only](/zh/resources/fundamentals/authentication#oauth-2-0) 认证。使用 Essential 访问级别时，你一次只能建立一个连接。对所有访问级别而言，每 15 分钟窗口内最多可发起 50 次连接请求。

这些高体量流端点支持已编辑的帖子。端点会传送已编辑的帖子及其编辑历史，其中包含一个帖子 ID 数组。对于没有编辑历史的帖子，该数组仅包含一个 ID；对于已被编辑的帖子，该数组包含多个 ID，按升序排列以反映编辑顺序，数组最后一个元素为最新版本。要了解帖子编辑的工作方式，请参阅[帖子编辑基础](/zh/x-api/fundamentals/edit-posts)页面。

使用这些 API 之前，你必须先与我们的企业销售团队完成账户设置。

<Note>
  **账户设置**

  要访问这些端点，你需要：

  - 一个已获批准的[开发者账户](https://developer.x.com/en/portal/petition/essential/basic-info)。
  - 使用位于某个[项目](/zh/resources/fundamentals/projects)中的[开发者应用](/zh/resources/fundamentals/developer-apps)所提供的密钥和令牌进行认证。

  在我们的[入门指南](/zh/x-api/getting-started/getting-access)中了解有关获取 X API v2 端点访问的更多信息。
</Note>
