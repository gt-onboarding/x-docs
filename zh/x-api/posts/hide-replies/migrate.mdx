  ## 比较 X API 的隐藏回复端点
</div>

v2 隐藏回复端点将取代 Labs 隐藏回复端点。如果你有使用该端点 Labs 版本的代码、应用或工具，并计划迁移到较新的 X API v2 端点，本指南将对你有所帮助。

要使用新的 X API v2（包括隐藏回复端点），你需要[选择加入新的开发者门户](https://developer.x.com/en/portal/opt-in)，创建一个[项目](/zh/resources/fundamentals/projects)，并向该项目添加一个应用。随后，你可以使用与该应用关联的凭据调用隐藏回复端点。将已用于 Labs v2 隐藏回复端点的同一应用添加到项目中，可保持你的用户身份验证状态不变。

下表比较了 Labs 与较新的 X API v2 端点之间的差异：
 

| **描述** | **Labs v2** | **X API v2** |
| :--- | :--- | :--- |
| 主机域名 | https://api.x.com | https://api.x.com |
| 端点路径 | /labs/2/tweets/:id/hidden | /2/tweets/:id/hidden |
| 认证方式 | OAuth 1.0a 用户上下文 | OAuth 1.0a 用户上下文 |
| 支持的 HTTP 方法 | PUT | PUT |
| 默认请求速率限制 | 每 15 分钟 10 次请求（在所有已认证用户间共享） | 每 15 分钟 50 次请求（针对每个已认证用户） |
| 可隐藏回复 | ✔︎  | ✔︎  |
| 可取消隐藏先前隐藏的回复 | ✔︎  | ✔︎  |
| 可多次隐藏或取消隐藏回复 | ✔︎  | ✔︎  |
| 需要使用与[项目](/zh/resources/fundamentals/projects)关联的[开发者应用](/zh/resources/fundamentals/developer-apps)的凭据 |     | ✔   |
