此端点使您能够以编程方式根据自定义条件隐藏或取消隐藏回复。与主版 X 体验中的功能相同，回复会从主对话中隐藏，但仍可在单独的页面上查看。您可以使用此端点创建应用，帮助人们隐藏他们认为没有价值的回复。

隐藏回复端点使用 [OAuth 1.0a 用户上下文](/zh/resources/fundamentals/authentication#oauth-1-0a-2) 或 [OAuth 2.0 Authorization Code with PKCE](zh/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) 进行身份验证。成功后，该端点会隐藏由已认证用户发起的帖子对话中发布的一条回复。每个对话最多可隐藏 725 条帖子。

<Note>
  **账户设置**

  要访问这些端点，您需要：

  - 一个已获批准的 [开发者账户](https://developer.x.com/en/portal/petition/essential/basic-info)。
  - 使用位于某个 [项目](/zh/resources/fundamentals/projects) 内的 [开发者应用](/zh/resources/fundamentals/developer-apps) 的密钥与令牌进行身份验证。

  在我们的 [入门指南](/zh/x-api/getting-started/getting-access) 中了解如何获取 X API v2 端点的访问权限。
</Note>
