此端点可基于关键词搜索并发现 Communities。

该端点接受一个或多个关键词作为查询。默认情况下，请求会返回与指定查询匹配的公开和封闭的 Communities。

<Note>
  **账户设置**

  要访问这些端点，你需要：

  - 一个已获批准的[开发者账号](https://developer.x.com/en/portal/petition/essential/basic-info)。
  - 使用位于[项目](/zh/resources/fundamentals/projects)中的[开发者 App](/zh/resources/fundamentals/developer-apps)所提供的密钥和令牌进行身份验证。

  在我们的[入门指南](/zh/x-api/getting-started/getting-access)中了解如何获取 X API v2 端点的访问权限。
</Note>

**开始使用**

<div id="authentication">
  ### 身份验证
</div>

你可以使用 [OAuth 1.0a 用户上下文](/zh/resources/fundamentals/authentication)、\[OAuth 2.0 仅限应用]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)，或 [OAuth 2.0（含 PKCE）的授权码](zh/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) 对此端点进行身份验证。
 

<div id="making-a-request">
  ### 发起请求
</div>

你可以按以下方式调用 Search Communities 端点：

```
      curl --location 'https://api.x.com/2/communities/search?query=anime' --header 'Authorization: ••••••'
```

<div id="community-fields">
  ### 社区字段
</div>

通过指定社区字段，您可以获取与某个社区关联的附加数据。

可用的社区字段如下：

| 字段值 | 类型 | 说明 |     |
| :--- | :--- | :--- | :--- |
| created\_at | date (ISO 8601) | 社区的创建时间。 |     |
| id  | string | 社区的唯一标识符。 |     |
| name | string | 社区的名称。 |     |
| description | string | 社区描述文本（如已提供）。 |     |
| access | string | 社区的访问级别。  <br />  <br />可能为以下之一：<br /><br />\* `Public`<br />\* `Closed` |     |
| join\_policy | string | 社区的加入策略。  <br />  <br />可能为以下之一：<br /><br />\* `Open`<br />\* `RestrictedJoinRequestsDisabled`<br />\* `RestrictedJoinRequestsRequireAdminApproval`<br />\* `RestrictedJoinRequestsRequireModeratorApproval`<br />\* `SuperFollowRequired` |     |
| member\_count | integer | 加入该社区的成员数。 |     |

<div id="example-request">
  #### 示例请求
</div>

您可以按照下方示例，调用 Search Communities 端点并请求额外的社区字段：

```
      curl --location 'https://api.x.com/2/communities/search?query=anime&community.fields=access,created_at,description,id,join_policy,member_count,name' --header 'Authorization: ••••••'
```
