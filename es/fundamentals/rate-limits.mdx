title: Límites de velocidad
icon: gauge-simple-high
---

Cada día, muchos miles de desarrolladores envían solicitudes a la plataforma para desarrolladores de X. Para ayudar a gestionar el gran volumen de solicitudes, establecemos límites en la cantidad que se pueden realizar. Estos límites nos ayudan a ofrecer la API confiable y escalable en la que confía nuestra comunidad de desarrolladores.

Cada una de nuestras API aplica límites de velocidad de diferentes maneras. Para obtener más información sobre estas diferencias entre plataformas, consulta las páginas de límites de velocidad dentro de las secciones específicas de cada API:

- [API de X v2](/es/x-api/fundamentals/rate-limits)
- [X API: Enterprise](/es/x-api/enterprise-gnip-2.0/fundamentals/rate-limits)
- [API de Anuncios de X](/es/x-ads-api/fundamentals/rate-limiting)
