<div id="basic-authentication">
  ## Autenticación básica
</div>

Muchas de las APIs empresariales de X requieren el uso de autenticación básica HTTP. Para realizar correctamente una solicitud a una API que requiere autenticación básica, debes incluir una combinación válida de dirección de correo electrónico y contraseña como encabezado de autorización en cada solicitud. La combinación de correo electrónico y contraseña es la misma que usarás para acceder a la [consola de la API empresarial](https://console.gnip.com/) y puede editarse desde esa consola.

Al construir una solicitud con autenticación básica, asegúrate de agregar el encabezado HTTP Authentication: Basic con las credenciales codificadas, sobre HTTPS.

En el siguiente ejemplo de solicitud cURL, debes reemplazar `<email_address>` y `<password>` con tus credenciales antes de enviar la solicitud:

```bash
curl -v --compressed -u<email_address>:<password>
    "https://gnip-api.x.com/search/30day/accounts/<account-name>/prod/counts.json?query=from%3Axdevelopers"
```
