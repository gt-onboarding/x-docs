Las API de X procesan enormes volúmenes de datos. La forma en que garantizamos la seguridad de estos datos para desarrolladores y usuarios es mediante la autenticación. Existen varios métodos de autenticación, cada uno descrito a continuación.

La mayoría de los desarrolladores no tendrán que ocuparse de las complejidades de la autenticación, ya que las bibliotecas cliente las gestionan automáticamente.

Puede encontrar una lista de bibliotecas cliente disponibles en nuestra página de [Herramientas y bibliotecas](/es/resources/tools-and-libraries).

    [**Más información**](/es/resources/fundamentals/authentication/oauth-2-0/authorization-code)

<Note>
  **Nota:**
  Las API Keys de tu App y el App only Access Token, así como tu Access Token personal y el Access Token Secret, se pueden obtener en la sección [X developer Apps](/es/resources/fundamentals/developer-apps) del [developer portal](/es/resources/fundamentals/developer-portal).

  **Si deseas realizar solicitudes en nombre de otro usuario**, deberás generar un conjunto independiente de Access Tokens para ese usuario utilizando el [flujo OAuth de 3 patas](https://developer.x.com/resources/fundamentals/authentication/obtaining-user-access-tokens) y enviar los tokens de ese usuario con tus solicitudes de OAuth 1.0a User Context o de OAuth 2.0 User Context.
</Note>
