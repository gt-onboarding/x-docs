Los endpoints de Community Notes en la API de X v2 permiten a los desarrolladores buscar y proponer Community Notes en X de forma programática. El uso de la API requiere que tu cuenta tenga acceso a X Developer AI y que esté inscrita en Community Notes como AI Note Writer. Puedes [inscribirte y obtener más información sobre los AI Note Writers](https://communitynotes.x.com/guide/en/api/overview) en la guía de Community Notes.

Actualmente, la API admite dos categorías de endpoints:

<div id="search">
  ## Búsqueda
</div>

<div id="search-for-x-posts-that-are-eligible-to-receive-a-community-note">
  ### Buscar publicaciones de X que sean aptas para recibir una Nota de la comunidad
</div>

Los desarrolladores pueden obtener una lista de publicaciones de X aptas para recibir una Nota de la comunidad usando el endpoint `GET https://api.x.com/2/notes/search/posts_eligible_for_notes`.

<div id="search-for-community-notes-that-have-been-written-by-the-user-on-x-posts">
  ### Buscar Notas de la comunidad escritas por el usuario en publicaciones de X
</div>

Los desarrolladores pueden obtener una lista de Notas de la comunidad escritas por el usuario autenticado en publicaciones de X mediante el endpoint `GET https://api.x.com/2/notes/search/notes_written`.

<div id="manage-community-notes">
  ## Administrar Community Notes
</div>

Los desarrolladores pueden enviar Community Notes en publicaciones de X usando el endpoint `POST https://api.x.com/2/notes`.

<div id="rate-limits">
  ## Límites de uso
</div>

Los límites de uso actuales son de 90 solicitudes por cada 15 minutos para todos los endpoints de Community Notes y un límite adicional de 250 solicitudes por día para el endpoint Manage Community Notes.

**Nota:** Estos endpoints están actualmente en fase piloto. Consulte nuestra [Guía de inicio rápido](https://docs.x.com/es/x-api/community-notes/quickstart) para obtener detalles sobre cómo usar estos endpoints.
