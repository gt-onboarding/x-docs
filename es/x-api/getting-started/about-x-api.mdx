title: "Acerca de la API de X"
---

La API de X puede utilizarse para recuperar y analizar datos de X de forma programática, así como para crear soluciones orientadas a la conversación en X.

A lo largo de los años, la API de X ha evolucionado incorporando niveles adicionales de acceso, lo que permite a los desarrolladores escalar su acceso para mejorar e investigar la conversación pública.

Recientemente lanzamos la API de X v2. La API de X v2 ofrece una base moderna, funciones nuevas y avanzadas, y un proceso de incorporación ágil al [acceso Básico](https://developer.x.com/en/portal/products/basic).

Las tres pestañas siguientes explican las diferentes versiones y niveles de acceso de la API de X, qué hay de nuevo en la v2 y qué recursos de X puedes recuperar, crear, eliminar y ajustar mediante la API.

  <Tab title="What's new with v2">
    ## Qué hay de nuevo con la v2

    La API de X v2 representa la mayor actualización de la API de X desde 2012. Con ella llega un conjunto de funciones nuevas y avanzadas, así como acceso rápido y gratuito a la API.

    Algunas de las funciones disponibles con la v2 incluyen lo siguiente:

    ### Nuevos endpoints

    Hemos publicado un conjunto de endpoints totalmente nuevos en la API de X v2. Puedes ver una lista completa de los endpoints de v2, incluidos los nuevos, en nuestra guía del mapa de endpoints de la API de X.

    [Visita el mapa de endpoints de la API de X >](/es/x-api/migrate/x-api-endpoint-map)

    ### Nuevos objetos de datos y más detallados

    Hemos modernizado nuestros objetos de datos con diversas mejoras nuevas que te permitirán navegar y analizar datos más fácilmente.

    [Visita el diccionario de datos >](/es/x-api/fundamentals/data-dictionary)

    ### Nuevos parámetros para ayudarte a recuperar solo los objetos y campos que deseas

    Hemos agregado los parámetros fields y expansions a nuestros endpoints de datos para permitirte solicitar objetos y campos relacionados más allá de los que se devuelven de forma predeterminada.

    [Aprende a usar fields y expansions >](/es/x-api/fundamentals/data-dictionary#how-to-use-fields-and-expansions)

    ### Métricas avanzadas

    Comprende más fácilmente el rendimiento de publicaciones, usuarios, contenido multimedia y encuestas directamente en tu payload solicitando métricas públicas y privadas, incluidas impresiones, reproducciones de video, perfil de usuario y clics en URL, algunas de las cuales se separan en contexto orgánico y promocionado.

    [Más información sobre métricas >](/es/x-api/fundamentals/metrics)

    ### Filtra e identifica qué publicaciones contienen distintos temas

    Al usar búsqueda de publicaciones o flujo filtrado, ahora puedes filtrar por tema con nuestros operadores de entidad y de contexto. También hemos incluido estos temas dentro del payload de la publicación para ayudar con el análisis.

    [Más información sobre anotaciones de publicaciones >](/es/x-api/fundamentals/post-annotations)

    ### Filtra e identifica qué publicaciones pertenecen a un hilo de respuestas

    Facilita la identificación de una publicación como parte de un hilo de conversación al usar búsqueda de publicaciones, flujo filtrado y consulta de publicaciones. También hemos añadido la capacidad de determinar si se han configurado los ajustes de respuesta de la conversación para una publicación con el campo de la publicación `reply_settings`.

    [Más información sobre el seguimiento de conversaciones >](/es/x-api/fundamentals/conversation-id)

    ### Y mucho más...

    - Filtrado de spam de alta confianza
    - Las URL acortadas se expanden por completo para facilitar el análisis de URL
    - Objetos de respuesta JSON simplificados al eliminar campos obsoletos y modernizar etiquetas
    - Funcionalidad de recuperación y redundancia para nuestros endpoints de streaming
    - Devolución del 100% de las publicaciones públicas y disponibles que coinciden en consultas de búsqueda
    - Reglas de streaming para que puedas hacer cambios sin perder conexiones
    - Lenguaje de consulta más expresivo para flujo filtrado y búsqueda
    - Especificación OpenAPI para crear nuevas bibliotecas y realizar un seguimiento de cambios con mayor transparencia
    - Compatibilidad de la API con nuevas funciones y endpoints con mayor rapidez a medida que nuestra plataforma evoluciona para satisfacer las necesidades de desarrolladores, investigadores, empresas y personas que usan X

  <Tab title="X API resources">
    ## Recursos de la plataforma de la API de X

    En el ámbito del diseño de APIs, un recurso es una entidad con datos asociados, relaciones con otros recursos y un conjunto de métodos que operan sobre ella. Por ejemplo, una Publicación es un recurso que puedes crear, eliminar o recuperar con distintas herramientas, como buscarla en el archivo histórico o obtenerla en tiempo real.

    La API de X permite crear, eliminar, recibir o ajustar diversos recursos de la plataforma, incluidos los siguientes:

    | Recurso             | Descripción                                                                                                                                                                               |
    | :------------------ | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
    | **Publicaciones**   | Accede a millones de Publicaciones para comprender la conversación pública o crea las tuyas para participar en ella.  Disponibilidad actual:  - API de X v2  - Enterprise  - Standard v1.1 |
    | **Usuarios**        | Administra o consulta usuarios de X para analizar redes, comprender a tu audiencia o fomentar relaciones positivas en línea.  Disponibilidad actual:  - API de X v2  - Enterprise  - Standard v1.1     |
    | **Spaces**          | Consulta y busca Spaces de X y sus asistentes para ayudar a las personas a encontrar conversaciones de audio interesantes y relevantes.  Disponibilidad actual:  - API de X v2                        |
    | **Mensajes directos** | Envía y recibe mensajes directos para gestionar incidencias de clientes, enviar mensajes de bienvenida o propiciar interacciones humanas positivas.  Disponibilidad actual:  - Standard v1.1                  |
    | **Listas**          | Crea y administra listas de cuentas para seguir de cerca a expertos del sector, voces influyentes u organizar a quién sigues.  Disponibilidad actual:  - API de X v2  - Standard v1.1       |
    | **Tendencias**      | Identifica primero las tendencias geográficas para detectar movimientos del sector, descubrir temas candentes o anticiparte a la última moda.  Disponibilidad actual:  - Standard v1.1           |
    | **Multimedia**      | Sube objetos multimedia para compartir tu creatividad, crear experiencias interactivas o desarrollar herramientas de accesibilidad.  Disponibilidad actual:  - Standard v1.1                   |
    | **Lugares**         | Busca lugares para entender qué sucede en tu vecindario y en todo el mundo.  Disponibilidad actual:  - Standard v1.1                                                          |
