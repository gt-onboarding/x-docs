<div id="overview">
  ## Descripción general
</div>

Cada día, muchos miles de desarrolladores realizan solicitudes a la API de X. Para ayudar a gestionar el gran volumen de estas solicitudes, se establecen límites en la cantidad de solicitudes que se pueden realizar. Estos límites nos ayudan a ofrecer una API fiable y escalable de la que depende nuestra comunidad de desarrolladores.

El número máximo de solicitudes permitidas se basa en un intervalo de tiempo, es decir, un período o ventana de tiempo determinado. Si un endpoint tiene un límite de solicitudes de 900 por cada 15 minutos, se permiten hasta 900 solicitudes en cualquier intervalo de 15 minutos.
