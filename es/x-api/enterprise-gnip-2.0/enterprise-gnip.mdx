Nuestras APIs empresariales ofrecen el nivel más alto de acceso y fiabilidad para quienes dependen de los datos de X. Son ideales cuando necesitas escalar más allá del nivel Premium y requieres un acceso más estable, paquetes a medida o contratos a más largo plazo. El acceso a la API empresarial incluye gestores de cuenta dedicados y soporte técnico personalizado.

Si has determinado que las soluciones Basic y Pro no cubren tus necesidades, puedes solicitar acceso a los productos empresariales en el siguiente enlace:

  Solicitar acceso
</Button>

<div id="available-endpoints">
  ## Endpoints disponibles
</div>

| En tiempo real | [PowerTrack API](/es/x-api/enterprise-gnip-2.0/powertrack-api) | Ofrece cobertura completa de las publicaciones públicas en tiempo real, con filtrado mediante una amplia variedad de operadores y reglas. |
| :--- | :--- | :--- |
| [Decahose API](/es/x-api/enterprise-gnip-2.0/fundamentals/decahose-api) | Transmite una muestra aleatoria del 10% de todas las publicaciones públicas y sus Me gusta. |
| :--- | :--- | :--- |
| [Account Activity API](/es/x-api/enterprise-gnip-2.0/fundamentals/account-activity) | Suscríbete a usuarios y recibe diversas actividades, incluidas publicaciones, Mensajes directos, Me gusta, seguimientos y más. |
| :--- | :--- | :--- |
| Históricos | [Search API](/es/x-api/enterprise-gnip-2.0/fundamentals/search-api) | Busca en los últimos 30 días o en el archivo completo de publicaciones públicas utilizando herramientas de filtrado avanzadas. |
| [Historical PowerTrack API](/es/x-api/enterprise-gnip-2.0/powertrack-api) | Filtra el archivo completo de publicaciones públicas mediante un proceso económico de acceso por lotes. |
| Insights | [Engagement API](/es/x-api/enterprise-gnip-2.0/fundamentals/engagement-api) | Mide el volumen de impresiones e interacciones de las publicaciones públicas. |
| Cumplimiento | [Compliance Firehose API](/es/x-api/enterprise-gnip-2.0/fundamentals/firehouse) | Transmite todos los eventos de cumplimiento. |
| Uso | [Usage API](/es/x-api/enterprise-gnip-2.0/fundamentals/usage) | Proporciona acceso programático al consumo de actividad en los productos empresariales. |

  ## Conceptos clave
</div>

<div id="gnip-console">
  ### Gnip Console
</div>

Puede gestionar sus productos empresariales y el acceso a su cuenta a través de la [Gnip Console](/es/x-api/enterprise-gnip-2.0/fundamentals/overview). 

<div id="enrichments">
  ### Enriquecimientos
</div>

Hemos añadido metadatos al cuerpo de la respuesta de las API premium para los desarrolladores que usan un plan de pago. Obtén más información sobre las URL expandidas, los datos de encuestas y la geolocalización de perfiles en nuestra [sección de enriquecimientos](/es/x-api/enterprise-gnip-2.0/fundamentals/data-enrichments).
 

<div id="advanced-filtering-tools">
  ### Herramientas avanzadas de filtrado
</div>

La Premium Search API ofrece un conjunto avanzado de operadores que no están disponibles en el endpoint de búsqueda estándar. Muchos de estos operadores están disponibles para los usuarios del sandbox gratuito, pero obtendrás un conjunto más completo de operadores si utilizas un plan de pago. Para más información, visita nuestra [sección de reglas y filtros](/es/x-api/enterprise-gnip-2.0/fundamentals/rules-filtering#getting-started-with-enterprise-rules-and-queries).
