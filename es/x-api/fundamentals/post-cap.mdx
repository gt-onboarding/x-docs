---
title= "Límites de publicaciones"
---

La API de X v2 tiene un límite de consumo de publicaciones que restringe la cantidad de publicaciones que se pueden recuperar desde endpoints específicos mensualmente. Los límites de publicaciones se aplican a nivel de [Proyecto](/es/resources/fundamentals/projects), lo que significa que cualquier solicitud a los endpoints enumerados a continuación que use las claves y los tokens de las [Apps de desarrollador](/es/resources/fundamentals/developer-apps) dentro de ese Proyecto contará para el límite de publicaciones del Proyecto.

<div id="limited-v2-endpoints">
  ## Endpoints v2 limitados
</div>

Las publicaciones obtenidas desde cualquiera de los siguientes endpoints cuentan para este límite mensual de publicaciones:

- [Posts Lookup](/es/x-api/posts/lookup/introduction)
- [Recent search](/es/x-api/posts/counts/introduction)
- [Full-archive search](/es/x-api/posts/counts/introduction)
- [Filtered stream](/es/x-api/posts/filtered-stream)
- [User Post timeline](/es/x-api/posts/timelines)
- [User mention timeline](/es/x-api/posts/timelines)
- [Likes lookup - Posts liked by a user](/es/x-api/posts/likes)
- [Bookmarks lookup - Posts bookmarked by a user](/es/x-api/posts/bookmarks/introduction)
- [List Posts lookup](/es/x-api/lists/list-tweets/introduction)
- [Spaces Lookup](/es/x-api/spaces/lookup/introduction)

<Note>
  **Nota:** Si la misma publicación se devuelve en varias consultas durante un día, solo cuenta una vez para el límite de publicaciones; las publicaciones se deduplican diariamente.
</Note>

<div id="post-cap-volumes">
  ## Volúmenes del límite de publicaciones
</div>

El volumen del límite de publicaciones varía según tu nivel de acceso:

| Nivel de acceso | Límite mensual de publicaciones |
|:---------------|:--------------------------------|
| **Free**       | 1,500 publicaciones por mes     |
| **Basic**      | 10,000 publicaciones por mes    |
| **Pro**        | 1,000,000 publicaciones por mes |
| **Enterprise** | 50+ millones de publicaciones por mes |

Puedes consultar tu uso respecto al límite mensual de publicaciones en la [página principal del panel](https://developer.x.com/content/developer-twitter/en/portal/dashboard) del [portal para desarrolladores](/es/resources/fundamentals/developer-portal). Debajo del nombre de tu Proyecto, verás una barra de estado que muestra el uso del mes actual en relación con el límite de publicaciones, incluido el total de publicaciones obtenidas, el porcentaje utilizado y la fecha de restablecimiento de tu uso.
