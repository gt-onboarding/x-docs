title: Mapa de endpoints de la API de X

La siguiente tabla relaciona los endpoints de la API de X v2 con sus correspondientes endpoints estándar v1.1 y enterprise. Para obtener más información sobre estas versiones y niveles, visita nuestra [guía de introducción a la API de X](/es/x-api/getting-started/about-x-api).

Notarás que aún tenemos varios elementos marcados como **\[Próximamente]**. Si ves algo en esta tabla marcado como **\[Reemplazo en evaluación]** o **\[Sin reemplazo previsto]**, y te gustaría que publiquemos una versión v2 de ese endpoint, háznoslo saber a través de nuestro [foro de la comunidad](https://devcommunity.x.com/) o mediante tu gerente de cuenta enterprise.
