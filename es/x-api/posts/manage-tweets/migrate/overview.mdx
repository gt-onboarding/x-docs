<div id="comparing-x-apis-manage-posts-endpoints">
  ## Comparación de los endpoints para gestionar publicaciones de la API de X
</div>

Los endpoints de gestión de publicaciones de la v2 reemplazarán a los endpoints estándar de la v1.1 [POST statuses/update](https://developer.x.com/en/docs/twitter-api/v1/tweets/post-and-engage/api-reference/post-statuses-update) y [POST statuses/destroy/:id](https://developer.x.com/en/docs/twitter-api/v1/tweets/post-and-engage/api-reference/post-statuses-destroy-id). Si dispone de código, apps o herramientas que utilizan la versión v1.1 de los endpoints de gestión de publicaciones y está considerando migrar al endpoint más reciente de la API de X v2, este conjunto de guías es para usted.

Las siguientes tablas comparan los endpoints estándar de la v1.1 y de la API de X v2 para la gestión de publicaciones:

<div id="create-a-post">
  ### Crear una publicación
</div>

| Descripción | v1.1 estándar | API de X v2 |
| :--- | :--- | :--- |
| Métodos HTTP compatibles | POST | POST |
| Dominio del host | https://api.x.com | https://api.x.com |
| Ruta del endpoint | /1.1/statuses/update.json | /2/tweets |
| [Autenticación](/es/resources/fundamentals/authentication) | Contexto de usuario de OAuth 1.0a | Contexto de usuario de OAuth 1.0a<br /><br />Código de autorización de OAuth 2.0 con PKCE |
| [Límites de tasa](/es/x-api/fundamentals/rate-limits) predeterminados de solicitud | Ninguno<br /><br />300 solicitudes por ventana de 3 horas por usuario y por app. Compartido con el endpoint POST Retweets de v1.1. | 200 solicitudes por 15 min por usuario<br /><br />300 solicitudes por ventana de 3 horas por usuario y por app. Compartido con el endpoint POST Retweets de v2. |
| Requiere el uso de credenciales de una [App de desarrollador](/es/resources/fundamentals/developer-apps) asociada a un [Proyecto](/es/resources/fundamentals/projects) |     | ✔   |

<div id="delete-a-post">
  ### Eliminar una publicación
</div>

| Descripción | v1.1 estándar | API de X v2 |
| :--- | :--- | :--- |
| Métodos HTTP compatibles | POST | DELETE |
| Dominio del host | https://api.x.com | https://api.x.com |
| Ruta del endpoint | /1.1/statuses/destroy/:id.json | /2/tweets/:id |
| [Autenticación](/es/resources/fundamentals/authentication) | Contexto de usuario de OAuth 1.0a | Contexto de usuario de OAuth 1.0a<br /><br />OAuth 2.0 con PKCE (código de autorización) |
| [Límites de tasa](/es/x-api/fundamentals/rate-limits) de solicitudes predeterminados | Ninguno | 50 solicitudes por usuario cada 15 min |
| Requiere el uso de credenciales de una [App de desarrollador](/es/resources/fundamentals/developer-apps) asociada con un [Proyecto](/es/resources/fundamentals/projects) |     | ✔   |
