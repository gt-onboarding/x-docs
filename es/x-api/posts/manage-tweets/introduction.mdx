Crear y eliminar publicaciones con la API de X es esencial para interactuar en la conversación pública. Los nuevos endpoints para gestionar publicaciones permiten hacer eso y mucho más.

Disponemos de dos métodos para gestionar publicaciones: POST y DELETE. El método POST permite publicar encuestas, citar Tweets, configurar respuestas, publicar con datos de ubicación, publicar con medios y etiquetar usuarios, y publicar para Super Followers, entre otras funciones. Del mismo modo, el método DELETE permite eliminar una publicación específica. Con el método POST, puedes incluir [los parámetros](/es/x-api/posts/creation-of-a-post) necesarios para personalizar aún más tu solicitud.

El método “eliminar publicación” se ha actualizado para admitir publicaciones editadas. Cuando se elimina cualquier publicación en una cadena de ediciones, todas las publicaciones de esa cadena también se eliminan. Para obtener más información sobre los metadatos de edición de publicaciones, consulta la página de [Fundamentos de edición de publicaciones](/es/x-api/fundamentals/edit-posts).

Existe un límite por usuario de 200 solicitudes cada 15 minutos para el método POST. El método DELETE tiene un límite de 50 solicitudes cada 15 minutos. Además, hay un límite de 300 solicitudes cada 3 horas, incluidas las publicaciones creadas con gestionar publicaciones o gestionar Retweets.

Dado que realizas solicitudes en nombre de un usuario con los endpoints de gestionar publicaciones, debes autenticarte con [OAuth 1.0a User Context](https://en.docs/authentication/oauth-1-0a) o [OAuth 2.0 Authorization Code with PKCE](https://en.docs/authentication/oauth-2-0/authorization-code), y usar tokens de acceso de usuario asociados a un usuario que haya autorizado tu App. Para generar este token de acceso de usuario con OAuth 1.0a, puedes usar el [flujo OAuth de 3 patas](https://en.docs/authentication/oauth-2-0/bearer-tokens). Para generar un token de acceso de usuario con OAuth 2.0, puedes usar el [flujo de concesión Authorization Code with PKCE](https://en.docs/authentication/oauth-2-0/user-access-token).

<Info>
  **Configuración de la cuenta**

  Para acceder a estos endpoints, necesitarás:

  - Una [cuenta de desarrollador](https://en/portal/petition/essential/basic-info) aprobada.
  - Autenticación con las claves y tokens de una [App de desarrollador](/es/resources/fundamentals/developer-apps) que se encuentre dentro de un [Proyecto](/es/resources/fundamentals/projects).

  Obtén más información sobre cómo acceder a los endpoints de la API de X v2 en nuestra [guía de primeros pasos](/es/x-api/getting-started/getting-access).
</Info>
