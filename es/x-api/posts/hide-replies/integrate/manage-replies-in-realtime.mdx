<div id="manage-replies-in-realtime">
  ### Gestiona las respuestas en tiempo real
</div>

Con el endpoint para ocultar respuestas, puedes crear un flujo de trabajo que ayude a tus usuarios a ocultar respuestas con una probabilidad muy alta de ser irrelevantes.

Las aplicaciones útiles a menudo combinan tecnologías para aportar valor a sus usuarios. En esta página se muestra cómo ocultar respuestas utilizando la [Perspective API](https://www.perspectiveapi.com/). Esta API es un sistema de inteligencia artificial entrenado por [Jigsaw](https://jigsaw.google.com/), una unidad de Google, para detectar comentarios tóxicos. La lógica de la aplicación funcionará de la siguiente manera:

1. Solicita el permiso del usuario para leer sus publicaciones y ocultar o volver a mostrar sus respuestas.
2. Utiliza la [Account Activity API](https://developer.x.com/en/docs/x-api/v1/accounts-and-users/subscribe-account-activity/overview) para detectar respuestas entrantes.
3. Solicita a la Perspective API una “puntuación” (un número entre 0 y 1) que indique el nivel de confianza del algoritmo en que un comentario sea similar a otros comentarios tóxicos que ha visto en el pasado. (Perspective no almacena el texto enviado al servicio).
4. Llama al endpoint de ocultar respuestas si la puntuación del algoritmo es muy alta.

<div id="strive-for-transparency">
  #### Procura la transparencia
</div>

Debido a que Perspective no está entrenado con publicaciones reales, ciertas sutilezas del lenguaje pueden hacer que esta app oculte una respuesta que un usuario desea mantener visible. Independientemente de la tecnología o el enfoque que uses al diseñar tu app, haz siempre el mejor esfuerzo posible para garantizar que tus usuarios entiendan qué ha ocultado tu app y que puedan hacer cambios.

* La mejor opción es confiar siempre en el usuario y darle control total sobre sus decisiones. Esto significa que tu experiencia de usuario debe incluir controles para deshacer cualquier acción que tu app haya realizado en nombre del usuario.
* Al usar inteligencia artificial, tu app debería emplear un umbral de confianza muy alto para detectar y ocultar publicaciones.
* No todas las personas usan las mismas palabras, y tu app debe estar diseñada para evitar sesgos. Ten en cuenta las palabras resignificadas y la jerga que pueden generar falsos positivos.
* Si estás entrenando una inteligencia artificial, considera adoptar un modelo que refleje de cerca el lenguaje que se usa con frecuencia en X.
