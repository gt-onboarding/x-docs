Este endpoint te permite ocultar o volver a mostrar respuestas de forma programática según criterios que definas. Al igual que en la experiencia principal de X, las respuestas se ocultarán de la conversación principal, pero seguirán siendo visibles en una página separada. Puedes usar el endpoint para crear apps que ayuden a las personas a ocultar respuestas que no consideran valiosas.

El endpoint para ocultar respuestas usa [OAuth 1.0a User Context](/es/resources/fundamentals/authentication#oauth-1-0a-2) o [OAuth 2.0 Authorization Code with PKCE](es/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) para la autenticación. Si la solicitud se completa correctamente, el endpoint oculta una única respuesta publicada en una conversación de una publicación iniciada por un usuario autenticado. Cada conversación admite ocultar hasta 725 publicaciones.

<Note>
  **Configuración de la cuenta**

  Para acceder a estos endpoints, necesitarás:

  - Una [cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info) aprobada.
  - Autenticarte con las claves y los tokens de una [App de desarrollador](/es/resources/fundamentals/developer-apps) que se encuentra dentro de un [Project](/es/resources/fundamentals/projects).

  Obtén más información sobre cómo acceder a los endpoints de la API de X v2 en nuestra [guía de introducción](/es/x-api/getting-started/getting-access).
</Note>
