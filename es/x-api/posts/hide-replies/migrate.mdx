  ## Comparación de los endpoints para ocultar respuestas de la API de X
</div>

El endpoint de ocultar respuestas de v2 está reemplazando al endpoint de ocultar respuestas de Labs. Si tienes código, apps o herramientas que usan la versión de Labs de este endpoint y estás considerando migrar al endpoint más reciente de la API de X v2, esta guía es para ti.

Para usar la nueva API de X v2 (incluido el endpoint de ocultar respuestas), deberás [optar por el nuevo portal para desarrolladores](https://developer.x.com/en/portal/opt-in), crear un [Project](/es/resources/fundamentals/projects) y añadir una App a ese Project. Luego podrás usar las credenciales asociadas con esa App para hacer solicitudes al endpoint de ocultar respuestas. Agregar la misma App que está inscrita en el endpoint de ocultar respuestas de Labs v2 mantendrá a tus usuarios autenticados.

La siguiente tabla compara las diferencias entre Labs y el endpoint más reciente de la API de X v2:
 

| **Descripción** | **Labs v2** | **API de X v2** |
| :--- | :--- | :--- |
| Dominio host | https://api.x.com | https://api.x.com |
| Ruta del endpoint | /labs/2/tweets/:id/hidden | /2/tweets/:id/hidden |
| Autenticación | OAuth 1.0a en contexto de usuario | OAuth 1.0a en contexto de usuario |
| Métodos HTTP compatibles | PUT | PUT |
| Límites de tasa de solicitud predeterminados | 10 solicitudes por 15 minutos (compartidas entre todos los usuarios autenticados) | 50 solicitudes por 15 minutos (para cada usuario autenticado) |
| Puede ocultar respuestas | ✔︎  | ✔︎  |
| Puede dejar de ocultar una respuesta previamente oculta | ✔︎  | ✔︎  |
| Puede ocultar o dejar de ocultar respuestas múltiples veces | ✔︎  | ✔︎  |
| Requiere el uso de credenciales de una [developer App](/es/resources/fundamentals/developer-apps) asociada con un [project](/es/resources/fundamentals/projects) |     | ✔   |
