<div id="comparing-x-apis-filtered-stream-endpoints">
  ## Comparación de los endpoints de flujo filtrado de la API de X
</div>

El grupo de endpoints de flujo filtrado de la v2 está reemplazando [statuses/filter estándar v1.1](https://developer.x.com/en/docs/x-api/v1/tweets/filter-realtime/api-reference/post-statuses-filter) y la [PowerTrack API](/es/x-api/enterprise-gnip-2.0/powertrack-api). Si tiene código, apps o herramientas que utilizan una versión anterior del endpoint de flujo filtrado y está considerando migrar al nuevo endpoint de la API de X v2, esta comparación puede ayudarle a comenzar.

Consulte nuestras guías de migración más detalladas:

[Migración desde Standard v1.1 en comparación con la API de X v2](es/x-api/posts/filtered-stream#standard-v1-1-compared-to-x-api-v2)

[Migración desde la PowerTrack API a la API de X v2](/es/x-api/enterprise-gnip-2.0/powertrack-api#powertrack-operators)

La siguiente tabla compara los endpoints de transmisión filtrada que ofrece X:
