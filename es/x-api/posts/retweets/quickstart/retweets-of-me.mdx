La consulta Reposts of Me te permite ver cuáles de tus propias publicaciones han sido compartidas por otros.

Esto ofrece a los usuarios visibilidad sobre qué publicaciones se comparten y resuenan con su audiencia, lo que puede ayudar a impulsar interacciones más dirigidas y ajustes en la estrategia de contenido.

<div id="authentication">
  #### Autenticación
</div>

Dado que realiza solicitudes en nombre de un usuario, debe autenticar estos endpoints con [OAuth 1.0a (User Context)](/es/resources/fundamentals/authentication) o [OAuth 2.0 Authorization Code con PKCE](/es/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2), y utilizar los tokens de acceso de usuario asociados al usuario en cuyo nombre realiza la solicitud. Puede generar este token de acceso de usuario mediante el [flujo OAuth de 3 participantes](https://developer.x.com/resources/fundamentals/authentication/obtaining-user-access-tokens) (OAuth 1.0a) o mediante el [flujo de concesión Authorization Code con PKCE](/es/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-3) (OAuth 2.0).

<div id="making-a-request">
  #### Realizar una solicitud
</div>

Una vez que un usuario se haya autenticado con tu app, puedes llamar a la API Retweets of Me en nombre del usuario como se muestra a continuación:

```bash
      curl 'https://api.x.com/2/users/reposts_of_me' --header 'Authorization: ••••••'
```
