La API de Uso en la API de X v2 permite a los desarrolladores recuperar de forma programática el uso de su proyecto. Con este endpoint, los desarrolladores pueden llevar un registro y supervisar la cantidad de publicaciones que han obtenido durante un ciclo de facturación determinado.

Los desarrolladores pueden usar el endpoint GET para obtener el uso diario del proyecto de hasta los últimos 90 días. El uso también se puede agregar por app cliente conectada a tu proyecto.

Existe un límite de 50 solicitudes por cada 15 minutos para este endpoint GET.

<div id="getting-started">
  ### Introducción
</div>

Para usar este endpoint, necesitas un \[token bearer]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token) from the [developer portal](https://developer.x.com/en/portal/dashboard). Una vez que tengas el token bearer, puedes llamar a la API de uso como se muestra a continuación:

```
      curl 'https://api.x.com/2/usage/tweets' --header 'Authorization: Bearer XXXXX'
```

Si la solicitud se realiza correctamente, verás una respuesta JSON como la siguiente:

```
{
    "data": {
        "cap_reset_day": 10,
        "project_cap": "5000000000",
        "project_id": "1369785403853424",
        "project_usage": "43435"
    }
}
```

<Note>
  **Configuración de la cuenta**

  Para acceder a estos endpoints, necesitarás:

  - Una [cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info) aprobada.
  - Autenticación con las claves y los tokens de una [App de desarrollador](/es/resources/fundamentals/developer-apps) que esté dentro de un [Proyecto](/es/resources/fundamentals/projects). 

  Obtén más información sobre cómo acceder a los endpoints de la API de X v2 en nuestra [guía de introducción](/es/x-api/getting-started/getting-access).
</Note>
