Disponible para desarrolladores con acceso Pro o superior

El endpoint Users Search ofrece una interfaz de búsqueda sencilla basada en relevancia para cuentas de usuario públicas en X. Pruebe a consultar por interés temático, nombre completo, nombre de la empresa, ubicación u otros criterios.

Este endpoint actualmente admite [autenticación de usuario](/es/resources/fundamentals/authentication). De forma predeterminada, obtiene 100 usuarios por solicitud y puede especificar hasta 1000 usuarios por solicitud. Tiene un límite de 10.000 solicitudes por cada 24 horas (además de 300 solicitudes por cada 15 minutos).

<Note>
  **Configuración de la cuenta**

  Para acceder a estos endpoints, necesitará:

  - Una [cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info) aprobada.
  - Autenticarse usando las claves y los tokens de una [App de desarrollador](/es/resources/fundamentals/developer-apps) que se encuentra dentro de un [Proyecto](/es/resources/fundamentals/projects).

  Obtenga más información sobre cómo acceder a los endpoints de la API de X v2 en nuestra [guía de introducción](/es/x-api/getting-started/getting-access).
</Note>
