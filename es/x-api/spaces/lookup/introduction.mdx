Los endpoints de consulta por ID devuelven datos que puede usar para crear experiencias que muestren información de Spaces a sus usuarios, o para obtener detalles sobre uno o más Spaces. Puede consultar uno o varios Spaces en la misma solicitud.

También puede consultar Spaces por el ID de su creador. Este endpoint es útil para obtener Spaces de usuarios incluidos en una lista o para determinar si varios anfitriones recurrentes están en vivo ahora o tienen un Space próximo programado. Puede consultar hasta 100 usuarios en una sola solicitud.

**Configuración de la cuenta**

Para acceder a estos endpoints, necesitará:

- Una [cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info) aprobada.
- Autenticarse con las claves y tokens de una [App de desarrollador](/es/resources/fundamentals/developer-apps) que esté dentro de un [Proyecto](/es/resources/fundamentals/projects).

Obtenga más información sobre cómo acceder a los endpoints de la API de X v2 en nuestra [guía de introducción](/es/x-api/getting-started/getting-access).
