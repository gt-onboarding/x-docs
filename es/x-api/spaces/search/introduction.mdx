Este endpoint permite buscar Spaces por su título y filtrar los resultados por estado. Es útil para descubrir Spaces en directo o próximos a partir de palabras clave, usuarios mencionados o hashtags en el título.

El endpoint acepta una o varias palabras clave como consulta. Los resultados pueden filtrarse por el estado de un Space (en directo o programado). De forma predeterminada, una solicitud devolverá tanto Spaces en directo como programados que coincidan con la consulta especificada.

<Note>
  **Configuración de la cuenta**

  Para acceder a estos endpoints, necesitará:

  - Una [cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info) aprobada.
  - Autenticarse con las claves y los tokens de una [App de desarrollador](/es/resources/fundamentals/developer-apps) que se encuentre dentro de un [Project](/es/resources/fundamentals/projects).

  Obtenga más información sobre cómo obtener acceso a los endpoints de la API de X v2 en nuestra [guía de introducción](/es/x-api/getting-started/getting-access).
</Note>
