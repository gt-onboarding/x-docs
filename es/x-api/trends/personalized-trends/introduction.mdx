El endpoint Personalized Trends te permite obtener tendencias personalizadas para una cuenta de usuario autenticada.

<Note>
  **Nota:** Este endpoint solo devolverá tendencias personalizadas para cuentas de usuario suscritas a X Premium.
</Note>

<Note>
  **Configuración de la cuenta**

  Para acceder a este endpoint, necesitarás:

  - Una [cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info) aprobada.
  - Autenticarte usando las claves y los tokens de una [App de desarrollador](/es/resources/fundamentals/developer-apps) que se encuentre dentro de un [Proyecto](/es/resources/fundamentals/projects).

  Obtén más información sobre cómo acceder a los endpoints de la API de X v2 en nuestra [guía de inicio](/es/x-api/getting-started/getting-access).
</Note>

<div id="getting-started">
  ### Comenzar
</div>

<div id="authentication">
  #### Autenticación
</div>

Puede autenticar este endpoint con [OAuth 1.0a en contexto de usuario](/es/resources/fundamentals/authentication) o con [OAuth 2.0 con código de autorización y PKCE](es/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2).
 

<div id="making-a-request">
  #### Realizar una solicitud
</div>

Puedes llamar al endpoint de Tendencias personalizadas como se muestra a continuación:

`curl 'https://api.x.com/2/users/personalized_trends' --header 'Authorization: ••••••'`

Si la solicitud se realiza correctamente, verás la respuesta JSON que se muestra a continuación:

```
{
    "data": [
        {
            "category": "Bitcoin",
            "post_count": "42K posts",
            "trend_name": "MicroStrategy's $42 Billion Bitcoin Investment",
            "trending_since": "5 hours ago"
        },
        {
            "category": "Music",
            "post_count": "2K posts",
            "trend_name": "Drake's 'No Face' Music Video Release",
            "trending_since": "Trending now"
        },
        {
            "category": "Weather",
            "post_count": "1.2K posts",
            "trend_name": "Super Typhoon Leon Impacts Taiwan and Batanes",
            "trending_since": "Trending now"
        }
    ]
}
```

**Nota:** Si el usuario autenticado de la solicitud no está suscrito a X Premium, recibirá la siguiente respuesta:

```
{
    "data": [
        {
            "category": "Unknown",
            "post_count": "Unknown",
            "trending_since": ""
        }
    ]
}
```
