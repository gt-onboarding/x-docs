El endpoint de consulta de Tendencias permite a los desarrolladores obtener las Tendencias de una ubicación, especificada mediante el where-on-earth ID (WOEID).

<Note>
  **Nota**: WOEID es un identificador legado creado por Yahoo que ha sido desaprobado. La API de X utiliza el valor numérico para identificar ubicaciones de tendencias de ciudades y países. Consulte nuestra [entrada de blog de legado](https://blog.x.com/engineering/en_us/a/2010/woeids-in-twitters-trends.html) o los [datos archivados](https://archive.org/details/geoplanet_data_7.10.0.zip0.)
</Note>

El `tweet_count` de las últimas 24 horas también se devuelve para muchas tendencias si está disponible.

Este endpoint admite autenticación de app y tiene un límite de 75 solicitudes por ventana de 15 minutos.

<div id="getting-started">
  ### Primeros pasos
</div>

Para usar este endpoint, necesitas un \[token de portador]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token) de el [portal para desarrolladores](https://developer.x.com/en/portal/dashboard). Una vez que tengas el token de portador, puedes llamar a la API de uso como se muestra a continuación:

```
      curl 'https://api.x.com/2/trends/by/woeid/26062' --header 'Authorization: Bearer XXXXX'
```

Si la solicitud se realiza correctamente, deberías ver la respuesta JSON como se muestra a continuación:

**Configuración de la cuenta**

Para acceder a estos endpoints, necesitarás:

- Una [cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info) aprobada.
- Autenticación con las claves y los tokens de una [App de desarrollador](/es/resources/fundamentals/developer-apps) que esté dentro de un [Proyecto](/es/resources/fundamentals/projects).

Obtén más información sobre cómo acceder a los endpoints de la API de X v2 en nuestra [guía de primeros pasos](/es/x-api/getting-started/getting-access).
