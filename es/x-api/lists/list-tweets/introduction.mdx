La consulta de publicaciones de Listas ofrece un único endpoint para recuperar publicaciones de una Lista específica. Con este endpoint, puedes crear soluciones que permitan a los usuarios personalizar, organizar y priorizar las publicaciones que ven en su cronología.

Este endpoint tiene un límite de 900 solicitudes cada 15 minutos y la respuesta permite consultar las últimas 800 publicaciones de una Lista determinada.

Puedes usar [OAuth 1.0a User Context](/es/resources/fundamentals/authentication), \[App only]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token), o [OAuth 2.0 Authorization Code with PKCE](es/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) para autenticar tus solicitudes a este endpoint.

**Configuración de la cuenta**

Para acceder a estos endpoints, necesitarás:

- Una [cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info) aprobada.
- Autenticarte usando las claves y los tokens de una [App de desarrollador](/es/resources/fundamentals/developer-apps) que se encuentra dentro de un [Proyecto](/es/resources/fundamentals/projects).

Obtén más información sobre cómo conseguir acceso a los endpoints de la API de X v2 en nuestra [guía de introducción](/es/x-api/getting-started/getting-access).
