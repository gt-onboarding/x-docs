<div id="steps-to-build-a-manage-pinned-list-request">
  ### Pasos para crear una solicitud para gestionar una Lista fijada
</div>

**Paso uno: Elige la colección de endpoints de List en Postman**

Una vez que tengas cargada la colección de la API de X v2 en Postman, navega a la carpeta “List”, selecciona la carpeta “Pinned Lists” y luego elige "Pin a List".
 

**Paso dos: Especifica la Lista que se fijará**

Los endpoints para gestionar Listas fijadas requieren dos IDs: uno para el usuario (el usuario autenticado que fijará una Lista) y otro para la Lista de destino (la Lista que se fijará). El ID del usuario debe corresponder al ID del usuario autenticado; esto significa que debes pasar los Access Tokens asociados con ese ID al autenticar tu solicitud. En este caso, puedes especificar el ID de tu propio usuario. Puedes encontrar tu ID de dos maneras:

1. Usando el endpoint [users lookup](/es/x-api/users/lookup/introduction) por nombre de usuario, puedes pasar un nombre de usuario y recibir el campo id.
2. Consultando tu Access Token, verás que la parte numérica es tu ID de usuario.

El ID de la Lista de destino puede ser cualquier lista válida. En Postman, navega a la pestaña "Params" e ingresa el ID de usuario en la columna "Value" de la variable de ruta id. Navega a la pestaña “Body” e ingresa el ID de la Lista que deseas fijar como valor del parámetro list\_id. Asegúrate de no incluir espacios antes o después de ningún ID.

|     |     |     |
| :--- | :--- | :--- |
| **Key** | **Value** | **Parameter type** |
| `id` | El ID del usuario | path |
| list\_id | El ID de la Lista de destino a fijar | body |

Ahora deberías ver una URL similar junto al botón "Send":

```
      https://api.x.com/2/users/2244994945/pinned_lists
```

Paso tres: Realiza tu solicitud y revisa la respuesta

Una vez que tengas todo configurado, presiona el botón "Send" y recibirás una respuesta similar al siguiente ejemplo:

```
{
  "data": {
    "pinned": true
  }
}
```

Habrás fijado correctamente la Lista de destino si el objeto de respuesta devuelto contiene el valor true para la clave pinned. 

Para desfijar una Lista, selecciona la solicitud “Unpin List”, que también se encuentra en la carpeta “Lists” de la colección de la API de X v2 cargada en Postman. Este endpoint requiere el ID del usuario autenticado y el ID de la Lista que se va a desfijar. En la pestaña “Params”, ingresa el ID de usuario como valor en la columna id y el ID de la Lista que se va a desfijar como valor en la columna list\_id. 

Si la solicitud de eliminación se realiza correctamente, recibirás una respuesta similar al siguiente ejemplo:

```
{
  "data": {
    "pinned": false
  }
}
```
