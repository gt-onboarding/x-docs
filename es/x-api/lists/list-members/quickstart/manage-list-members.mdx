<div id="manage-list-members-quick-start-guide">
  ### Guía de inicio rápido para administrar miembros de listas
</div>

<Note>
  **Tenga en cuenta:** Esta guía asume que ha completado los requisitos previos del [resumen de inicio rápido](/es/x-api/lists/list-members#getting-started-with-the-list-members-endpoint-group).
</Note>

<div id="steps-to-build-a-manage-list-member-request">
  ### Pasos para crear una solicitud para gestionar miembros de una lista
</div>

<div id="step-one-choose-the-list-endpoint-collection-from-postman">
  #### Paso uno: Elige la colección del endpoint de List en Postman
</div>

Una vez que tengas cargada la colección de la API de X v2 en Postman, navega a la carpeta "List", selecciona la carpeta "List members" y luego elige "Add a member".
 

<div id="step-two-specify-the-user-to-add">
  #### Paso dos: especificar el usuario que se va a agregar
</div>

Los endpoints para administrar miembros de una Lista requieren dos IDs: el ID de la Lista y el ID del usuario que se va a agregar. Puedes encontrar el ID del usuario usando la [búsqueda de usuarios](/es/x-api/users/lookup/introduction) y pasar un nombre de usuario para recibir el campo id.

El ID de destino puede ser cualquier ID de usuario válido. En Postman, ve a la pestaña "Params" y escribe tu ID en la columna "Value" de la variable de ruta id. Ve a la pestaña “Body” e introduce el ID del usuario que deseas agregar como valor del parámetro user\_id. Asegúrate de no incluir espacios antes o después de ningún ID.

|     |     |     |
| :--- | :--- | :--- |
| **Key** | **Value** | **Parameter type** |
| `id` | El ID de la Lista | path |
| user\_id | El ID del usuario de destino que deseas agregar como miembro | body |

Ahora deberías ver una URL similar junto al botón "Send":

`https://api.x.com/2/lists/1441162269824405510/members`

Paso tres: realiza tu solicitud y revisa tu respuesta

Una vez que tengas todo configurado, presiona el botón "Send" y recibirás una respuesta similar al siguiente ejemplo:

```
{
  "data": {
    "is_member": true
  }
}
```

Si el objeto de respuesta contiene el valor true para la clave is\_member, habrás agregado correctamente al usuario como miembro de la Lista.

Para eliminar a un miembro de una Lista, selecciona la solicitud “Remove a member”, que también se encuentra en la carpeta “Lists” de la colección de la API de X v2 cargada en Postman. Este endpoint requiere el ID de la Lista y el ID de usuario del miembro que deseas eliminar. En la pestaña “Params”, introduce el ID de la Lista como valor de la columna id y el ID del usuario que se eliminará como valor de la columna user\_id.

Si la solicitud de eliminación se completa correctamente, recibirás una respuesta similar al siguiente ejemplo:

```
{
  "data": {
    "is_member": false
  }
}
```
