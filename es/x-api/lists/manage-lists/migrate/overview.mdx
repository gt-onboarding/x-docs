<div id="comparing-x-apis-lists-endpoints">
  ## Comparación de los endpoints de Listas de la API de X
</div>

Los endpoints de gestión de Listas de la v2 reemplazarán eventualmente [POST lists/create](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-create), [POST lists/destroy](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-destroy) y [POST lists/update](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-update). Si cuenta con código, apps o herramientas que usan una versión anterior de estos endpoints y está considerando migrar a la API de X v2 más reciente, esta guía es para usted.

Las siguientes tablas comparan los endpoints de Listas estándar v1.1 y de la API de X v2:

**Crear una Lista**

| Descripción | Estándar v1.1 | API de X v2 |
| :--- | :--- | :--- |
| Métodos HTTP compatibles | POST | POST |
| Dominio host | https://api.x.com | https://api.x.com |
| Ruta del endpoint | /1.1/lists/create.json | /2/lists |
| [Autenticación](/es/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context |
| [Límites de frecuencia](/es/resources/fundamentals/rate-limits) predeterminados de la solicitud | Ninguno | 300 solicitudes por 15 min (por usuario) |

**Eliminar una Lista**

| Descripción | Estándar v1.1 | API de X v2 |
| :--- | :--- | :--- |
| Métodos HTTP compatibles | POST | DELETE |
| Dominio host | https://api.x.com | https://api.x.com |
| Ruta del endpoint | /1.1/lists/destroy.json | /2/lists/:id |
| [Autenticación](/es/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context |
| [Límites de frecuencia](/es/resources/fundamentals/rate-limits) predeterminados de la solicitud | Ninguno | 300 solicitudes por 15 min (por usuario) |

**Actualizar una Lista**

| Descripción | Estándar v1.1 | API de X v2 |
| :--- | :--- | :--- |
| Métodos HTTP compatibles | POST | PUT |
| Dominio host | https://api.x.com | https://api.x.com |
| Ruta del endpoint | /1.1/lists/update.json | /2/lists/:id |
| [Autenticación](/es/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context |
| [Límites de frecuencia](/es/resources/fundamentals/rate-limits) predeterminados de la solicitud | Ninguno | 300 solicitudes por 15 min (por usuario) |

Para acceder a los endpoints de la API de X v2, debe [registrarse para obtener una cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info). Al autenticarse, debe usar claves y tokens de una [App de desarrollador](/es/resources/fundamentals/developer-apps) que esté dentro de un [Proyecto](/es/resources/fundamentals/projects).

Obtenga más información sobre cómo acceder a los endpoints de la API de X v2 en nuestra página de [primeros pasos](/es/x-api/getting-started/getting-access).
