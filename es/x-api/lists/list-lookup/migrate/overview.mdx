  ## Comparación de los endpoints de búsqueda de listas de la API de X
</div>

El grupo de endpoints de búsqueda de listas de v2 reemplazará los endpoints estándar de v1.1 [GET lists/show](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-show) y [GET lists/ownership](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-ownerships). Si tienes código, apps o herramientas que usan una de estas versiones de los endpoints de búsqueda de listas y estás considerando migrar al endpoint más reciente de la API de X v2, este conjunto de guías es para ti.

Las siguientes tablas comparan los endpoints de listas de v1.1 estándar y de la API de X v2:

**Búsqueda de lista por ID**

|     |     |     |
| :--- | :--- | :--- |
| Descripción | v1.1 estándar | API de X v2 |
| Métodos HTTP compatibles | `GET` | `GET` |
| Dominio host | `https://api.x.com` | `https://api.x.com` |
| Ruta del endpoint | `/1.1/lists/show.json` | `/2/lists/:id` |
| [Autenticación](/es/resources/fundamentals/authentication) | OAuth 1.0a User Context<br /><br />App only | OAuth 1.0a User Context<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />App only |
| [Límites de frecuencia](/es/resources/fundamentals/rate-limits) de solicitud predeterminados | 75 solicitudes por 15 min con OAuth 1.0a<br /><br />75 solicitudes por 15 min con OAuth 2.0<br /><br />75 solicitudes por 15 min con App only | 75 solicitudes por 15 min con OAuth 1.0a<br /><br />75 solicitudes por 15 min con OAuth 2.0<br /><br />75 solicitudes por 15 min con App only |

**Búsqueda de listas propiedad del usuario**

|     |     |     |
| :--- | :--- | :--- |
| Descripción | v1.1 estándar | API de X v2 |
| Métodos HTTP compatibles | `GET` | `GET` |
| Dominio host | `https://api.x.com` | `https://api.x.com` |
| Ruta del endpoint | `/1.1/lists/ownerships.json` | `/2/users/:id/owned_lists` |
| [Autenticación](/es/resources/fundamentals/authentication) | OAuth 1.0a User Context<br /><br />App only | OAuth 1.0a User Context<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />App only |
| [Límites de frecuencia](/es/resources/fundamentals/rate-limits) de solicitud predeterminados | 15 solicitudes por 15 min con OAuth 1.0a<br /><br />15 solicitudes por 15 min con App only | 15 solicitudes por 15 min con OAuth 1.0a<br /><br />15 solicitudes por 15 min con OAuth 2.0<br /><br />15 solicitudes por 15 min con App only |

Para acceder a los endpoints de la API de X v2, debes [registrarte para obtener una cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info). Al autenticarte, debes usar claves y tokens de una [App de desarrollador](/es/resources/fundamentals/developer-apps) que esté ubicada dentro de un [Proyecto](/es/resources/fundamentals/projects).

Obtén más información sobre cómo obtener acceso a los endpoints de la API de X v2 en nuestra página de [primeros pasos](/es/x-api/getting-started/getting-access).
