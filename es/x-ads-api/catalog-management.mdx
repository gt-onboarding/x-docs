<div id="overview">
  ## Descripción general
</div>

La API de Catálogo es una solución de comercio que brinda a los anunciantes la capacidad de configurar feeds de productos, agrupar productos en conjuntos y gestionar de forma integral los productos del catálogo. La API de Catálogo será una herramienta fundamental que habilita la gestión programática del catálogo y otorga a los anunciantes mayor control sobre cómo se incorporan y actualizan sus catálogos.

Un catálogo contiene todo lo relacionado con los productos y los conjuntos de productos del usuario y está asociado al identificador del usuario. Actualmente, **un usuario solo puede crear un catálogo**. 

La API de Catálogo admite 2 formas de incorporar productos:

1.  Feed programado: el usuario puede añadir una URL de feed (CSV, TSV y XML) que se recuperará periódicamente (X actualmente admite archivos de hasta 8 GB)

2.  API de productos por lotes: el usuario puede ver, crear, actualizar y eliminar atributos de productos con solicitudes por lotes (JSON)

Los comercios de X pueden crear conjuntos de productos basados en reglas de filtrado. Al hacerlo, podrán adjuntar metadatos adicionales a sus productos y crear conjuntos de productos especiales para exhibirlos mediante funciones de comercio orgánico o anuncios de productos dinámicos. La API de Catálogo y el administrador de compras de X admiten dos tipos de conjuntos de productos. 

1.  Manual: el usuario puede seleccionar hasta 50 productos y establecer un nombre y una descripción

2.  Filtro: el usuario puede añadir hasta 30 filtros para generar automáticamente conjuntos con productos que cumplan estos filtros. Los tipos de filtros incluyen precio, categoría de producto de Google, tipo de producto, inventario, precio rebajado y campos personalizados.

La API de Catálogo incluye 4 endpoints para gestionar el catálogo:

* [Product Catalogs](https://developer.x.com/en/docs/x-ads-api/catalog-management/api-reference/product-catalogs) \- Creación de un contenedor para la información de productos
* [Products](https://developer.x.com/en/docs/x-ads-api/catalog-management/api-reference/products) -  Adición, actualización, visualización y eliminación sincrónica de productos de los catálogos
* [Product Sets](https://developer.x.com/en/docs/x-ads-api/catalog-management/api-reference/product-sets) -  Clasificación de productos en grupos según la información del producto
* [Schedule Feeds](https://developer.x.com/en/docs/twitter-ads-api/catalog-management/api-reference/scheduled-feeds) \- Configuración de la incorporación asíncrona de datos de productos basados en archivos en los catálogos

<div id="prerequisites-for-the-catalog-api">
  ### Requisitos previos para la API de Catálogo
</div>

Los endpoints de la API de Catálogo están disponibles actualmente solo mediante acceso anticipado. Para solicitar acceso, póngase en contacto con su representante de X.

Debe aceptar los Términos del servicio a través de [X Shopping Manager](https://ads.x.com/shopping_manager).

Para obtener más información sobre Shopping Manager y las especificaciones de productos, consulte la [guía de especificaciones de productos](https://business.x.com/en/help/shopping-specs.html).
