---
title: Sandbox
---

El sandbox es un entorno para probar y crear tu implementación de la API de Anuncios de X.

Las campañas creadas en el sandbox no se entregarán a los usuarios y se pueden crear dinámicamente registros de prueba para objetos como Funding Instruments.

Usar el sandbox es tan sencillo como dirigirse al hostname ads-api-sandbox.x.com, pero también incluye funcionalidades únicas disponibles solo en el sandbox.

Realiza una solicitud inicial a [POST accounts](/es/x-ads-api/campaign-management#post-accounts) para crear una nueva cuenta de anunciante. La respuesta de esta llamada incluirá un account\_id que podrás usar en solicitudes posteriores. Estos endpoints permiten crear y probar con múltiples cuentas de anunciante.

<div id="sandbox-only-endpoint-list">
  ### Lista de endpoints solo para el sandbox:
</div>

- [POST accounts](/es/x-ads-api/campaign-management#post-accounts) Crear una cuenta en el sandbox
- [POST accounts/:account\_id/features](/es/x-ads-api/campaign-management#post-accounts-account-id-features) Agregar una función de cuenta a una cuenta en el sandbox
- [POST accounts/:account\_id/funding\_instruments](/es/x-ads-api/campaign-management#post-accounts-account-id-funding-instruments) Agregar un instrumento de financiación a una cuenta en el sandbox
- [DELETE accounts/:account\_id](/es/x-ads-api/campaign-management#delete-accounts-account-id) Eliminar una cuenta en el sandbox
- [DELETE accounts/:account\_id/features](/es/x-ads-api/campaign-management#delete-accounts-account-id-features) Eliminar una función de cuenta de una cuenta en el sandbox
- [DELETE accounts/:account\_id/funding\_instruments/:funding\_instrument\_id](/es/x-ads-api/campaign-management#delete-accounts-account-id-funding-instruments-funding-instrument-id) Eliminar un instrumento de financiación de una cuenta en el sandbox
