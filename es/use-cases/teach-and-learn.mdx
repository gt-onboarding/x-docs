<div id="educators">
  ## Educators
</div>

Lleve la API a su aula para enseñar a sus estudiantes los fundamentos de la programación y
valiosas habilidades de procesamiento de datos.

**Recursos:**

¿Es docente y desea usar la API de X en su curso? Esta
[guía para enseñar con la API de X](https://dev.to/suhemparack/a-guide-to-teaching-with-the-twitter-api-v2-3n08)
le ayudará a comprender la API de X, por qué la utilizan los docentes, qué tipo de
clases se imparten con la API y todo lo que necesita saber para
empezar.

**Destacado:**

Los siguientes planes de estudio se han creado y ejecutado con la API de X v2:

- [Social Media Data Mining and Analysis](https://canvas.uw.edu/courses/1434897)
  por la Dra. Tanu Mitra

- [Data Science & Society](https://dssoc.github.io/) por el Dr. Chris Bail

- [Introduction to Cultural Analytics & Python](https://melaniewalsh.github.io/Intro-Cultural-Analytics/welcome.html)
  por la Dra. Melanie Walsh

- [Oxford Spring School in Advanced Research Methods, Analysing X Data](https://github.com/cjbarrie/oxspsch)
  por el Dr. Chris Barrie
