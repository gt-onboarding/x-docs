Die Lookup-Endpunkte nach ID liefern Daten, mit denen Sie Oberflächen zur Anzeige von Spaces-Informationen für Ihre Nutzer erstellen oder Details zu einem oder mehreren Spaces abrufen können. Sie können ein oder mehrere Spaces in derselben Anfrage abfragen.

Sie können Spaces auch anhand der Creator-ID abfragen. Dieser Endpoint ist nützlich, um Spaces von Nutzern aus einer Liste abzurufen oder festzustellen, ob mehrere wiederkehrende Hosts derzeit live sind oder eine bevorstehende Space geplant haben. Bis zu 100 Nutzer können in einer einzelnen Anfrage abgefragt werden.

**Kontoeinrichtung**

Für den Zugriff auf diese Endpunkte benötigen Sie:

- Ein genehmigtes [Developer-Konto](https://developer.x.com/en/portal/petition/essential/basic-info).
- Eine Authentifizierung mit den Schlüsseln und Tokens aus einer [Developer-App](/de/resources/fundamentals/developer-apps), die innerhalb eines [Projects](/de/resources/fundamentals/projects) liegt.

Erfahren Sie mehr über den Zugriff auf die Endpunkte der X-API v2 in unserem [Einstiegsleitfaden](/de/x-api/getting-started/getting-access).
