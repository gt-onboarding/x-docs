  ## Apps, die den Endpoint zum Ausblenden von Antworten verwenden
</div>

Hier ist eine Sammlung bemerkenswerter Apps, die von Entwicklern erstellt wurden, die den Endpoint zum Ausblenden von Antworten integriert haben. Durch die Kombination ihrer Technologie mit der X-API haben sie nützliche Funktionen geschaffen, die allen Nutzern helfen, die öffentliche Konversation zu verbessern.

![Clarabridge](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/clarabridge.jpg.twimg.1920.jpg)

**Clarabridge**

Clarabridge integriert das Ausblenden von Antworten, damit Unternehmen ihren X-Feed sauber und angenehm lesbar halten können. Antworten können manuell ausgeblendet werden; zudem können Agents die Textanalysen von Clarabridge nutzen, um automatisch Obszönitäten und andere Formulierungen zu erkennen, die nicht mit der Markenrichtlinie vereinbar sind.

[Zur Website ▸](https://www.clarabridge.com/ "Zur Website ▸")

[Clarabridge auf X](https://x.com/Clarabridge "Clarabridge auf X")

![Perspective API template app](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/perspective-logo.png.twimg.1920.png)

**Perspective API-Vorlagen-App**

Perspective ist eine von Jigsaw (einer Einheit innerhalb von Alphabet) trainierte KI zur Erkennung toxischer Kommentare. Diese App erkennt eingehende Antworten einer Person und blendet anhand eines „Scores“, der angibt, wie sicher Perspective ist, dass ein Kommentar ähnlichen toxischen Kommentaren aus der Vergangenheit entspricht, automatisch eine Antwort aus. Dies ist eine von X entwickelte Open-Source-Vorlagen-App. Wir empfehlen, sie zu personalisieren, um Tools für Ihre Nutzer zu entwickeln.

[App verwenden ▸](https://quintessential-yoke.glitch.me "App verwenden ▸")

[Jigsaw auf X](https://x.com/jigsaw "Jigsaw auf X")

![Reshuffle](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 150 150%22%3E%3C/svg%3E)

**Reshuffle**

Reshuffle ist eine Plattform, die Geschäftsapplikationen mit flexiblen Skripten verbindet. Sie haben ein Integrationsskript erstellt, das Antworten erkennt und ausblendet, die von Ihnen definierte Schlüsselwörter enthalten. Um es auszuprobieren, stellen Sie sicher, dass Sie eine gültige App für „Hide replies“ aktiviert haben, und folgen Sie dann den Anweisungen im Repository.

[Zur App ▸](https://github.com/reshufflehq/reshuffle "Zur App ▸")

[Reshuffle auf X](https://x.com/reshufflehq "Reshuffle auf X")

**Unerwünschte Antworten ausblenden**

Dara Oladosu, der Entwickler hinter der beliebten App QuotedReplies, hat eine App entwickelt, die automatisch Antworten ausblendet, die einige der Kriterien erfüllen, von denen er festgestellt hat, dass sie eher missbräuchliches Verhalten erkennen lassen – darunter Antworten mit bestimmten Schlüsselwörtern, die er in der Vergangenheit stummgeschaltet hat.

**Pandaflow**

Pandaflow ist eine No-Code-Plattform zur Automatisierung von Workflows. Sie integrieren den Endpoint, sodass Nutzer programmatisch Antworten basierend auf einem von ihnen definierten individuellen Flow ausblenden können.
