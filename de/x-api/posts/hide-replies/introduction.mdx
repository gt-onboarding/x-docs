Dieser Endpoint ermöglicht es Ihnen, Antworten anhand Ihrer definierten Kriterien programmatisch zu verbergen oder wieder einzublenden. Wie in der Hauptansicht von X werden Antworten aus der Hauptunterhaltung ausgeblendet, bleiben jedoch auf einer separaten Seite sichtbar. Sie können den Endpoint verwenden, um Apps zu entwickeln, die Menschen helfen, Antworten auszublenden, die sie nicht hilfreich finden.

Der Endpoint zum Verbergen von Antworten verwendet entweder [OAuth 1.0a User Context](/de/resources/fundamentals/authentication#oauth-1-0a-2) oder [OAuth 2.0 Authorization Code mit PKCE](de/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) zur Authentifizierung. Bei Erfolg blendet der Endpoint eine einzelne Antwort aus, die in einer Post-Unterhaltung veröffentlicht wurde, die von einem authentifizierten Nutzer initiiert wurde. Jede Unterhaltung unterstützt das Ausblenden von bis zu 725 Posts.

<Note>
  **Kontoeinrichtung**

  Um auf diese Endpoints zuzugreifen, benötigen Sie:

  - Ein genehmigtes [Developer-Konto](https://developer.x.com/en/portal/petition/essential/basic-info).
  - Eine Authentifizierung mit den Schlüsseln und Token aus einer [Developer-App](/de/resources/fundamentals/developer-apps), die sich innerhalb eines [Projects](/de/resources/fundamentals/projects) befindet.

  Erfahren Sie mehr über den Zugriff auf die Endpoints der X-API v2 in unserem [Einstiegshandbuch](/de/x-api/getting-started/getting-access).
</Note>
