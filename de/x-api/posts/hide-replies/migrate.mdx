  ## Vergleich der Endpunkte zum Ausblenden von Antworten der X-API
</div>

Der v2-Endpunkt zum Ausblenden von Antworten ersetzt den Labs-Endpunkt zum Ausblenden von Antworten. Wenn Sie Code, Apps oder Tools verwenden, die die Labs-Version dieses Endpunkts nutzen und eine Migration zum neueren X-API-v2-Endpunkt erwägen, ist dieser Leitfaden für Sie.

Um die neue X-API v2 (einschließlich des Endpunkts zum Ausblenden von Antworten) zu verwenden, müssen Sie [dem neuen Developer-Portal beitreten](https://developer.x.com/en/portal/opt-in), ein [Projekt](/de/resources/fundamentals/projects) erstellen und diesem Projekt eine App hinzufügen. Anschließend können Sie die zu dieser App gehörenden Zugangsdaten verwenden, um Anfragen an den Endpunkt zum Ausblenden von Antworten zu stellen. Wenn Sie dieselbe App hinzufügen, die für den Labs-v2-Endpunkt zum Ausblenden von Antworten registriert ist, bleiben Ihre Benutzer authentifiziert.

Die folgende Tabelle stellt die Unterschiede zwischen Labs und dem neueren X-API-v2-Endpunkt gegenüber:
 

| **Beschreibung** | **Labs v2** | **X-API v2** |
| :--- | :--- | :--- |
| Host-Domain | https://api.x.com | https://api.x.com |
| Endpunktpfad | /labs/2/tweets/:id/hidden | /2/tweets/:id/hidden |
| Authentifizierung | OAuth 1.0a User context | OAuth 1.0a User context |
| Unterstützte HTTP-Methoden | PUT | PUT |
| Standardmäßige Anfrage-Limits | 10 Anfragen pro 15 Minuten (gemeinsam genutzt von allen authentifizierten Nutzern) | 50 Anfragen pro 15 Min. (für jeden authentifizierten Nutzer) |
| Kann Antworten ausblenden | ✔︎  | ✔︎  |
| Kann eine zuvor ausgeblendete Antwort wieder einblenden | ✔︎  | ✔︎  |
| Kann Antworten mehrfach aus- oder einblenden | ✔︎  | ✔︎  |
| Erfordert die Verwendung von Zugangsdaten aus einer [Developer-App](/de/resources/fundamentals/developer-apps), die einem [Projekt](/de/resources/fundamentals/projects) zugeordnet ist |     | ✔   |
