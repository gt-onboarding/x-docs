<div id="comparing-x-apis-filtered-stream-endpoints">
  ## Vergleich der gefilterten Stream-Endpunkte der X-API
</div>

Die v2-Gruppe der gefilterten Stream-Endpunkte ersetzt die [Standard-v1.1 statuses/filter](https://developer.x.com/en/docs/x-api/v1/tweets/filter-realtime/api-reference/post-statuses-filter) und die [PowerTrack-API](/de/x-api/enterprise-gnip-2.0/powertrack-api). Wenn Sie Code, Apps oder Tools verwenden, die eine ältere Version des gefilterten Stream-Endpunkts nutzen und eine Migration zum neueren X-API-v2-Endpunkt erwägen, kann Ihnen dieser Vergleich beim Einstieg helfen.

Weitere ausführliche Migrationsleitfäden:

[Migration von Standard v1.1 im Vergleich zur X-API v2](de/x-api/posts/filtered-stream#standard-v1-1-compared-to-x-api-v2)

[Migration von der PowerTrack-API zur X-API v2](/de/x-api/enterprise-gnip-2.0/powertrack-api#powertrack-operators)
