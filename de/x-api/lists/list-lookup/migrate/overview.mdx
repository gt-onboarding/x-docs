<div id="comparing-x-apis-list-lookup-endpoints">
  ## Vergleich der Endpunkte zur Listenabfrage der X-API
</div>

Die Endpunktgruppe zur Listenabfrage der v2 wird die Standard-v1.1-Endpunkte [GET lists/show](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-show) und [GET lists/ownership](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-ownerships) ersetzen. Wenn Sie Code, Apps oder Tools verwenden, die eine dieser Versionen der Listenabfrage-Endpunkte nutzen, und eine Migration zum neueren Endpunkt der X-API v2 in Betracht ziehen, sind diese Leitfäden für Sie.

Die folgenden Tabellen vergleichen die Standard-v1.1- und X-API-v2-Listenendpunkte:

**Listenabfrage nach ID**

|     |     |     |
| :--- | :--- | :--- |
| Beschreibung | Standard v1.1 | X-API v2 |
| Unterstützte HTTP-Methoden | `GET` | `GET` |
| Host-Domain | `https://api.x.com` | `https://api.x.com` |
| Endpunktpfad | `/1.1/lists/show.json` | `/2/lists/:id` |
| [Authentifizierung](/de/resources/fundamentals/authentication) | OAuth 1.0a User Context<br /><br />App-only | OAuth 1.0a User Context<br /><br />OAuth 2.0 Authorization Code mit PKCE<br /><br />App-only |
| Standardmäßige [Ratenlimits](/de/resources/fundamentals/rate-limits) für Anfragen | 75 Anfragen pro 15 Min mit OAuth 1.0a<br /><br />75 Anfragen pro 15 Min mit OAuth 2.0<br /><br />75 Anfragen pro 15 Min mit App-only | 75 Anfragen pro 15 Min mit OAuth 1.0a<br /><br />75 Anfragen pro 15 Min mit OAuth 2.0<br /><br />75 Anfragen pro 15 Min mit App-only |

**Vom Nutzer besessene Listenabfrage**

|     |     |     |
| :--- | :--- | :--- |
| Beschreibung | Standard v1.1 | X-API v2 |
| Unterstützte HTTP-Methoden | `GET` | `GET` |
| Host-Domain | `https://api.x.com` | `https://api.x.com` |
| Endpunktpfad | `/1.1/lists/ownerships.json` | `/2/users/:id/owned_lists` |
| [Authentifizierung](/de/resources/fundamentals/authentication) | OAuth 1.0a User Context<br /><br />App-only | OAuth 1.0a User Context<br /><br />OAuth 2.0 Authorization Code mit PKCE<br /><br />App-only |
| Standardmäßige [Ratenlimits](/de/resources/fundamentals/rate-limits) für Anfragen | 15 Anfragen pro 15 Min mit OAuth 1.0a<br /><br />15 Anfragen pro 15 Min mit App-only | 15 Anfragen pro 15 Min mit OAuth 1.0a<br /><br />15 Anfragen pro 15 Min mit OAuth 2.0<br /><br />15 Anfragen pro 15 Min mit App-only |

Um auf die Endpunkte der X-API v2 zuzugreifen, müssen Sie sich für ein [Developer-Konto anmelden](https://developer.x.com/en/portal/petition/essential/basic-info). Bei der Authentifizierung müssen Sie Schlüssel und Tokens aus einer [Developer-App](/de/resources/fundamentals/developer-apps) verwenden, die sich innerhalb eines [Projects](/de/resources/fundamentals/projects) befindet.

Erfahren Sie mehr über den Zugang zu den Endpunkten der X-API v2 auf unserer Seite [Erste Schritte](/de/x-api/getting-started/getting-access).
