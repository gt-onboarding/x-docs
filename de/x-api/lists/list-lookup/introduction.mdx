Es gibt ein Ratenlimit von 75 Anfragen pro 15 Minuten beim Abrufen einer bestimmten Liste nach ID und ein Limit von 15 Anfragen pro 15 Minuten beim Abrufen von benutzereigenen Listen.

Sie können [OAuth 1.0a User Context](/de/resources/fundamentals/authentication), \[App only]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token), oder [OAuth 2.0 Authorization Code with PKCE](de/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) verwenden, um Ihre Anfragen an diese Endpunkte zu authentifizieren.

<Note>
  **Konto-Einrichtung**

  Um auf diese Endpunkte zugreifen zu können, benötigen Sie:

  - Ein genehmigtes [Entwicklerkonto](https://developer.x.com/en/portal/petition/essential/basic-info).
  - Authentifizierung mit den Schlüsseln und Token einer [Entwickler-App](/de/resources/fundamentals/developer-apps), die sich innerhalb eines [Projekts](/de/resources/fundamentals/projects) befindet.

  Erfahren Sie mehr über den Zugang zu den X-API v2-Endpunkten in unserem [Erste-Schritte-Leitfaden](/de/x-api/getting-started/getting-access).
</Note>
