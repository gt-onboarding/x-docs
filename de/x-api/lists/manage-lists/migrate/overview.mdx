<div id="comparing-x-apis-lists-endpoints">
  ## Vergleich der Listen-Endpunkte der X-API
</div>

Die v2-Endpunkte zur Verwaltung von Listen werden schließlich [POST lists/create](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-create), [POST lists/destroy](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-destroy) und [POST lists/update](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-update) ersetzen. Wenn Sie Code, Apps oder Tools verwenden, die eine ältere Version dieser Endpunkte nutzen und eine Migration zur neueren X-API v2 in Betracht ziehen, ist dieser Leitfaden für Sie.

Die folgenden Tabellen vergleichen die Standard-v1.1- und X-API-v2-Listen-Endpunkte:

**Liste erstellen**

| Beschreibung | Standard v1.1 | X-API v2 |
| :--- | :--- | :--- |
| Unterstützte HTTP-Methoden | POST | POST |
| Hostdomain | https://api.x.com | https://api.x.com |
| Endpunktpfad | /1.1/lists/create.json | /2/lists |
| [Authentifizierung](/de/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context |
| Standard-[Rate Limits](/de/resources/fundamentals/rate-limits) für Anfragen | Keine | 300 Anfragen pro 15 Min. (pro Nutzer) |

**Liste löschen**

| Beschreibung | Standard v1.1 | X-API v2 |
| :--- | :--- | :--- |
| Unterstützte HTTP-Methoden | POST | DELETE |
| Hostdomain | https://api.x.com | https://api.x.com |
| Endpunktpfad | /1.1/lists/destroy.json | /2/lists/:id |
| [Authentifizierung](/de/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context |
| Standard-[Rate Limits](/de/resources/fundamentals/rate-limits) für Anfragen | Keine | 300 Anfragen pro 15 Min. (pro Nutzer) |

**Liste aktualisieren**

| Beschreibung | Standard v1.1 | X-API v2 |
| :--- | :--- | :--- |
| Unterstützte HTTP-Methoden | POST | PUT |
| Hostdomain | https://api.x.com | https://api.x.com |
| Endpunktpfad | /1.1/lists/update.json | /2/lists/:id |
| [Authentifizierung](/de/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context |
| Standard-[Rate Limits](/de/resources/fundamentals/rate-limits) für Anfragen | Keine | 300 Anfragen pro 15 Min. (pro Nutzer) |

Um auf die Endpunkte der X-API v2 zuzugreifen, müssen Sie sich für ein [Entwicklerkonto registrieren](https://developer.x.com/en/portal/petition/essential/basic-info). Bei der Authentifizierung müssen Sie Schlüssel und Token aus einer [Developer-App](/de/resources/fundamentals/developer-apps) verwenden, die einem [Projekt](/de/resources/fundamentals/projects) zugeordnet ist.

Weitere Informationen zum Zugriff auf die Endpunkte der X-API v2 finden Sie auf unserer Seite [Erste Schritte](/de/x-api/getting-started/getting-access).
