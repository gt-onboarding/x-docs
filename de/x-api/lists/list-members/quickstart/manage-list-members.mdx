<Note>

<div id="steps-to-build-a-manage-list-member-request">
  ### Schritte zum Erstellen einer Anfrage zur Verwaltung von Listenmitgliedern
</div>

<div id="step-one-choose-the-list-endpoint-collection-from-postman">
  #### Schritt eins: Wählen Sie die List-Endpunkt-Sammlung in Postman
</div>

Sobald Sie die X-API v2-Sammlung in Postman geladen haben, navigieren Sie zum Ordner „List“, wählen Sie den Unterordner „List members“ und anschließend „Add a member“.
 

<div id="step-two-specify-the-user-to-add">
  #### Schritt zwei: Den hinzuzufügenden Nutzer angeben
</div>

Die Endpunkte zum Verwalten von Listenmitgliedern benötigen zwei IDs: die ID der Liste und die ID des hinzuzufügenden Nutzers. Die Nutzer-ID können Sie über die [users lookup](/de/x-api/users/lookup/introduction) ermitteln, indem Sie einen Benutzernamen übergeben und das Feld id aus der Antwort verwenden.

Die Ziel-ID kann jede gültige Nutzer-ID sein. Navigieren Sie in Postman zur Registerkarte „Params“ und geben Sie Ihre ID in die Spalte „Value“ der id-Pfadvariable ein. Wechseln Sie zur Registerkarte „Body“ und tragen Sie die ID des Nutzers, den Sie hinzufügen möchten, als Wert für den Parameter user\_id ein. Achten Sie darauf, vor oder nach einer ID keine Leerzeichen einzufügen.

|     |     |     |
| :--- | :--- | :--- |
| **Key** | **Value** | **Parameter type** |
| `id` | Die Listen-ID | path |
| user\_id | Die Ziel-Nutzer-ID, die Sie als Mitglied hinzufügen möchten | body |

Neben der Schaltfläche „Send“ sollte nun eine ähnliche URL angezeigt werden:

`https://api.x.com/2/lists/1441162269824405510/members`

Schritt drei: Anfrage senden und Antwort prüfen

Sobald alles eingerichtet ist, klicken Sie auf „Send“. Sie erhalten eine Antwort ähnlich dem folgenden Beispiel:

```
{
  "data": {
    "is_member": true
  }
}
```

Wenn das zurückgegebene Response-Objekt für den Schlüssel is\_member den Wert true enthält, haben Sie den Nutzer erfolgreich als Mitglied der Liste hinzugefügt.

Um ein Mitglied aus einer Liste zu entfernen, wählen Sie die Anfrage „Remove a member“, die sich ebenfalls im Ordner „Lists“ der in Postman geladenen X-API-v2-Collection befindet. Dieser Endpunkt erfordert die ID der Liste und die Nutzer-ID des Mitglieds, das Sie entfernen möchten. Geben Sie auf der Registerkarte „Params“ die ID der Liste als Wert für die Spalte id und die ID des zu entfernenden Nutzers als Wert für die Spalte user\_id ein.

Bei einer erfolgreichen Löschanfrage erhalten Sie eine Antwort ähnlich dem folgenden Beispiel:

```
{
  "data": {
    "is_member": false
  }
}
```
