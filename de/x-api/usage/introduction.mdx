Die Usage-API in der X-API v2 ermöglicht es Entwicklerinnen und Entwicklern, ihre Projektnutzung programmgesteuert abzurufen. Mit diesem Endpoint können sie die Anzahl der abgerufenen Posts für einen bestimmten Abrechnungszeitraum nachverfolgen und überwachen.

Entwicklerinnen und Entwickler können den GET-Endpoint verwenden, um die tägliche Projektnutzung für bis zu die letzten 90 Tage abzurufen. Die Nutzung kann außerdem pro mit Ihrem Projekt verbundener Client-App aggregiert werden.

Für diesen GET-Endpoint gilt ein App-Rate-Limit von 50 Anfragen pro 15 Minuten.

<div id="getting-started">
  ### Erste Schritte
</div>

Um diesen Endpoint zu verwenden, benötigen Sie ein \[Bearer-Token]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token) aus dem [Developer-Portal](https://developer.x.com/en/portal/dashboard). Sobald Sie das Bearer-Token haben, können Sie die Usage-API wie unten gezeigt aufrufen:

```
      curl 'https://api.x.com/2/usage/tweets' --header 'Authorization: Bearer XXXXX'
```

Wenn die Anfrage erfolgreich ist, sehen Sie die JSON-Antwort wie unten gezeigt:

```
{
    "data": {
        "cap_reset_day": 10,
        "project_cap": "5000000000",
        "project_id": "1369785403853424",
        "project_usage": "43435"
    }
}
```

<Note>
  **Kontoeinrichtung**

  Um auf diese Endpoints zuzugreifen, benötigen Sie:

  - Ein genehmigtes [Developer-Konto](https://developer.x.com/en/portal/petition/essential/basic-info).
  - Eine Authentifizierung mit den Schlüsseln und Tokens aus einer [Developer-App](/de/resources/fundamentals/developer-apps) , die sich innerhalb eines [Projekts](/de/resources/fundamentals/projects) befindet. 

  Erfahren Sie mehr über den Zugriff auf die Endpoints der X-API v2 in unserem [Einstiegsleitfaden](/de/x-api/getting-started/getting-access).
</Note>
