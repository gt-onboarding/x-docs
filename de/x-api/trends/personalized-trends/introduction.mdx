Der Endpoint für personalisierte Trends ermöglicht es Ihnen, personalisierte Trends für ein authentifiziertes Nutzerkonto abzurufen.

<Note>
  **Hinweis:** Personalisierte Trends werden von diesem Endpoint nur für Nutzerkonten zurückgegeben, die X Premium abonniert haben.
</Note>

<Note>
  **Kontoeinrichtung**

  Um auf diesen Endpoint zuzugreifen, benötigen Sie:

  - Ein genehmigtes [Developer-Konto](https://developer.x.com/en/portal/petition/essential/basic-info).
  - Eine Authentifizierung mit den Schlüsseln und Tokens aus einer [Developer-App](/de/resources/fundamentals/developer-apps), die innerhalb eines [Projekts](/de/resources/fundamentals/projects) liegt.

  Erfahren Sie mehr über den Zugriff auf die Endpunkte der X-API v2 in unserem [Einstiegsleitfaden](/de/x-api/getting-started/getting-access).
</Note>

<div id="getting-started">
  ### Erste Schritte
</div>

<div id="authentication">
  #### Authentifizierung
</div>

Sie können diesen Endpunkt entweder mit [OAuth 1.0a User Context](/de/resources/fundamentals/authentication) oder mit [OAuth 2.0 Authorization Code mit PKCE](de/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) authentifizieren.
 

<div id="making-a-request">
  #### Eine Anfrage stellen
</div>

Sie können den Endpoint für personalisierte Trends wie unten gezeigt aufrufen:

`curl 'https://api.x.com/2/users/personalized_trends' --header 'Authorization: ••••••'`

Wenn die Anfrage erfolgreich ist, wird die JSON-Antwort wie unten gezeigt zurückgegeben:

```
{
    "data": [
        {
            "category": "Bitcoin",
            "post_count": "42K posts",
            "trend_name": "MicroStrategy's $42 Billion Bitcoin Investment",
            "trending_since": "5 hours ago"
        },
        {
            "category": "Music",
            "post_count": "2K posts",
            "trend_name": "Drake's 'No Face' Music Video Release",
            "trending_since": "Trending now"
        },
        {
            "category": "Weather",
            "post_count": "1.2K posts",
            "trend_name": "Super Typhoon Leon Impacts Taiwan and Batanes",
            "trending_since": "Trending now"
        }
    ]
}
```

**Hinweis:** Wenn der authentifizierte Nutzer der Anfrage kein X Premium abonniert hat, erhält er stattdessen die folgende Antwort:

```
{
    "data": [
        {
            "category": "Unknown",
            "post_count": "Unknown",
            "trending_since": ""
        }
    ]
}
```
