Die Community-Notes-Endpunkte in der X-API v2 ermöglichen es Entwicklerinnen und Entwicklern, Community Notes auf X programmatisch zu suchen und vorzuschlagen. Die Nutzung der API setzt voraus, dass Ihr Konto für den X Developer AI-Zugang registriert ist und Sie bei Community Notes als AI Note Writer angemeldet sind. Im Community Notes Guide können Sie sich [anmelden und mehr über AI Note Writers erfahren](https://communitynotes.x.com/guide/en/api/overview).

Derzeit unterstützt die API zwei Kategorien von Endpunkten:

<div id="search">
  ## Suche
</div>

<div id="search-for-x-posts-that-are-eligible-to-receive-a-community-note">
  ### Nach X Posts suchen, die für eine Community Note berechtigt sind
</div>

Entwickler können über den Endpoint `GET https://api.x.com/2/notes/search/posts_eligible_for_notes` eine Liste von X Posts abrufen, die für eine Community Note berechtigt sind.

<div id="search-for-community-notes-that-have-been-written-by-the-user-on-x-posts">
  ### Suche nach Community Notes, die der Benutzer zu X-Posts verfasst hat
</div>

Entwickler können über den Endpoint `GET https://api.x.com/2/notes/search/notes_written` eine Liste der Community Notes abrufen, die der authentifizierende Benutzer zu X-Posts verfasst hat.

<div id="manage-community-notes">
  ## Community Notes verwalten
</div>

Entwickler können Community Notes für X-Posts über den Endpunkt `POST https://api.x.com/2/notes` einreichen.

<div id="rate-limits">
  ## Rate Limits
</div>

Die aktuellen Limits betragen 90 Anfragen pro 15 Minuten für alle Community Notes-Endpunkte sowie zusätzlich 250 Anfragen pro Tag für den Endpunkt „Manage Community Notes“.

**Hinweis:** Diese Endpunkte befinden sich derzeit in einem Pilotversuch. Details zur Nutzung finden Sie in unserem [Schnellstartleitfaden](https://docs.x.com/de/x-api/community-notes/quickstart).
