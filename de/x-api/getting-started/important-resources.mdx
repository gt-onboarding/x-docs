<div id="learn-about-whats-possible">
  ## Erfahren Sie, was möglich ist
</div>

Unsere Use Cases, Lösungen und Produktseiten, die über die obere Navigation erreichbar sind, bieten eine hervorragende Möglichkeit, sich einen überblicksartigen Eindruck davon zu verschaffen, was auf der Plattform möglich ist.

Wir haben außerdem mehrere [Tutorials](/de/tutorials) erstellt, die Ihnen zeigen, wie Sie ein oder mehrere Plattformtools nutzen können, um verschiedene Use Cases abzudecken.

Auf der untersten Ebene unserer Dokumentation finden Sie die Endpoint- und Tool-Dokumentation. Jede davon enthält eine Einführung mit einem kurzen Überblick über die jeweilige Funktionalität und typische Anwendungsfälle. Zum Beispiel finden Sie hier unseren X-API v2-Endpoint-Bereich für [Search Posts](/de/x-api/posts/search/introduction).

<div id="integrate-an-endpoint-into-your-system">
  ### Einen Endpoint in Ihr System integrieren
</div>

Um die Integration mit unseren Endpoints nahtlos zu gestalten, haben wir eine Reihe von Tools, Bibliotheken und Integrationsleitfäden bereitgestellt. Der einfachste Weg zur Integration ist die Nutzung unserer [Tools und Bibliotheken](/de/x-api/tools-and-libraries/overview), die komplexe Funktionen wie [OAuth 1.0a User Context-Authentifizierung](/de/resources/fundamentals/authentication#oauth-1-0a-2), Paginierung und [Umgang mit Rate Limits](/de/resources/fundamentals/rate-limits) automatisch übernehmen.

Wenn Sie lieber eine Lösung von Grund auf entwickeln möchten, beschreiben unsere Leitfäden zur Endpoint-Integration die Funktionalität und Best Practices für die Integration unserer Endpoints in Ihr System. Diese Leitfäden finden Sie in der Dokumentation zu den verschiedenen Endpoints und Tools, gekennzeichnet als „integrate“ oder einfach „guides“. Ein Beispiel ist dieser Leitfaden dazu, [wie Sie eine Regel für den Gefilterten Stream erstellen](/de/x-api/posts/filtered-stream/integrate/build-a-rule).

<div id="troubleshoot-an-issue">
  ### Probleme beheben
</div>

Wenn ein Fehler auftritt oder Sie eine Frage haben, besuchen Sie bitte unseren [Support-Hub](https://developer.x.com/en/support/twitter-api). Dort finden Sie Tipps zur Fehlerbehebung, Antworten auf häufig gestellte Fragen und weitere hilfreiche Ressourcen.

<div id="keep-up-to-date-with-the-latest-releases">
  ### Bleiben Sie über die neuesten Releases auf dem Laufenden
</div>

Wir veröffentlichen regelmäßig Updates für die Plattform, um neue X‑Funktionen über die API bereitzustellen. Außerdem planen wir, mindestens einmal pro Jahr eine neue Hauptversion zu veröffentlichen, die einige inkompatible Änderungen enthält.

Um die neuesten Updates zu erhalten, abonnieren Sie bitte die [Kategorie „Announcements“ in unserem Forum](https://devcommunity.x.com/c/announcements/22), folgen Sie [@API](https://x.com/api) und [@XDevelopers](https://x.com/XDevelopers) auf X und aktivieren Sie Benachrichtigungen, und sehen Sie sich unsere verschiedenen Ressourcen auf der Seite [Auf dem Laufenden bleiben](https://developer.x.com/en/stay-informed) an.

Zusätzlich haben wir ein neues [Migrations-Hub](/de/x-api/migrate/overview) erstellt, das Sie durch die Aktualisierung auf die neueste Version der API führt.
