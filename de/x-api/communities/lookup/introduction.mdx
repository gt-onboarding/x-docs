**Erste Schritte**

<div id="authentication">
  ### Authentifizierung
</div>

Sie können diesen Endpunkt mit [OAuth 1.0a User Context](/de/resources/fundamentals/authentication), \[OAuth 2.0 App-Only]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token) oder [OAuth 2.0 mit PKCE](de/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) authentifizieren.
 

<div id="making-a-request">
  ### Eine Anfrage senden
</div>

Sie können den Communities-Lookup-Endpoint aufrufen, indem Sie eine ID angeben, wie unten gezeigt (hier mit der ID `1758747817642700922`):

```
      curl --location 'https://api.x.com/2/communities/1758747817642700922' --header 'Authorization: ••••••'
```

Wenn die Anfrage erfolgreich ist, sehen Sie die JSON-Antwort wie unten:

```
{
    "data": {
        "access": "Public",
        "description": "Welcome to the Anime Community! Where anime fans gather to share their favorite shows and discuss everything anime-related.",
        "join_policy": "Open",
        "name": "Anime Community"
    }
}
```

<div id="community-fields">
  ### Community-Felder
</div>

Sie können zusätzliche Daten abrufen, die einer Community zugeordnet sind, indem Sie Community-Felder angeben.

Die folgenden Community-Felder sind verfügbar:

| Feldwert | Typ | Beschreibung |     |
| :--- | :--- | :--- | :--- |
| created\_at | Datum (ISO 8601) | Erstellungszeitpunkt der Community. |     |
| id  | string | Der eindeutige Bezeichner der Community. |     |
| name | string | Der Name der Community. |     |
| description | string | Der Beschreibungstext der Community, falls vorhanden. |     |
| access | string | Die Zugriffsebene der Community.  <br />  <br />Mögliche Werte:<br /><br />\* `Public`<br />\* `Closed` |     |
| join\_policy | string | Die Beitrittsrichtlinie der Community.  <br />  <br />Mögliche Werte:<br /><br />\* `Open`<br />\* `RestrictedJoinRequestsDisabled`<br />\* `RestrictedJoinRequestsRequireAdminApproval`<br />\* `RestrictedJoinRequestsRequireModeratorApproval`<br />\* `SuperFollowRequired` |     |
| member\_count | integer | Die Zahl der Mitglieder, die der Community beigetreten sind. |     |

<div id="example-request">
  #### Beispielanfrage
</div>

Sie können den Communities-Lookup-Endpunkt mit zusätzlichen Community-Feldern wie unten gezeigt aufrufen (mit der ID `1758747817642700922`):

```
curl --location 'https://api.x.com/2/communities/1758747817642700922?community.fields=access,created_at,description,id,join_policy,member_count,name' --header 'Authorization: ••••••'
```

**Beispielantwort**

Wenn die Anfrage erfolgreich ist, sehen Sie die JSON-Antwort wie unten:

```
{
    "data": {
        "description": "Welcome to the Anime Community! Where anime fans gather to share their favorite shows and discuss everything anime-related.",
        "access": "Public",
        "member_count": 40369,
        "created_at": "2024-02-17T06:58:50.000Z",
        "join_policy": "Open",
        "name": "Anime Community",
        "id": "Q29tbXVuaXR5OjE3NTg3NDc4MTc2NDI3MDA5MjI="
    }
}
```
