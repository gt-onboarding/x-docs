Verfügbar für Entwickler mit Pro-Zugang und höher

Der Endpoint Users Search bietet eine einfache, relevanzbasierte Suchoberfläche für öffentliche Nutzerkonten auf X. Probieren Sie Suchanfragen nach thematischen Interessen, vollständigem Namen, Firmenname, Standort oder anderen Kriterien aus.

Dieser Endpoint unterstützt derzeit [User Auth](/de/resources/fundamentals/authentication). Standardmäßig erhalten Sie 100 Nutzer pro Anfrage, und Sie können bis zu 1.000 Nutzer pro Anfrage anfordern. Es gilt ein Limit von 10.000 Anfragen pro 24 Stunden (zusätzlich zu 300 Anfragen pro 15 Minuten).

<Note>
  **Kontoeinrichtung**

  Um auf diese Endpoints zuzugreifen, benötigen Sie:

  - Ein genehmigtes [Developer-Konto](https://developer.x.com/en/portal/petition/essential/basic-info).
  - Eine Authentifizierung mit den Schlüsseln und Token aus einer [Developer App](/de/resources/fundamentals/developer-apps), die sich innerhalb eines [Project](/de/resources/fundamentals/projects) befindet.

  Erfahren Sie mehr über den Zugriff auf die Endpoints der X-API v2 in unserem [Einstiegshandbuch](/de/x-api/getting-started/getting-access).
</Note>
