<div id="overview">
  ## Übersicht
</div>

Jeden Tag stellen viele Tausend Entwickler Anfragen an die X-API. Um das enorme Volumen dieser Anfragen zu steuern, werden Beschränkungen für die Anzahl der zulässigen Anfragen festgelegt. Diese Limits helfen uns, die zuverlässige und skalierbare API bereitzustellen, auf die sich unsere Entwicklercommunity verlässt. 

Die maximal zulässige Anzahl von Anfragen richtet sich nach einem Zeitintervall, also einem bestimmten Zeitraum oder Zeitfenster. Wenn ein Endpoint ein Rate-Limit von 900 Anfragen/15 Minuten hat, sind bis zu 900 Anfragen in einem beliebigen 15-Minuten-Intervall erlaubt.
