Unsere Enterprise-APIs bieten die höchste Stufe an Zugriff und Zuverlässigkeit für alle, die auf X-Daten angewiesen sind. Ideal, wenn Sie über Premium hinaus skalieren und verlässlicheren Zugriff, maßgeschneiderte Pakete oder längerfristige Verträge benötigen. Der Zugang zur Enterprise-API umfasst dedizierte Account Manager und persönlichen technischen Support.

Wenn Sie festgestellt haben, dass die Basic- und Pro-Lösungen nicht ausreichen, um Ihre Anforderungen zu erfüllen, können Sie über den folgenden Link Zugriff auf die Enterprise-Produkte beantragen:

  Zugriff beantragen

<div id="available-endpoints">
  ## Verfügbare Endpunkte
</div>

| Echtzeit | [PowerTrack API](/de/x-api/enterprise-gnip-2.0/powertrack-api) | Bietet vollständige Abdeckung öffentlicher Posts in Echtzeit, filterbar mit einer Vielzahl von Operatoren und Regeln. |
| :--- | :--- | :--- |
| [Decahose API](/de/x-api/enterprise-gnip-2.0/fundamentals/decahose-api) | Streamen Sie eine zufällige Stichprobe von 10 % aller öffentlichen Posts und ihrer Likes. |
| :--- | :--- | :--- |
| [Account Activity API](/de/x-api/enterprise-gnip-2.0/fundamentals/account-activity) | Abonnieren Sie Nutzer und erhalten Sie eine Vielzahl unterschiedlicher Aktivitäten, darunter Posts, Direct Messages, Likes, Follows und mehr. |
| :--- | :--- | :--- |
| Historisch | [Search API](/de/x-api/enterprise-gnip-2.0/fundamentals/search-api) | Durchsuchen Sie entweder die letzten 30 Tage oder das vollständige Archiv öffentlicher Posts mit erweiterten Filterwerkzeugen. |
| [Historical PowerTrack API](/de/x-api/enterprise-gnip-2.0/powertrack-api) | Filtern Sie das vollständige Archiv öffentlicher Posts über einen kosteneffizienten Batch-Zugriffsprozess. |
| Einblicke | [Engagement API](/de/x-api/enterprise-gnip-2.0/fundamentals/engagement-api) | Messen Sie die Anzahl von Impressions und Engagements für öffentliche Posts. |
| Compliance | [Compliance Firehose API](/de/x-api/enterprise-gnip-2.0/fundamentals/firehouse) | Streamen Sie alle Compliance-Ereignisse. |
| Nutzung | [Usage API](/de/x-api/enterprise-gnip-2.0/fundamentals/usage) | Bietet programmatischen Zugang zur Nutzung von Aktivitäten über Enterprise-Produkte hinweg. |

  ## Wichtige Konzepte

<div id="gnip-console">
  ### Gnip-Konsole
</div>

Über die [Gnip-Konsole](/de/x-api/enterprise-gnip-2.0/fundamentals/overview) können Sie Ihre Enterprise-Produkte und den Zugriff auf Konten verwalten. 

<div id="enrichments">
  ### Enrichments
</div>

Wir haben die Antwortnutzlast der Premium-APIs für Entwicklerinnen und Entwickler mit kostenpflichtigem Tarif um Metadaten ergänzt. Erfahren Sie mehr über expandierte URLs, Umfragedaten und Profil-Geodaten in unserem [Enrichments-Abschnitt](/de/x-api/enterprise-gnip-2.0/fundamentals/data-enrichments).
 

<div id="advanced-filtering-tools">
  ### Erweiterte Filtertools
</div>

Die Premium Search API bietet einen erweiterten Satz an Operatoren, die beim Standard-Suchendpoint nicht verfügbar sind. Viele dieser Operatoren stehen auch kostenlosen Sandbox-Nutzern zur Verfügung, aber mit einer kostenpflichtigen Stufe erhalten Sie einen noch umfassenderen Satz an Operatoren. Weitere Informationen finden Sie im [Abschnitt zu Regeln und Filtern](/de/x-api/enterprise-gnip-2.0/fundamentals/rules-filtering#getting-started-with-enterprise-rules-and-queries).
