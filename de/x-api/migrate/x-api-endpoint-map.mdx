Die folgende Tabelle ordnet die Endpunkte der X-API v2 den entsprechenden Standard-v1.1- und Enterprise-Endpunkten zu. Weitere Informationen zu den einzelnen Versionen und Zugangsstufen finden Sie in unserem [Leitfaden für den Einstieg in die X-API](/de/x-api/getting-started/about-x-api).

Sie werden feststellen, dass einige Einträge noch mit **\[Coming Soon]** gekennzeichnet sind. Wenn Ihnen in dieser Tabelle etwas auffällt, das mit **\[Replacement Under Consideration]** oder **\[No Replacement Planned]** markiert ist und Sie sich eine v2-Version dieses Endpunkts wünschen, lassen Sie es uns bitte über unser [Community-Forum](https://devcommunity.x.com/) oder Ihre Enterprise-Ansprechpartnerin bzw. Ihren -Ansprechpartner wissen.
