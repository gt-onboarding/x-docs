---
title: Währung
---

Der Typ einer Währung wird gemäß [ISO-4217](http://en.wikipedia.org/wiki/ISO_4217) identifiziert. Dies ist eine dreibuchstabige Zeichenfolge wie `USD` oder `EUR`. Der Wert einer Währung wird in Micros dargestellt. Für USD entspricht $5.50 der Kodierung 5.50\*1e6, also 5.500.000. Um einen „Ganzzahlwert“ darzustellen, müssen Sie den lokalen Micro-Betrag für alle Währungen mit 1e6 (1\_000\_000) multiplizieren.

Die X-Ads-API erkennt die offiziellen Währungen aller Länder, in denen das X-SMB-Ads-Produkt verfügbar ist. Eine vollständige Liste dieser Länder finden Sie [hier](https://business.x.com/en/help/overview/about-eligibility-for-twitter-ads.html).

Die meisten Währungen haben eine „Genauigkeit“ von zwei (z. B. USD: $1.23).

- `AED`, `VEF`, `PKR`, `IDR`, `HKD`, `QAR`, `PHP`, `NZD`, `INR`, `EUR`, `ARS`, `THB`, `MXN`, `EGP`, `DZD`, `SAR`, `LBP`, `COP`, `TRY`, `SGD`, `MAD`, `AUD`, `USD`, `MYR`, `CAD`, `BRL`, `GBP`.

Die folgenden Währungen verwenden keine Dezimalstellen:

- `VND`, `KRW`, `CLP`, `JPY`

Und diese verwenden drei Dezimalstellen:

- `KWD`, `IQD`, `BHD`, `TND`

Die Verwendung von Währungen erfolgt über [Zahlungsmittel](/de/x-ads-api/introduction).
