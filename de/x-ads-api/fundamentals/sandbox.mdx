---
title: Sandbox
---

Die Sandbox ist eine Umgebung, in der Sie Ihre Implementierung der X-Ads-API testen und entwickeln können.

In der Sandbox erstellte Kampagnen werden nicht an Nutzer ausgeliefert, und Testdatensätze für Objekte wie Funding Instruments können dynamisch erstellt werden.

Die Nutzung der Sandbox ist so einfach wie das Ansprechen des entsprechenden Hostnamens ads-api-sandbox.x.com, sie bietet jedoch auch einige Funktionen, die ausschließlich in der Sandbox verfügbar sind.

Senden Sie zunächst eine Anfrage an [POST accounts](/de/x-ads-api/campaign-management#post-accounts), um ein neues Advertiser-Konto zu erstellen. Die Antwort enthält eine account\_id, die für nachfolgende Anfragen verwendet werden kann. Über diese Endpunkte können Sie mehrere Advertiser-Konten erstellen und dagegen testen.

<div id="sandbox-only-endpoint-list">
  ### Nur-Sandbox-Endpunktliste:
</div>

- [POST accounts](/de/x-ads-api/campaign-management#post-accounts) Ein Konto in der Sandbox erstellen
- [POST accounts/:account\_id/features](/de/x-ads-api/campaign-management#post-accounts-account-id-features) Eine Kontofunktion zu einem Konto in der Sandbox hinzufügen
- [POST accounts/:account\_id/funding\_instruments](/de/x-ads-api/campaign-management#post-accounts-account-id-funding-instruments) Ein Finanzierungsinstrument zu einem Konto in der Sandbox hinzufügen
- [DELETE accounts/:account\_id](/de/x-ads-api/campaign-management#delete-accounts-account-id) Ein Konto in der Sandbox löschen
- [DELETE accounts/:account\_id/features](/de/x-ads-api/campaign-management#delete-accounts-account-id-features) Eine Kontofunktion aus einem Konto in der Sandbox löschen
- [DELETE accounts/:account\_id/funding\_instruments/:funding\_instrument\_id](/de/x-ads-api/campaign-management#delete-accounts-account-id-funding-instruments-funding-instrument-id) Ein Finanzierungsinstrument aus einem Konto in der Sandbox löschen
