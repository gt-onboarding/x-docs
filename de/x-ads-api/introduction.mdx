<div id="connect-to-the-x-ads-platform">
  ## Verbindung zur X-Ads-Plattform
</div>

Die X-Ads-API verbindet Entwickler mit der Werbeplattform von X, um Lösungen zu bauen, die den Anforderungen der Werbetreibenden auf X weltweit gerecht werden.
