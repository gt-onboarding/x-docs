<div id="basic-authentication">
  ## Basisauthentifizierung
</div>

Viele Enterprise-APIs von X erfordern HTTP Basic Authentication. Um eine Anfrage an eine API zu stellen, die Basic Authentication voraussetzt, müssen Sie für jede Anfrage eine gültige Kombination aus E-Mail-Adresse und Passwort als Authorization-Header übermitteln. Diese E-Mail-/Passwort-Kombination entspricht den Zugangsdaten für die [Enterprise-API-Konsole](https://console.gnip.com/) und kann dort geändert werden.

Wenn Sie eine Anfrage mit Basic Authentication erstellen, fügen Sie den HTTP-Header Authorization: Basic mit codierten Zugangsdaten über HTTPS hinzu.

Im folgenden cURL-Beispiel ersetzen Sie vor dem Senden der Anfrage `<email_address>` und `<password>` durch Ihre Anmeldedaten:

```bash
curl -v --compressed -u<email_address>:<password>
    "https://gnip-api.x.com/search/30day/accounts/<account-name>/prod/counts.json?query=from%3Axdevelopers"
```
