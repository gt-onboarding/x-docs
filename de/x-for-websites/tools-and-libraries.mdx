<div id="official-resources">
  ## Offizielle Ressourcen
</div>

Die folgenden Tools helfen Ihnen beim Einstieg in X für Websites.

<CardGroup cols={3}>
<Card
  title="publish.x.com"
  icon="x-twitter"
  href="https://publish.x.com/"
>
Erstellen Sie eingebettete Tweets, Timelines oder Schaltflächen.
</Card>
<Card
  title="Twitter for WordPress"
  icon="x-twitter"
  href="https://github.com/twitter/wordpress"
>
Offizielles Plugin zur Optimierung Ihrer WordPress-Website für X, einschließlich Tweet-Einbettungen und X Cards.
</Card>
<Card
  title="twemoji"
  icon="x-twitter"
  href="https://twemoji.x.com/"
>
Der kostenlose Open-Source-Emoji-Zeichensatz von X, einschließlich einer JavaScript-Bibliothek für plattformübergreifende Unterstützung.
</Card>
</CardGroup>
