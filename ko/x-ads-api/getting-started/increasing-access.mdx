<div id="requesting-additional-app-level-permissions">
  ### 추가 앱 수준 권한 요청
</div>

애플리케이션이 아래의 앱 수준 권한 중 하나로 제한되어 있고 추가 엔드포인트에 접근하려는 경우, **Standard Access** 신청을 위해 X 담당자에게 문의하세요.

2023년 7월 이전에 Ads API 접근을 요청한 개발자는 다음 앱 수준 권한 중 하나를 신청했을 수 있으며, 이는 이제 Standard Access 레벨에 포함됩니다.

- **Analytics**(읽기 전용): 사용자는 [읽기 전용](https://developer.x.com/en/docs/apps/app-permissions) 권한으로 [Analytics](https://developer.x.com/en/docs/twitter-ads-api/analytics/overview) 엔드포인트에 접근할 수 있습니다. 사용자는 Campaign Management, Creative, 맞춤 잠재고객 엔드포인트에는 접근할 수 없지만, Analytics 엔드포인트 요청 매개변수에 크리에이티브 엔터티를 전달하여 지표를 수집할 수 있습니다. media_id 또는 media_name 단위의 세분화 수준을 조회하려면 사용자 애플리케이션에 Ads API 쓰기 권한이 필요합니다.

- **Campaign Management & Creative**(읽기 및 쓰기): 사용자는 [읽기 및 쓰기](https://developer.x.com/en/docs/apps/app-permissions) 권한으로 [Campaign Management](https://developer.x.com/en/docs/twitter-ads-api/campaign-management/overview) 및 [Creative](https://developer.x.com/en/docs/twitter-ads-api/creatives/overview) 엔드포인트에 접근할 수 있습니다. 또한 Analytics 엔드포인트에도 접근할 수 있습니다.

- **맞춤 잠재고객**(읽기 및 쓰기): 사용자는 [읽기 및 쓰기](https://developer.x.com/en/docs/apps/app-permissions) 권한으로 [맞춤 잠재고객](https://developer.x.com/en/docs/twitter-ads-api/audiences/overview) 엔드포인트에 접근할 수 있습니다. 또한 Campaign Management, Creative, Analytics 엔드포인트에도 접근할 수 있습니다.

- **Conversion**(읽기 및 쓰기): 사용자는 [읽기 및 쓰기](https://developer.x.com/en/docs/apps/app-permissions) 권한으로 [모바일](https://developer.x.com/en/docs/twitter-ads-api/measurement/mobile-conversions/overview) 및 [웹 전환](https://developer.x.com/en/docs/twitter-ads-api/measurement/web-conversions/overview) 엔드포인트에 접근할 수 있습니다. 사용자는 다른 Ads API 엔드포인트에 대한 접근 권한이 있을 수도 없을 수도 있습니다. 

<div id="requesting-an-increase-to-user-oauth-token-limits">
  ### 사용자 OAuth 토큰 한도 상향 요청
</div>

2023년 7월 이전에 Ads API 액세스를 요청한 개발자는 애플리케이션의 사용자 OAuth 토큰 한도가 5개로 설정되어 있을 수 있습니다. 개발자 앱의 토큰 한도를 늘려야 하는 경우 X 담당자에게 문의하시기 바랍니다.

<Note>**참고:** 어떤 Ads API 엔드포인트에 대해서도 [요청 속도 제한](https://developer.x.com/en/docs/twitter-ads-api/rate-limiting)은 상향하지 않습니다. 제한 내에서 운영하는 방법은 [모범 사례](https://developer.x.com/en/docs/twitter-ads-api/rate-limiting)를 참고하세요.</Note>
