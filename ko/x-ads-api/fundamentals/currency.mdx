---
title: 통화
---

통화 유형은 [ISO-4217](http://en.wikipedia.org/wiki/ISO_4217)로 식별합니다. `USD`, `EUR`와 같은 세 글자 코드입니다. 통화 값은 마이크로 단위로 표현됩니다. USD의 경우 $5.50은 5.50\*1e6, 즉 5,500,000으로 인코딩됩니다. “정수 값”을 표현하려면 모든 통화에서 로컬 마이크로 값을 1e6(1\_000\_000)으로 곱해야 합니다.

X Ads API는 X SMB Ads 제품이 제공되는 모든 국가의 공식 통화를 인식합니다. 해당 국가의 전체 목록은 [여기](https://business.x.com/en/help/overview/about-eligibility-for-twitter-ads.html)를 확인하세요.

대부분의 통화는 소수 “자릿수”가 2입니다(예: USD: $1.23).

- `AED`, `VEF`, `PKR`, `IDR`, `HKD`, `QAR`, `PHP`, `NZD`, `INR`, `EUR`, `ARS`, `THB`, `MXN`, `EGP`, `DZD`, `SAR`, `LBP`, `COP`, `TRY`, `SGD`, `MAD`, `AUD`, `USD`, `MYR`, `CAD`, `BRL`, `GBP`.

다음 통화는 소수를 사용하지 않습니다:

- `VND`, `KRW`, `CLP`, `JPY`

다음 통화는 소수 셋째 자리까지 사용합니다:

- `KWD`, `IQD`, `BHD`, `TND`

통화는 [funding instruments](/ko/x-ads-api/introduction)를 통해 사용됩니다.
