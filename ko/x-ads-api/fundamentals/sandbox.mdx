---
title: 샌드박스
---

샌드박스는 Ads API 구현을 테스트하고 빌드하기 위한 환경입니다.

샌드박스에서 생성된 캠페인은 사용자에게 노출되지 않으며, Funding Instruments와 같은 객체의 테스트 레코드를 동적으로 생성할 수 있습니다.

샌드박스 사용은 ads-api-sandbox.x.com의 해당 호스트명을 호출하는 것만큼 간단하지만, 샌드박스에만 있는 고유 기능도 포함되어 있습니다.

새 광고주 계정을 생성하려면 [POST accounts](/ko/x-ads-api/campaign-management#post-accounts)로 초기 요청을 보내세요. 이 호출의 응답에는 이후 요청에 사용할 수 있는 account\_id가 포함됩니다. 이러한 엔드포인트를 통해 여러 광고주 계정을 생성하고 테스트할 수 있습니다.

<div id="sandbox-only-endpoint-list">
  ### 샌드박스 전용 엔드포인트 목록:
</div>

- [POST accounts](/ko/x-ads-api/campaign-management#post-accounts) 샌드박스에서 계정 만들기
- [POST accounts/:account\_id/features](/ko/x-ads-api/campaign-management#post-accounts-account-id-features) 샌드박스의 계정에 기능 추가
- [POST accounts/:account\_id/funding\_instruments](/ko/x-ads-api/campaign-management#post-accounts-account-id-funding-instruments) 샌드박스의 계정에 결제 수단 추가
- [DELETE accounts/:account\_id](/ko/x-ads-api/campaign-management#delete-accounts-account-id) 샌드박스에서 계정 삭제
- [DELETE accounts/:account\_id/features](/ko/x-ads-api/campaign-management#delete-accounts-account-id-features) 샌드박스의 계정에서 기능 삭제
- [DELETE accounts/:account\_id/funding\_instruments/:funding\_instrument\_id](/ko/x-ads-api/campaign-management#delete-accounts-account-id-funding-instruments-funding-instrument-id) 샌드박스의 계정에서 결제 수단 삭제
