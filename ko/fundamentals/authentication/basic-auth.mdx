<div id="basic-authentication">
  ## 기본 인증
</div>

X의 엔터프라이즈 API 중 상당수는 HTTP Basic 인증을 요구합니다. Basic 인증이 필요한 API에 정상적으로 요청하려면 각 요청의 인증 헤더에 유효한 이메일 주소와 비밀번호 조합을 전달해야 합니다. 이 이메일과 비밀번호 조합은 [enterprise API console](https://console.gnip.com/)에 액세스할 때 사용하는 것과 동일하며, 해당 콘솔에서 수정할 수 있습니다. 

Basic 인증으로 요청을 구성할 때는 HTTPS를 통해 인코딩된 자격 증명을 포함해 Authentication: Basic HTTP 헤더를 추가해야 합니다.

다음 cURL 요청 예시에서 요청을 보내기 전에 `<email_address>`와 `<password>`를 본인의 자격 증명으로 바꾸십시오:

```bash
curl -v --compressed -u<email_address>:<password>
    "https://gnip-api.x.com/search/30day/accounts/<account-name>/prod/counts.json?query=from%3Axdevelopers"
```
