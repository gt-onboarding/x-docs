title: 요청 한도
icon: gauge-simple-high
---

매일 수천 명의 개발자가 X 개발자 플랫폼에 요청을 보냅니다. 이러한 방대한 요청량을 관리하기 위해 수행 가능한 요청 수에 한도를 두고 있습니다. 이 한도는 개발자 커뮤니티가 신뢰하는 안정적이고 확장 가능한 API를 제공하는 데 도움이 됩니다.

각 API는 요청 한도를 서로 다르게 적용합니다. 플랫폼 간 차이에 대해 자세히 알아보려면 각 API 섹션의 해당 요청 한도 페이지를 확인하세요.

- [X API v2](/ko/x-api/fundamentals/rate-limits)
- [X API: Enterprise](/ko/x-api/enterprise-gnip-2.0/fundamentals/rate-limits)
- [X Ads API](/ko/x-ads-api/fundamentals/rate-limiting)
