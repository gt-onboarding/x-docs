<div id="educators">
  ## 교육자
</div>

API를 수업에 도입해 학생들에게 프로그래밍 기초와 실무적인 데이터 처리 역량을 가르치세요.

**리소스:**

강의에서 X API를 사용하려는 교육자이신가요? 이
[X API 활용 교육 가이드](https://dev.to/suhemparack/a-guide-to-teaching-with-the-twitter-api-v2-3n08)는
X API의 개요, 교육자가 이를 사용하는 이유, API로 진행되는 수업 유형, 그리고 시작하는 데 필요한 모든 정보를 제공합니다.

**스포트라이트:**
