<div id="steps-to-build-a-pinned-list-lookup-request">
  ### 고정된 리스트 조회 요청을 만드는 단계
</div>

**1단계: Postman에서 List 엔드포인트 컬렉션 선택**

Postman에 X API v2 컬렉션을 로드한 후 “List” 폴더로 이동해 “Pinned Lists” 폴더를 선택한 다음 "User's pinned Lists"를 선택합니다.
 

**2단계: 사용자를 식별하고 지정**

사용자의 고정된 리스트를 가져오려면 요청에 해당 사용자의 사용자 ID를 지정해야 합니다. 사용자 ID는 인증에 사용한 사용자(Authenticating user)의 ID와 일치해야 하며, 즉 요청을 인증할 때 해당 사용자 ID와 연결된 Access Token을 전달해야 합니다. 이 예에서는 본인의 사용자 ID를 지정할 수 있습니다. 본인의 ID는 다음 두 가지 방법으로 확인할 수 있습니다.

1. [users lookup](/ko/x-api/users/lookup/introduction) by username 엔드포인트를 사용해 사용자 이름을 전달하면 id 필드를 받을 수 있습니다.
2. Access Token의 숫자 부분이 사용자 ID입니다.

Postman에서 "Params" 탭으로 이동해 `id` 매개변수의 "Value" 열에 이 사용자 ID를 입력합니다.

|     |     |
| :--- | :--- |
| **Key** | **Value** |
| `id` | 2244994945 (user ID) |

**3단계: 가져올 필드를 식별하고 지정**

이 단계까지 완료한 뒤 "Send" 버튼을 클릭하면 응답에서 기본 [List object](/ko/x-api/fundamentals/data-dictionary#list) 필드인 `id`와 `name`을 받게 됩니다.

`id`와 `name` 외에 추가 필드를 받으려면 요청에서 [`field`](/ko/x-api/fundamentals/fields) 및/또는 [`expansion`](/ko/x-api/fundamentals/expansions) 매개변수로 해당 필드를 지정해야 합니다.

이 연습에서는 서로 다른 오브젝트에서 다음 세 가지 추가 필드 세트를 요청합니다.

- 기본 List 오브젝트의 추가 `follower_count` 필드
- expansion 매개변수를 사용한 전체 [user object](/ko/x-api/fundamentals/data-dictionary#user)
- 관련 user 오브젝트의 추가 `tweet.created_at` 필드

Postman에서 "Params" 탭으로 이동해 "Query Params" 표에 다음 key:value 쌍을 추가합니다.

|     |     |     |
| :--- | :--- | :--- |
| Key | Value | Returned fields |
| list.fields | follower\_count | `follower_count` |
| expansions | owner\_id | `includes.users.id,  <br/>includes.users.name,  <br/>includes.users.username` |
| user.fields | created\_at | `includes.users.created_at` |

이제 “Send” 버튼 옆에 다음과 유사한 URL이 표시됩니다.

`https://api.x.com/2/users/2244994945/pinned_lists?expansions=owner_id&list.fields=follower_count&user.fields=created_at`

**4단계: 요청을 보내고 응답 검토**

모든 설정이 완료되면 "Send" 버튼을 눌러 다음 예시와 유사한 응답을 받습니다.

```
{
  "data": [
    {
      "follower_count": 0,
      "id": "1454155907651158017",
      "name": "test list",
      "owner_id": "2244994945"
    }
  ],
  "includes": {
    "users": [
      {
        "username": "TwitterDev",
        "id": "2244994945",
        "created_at": "2013-12-14T04:35:55.000Z",
        "name": "Twitter Dev"
      }
    ]
  },
  "meta": {
    "result_count": 1
  }
}
```
