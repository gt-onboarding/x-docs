ID로 지정된 리스트를 조회할 때는 15분당 75개의 요청 제한이 있으며, 사용자 소유 리스트를 조회할 때는 15분당 15개의 요청 제한이 있습니다.

이 엔드포인트에 대한 요청을 인증하려면 [OAuth 1.0a User Context](/ko/resources/fundamentals/authentication), \[App only]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token), 또는 [OAuth 2.0 Authorization Code with PKCE](ko/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2)를 사용할 수 있습니다.

<Note>
  **계정 설정**

  이 엔드포인트에 액세스하려면 다음이 필요합니다:

  - 승인된 [개발자 계정](https://developer.x.com/en/portal/petition/essential/basic-info).
  - [프로젝트](/ko/resources/fundamentals/projects) 내 [개발자 앱](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용한 인증.

  X API v2 엔드포인트에 액세스하는 방법은 [시작하기 가이드](/ko/x-api/getting-started/getting-access)에서 자세히 알아보세요.
</Note>
