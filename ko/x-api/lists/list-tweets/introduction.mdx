리스트 게시물 조회에는 지정된 리스트의 게시물을 가져오는 하나의 엔드포인트가 제공됩니다. 이 엔드포인트를 사용하면 사용자가 타임라인에서 보는 게시물을 사용자화하고 체계화하며 우선순위를 지정할 수 있는 솔루션을 구축할 수 있습니다.

이 엔드포인트의 호출 한도는 15분당 900건이며, 응답에서는 지정된 리스트에 대해 최신 게시물 최대 800건까지 조회할 수 있습니다.

이 엔드포인트에 대한 요청 인증에는 [OAuth 1.0a User Context](/ko/resources/fundamentals/authentication), \[App only]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token), 또는 [OAuth 2.0 Authorization Code with PKCE](ko/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2)를 사용할 수 있습니다.

**계정 설정**

이 엔드포인트에 액세스하려면 다음이 필요합니다.

- 승인된 [developer account](https://developer.x.com/en/portal/petition/essential/basic-info)
- [Project](/ko/resources/fundamentals/projects) 내 [developer App](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용한 인증

X API v2 엔드포인트 액세스 방법에 대해서는 [getting started guide](/ko/x-api/getting-started/getting-access)를 참조하세요.
