  ## X API의 리스트 게시물 조회 엔드포인트 비교
</div>

v2 리스트 게시물 조회 엔드포인트는 표준 v1.1 [GET lists/statuses](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-statuses)를 대체합니다. 이 버전의 엔드포인트를 사용하는 코드, 앱, 도구가 있고 최신 X API v2 엔드포인트로 마이그레이션을 고려 중이라면 이 가이드 세트를 확인하세요.

다음 표는 표준 v1.1과 X API v2 리스트 엔드포인트를 비교합니다:

**ID로 리스트 조회**

|     |     |     |
| :--- | :--- | :--- |
| Description | Standard v1.1 | X API v2 |
| HTTP methods supported | `GET` | `GET` |
| Host domain | `https://api.x.com` | `https://api.x.com` |
| Endpoint path | `/1.1/lists/statuses.json` | `/2/lists/:id/tweets` |
| [Authentication](/ko/resources/fundamentals/authentication) | OAuth 1.0a 사용자 컨텍스트<br /><br />앱 전용 | OAuth 1.0a 사용자 컨텍스트<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />앱 전용 |
| Default request [rate limits](/ko/resources/fundamentals/rate-limits) | OAuth 1.0a: 15분당 900회 요청<br /><br />앱 전용: 15분당 900회 요청 | OAuth 1.0a: 15분당 900회 요청<br /><br />OAuth 2.0: 15분당 900회 요청<br /><br />앱 전용: 15분당 900회 요청 |

X API v2 엔드포인트에 접근하려면 [개발자 계정을 신청](https://developer.x.com/en/portal/petition/essential/basic-info)해야 합니다. 인증 시 [프로젝트](/ko/resources/fundamentals/projects) 내에 있는 [개발자 앱](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용해야 합니다.

X API v2 엔드포인트에 대한 접근 권한 획득 방법은 [시작하기](/ko/x-api/getting-started/getting-access) 페이지에서 자세히 알아보세요.
