<div id="comparing-x-apis-list-members-endpoints">
  ## X API의 목록 멤버 엔드포인트 비교
</div>

v2 목록 멤버 엔드포인트 그룹은 표준 v1.1 [GET lists/members](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-members), [GET lists/memberships](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-memberships), [POST lists/members/create](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-members-create) 및 [POST lists/members/destroy](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-members-destroy) 엔드포인트를 대체합니다. 이러한 목록 멤버 엔드포인트 중 하나를 사용하는 코드, 앱, 또는 도구가 있고 최신 X API v2 엔드포인트로 마이그레이션을 고려 중이라면, 이 가이드 모음이 도움이 될 것입니다. 

<div id="list-members-lookup">
  ### 리스트 멤버 조회
</div>

v2 리스트 멤버 조회 엔드포인트는 표준 [GET lists/members](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-members), [GET lists/memberships](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-memberships) 엔드포인트를 대체합니다.

다음 표는 표준 v1.1과 X API v2의 리스트 엔드포인트를 비교합니다:

**리스트 멤버 조회**

|     |     |     |
| :--- | :--- | :--- |
| 설명 | Standard v1.1 | X API v2 |
| 지원 HTTP 메서드 | `GET` | `GET` |
| 호스트 도메인 | `https://api.x.com` | `https://api.x.com` |
| 엔드포인트 경로 | `/1.1/lists/members.json` | `/2/lists/:id/members` |
| [Authentication](/ko/resources/fundamentals/authentication) | OAuth 1.0a User Context<br /><br />App only | OAuth 1.0a User Context<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />App only |
| 기본 요청 [rate limits](/ko/resources/fundamentals/rate-limits) | OAuth 1.0a: 15분당 900회 요청<br /><br />App only: 15분당 75회 요청 | OAuth 1.0a: 15분당 900회 요청<br /><br />OAuth 2.0: 15분당 900회 요청<br /><br />App only: 15분당 900회 요청 |

**리스트 멤버십 조회**

|     |     |     |
| :--- | :--- | :--- |
| 설명 | Standard v1.1 | X API v2 |
| 지원 HTTP 메서드 | `GET` | `GET` |
| 호스트 도메인 | `https://api.x.com` | `https://api.x.com` |
| 엔드포인트 경로 | `/1.1/lists/memberships.json` | `/2/users/:id/list_memberships` |
| [Authentication](/ko/resources/fundamentals/authentication) | OAuth 1.0a User Context<br /><br />App only | OAuth 1.0a User Context<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />App only |
| 기본 요청 [rate limits](/ko/resources/fundamentals/rate-limits) | OAuth 1.0a: 15분당 75회 요청<br /><br />App only: 15분당 75회 요청 | OAuth 1.0a: 15분당 75회 요청<br /><br />OAuth 2.0: 15분당 75회 요청<br /><br />App only: 15분당 75회 요청 |

<div id="manage-list-members">
  ### 목록 멤버 관리
</div>

v2 목록 멤버 관리 엔드포인트는 표준 [POST lists/members/create](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-members-create), [POST lists/members/destroy](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-members-destroy) 엔드포인트를 대체합니다.

다음 표는 표준 v1.1과 X API v2의 목록 엔드포인트를 비교합니다:

**멤버 추가**

| 설명 | 표준 v1.1 | X API v2 |
| :--- | :--- | :--- |
| 지원 HTTP 메서드 | POST | POST |
| 호스트 도메인 | https://api.x.com | https://api.x.com |
| 엔드포인트 경로 | /1.1/lists/members/create.json | /2/lists/:id/members |
| [인증](/ko/resources/fundamentals/authentication) | OAuth 1.0a 사용자 컨텍스트 | OAuth 1.0a 사용자 컨텍스트<br /><br />OAuth 2.0 Authorization Code with PKCE |
| 기본 요청 [레이트 리밋](/ko/resources/fundamentals/rate-limits) | 없음 | 15분당 1인 기준 300회 |

**멤버 제거**

| 설명 | 표준 v1.1 | X API v2 |
| :--- | :--- | :--- |
| 지원 HTTP 메서드 | POST | DELETE |
| 호스트 도메인 | https://api.x.com | https://api.x.com |
| 엔드포인트 경로 | /1.1/lists/members/destroy.json | /2/lists/:id/:user\_id |
| [인증](/ko/resources/fundamentals/authentication) | OAuth 1.0a 사용자 컨텍스트 | OAuth 1.0a 사용자 컨텍스트<br /><br />OAuth 2.0 Authorization Code with PKCE |
| 기본 요청 [레이트 리밋](/ko/resources/fundamentals/rate-limits) | 없음 | 15분당 1인 기준 300회 |

X API v2 엔드포인트에 액세스하려면 [개발자 계정에 가입](https://developer.x.com/en/portal/petition/essential/basic-info)해야 합니다. 인증 시에는 [프로젝트](/ko/resources/fundamentals/projects) 내에 있는 [개발자 앱](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용해야 합니다.

X API v2 엔드포인트 액세스에 대한 자세한 내용은 [시작하기](/ko/x-api/getting-started/getting-access) 페이지를 참조하세요.
