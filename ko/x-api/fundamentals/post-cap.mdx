---
title= "게시 한도"
---

X API v2에는 월별로 특정 엔드포인트에서 조회할 수 있는 게시물 수를 제한하는 게시 소비 한도가 있습니다. 게시 한도는 [프로젝트](/ko/resources/fundamentals/projects) 단위로 적용되며, 해당 프로젝트의 [개발자 앱](/ko/resources/fundamentals/developer-apps)에서 발급된 키와 토큰을 사용해 아래 나열된 엔드포인트로 보내는 모든 요청은 그 프로젝트의 게시 한도에 합산됩니다.

<div id="limited-v2-endpoints">
  ## 제한된 v2 엔드포인트
</div>

다음 엔드포인트에서 수신한 게시물은 월간 트윗 한도(게시 한도)에 포함됩니다:

- [Posts Lookup](/ko/x-api/posts/lookup/introduction)
- [Recent search](/ko/x-api/posts/counts/introduction)
- [Full-archive search](/ko/x-api/posts/counts/introduction)
- [Filtered stream](/ko/x-api/posts/filtered-stream)
- [User Post timeline](/ko/x-api/posts/timelines)
- [User mention timeline](/ko/x-api/posts/timelines)
- [Likes lookup - Posts liked by a user](/ko/x-api/posts/likes)
- [Bookmarks lookup - Posts bookmarked by a user](/ko/x-api/posts/bookmarks/introduction)
- [List Posts lookup](/ko/x-api/lists/list-tweets/introduction)
- [Spaces Lookup](/ko/x-api/spaces/lookup/introduction)

<Note>
  **참고:** 동일한 게시물이 하루 동안 여러 쿼리에서 반환되더라도 트윗 한도(게시 한도)에는 해당 일자에 한 번만 반영됩니다. 게시물은 일 단위로 중복 제거됩니다.
</Note>

<div id="post-cap-volumes">
  ## 게시 한도 볼륨
</div>

게시 한도 볼륨은 액세스 수준에 따라 달라집니다:

| 액세스 티어   | 월간 게시 한도                |
|:---------------|:---------------------------------|
| **Free**      | 월 1,500개 게시물           |
| **Basic**     | 월 10,000개 게시물          |
| **Pro**       | 월 1,000,000개 게시물       |
| **Enterprise**| 월 5천만 개 이상의 게시물     |

[개발자 포털](/ko/resources/fundamentals/developer-portal)의 [메인 대시보드 페이지](https://developer.x.com/content/developer-twitter/en/portal/dashboard)에서 월간 트윗 한도(게시 한도) 사용량을 확인할 수 있습니다. 프로젝트 이름 아래의 상태 표시줄에서 현재 월 사용량을 게시 한도 대비로 확인할 수 있으며, 가져온 게시물 총계, 사용 비율, 사용량 재설정 날짜가 표시됩니다.
