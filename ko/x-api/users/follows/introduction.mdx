팔로우 관계를 조회, 생성, 삭제할 수 있도록 두 가지 엔드포인트 그룹을 제공합니다: 팔로우 조회와 팔로우 관리.

<div id="follows-lookup">
  ### 팔로우 조회
</div>

Only available to those with [Enterprise access](/ko/x-api/getting-started/about-x-api)

팔로우 조회 엔드포인트는 사용자 간 관계를 탐색하고 분석할 수 있도록 하며, 이를 네트워크 분석이라고도 합니다. 구체적으로, 지정한 사용자가 누구를 팔로우하는지 또는 누가 해당 사용자를 팔로우하는지를 나타내는 [user objects](/ko/x-api/fundamentals/data-dictionary#user)를 반환하는 두 가지 REST 엔드포인트가 있습니다.

You can authenticate this endpoint with either [OAuth 1.0a User Context](/ko/resources/fundamentals/authentication), \[App only]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token), or [OAuth 2.0 Authorization Code with PKCE](ko/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2). You can request up to 1,000 users per request, and pagination tokens will be provided for paging through large sets of results.
 

<div id="manage-follows">
  ### 팔로우 관리
</div>

팔로우 관리 엔드포인트를 사용하면 사용자를 팔로우하거나 언팔로우할 수 있습니다.

사용자를 대신해 요청을 보내므로, 이 엔드포인트는 [OAuth 1.0a User Context](/ko/resources/fundamentals/authentication) 또는 [OAuth 2.0 Authorization Code with PKCE](ko/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2)로 인증해야 하며, 요청 대상 사용자와 연결된 사용자 액세스 토큰을 사용해야 합니다. 이 사용자 액세스 토큰은 [3-legged OAuth flow](/ko/resources/fundamentals/authentication#obtaining-access-tokens-using-3-legged-oauth-flow) (OAuth 1.0a) 또는 [Authorization Code with PKCE grant flow](/ko/resources/fundamentals/authentication#how-to-connect-to-endpoints-using-oauth-2-0-authorization-code-flow-with-pkce)) (OAuth 2.0)를 사용해 생성할 수 있습니다.

인증된 각 사용자를 대신해 하루 최대 400건의 팔로우 작업이 가능하며, 인증된 모든 사용자를 합산해 앱당 하루 최대 1,000건으로 제한됩니다. 예를 들어 인증된 사용자가 다섯 명이면, 그중 두 명으로 각 사용자별 하루 400건씩 총 800건을 수행할 수 있고, 나머지 세 사용자에게 남은 200건(앱당)을 배분해야 합니다. 이 한도는 언팔로우 엔드포인트에는 적용되지 않으며, 언팔로우는 별도로 앱당 하루 500건의 한도가 있습니다.

<Note>
  **계정 설정**

  이 엔드포인트에 액세스하려면 다음이 필요합니다:

  - 승인된 [developer account](https://developer.x.com/en/portal/petition/essential/basic-info)
  - [developer App](/ko/resources/fundamentals/developer-apps)이 포함된 [Project](/ko/resources/fundamentals/projects)의 키와 토큰을 사용한 인증
</Note>

\[X API v2] 엔드포인트에 대한 액세스 권한 얻기에 대해 자세히 알아보려면 [시작하기 가이드](/ko/x-api/getting-started/getting-access)를 참고하세요.
