<div id="blocks-lookup">
  ### 차단 목록 조회
</div>

차단 목록 조회 GET 엔드포인트는 권한이 부여된 사용자를 대신하여 해당 사용자가 차단한 계정을 확인할 수 있도록 합니다. 이 엔드포인트의 요청 한도는 사용자당 15분에 15회입니다. 

차단 목록 조회는 비공개 정보를 사용자(차단 관리 권한 보유자)를 대신해 요청하므로, [OAuth 1.0a User Context](/ko/resources/fundamentals/authentication) 또는 [OAuth 2.0 Authorization Code with PKCE](ko/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2)로 인증해야 하며, 귀하의 앱을 승인한 사용자와 연결된 사용자 액세스 토큰을 사용해야 합니다. 해당 토큰은 [3-legged OAuth flow](/ko/resources/fundamentals/authentication#obtaining-access-tokens-using-3-legged-oauth-flow) (OAuth 1.0a) 또는 [Authorization Code with PKCE grant flow](/ko/resources/fundamentals/authentication#how-to-connect-to-endpoints-using-oauth-2-0-authorization-code-flow-with-pkce) (OAuth 2.0)를 통해 생성할 수 있습니다.

<Note>
  **계정 설정**

  이 엔드포인트에 접근하려면 다음이 필요합니다:

  - 승인된 [개발자 계정](https://developer.x.com/en/portal/petition/essential/basic-info)
  - [프로젝트](/ko/resources/fundamentals/projects) 내에 있는 [개발자 앱](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용한 인증

  [X API v2 엔드포인트 접근](/ko/x-api/getting-started/getting-access)에 대한 자세한 내용은 시작 가이드를 참조하세요.
</Note>
