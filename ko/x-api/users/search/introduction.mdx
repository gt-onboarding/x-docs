Pro 액세스 이상 보유 개발자 이용 가능

Users Search 엔드포인트는 X의 공개 사용자 계정에 대해 간단한 관련성 기반 검색 인터페이스를 제공합니다. 주제 관심사, 실명, 회사명, 위치 등 다양한 기준으로 쿼리해 보세요.

이 엔드포인트는 현재 [User Auth](/ko/resources/fundamentals/authentication)를 지원합니다. 기본적으로 요청당 사용자 100명을 반환하며, 요청당 최대 1,000명까지 지정할 수 있습니다. 24시간 기준 10,000건의 요청 한도가 있습니다(15분당 300건 한도와는 별개).

<Note>
  **계정 설정**

  이 엔드포인트에 액세스하려면 다음이 필요합니다:

  - 승인된 [개발자 계정](https://developer.x.com/en/portal/petition/essential/basic-info)
  - [프로젝트](/ko/resources/fundamentals/projects) 내 [개발자 앱](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용한 인증

  X API v2 엔드포인트 액세스 방법은 [시작하기 가이드](/ko/x-api/getting-started/getting-access)에서 자세히 확인하세요.
</Note>
