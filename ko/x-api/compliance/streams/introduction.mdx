X는 X API로 구축하는 개발자 커뮤니티를 위해 최선을 다하고 있습니다. 이러한 약속의 일환으로, 당사는 API를 개발자에게 개방적이고 공정하게 제공하며, X 이용자에게 안전하고 X 플랫폼 전체에 이롭도록 하는 것을 목표로 합니다. X 콘텐츠를 오프라인으로 저장하는 모든 개발자는 데이터가 이용자 의도와 X의 현재 콘텐츠 상태를 반영하도록 보장해야 합니다. 예를 들어, X에서 누군가가 게시물을 삭제하거나 계정을 삭제하거나, 게시물을 보호로 전환하거나, 게시물을 편집한 경우, 그 사람의 기대와 의도를 X와 개발자 모두가 존중하는 것이 매우 중요합니다.

컴플라이언스 이벤트의 실시간 스트림은 개발자가 [X Developer Agreement and Policy](https://developer.x.com/en/developer-terms/policy)를 준수하여 X 데이터를 유지할 수 있도록 도구를 제공합니다. 

컴플라이언스 이벤트 스트림은 두 가지가 있으며, 하나는 게시물 컴플라이언스 이벤트용이고 다른 하나는 사용자 컴플라이언스 이벤트용입니다. 이 스트림은 Enterprise 액세스에서 제공되며, 대량의 데이터를 수집하는 파트너가 게시물 편집 이벤트와 같은 컴플라이언스 이벤트를 수신해 ‘리스닝’할 수 있도록 설계되었습니다.

이 스트림에서는 다음 이벤트가 제공됩니다: 

**게시물 컴플라이언스 스트림:** 

- delete - 게시물이 삭제되었음을 나타냅니다.

- tweet\_edit - 게시물이 편집되었음을 나타내며 업데이트된 게시물의 ID를 제공합니다. 

- withheld - 게시물이 하나 이상의 국가에서 보류되었음을 나타냅니다. 

- drop - 게시물을 공개 보기에서 제거해야 함을 나타냅니다.

- undrop - 게시물을 다시 표시하고 공개로 취급할 수 있음을 나타냅니다.

**사용자 컴플라이언스 스트림:** 

- **user\_delete** - 사용자 계정이 삭제되었음을 나타냅니다

- **user\_undelete** - 사용자 계정 삭제가 취소되었음을 나타냅니다

- **user\_protect** - 사용자 계정이 비공개로 전환되었음을 나타냅니다

- **user\_unprotect** - 사용자 계정이 공개로 전환되었음을 나타냅니다

- **user\_withheld** - 사용자 계정이 하나 이상의 국가에서 보류되었음을 나타냅니다. 

- **user\_suspend** - 사용자 계정이 정지되었음을 나타냅니다

- **user\_unsuspend** - 사용자 계정 정지가 해제되었음을 나타냅니다

- **user\_profile\_modification** - 사용자 프로필이 업데이트되었음을 나타냅니다. 여기에는 설명, 이름, 위치, URL 업데이트가 포함됩니다.

**계정 설정**

이 엔드포인트에 액세스하려면 다음이 필요합니다:

- 승인된 [developer account](https://developer.x.com/en/portal/petition/essential/basic-info)
- [Project](/ko/resources/fundamentals/projects) 내에 있는 [developer App](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용한 인증 

X API v2 엔드포인트에 대한 액세스 권한 얻기에 대해 더 알아보려면 [getting started guide](/ko/x-api/getting-started/getting-access)를 참고하세요.
