<div id="community-tools-and-libraries-for-v2">
  ## v2용 커뮤니티 도구 및 라이브러리
</div>

여기에 나열된 라이브러리는 개발자 커뮤니티 구성원이 제작한 것입니다. API 지원 범위와 완성도는 각기 다를 수 있습니다.

직접 만든 X API 라이브러리나 유용한 도구가 있다면 [알려 주세요](https://devcommunity.x.com/c/libraries-and-sdks/63). 다른 분들이 찾을 수 있도록 이 목록에 추가하겠습니다. 또한 README 파일에 사용할 수 있는 [버전 배지](https://twbadges.glitch.me/)도 제공하고 있습니다.

아이디어가 필요하신가요? GitHub의 [X](https://github.com/topics/twitter?o=desc&s=updated) 및 [X-api-v2](https://github.com/topics/twitter-api-v2?o=desc&s=updated) 토픽을 둘러보며 다른 개발자들의 유용한 코드 예제를 검색해 보세요.

**바로 가기:** [C# / .NET](#csharp), [Dart / Flutter](#dart), [Go](#go), [Java](#java), [JavaScript (Node.JS) / TypeScript](#nodejs), [Kotlin](#kotlin), [PHP](#php), [PowerShell](#powershell), [Python](#python), [R](#r), [Ruby](#ruby), [Rust](#rust), [Swift](#swift)

<div id="community-tools-and-libraries">
  ## 커뮤니티 도구 및 라이브러리
</div>

여기는 다양한 프로그래밍 언어와 플랫폼에서 X API를 다루는 커뮤니티 지원 라이브러리 중 일부를 소개합니다. 이 리소스가 모두 Twitter 팀에 의해 테스트된 것은 아닙니다.

여기에 나열된 라이브러리는 별도 표기가 없는 한 Standard API **v1.1**의 대부분 기능을 구현해야 합니다. 자세한 내용과 추가 지원은 각 작성자에게 문의하세요.\
X API **v2**를 지원하는 라이브러리를 개발하셨다면, 이 페이지에 반영될 수 있도록 [커뮤니티 포럼](https://devcommunity.x.com/c/libraries-and-sdks/63)에서 알려주세요. 또한 포럼을 통해 이 목록의 변경 사항을 신고할 수도 있습니다.

선호하는 프로그래밍 언어용 라이브러리나 도구가 없다면 [피드백 플랫폼](https://twitterdevfeedback.uservoice.com/)으로 알려주세요. 그곳에서 아이디어에 투표하거나, 새롭게 만들어 제출할 영감을 얻을 수도 있습니다.
