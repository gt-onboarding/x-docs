<div id="overview">
  ## 개요
</div>

매일 수천 명의 개발자가 X API에 요청을 보냅니다. 이러한 대규모 요청을 관리하기 위해 요청 횟수에 제한을 둡니다. 이 제한은 우리 개발자 커뮤니티가 신뢰하는 안정적이고 확장 가능한 API를 제공하는 데 도움이 됩니다.

허용되는 최대 요청 수는 특정 기간, 즉 일정한 시간 창을 기준으로 합니다. 예를 들어, 어떤 엔드포인트의 레이트 리밋이 15분당 900건이라면, 임의의 15분 구간 동안 최대 900건의 요청이 허용됩니다.
