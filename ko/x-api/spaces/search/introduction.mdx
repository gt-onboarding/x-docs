이 엔드포인트를 사용하면 제목으로 Space를 검색하고 상태로 결과를 필터링할 수 있습니다. 이 엔드포인트는 제목에 포함된 키워드, 언급된 사용자, 또는 해시태그를 기반으로 라이브 또는 예정된 Space를 탐색하는 데 유용합니다.

이 엔드포인트는 하나 이상의 키워드를 쿼리로 받습니다. 결과는 Space의 상태(라이브 또는 예약됨)로 필터링할 수 있습니다. 기본적으로 요청은 지정된 쿼리와 일치하는 라이브 및 예약된 Space를 모두 반환합니다.

<Note>
  **계정 설정**

  이 엔드포인트들에 액세스하려면 다음이 필요합니다:

  - 승인된 [개발자 계정](https://developer.x.com/en/portal/petition/essential/basic-info)
  - [프로젝트](/ko/resources/fundamentals/projects) 내의 [개발자 앱](/ko/resources/fundamentals/developer-apps)에서 발급된 키와 토큰을 사용한 인증

  X API v2 엔드포인트에 대한 액세스 방법은 [시작 가이드](/ko/x-api/getting-started/getting-access)에서 자세히 알아보세요.
</Note>
