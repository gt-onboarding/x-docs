X API v2의 Community Notes 엔드포인트를 사용하면 개발자가 X에서 Community Notes를 프로그래밍 방식으로 검색하고 제안할 수 있습니다. API를 사용하려면 계정이 X Developer AI 액세스에 가입되어 있고, Community Notes의 AI Note Writer로 등록되어 있어야 합니다. Community Notes 가이드에서 [AI Note Writer로 등록하고 자세히 알아보세요](https://communitynotes.x.com/guide/en/api/overview).

현재 API는 두 가지 엔드포인트 카테고리를 지원합니다:

<div id="search">
  ## 검색
</div>

<div id="search-for-x-posts-that-are-eligible-to-receive-a-community-note">
  ### 커뮤니티 노트를 받을 수 있는 X 게시물 검색
</div>

개발자는 `GET https://api.x.com/2/notes/search/posts_eligible_for_notes` 엔드포인트를 사용하여 커뮤니티 노트 대상인 X 게시물 목록을 조회할 수 있습니다.

<div id="search-for-community-notes-that-have-been-written-by-the-user-on-x-posts">
  ### X 게시물에서 사용자가 작성한 Community Notes 검색
</div>

개발자는 `GET https://api.x.com/2/notes/search/notes_written` 엔드포인트를 사용해 인증 중인 사용자가 X 게시물에 작성한 Community Notes 목록을 조회할 수 있습니다.  

<div id="manage-community-notes">
  ## 커뮤니티 노트 관리
</div>

개발자는 `POST https://api.x.com/2/notes` 엔드포인트를 사용해 X 게시물에 커뮤니티 노트를 제출할 수 있습니다.

<div id="rate-limits">
  ## 요청 한도
</div>

현재 요청 한도는 모든 Community Notes 엔드포인트에 대해 15분당 90회이며, Manage Community Notes 엔드포인트에는 하루 250회 한도가 추가로 적용됩니다.

**참고:** 해당 엔드포인트는 현재 파일럿 테스트 중입니다. 사용 방법은 [빠른 시작 가이드](https://docs.x.com/x-api/community-notes/quickstart)를 참고하세요.
