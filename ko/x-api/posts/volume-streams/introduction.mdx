- 1% 샘플 스트림.
- 10% 샘플 스트림. 일반적으로 “Decahose”라고 합니다.

둘 다 대량의 데이터를 전송하므로 “볼륨 스트림”이라고 부릅니다. 1% 스트림만으로도 매초 수십 개의 게시물을 내보낼 수 있습니다. 이러한 스트림을 사용하면 트렌드를 식별·추적하고, 전반적인 감성을 파악하고, 전 세계 이벤트를 모니터링하는 등 다양한 작업을 수행할 수 있습니다. 

이 스트리밍 엔드포인트는 지속적인 HTTP GET 연결을 통해 [Post 객체](/ko/x-api/fundamentals/data-dictionary#tweet)를 전달하며 [OAuth 2.0 App-Only](/ko/resources/fundamentals/authentication#oauth-2-0) 인증을 사용합니다. Essential 액세스에서는 동시에 하나의 연결만 유지할 수 있습니다. 모든 액세스 수준에서 15분당 최대 50회까지 연결 요청이 가능합니다.

이 볼륨 스트림 엔드포인트는 게시물 편집을 지원합니다. 편집된 게시물은 편집 이력과 함께, Post ID 배열을 포함하여 전달됩니다. 편집 이력이 없는 게시물의 경우 이 배열에는 단일 ID가 포함됩니다. 편집된 게시물의 경우 이 배열에는 여러 ID가 포함되며, 편집 순서를 반영해 오름차순으로 정렬되고 배열의 마지막에 최신 버전이 위치합니다. 게시물 편집 방식에 대해 자세히 알아보려면 [게시물 편집 기본 사항](/ko/x-api/fundamentals/edit-posts) 페이지를 참조하세요. 

_이 API를 사용하려면 먼저 엔터프라이즈 영업팀과 계정을 설정해야 합니다._

<Note>
  **계정 설정**

  이 엔드포인트에 액세스하려면 다음이 필요합니다:

  - 승인된 [개발자 계정](https://developer.x.com/en/portal/petition/essential/basic-info)
  - [프로젝트](/ko/resources/fundamentals/projects) 내의 [개발자 앱](/ko/resources/fundamentals/developer-apps)에서 발급된 키와 토큰을 사용한 인증

  [시작하기 가이드](/ko/x-api/getting-started/getting-access)에서 X API v2 엔드포인트 액세스 방법을 자세히 알아보세요.
</Note>
