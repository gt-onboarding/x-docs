<div id="getting-started-with-the-recent-post-counts-endpoint">
  ### 최신 게시물 수 카운트 엔드포인트 시작하기
</div>

이 빠른 시작 가이드는 HTTP 요청을 보낼 수 있는 그래픽 도구인 Postman을 사용해 최신 게시물 수 카운트 엔드포인트에 첫 요청을 보내는 방법을 안내합니다.

다양한 프로그래밍 언어의 샘플 코드를 보려면 [X API v2 sample code](https://github.com/xdevplatform/Twitter-API-v2-sample-code) GitHub 리포지토리를 방문하세요.

<Note>
  ### 사전 준비 사항

  이 가이드를 완료하려면 요청 인증을 위한 [keys and tokens](/ko/resources/fundamentals/authentication)이 필요합니다. 다음 단계를 따라 이러한 키와 토큰을 생성할 수 있습니다:

  - [개발자 계정에 신청](https://developer.x.com/en/apply-for-access)하고 승인을 받습니다.
  - 개발자 포털에서 [프로젝트](/ko/resources/fundamentals/projects)와 연결된 [개발자 앱](/ko/resources/fundamentals/developer-apps)을 생성합니다.
  - 앱의 “Keys and tokens” 페이지로 이동하여 필요한 자격 증명을 생성합니다. 모든 자격 증명은 안전한 위치에 보관하세요.
</Note>

<div id="steps-to-build-a-recent-post-counts-request">
  #### 최근 Post 수 집계 요청 빌드 단계
</div>

**1단계: 도구 또는 라이브러리로 시작**

이 엔드포인트에 요청을 보내는 데 사용할 수 있는 다양한 도구, 코드 예제, 라이브러리가 있지만, 여기서는 과정을 단순화하기 위해 Postman을 사용하겠습니다.

X API v2 Postman 컬렉션을 환경에 로드하려면 다음 버튼을 클릭하세요:

<Button href="https://app.getpostman.com/run-collection/9956214-784efcda-ed4c-4491-a4c0-a26470a67400">
  Postman에 X API v2 추가
</Button>

Postman에 X API v2 컬렉션을 로드한 뒤, Post counts > Recent Post counts 요청으로 이동하세요.

**2단계: 요청 인증**

X API에 올바르게 요청하려면 권한이 있는지 확인해야 합니다. 이 엔드포인트의 경우 [OAuth 2.0 App-Only](/ko/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token) 인증 방식으로 요청을 인증해야 합니다.

키와 토큰, 특히 [App Access Token](/ko/resources/fundamentals/authentication#using-and-generating-an-app-only-bearer-token)(App-only Bearer Token이라고도 함)을 Postman에 추가해야 합니다. Postman 오른쪽 상단에서 “X API v2”라는 이름의 환경을 선택한 다음, 환경 드롭다운 옆의 눈 아이콘을 클릭해 "initial value" 및 "current value" 필드에 키와 토큰을 추가하세요.

이 변수가 올바르게 설정되었다면 요청의 Authorization 탭에 자동으로 적용됩니다.
 

**3단계: 쿼리 생성**

각 최근 Post 수 집계 요청에는 단일 [query](/ko/x-api/posts/counts/integrate/build-a-query)가 필요합니다. 이 예에서는 @XDevelopers 계정이 게시한 Post에 매칭되는 쿼리를 사용하겠습니다. 이 쿼리에서는 from: 연산자를 사용하고 값을 XDevelopers로 설정합니다(대소문자 구분 없음):

`from:XDevelopers`

Postman에서 "Params" 탭으로 이동하여 이 값을 `query` 매개변수의 "Value" 열에 입력하세요.

|     |     |     |
| :--- | :--- | :--- |
| **Key** | **Value** | **Description** |
| `query` | from:XDevelopers | 최근 Post 수 집계 엔드포인트에 제출할 쿼리 |

**4단계(선택): 요청의 세분화 수준 지정**

3단계 후 ‘Send’ 버튼을 클릭하면 기본 최근 Post 수 집계 결과(지난 7일 기준 시간 단위)를 받게 됩니다. 일 단위로 받으려면 세분화 수준 매개변수에 day 값을 추가하세요.

Postman에서 "Params" 탭으로 이동하여 granularity 매개변수의 "Value" 열에 day를 입력하세요.

|     |     |     |
| :--- | :--- | :--- |
| **Key** | **Value** | **Description** |
| granularity | day | Post 수 집계 결과의 세분화 수준. 가능한 값: day, hour, minute |

이제 "Send" 버튼 옆에 다음 URL이 표시됩니다:

`https://api.x.com/2/tweets/counts/recent?query=from%3AXDevelopers&granularity=day`
