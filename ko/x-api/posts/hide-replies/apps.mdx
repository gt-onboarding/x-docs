  ## 답글 숨기기 엔드포인트를 사용하는 앱
</div>

다음은 답글 숨기기 엔드포인트를 통합한 개발자들이 만든 주목할 만한 앱 모음입니다. 이들이 X API와 자사 기술을 결합하여 모든 사용자가 공개 대화를 더욱 건전하게 만들 수 있는 유용한 경험을 제공하고 있습니다.

![Clarabridge](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/clarabridge.jpg.twimg.1920.jpg)

**Clarabridge**

Clarabridge는 답글 숨기기를 통합하여 기업이 X 피드를 깔끔하고 읽기 좋게 유지하도록 돕습니다. 답글은 수동으로 숨길 수 있으며, 상담원은 Clarabridge의 텍스트 분석을 활용해 비속어 및 브랜드 정책에 어긋나는 표현을 자동으로 감지할 수도 있습니다.

[웹사이트로 이동 ▸](https://www.clarabridge.com/ "웹사이트로 이동 ▸")

[X의 Clarabridge](https://x.com/Clarabridge "X의 Clarabridge")

![Perspective API template app](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/perspective-logo.png.twimg.1920.png)

**Perspective API 템플릿 앱**

Perspective는 Jigsaw(Alphabet 산하 조직)가 유해 댓글을 감지하도록 학습시킨 인공지능입니다. 이 앱은 사용자가 받은 답글을 감지하고, 과거에 본 유해 댓글과 유사할 가능성에 대한 Perspective의 확신도를 나타내는 “점수”에 따라 해당 답글을 자동으로 숨깁니다. 이는 X가 개발한 오픈 소스 템플릿 앱입니다. 사용자를 위한 도구를 구축할 수 있도록 자유롭게 맞춤화해 보시기 바랍니다.

[앱 사용 ▸](https://quintessential-yoke.glitch.me "앱 사용 ▸")

[X의 Jigsaw](https://x.com/jigsaw "X의 Jigsaw")

![Reshuffle](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 150 150%22%3E%3C/svg%3E)

**Reshuffle**

Reshuffle은 유연한 스크립트로 비즈니스 애플리케이션을 연결하는 플랫폼입니다. 이들은 사용자가 정의한 키워드를 포함하는 답글을 감지해 숨기는 통합 스크립트를 만들었습니다. 사용해 보려면 먼저 답글 숨기기가 활성화된 유효한 앱이 있는지 확인한 뒤, 저장소의 안내를 따르세요.

[앱으로 이동 ▸](https://github.com/reshufflehq/reshuffle "앱으로 이동 ▸")

[X의 Reshuffle](https://x.com/reshufflehq "X의 Reshuffle")

![Hide unwanted replies](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 396 397%22%3E%3C/svg%3E)

**원치 않는 답글 숨기기**

인기 앱 QuotedReplies의 개발자인 Dara Oladosu는 과거에 자신이 음소거한 특정 키워드를 포함하는 답글 등, 학대적 행동을 보일 가능성이 높다고 자신이 판단한 몇 가지 기준을 충족하는 답글을 자동으로 숨기는 앱을 만들었습니다.

[앱으로 이동 ▸](https://hideunwantedreplies.com/ "앱으로 이동 ▸")

[X의 Dara Oladosu](https://x.com/dara_tobi "X의 Dara Oladosu")

![Pandaflow](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 94 94%22%3E%3C/svg%3E)

**Pandaflow**

Pandaflow는 워크플로를 자동화하는 노코드 플랫폼입니다. 사용자가 정의한 커스텀 플로에 따라 프로그래매틱하게 답글을 숨길 수 있도록 이 엔드포인트를 통합했습니다.

[앱으로 이동 ▸](https://pandaflow.io/ "앱으로 이동 ▸")

[X의 Pandaflow](https://x.com/pandaflowio "X의 Pandaflow")
