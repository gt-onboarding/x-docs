  ## X API의 답글 숨기기 엔드포인트 비교
</div>

v2 답글 숨기기 엔드포인트가 Labs 답글 숨기기 엔드포인트를 대체합니다. 해당 엔드포인트의 Labs 버전을 사용하는 코드, 앱, 또는 도구가 있고, 최신 X API v2 엔드포인트로 마이그레이션을 고려 중이라면 이 가이드를 참고하세요.

새로운 X API v2(답글 숨기기 엔드포인트 포함)를 사용하려면 [새 개발자 포털에 옵트인](https://developer.x.com/en/portal/opt-in)하고, [Project](/ko/resources/fundamentals/projects)를 생성한 뒤 해당 Project에 App을 추가해야 합니다. 그런 다음 그 App과 연결된 자격 증명을 사용해 답글 숨기기 엔드포인트에 요청을 보낼 수 있습니다. Labs v2 답글 숨기기 엔드포인트에 등록된 동일한 App을 추가하면 사용자의 인증 상태가 유지됩니다.

다음 표는 Labs와 최신 X API v2 엔드포인트의 차이를 비교합니다:
 

| **설명** | **Labs v2** | **X API v2** |
| :--- | :--- | :--- |
| 호스트 도메인 | https://api.x.com | https://api.x.com |
| 엔드포인트 경로 | /labs/2/tweets/:id/hidden | /2/tweets/:id/hidden |
| 인증 | OAuth 1.0a User context | OAuth 1.0a User context |
| 지원 HTTP 메서드 | PUT | PUT |
| 기본 요청 속도 제한 | 15분당 10회 요청(모든 인증 사용자 간 공유) | 15분당 50회 요청(인증 사용자별) |
| 답글 숨기기 가능 | ✔︎  | ✔︎  |
| 이전에 숨긴 답글 숨김 해제 가능 | ✔︎  | ✔︎  |
| 답글을 여러 번 숨기거나 숨김 해제 가능 | ✔︎  | ✔︎  |
| [developer App](/ko/resources/fundamentals/developer-apps)의 자격 증명이 [project](/ko/resources/fundamentals/projects)와 연결되어 있어야 함 |     | ✔   |
