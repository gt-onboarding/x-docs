다음 표는 X API v2 엔드포인트를 표준 v1.1 및 엔터프라이즈 엔드포인트에 대응시켜 보여줍니다. 각 버전과 티어에 대해 자세히 알아보려면 [X API 시작하기 가이드](/ko/x-api/getting-started/about-x-api)를 참조하세요.

여러 항목이 아직 \*\*\[곧 제공 예정]\*\*으로 표시되어 있을 수 있습니다. 이 표에서 **\[대체 검토 중]** 또는 \*\*\[대체 계획 없음]\*\*으로 표시된 항목에 대해 해당 엔드포인트의 v2 버전 제공을 원하신다면, [커뮤니티 포럼](https://devcommunity.x.com/) 또는 엔터프라이즈 계정 관리자를 통해 의견을 보내주세요.
