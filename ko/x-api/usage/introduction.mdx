X API v2의 Usage API를 사용하면 개발자가 프로젝트 사용량을 프로그래밍 방식으로 조회할 수 있습니다. 이 엔드포인트를 사용하면 특정 청구 주기에 가져온 게시물(Post) 수를 추적하고 모니터링할 수 있습니다.

개발자는 GET 엔드포인트를 통해 최대 지난 90일간의 일별 프로젝트 사용량을 확인할 수 있습니다. 또한 프로젝트에 연결된 각 클라이언트 앱별로 사용량을 집계할 수도 있습니다.

이 GET 엔드포인트에는 15분당 50회라는 앱 수준의 요청 한도가 적용됩니다.

<div id="getting-started">
  ### 시작하기
</div>

이 엔드포인트를 사용하려면 [개발자 포털](https://developer.x.com/en/portal/dashboard)에서 \[bearer 토큰]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)이 필요합니다. bearer 토큰을 획득한 후, 아래와 같이 사용량 API를 호출할 수 있습니다:

```
      curl 'https://api.x.com/2/usage/tweets' --header 'Authorization: Bearer XXXXX'
```

요청이 성공하면 아래와 같은 JSON 응답을 확인할 수 있습니다:

```
{
    "data": {
        "cap_reset_day": 10,
        "project_cap": "5000000000",
        "project_id": "1369785403853424",
        "project_usage": "43435"
    }
}
```

<Note>
  **계정 설정**

  이러한 엔드포인트에 액세스하려면 다음이 필요합니다:

  - 승인된 [개발자 계정](https://developer.x.com/en/portal/petition/essential/basic-info)
  - [프로젝트](/ko/resources/fundamentals/projects) 내에 위치한 [개발자 앱](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용한 인증

  X API v2 엔드포인트에 대한 액세스 권한을 얻는 방법에 대한 자세한 내용은 [시작하기 가이드](/ko/x-api/getting-started/getting-access)를 참조하세요.
</Note>
