Trends 조회 엔드포인트는 개발자가 where-on-earth ID(WOEID)를 사용해 지정한 위치의 트렌드를 가져올 수 있도록 합니다.

<Note>
  **참고**: WOEID는 Yahoo에서 만든 레거시 식별자이며 더 이상 사용되지 않습니다. X API는 도시 및 국가 트렌드 위치를 식별하기 위해 이 숫자 값을 사용합니다. 레거시 [블로그 게시물](https://blog.x.com/engineering/en_us/a/2010/woeids-in-twitters-trends.html) 또는 [아카이브 데이터](https://archive.org/details/geoplanet_data_7.10.0.zip0.)를 참고하세요.
</Note>

가능한 경우 다수의 트렌드에 대해 지난 24시간의 `tweet_count`도 반환됩니다.

이 엔드포인트는 앱 인증(app-auth)을 지원하며, 15분당 75건의 요청 한도가 적용됩니다.

<div id="getting-started">
  ### 시작하기
</div>

이 엔드포인트를 사용하려면 \[베어러 토큰]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token) from the [developer portal](https://developer.x.com/en/portal/dashboard). Once you have the bearer token,  you can call the usage API as shown below:

```
      curl 'https://api.x.com/2/trends/by/woeid/26062' --header 'Authorization: Bearer XXXXX'
```

요청이 성공하면 아래와 같은 JSON 응답을 확인할 수 있습니다:
