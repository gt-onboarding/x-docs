title: "X API 소개"
---

X API는 X 데이터를 프로그래밍 방식으로 가져와 분석할 수 있을 뿐 아니라, X에서 이루어지는 대화를 기반으로 제품을 구축하는 데에도 사용할 수 있습니다.

수년간 X API는 개발자가 공개 대화를 확장해 분석·연구할 수 있도록 추가적인 액세스 수준을 도입하며 발전해 왔습니다.

최근 X API v2를 출시했습니다. X API v2는 현대적인 기반 위에 새롭고 고급 기능을 제공하며, [Basic access](https://developer.x.com/en/portal/products/basic)를 빠르게 온보딩할 수 있도록 지원합니다.

아래의 세 개 탭에서는 X API의 버전과 액세스 수준, v2의 변경 사항, 그리고 API로 조회, 생성, 삭제, 조정할 수 있는 X 리소스를 설명합니다.

  <Tab title="What's new with v2">
    ## v2의 새로운 사항

    X API v2는 2012년 이후 X API의 가장 큰 업그레이드입니다. 이를 통해 다양한 신규·고급 기능이 도입되었으며, API에 빠르고 무료로 접근할 수 있습니다.

    v2에서 제공되는 기능은 다음과 같습니다:

    ### 새로운 엔드포인트

    X API v2에 완전히 새로운 엔드포인트 세트를 출시했습니다. 새로운 엔드포인트를 포함한 v2 엔드포인트 전체 목록은 X API 엔드포인트 맵 가이드에서 확인할 수 있습니다.

    [Visit the X API endpoint map >](/ko/x-api/migrate/x-api-endpoint-map)

    ### 더욱 상세해진 데이터 오브젝트

    데이터를 더 쉽게 탐색하고 파싱할 수 있도록 다양한 개선을 통해 데이터 오브젝트를 현대화했습니다.

    [Visit the data dictionary >](/ko/x-api/fundamentals/data-dictionary)

    ### 원하는 오브젝트와 필드만 가져오도록 돕는 신규 파라미터

    기본적으로 반환되는 필드 외에도 연관된 오브젝트와 필드를 요청할 수 있도록 데이터 엔드포인트에 fields 및 expansions 파라미터를 추가했습니다.

    [Learn how to use fields and expansions >](/ko/x-api/fundamentals/data-dictionary#how-to-use-fields-and-expansions)

    ### 고급 지표

    노출 수, 동영상 조회수, 사용자 프로필, URL 클릭 등 공용 및 비공개 지표를 함께 요청하여 페이로드 내에서 Posts, 사용자, 미디어, 설문의 성과를 더 쉽게 파악할 수 있습니다. 일부 지표는 오가닉과 프로모션 맥락으로 구분됩니다.

    [Learn more about metrics >](/ko/x-api/fundamentals/metrics)

    ### 다양한 주제를 포함한 Posts 필터링 및 식별

    search Posts 또는 filtered stream을 사용할 때 엔터티 및 컨텍스트 연산자를 통해 주제별로 필터링할 수 있습니다. 또한 분석을 돕기 위해 Post 페이로드에 이러한 주제를 포함했습니다.

    [Learn more about Post annotations >](/ko/x-api/fundamentals/post-annotations)

    ### 답글 스레드에 속한 Posts 필터링 및 식별

    search Posts, filtered stream, Post lookup을 사용할 때 대화 스레드의 일부인 Post를 더 쉽게 식별할 수 있습니다. 또한 Post 필드 `reply_settings`로 해당 Post에 대화 답글 설정이 적용되었는지 판단할 수 있도록 기능을 추가했습니다.

    [Learn more about conversation tracking >](/ko/x-api/fundamentals/conversation-id)

    ### 그 밖에도...

    - 높은 신뢰도의 스팸 필터링
    - URL 분석을 위해 단축 URL 완전 해제
    - 사용 중단 필드 제거 및 라벨 현대화를 통한 간소화된 JSON 응답 오브젝트
    - 스트리밍 엔드포인트의 복구 및 중복성 기능
    - 검색 쿼리에서 일치하는 공개·사용 가능한 Posts 100% 반환
    - 연결을 끊지 않고 변경할 수 있는 스트리밍 "rules"
    - filtered stream과 검색을 위한 더 표현력 있는 쿼리 언어
    - 새로운 라이브러리 구축과 변경 사항의 투명한 추적을 위한 OpenAPI 사양
    - 개발자, 연구자, 기업, X 사용자 요구에 맞춰 플랫폼이 발전함에 따라 새로운 기능과 엔드포인트에 대한 더 빠른 API 지원
  </Tab>

  <Tab title="X API resources">
    ## X API 플랫폼 리소스

    API 설계 맥락에서 리소스는 관련 데이터, 다른 리소스와의 관계, 그리고 해당 리소스에 작동하는 메서드 집합을 갖는 엔터티를 의미합니다. 예를 들어, 게시물(Post)은 다양한 도구를 사용해 생성, 삭제, 조회할 수 있는 리소스이며, 전체 기록을 검색하거나 실시간으로 가져오는 방식이 있습니다.

    X API는 플랫폼의 다양한 리소스를 생성, 삭제, 수신 또는 조정할 수 있는 액세스를 제공하며, 다음이 포함됩니다:

    | Resource            | Description                                                                                                                                                                              |
    | :------------------ | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
    | **Posts**           | 수백만 건의 게시물을 활용해 공개 대화를 이해하거나, 직접 게시물을 만들어 대화에 참여할 수 있습니다.  현재 사용 가능:  - X API v2  - Enterprise  - Standard v1.1 |
    | **Users**           | 네트워크 분석, 오디언스 이해, 긍정적인 온라인 관계 형성을 위해 X 사용자를 관리하거나 조회할 수 있습니다.  현재 사용 가능:  - X API v2  - Enterprise  - Standard v1.1     |
    | **Spaces**          | 사람들이 흥미롭고 관련성 높은 오디오 대화를 찾을 수 있도록 X Spaces와 참석자를 조회하고 검색할 수 있습니다.  현재 사용 가능:  - X API v2                                    |
    | **Direct Messages** | 고객 이슈 분류, 환영 메시지 전송, 긍정적인 상호작용을 위해 다이렉트 메시지를 보내고 받을 수 있습니다.  현재 사용 가능:  - Standard v1.1                        |
    | **Lists**           | 업계 전문가와 영향력 있는 목소리를 파악하거나 팔로우 대상을 정리하기 위해 계정 목록을 선별하고 관리할 수 있습니다.  현재 사용 가능:  - X API v2  - Standard v1.1               |
    | **Trends**          | 지리적 트렌드를 조기에 파악해 업계 동향을 짚고, 화제 주제를 발견하며, 최신 유행을 앞서갈 수 있습니다.  현재 사용 가능:  - Standard v1.1                           |
    | **Media**           | 창의성을 공유하고 인터랙티브한 경험을 만들거나 접근성 도구를 구축하기 위해 미디어 객체를 업로드할 수 있습니다.  현재 사용 가능:  - Standard v1.1                               |
    | **Places**          | 내 주변과 전 세계에서 일어나는 일을 이해하기 위해 장소를 검색할 수 있습니다.  현재 사용 가능:  - Standard v1.1                                                     |
  </Tab>
