<div id="learn-about-whats-possible">
  ## 가능한 사항 알아보기
</div>

상단 내비게이션에서 접근할 수 있는 사용 사례, 솔루션, 제품 페이지는 플랫폼에서 무엇을 할 수 있는지 고수준에서 빠르게 파악하는 데 유용합니다.

또한 다양한 사용 사례를 충족하기 위해 하나 또는 여러 플랫폼 도구를 어떻게 활용할 수 있는지 알아보는 데 도움이 되는 [튜토리얼](/ko/tutorials)도 다수 제공합니다.

가장 하위 수준의 문서에는 엔드포인트 및 도구 문서가 있으며, 모두 해당 도구의 기능과 일반적인 사용 사례를 간략히 소개하는 도입 콘텐츠를 포함합니다. 예를 들어, [Search Posts](/ko/x-api/posts/search/introduction)에 대한 X API v2 엔드포인트 섹션이 있습니다.

<div id="integrate-an-endpoint-into-your-system">
  ### 엔드포인트를 시스템에 통합하기
</div>

엔드포인트와의 통합을 원활하게 할 수 있도록 다양한 도구, 라이브러리, 통합 가이드를 제공하고 있습니다. 가장 간단한 통합 방법은 [도구와 라이브러리](/ko/x-api/tools-and-libraries/overview)를 사용하는 것으로, [OAuth 1.0a 사용자 컨텍스트 인증](/ko/resources/fundamentals/authentication#oauth-1-0a-2), 페이지네이션, [요율 제한 처리](/ko/resources/fundamentals/rate-limits)와 같은 복잡한 기능을 자동으로 처리합니다.

처음부터 솔루션을 구축하려면, 엔드포인트 통합 가이드에서 엔드포인트를 시스템에 통합하는 데 필요한 기능과 모범 사례를 설명합니다. 이러한 가이드는 다양한 엔드포인트 및 도구 문서에서 ‘integrate’ 또는 ‘guides’로 표시되어 있습니다. 예를 들어, [필터드 스트림을 위한 규칙을 만드는 방법](/ko/x-api/posts/filtered-stream/integrate/build-a-rule)에 대한 통합 가이드가 있습니다.

<div id="troubleshoot-an-issue">
  ### 문제 해결
</div>

오류가 발생했거나 문의 사항이 있다면 [지원 허브](https://developer.x.com/en/support/twitter-api)를 확인하세요. 문제 해결 팁, 자주 묻는 질문(FAQ) 답변, 기타 유용한 자료를 확인하실 수 있습니다.

<div id="keep-up-to-date-with-the-latest-releases">
  ### 최신 릴리스를 계속 확인하세요
</div>

우리는 API를 통해 새로운 X 기능을 제공하기 위해 플랫폼 업데이트를 자주 배포합니다. 이와 함께 매년 최소 한 번은 주요 버전 업그레이드를 목표로 하며, 이 과정에서 하위 호환성이 깨지는 변경 사항이 포함될 수 있습니다.

최신 업데이트를 받으려면 [포럼의 공지사항 카테고리](https://devcommunity.x.com/c/announcements/22)를 구독하고, X에서 [@API](https://x.com/api)와 [@XDevelopers](https://x.com/XDevelopers)를 팔로우한 뒤 알림을 켜며, [최신 소식](https://developer.x.com/en/stay-informed) 페이지에서 다양한 자료를 확인하세요.

또한 최신 API 버전으로 업데이트하는 과정을 안내하기 위해 새로운 [마이그레이션 허브](/ko/x-api/migrate/overview)를 구축했습니다.
