<div id="official-resources">
  ## Ressources officielles
</div>

Les outils ci-dessous peuvent vous aider à démarrer avec X pour les sites web.

<CardGroup cols={3}>
<Card
  title="publish.x.com"
  icon="x-twitter"
  href="https://publish.x.com/"
>
Créez votre propre Tweet intégré, fil chronologique ou bouton.
</Card>
<Card
  title="Twitter for WordPress"
  icon="x-twitter"
  href="https://github.com/twitter/wordpress"
>
Une extension officielle pour optimiser votre site WordPress pour X, notamment l’intégration de Tweets et les X Cards.
</Card>
<Card
  title="twemoji"
  icon="x-twitter"
  href="https://twemoji.x.com/"
>
Jeu de caractères emoji gratuit et open source de X, avec une bibliothèque JavaScript pour la compatibilité multiplateforme.
</Card>
</CardGroup>
