<div id="community-tools-and-libraries">
  ## Outils et bibliothèques de la communauté
</div>

Voici quelques-unes des nombreuses bibliothèques maintenues par la communauté qui couvrent l’API publicitaire X, dans plusieurs langages et sur plusieurs plateformes. Sauf indication contraire, les bibliothèques listées ici devraient prendre en charge la plupart des fonctionnalités de l’API publicitaire ; consultez les auteurs pour plus de détails.

Si vous avez créé une bibliothèque compatible avec les API X, veuillez nous en informer via nos [forums de la communauté des développeurs](https://devcommunity.x.com/c/libraries-and-sdks/63).
