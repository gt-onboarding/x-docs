---
title: Sandbox
---

Le sandbox est un environnement qui vous permet de tester et de développer votre implémentation de l’API publicitaire X.

Les campagnes créées via le sandbox ne seront pas diffusées auprès des utilisateurs et des enregistrements de test pour des objets tels que les instruments de financement peuvent être créés dynamiquement.

L’utilisation du sandbox est aussi simple que d’appeler le nom d’hôte approprié ads-api-sandbox.x.com, mais il inclut également des fonctionnalités uniques propres au sandbox.

Envoyez d’abord une requête [POST accounts](/fr/x-ads-api/campaign-management#post-accounts) pour créer un nouveau compte annonceur. La réponse à cet appel contiendra un account\_id qui pourra être utilisé pour les requêtes suivantes. Ces endpoints permettent de créer et de tester plusieurs comptes annonceurs.

<div id="sandbox-only-endpoint-list">
  ### Liste des endpoints réservés à l’environnement sandbox :
</div>

- [POST accounts](/fr/x-ads-api/campaign-management#post-accounts) Créer un compte dans l’environnement sandbox
- [POST accounts/:account\_id/features](/fr/x-ads-api/campaign-management#post-accounts-account-id-features) Ajouter une fonctionnalité de compte à un compte dans l’environnement sandbox
- [POST accounts/:account\_id/funding\_instruments](/fr/x-ads-api/campaign-management#post-accounts-account-id-funding-instruments) Ajouter un instrument de financement à un compte dans l’environnement sandbox
- [DELETE accounts/:account\_id](/fr/x-ads-api/campaign-management#delete-accounts-account-id) Supprimer un compte dans l’environnement sandbox
- [DELETE accounts/:account\_id/features](/fr/x-ads-api/campaign-management#delete-accounts-account-id-features) Supprimer une fonctionnalité de compte d’un compte dans l’environnement sandbox
- [DELETE accounts/:account\_id/funding\_instruments/:funding\_instrument\_id](/fr/x-ads-api/campaign-management#delete-accounts-account-id-funding-instruments-funding-instrument-id) Supprimer un instrument de financement d’un compte dans l’environnement sandbox
