---
title: Devise
---

Le type d’une devise est identifié à l’aide de la norme [ISO 4217](http://en.wikipedia.org/wiki/ISO_4217). Il s’agit d’une chaîne de trois lettres, comme `USD` ou `EUR`. La valeur d’une devise est représentée en micros. Pour l’USD, 5,50 $ est encodé comme 5,50\*1e6, soit 5 500 000. Pour représenter une « valeur entière », vous devez multiplier la valeur locale en micro par 1e6 (1 000 000) pour toutes les devises.

L’API publicitaire X reconnaît les devises officielles de tous les pays où le produit X SMB Ads est disponible. Pour la liste complète de ces pays, cliquez [ici](https://business.x.com/en/help/overview/about-eligibility-for-twitter-ads.html).

La plupart des devises ont une « précision » de deux (par exemple USD : 1,23 $).

- `AED`, `VEF`, `PKR`, `IDR`, `HKD`, `QAR`, `PHP`, `NZD`, `INR`, `EUR`, `ARS`, `THB`, `MXN`, `EGP`, `DZD`, `SAR`, `LBP`, `COP`, `TRY`, `SGD`, `MAD`, `AUD`, `USD`, `MYR`, `CAD`, `BRL`, `GBP`.

Les devises suivantes n’utilisent pas de décimales :

- `VND`, `KRW`, `CLP`, `JPY`

Et celles-ci utilisent trois décimales :

- `KWD`, `IQD`, `BHD`, `TND`

L’utilisation des devises se fait via les [instruments de financement](/fr/x-ads-api/introduction).
