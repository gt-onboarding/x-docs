<div id="basic-authentication">
  ## Authentification Basic
</div>

De nombreuses API Enterprise de X requièrent l’authentification HTTP Basic. Pour effectuer une requête vers une API qui exige l’authentification Basic, vous devez transmettre, dans l’en-tête d’autorisation de chaque requête, une combinaison valide d’adresse e‑mail et de mot de passe. Cette combinaison est la même que celle utilisée pour accéder à la [console des API Enterprise](https://console.gnip.com/), et elle peut y être modifiée.

Lors de la construction d’une requête avec l’authentification Basic, veillez à ajouter l’en-tête HTTP Authorization: Basic avec des identifiants encodés, et à utiliser HTTPS.

Dans l’exemple de requête cURL suivant, remplacez `<email_address>` et `<password>` par vos identifiants avant d’envoyer la requête :

```bash
curl -v --compressed -u<email_address>:<password>
    "https://gnip-api.x.com/search/30day/accounts/<account-name>/prod/counts.json?query=from%3Axdevelopers"
```

- [Decahose stream API](http://localhost:3000/x-api/enterprise-gnip-2.0/fundamentals/decahose-api) Enterprise
- [30-Day Search API](/fr/x-api/enterprise-gnip-2.0/fundamentals/search-api) Enterprise
- [Full-Archive Search API](/fr/x-api/enterprise-gnip-2.0/fundamentals/search-api) Enterprise
