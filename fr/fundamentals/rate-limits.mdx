title: Limites de débit
icon: gauge-simple-high
---

Chaque jour, des dizaines de milliers de développeurs envoient des requêtes vers la plateforme développeur X. Pour gérer le volume élevé de ces requêtes, des limites sont appliquées au nombre de requêtes pouvant être effectuées. Ces limites nous permettent de proposer une API fiable et évolutive sur laquelle notre communauté de développeurs s’appuie.

Chacune de nos API applique les limites de débit différemment. Pour en savoir plus sur ces variations entre plateformes, veuillez consulter les pages dédiées aux limites de débit dans les sections correspondantes de nos API :

- [API X v2](/fr/x-api/fundamentals/rate-limits)
- [X API : Enterprise](/fr/x-api/enterprise-gnip-2.0/fundamentals/rate-limits)
- [API publicitaire X](/fr/x-ads-api/fundamentals/rate-limiting)
