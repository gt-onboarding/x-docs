Le tableau suivant fait correspondre les endpoints de l’API X v2 aux endpoints standard v1.1 et Entreprise. Pour en savoir plus sur chacune de ces versions et niveaux d’accès, veuillez consulter notre [guide de prise en main de l’API X](/fr/x-api/getting-started/about-x-api).

Vous constaterez que plusieurs éléments sont encore marqués **\[Bientôt disponible]**. Si vous repérez dans ce tableau des éléments marqués **\[Remplacement à l’étude]** ou **\[Aucun remplacement prévu]** et que vous souhaitez qu’une version v2 de cet endpoint soit publiée, merci de nous le faire savoir via notre [forum communautaire](https://devcommunity.x.com/) ou votre responsable de compte Entreprise.
