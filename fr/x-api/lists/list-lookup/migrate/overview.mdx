<div id="comparing-x-apis-list-lookup-endpoints">
  ## Comparaison des endpoints de recherche de listes de l’API X
</div>

Le groupe d’endpoints v2 de recherche de listes remplacera les endpoints standard v1.1 [GET lists/show](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-show) et [GET lists/ownership](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-ownerships). Si vous avez du code, des applications ou des outils qui utilisent l’une de ces versions des endpoints de recherche de listes et que vous envisagez de migrer vers le nouvel endpoint de l’API X v2, cet ensemble de guides est fait pour vous.

Les tableaux suivants comparent les endpoints standard v1.1 et de l’API X v2 pour les listes :

**Recherche de liste par ID**

|     |     |     |
| :--- | :--- | :--- |
| Description | Standard v1.1 | API X v2 |
| Méthodes HTTP prises en charge | `GET` | `GET` |
| Domaine hôte | `https://api.x.com` | `https://api.x.com` |
| Chemin de l’endpoint | `/1.1/lists/show.json` | `/2/lists/:id` |
| [Authentication](/fr/resources/fundamentals/authentication) | Contexte utilisateur OAuth 1.0a<br /><br />App only | Contexte utilisateur OAuth 1.0a<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />App only |
| [Limites de débit](/fr/resources/fundamentals/rate-limits) de requête par défaut | 75 requêtes par 15 min avec OAuth 1.0a<br /><br />75 requêtes par 15 min avec OAuth 2.0<br /><br />75 requêtes par 15 min avec App only | 75 requêtes par 15 min avec OAuth 1.0a<br /><br />75 requêtes par 15 min avec OAuth 2.0<br /><br />75 requêtes par 15 min avec App only |

**Recherche de listes appartenant à l’utilisateur**

|     |     |     |
| :--- | :--- | :--- |
| Description | Standard v1.1 | API X v2 |
| Méthodes HTTP prises en charge | `GET` | `GET` |
| Domaine hôte | `https://api.x.com` | `https://api.x.com` |
| Chemin de l’endpoint | `/1.1/lists/ownerships.json` | `/2/users/:id/owned_lists` |
| [Authentication](/fr/resources/fundamentals/authentication) | Contexte utilisateur OAuth 1.0a<br /><br />App only | Contexte utilisateur OAuth 1.0a<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />App only |
| [Limites de débit](/fr/resources/fundamentals/rate-limits) de requête par défaut | 15 requêtes par 15 min avec OAuth 1.0a<br /><br />15 requêtes par 15 min avec App only | 15 requêtes par 15 min avec OAuth 1.0a<br /><br />15 requêtes par 15 min avec OAuth 2.0<br /><br />15 requêtes par 15 min avec App only |

Pour accéder aux endpoints de l’API X v2, vous devez [vous inscrire pour obtenir un compte développeur](https://developer.x.com/en/portal/petition/essential/basic-info). Lors de l’authentification, vous devez utiliser des clés et des jetons d’une [application développeur](/fr/resources/fundamentals/developer-apps) qui se trouve dans un [projet](/fr/resources/fundamentals/projects).

Pour en savoir plus sur l’accès aux endpoints de l’API X v2, consultez notre page [Pour commencer](/fr/x-api/getting-started/getting-access).
