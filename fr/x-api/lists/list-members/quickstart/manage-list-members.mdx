<div id="manage-list-members-quick-start-guide">
  ### Guide de démarrage rapide pour gérer les membres d’une liste
</div>

<Note>
  **Remarque :** ce guide part du principe que vous avez terminé les prérequis de la [vue d’ensemble du démarrage rapide](/fr/x-api/lists/list-members#getting-started-with-the-list-members-endpoint-group).
</Note>

<div id="steps-to-build-a-manage-list-member-request">
  ### Étapes pour créer une requête de gestion des membres de liste
</div>

<div id="step-one-choose-the-list-endpoint-collection-from-postman">
  #### Première étape : sélectionnez la collection d’endpoints List dans Postman
</div>

Une fois la collection de l’API X v2 chargée dans Postman, ouvrez le dossier « List », puis le sous-dossier « List members », et enfin choisissez « Add a member ».
 

<div id="step-two-specify-the-user-to-add">
  #### Deuxième étape : spécifier l’utilisateur à ajouter
</div>

Les endpoints de gestion des membres de liste requièrent deux identifiants : l’ID de la liste et l’ID de l’utilisateur à ajouter. Vous pouvez trouver l’ID de l’utilisateur à l’aide de [users lookup](/fr/x-api/users/lookup/introduction) en passant un nom d’utilisateur pour obtenir le champ id.

L’ID cible peut être n’importe quel ID d’utilisateur valide. Dans Postman, allez à l’onglet « Params » et saisissez votre ID dans la colonne « Value » de la variable de chemin id. Allez à l’onglet « Body » et indiquez l’ID de l’utilisateur à ajouter comme valeur du paramètre user\_id. Veillez à ne pas inclure d’espaces avant ou après un ID.

|     |     |     |
| :--- | :--- | :--- |
| **Key** | **Value** | **Parameter type** |
| `id` | L’ID de la liste | path |
| user\_id | L’ID de l’utilisateur cible à ajouter comme membre | body |

Vous devriez maintenant voir une URL similaire à côté du bouton « Send » :

`https://api.x.com/2/lists/1441162269824405510/members`

Troisième étape : envoyer la requête et vérifier la réponse

Une fois tout configuré, cliquez sur le bouton « Send ». Vous recevrez une réponse similaire à l’exemple suivant :

```
{
  "data": {
    "is_member": true
  }
}
```

Si l’objet de réponse contient la valeur true pour la clé is\_member, vous avez bien ajouté l’utilisateur comme membre de la liste. 

Pour supprimer un membre d’une liste, sélectionnez la requête « Remove a member », également présente dans le dossier « Lists » de la collection API X v2 chargée dans Postman. Cet endpoint requiert l’ID de la liste et l’ID de l’utilisateur à supprimer. Dans l’onglet « Params », saisissez l’ID de la liste comme valeur pour la colonne id et l’ID de l’utilisateur à supprimer comme valeur pour la colonne user\_id. 

En cas de suppression réussie, vous recevrez une réponse similaire à l’exemple suivant :

```
{
  "data": {
    "is_member": false
  }
}
```
