<div id="comparing-x-apis-lists-endpoints">
  ## Comparaison des endpoints Lists de l’API X
</div>

Les endpoints v2 de gestion des Lists remplaceront à terme [POST lists/create](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-create), [POST lists/destroy](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-destroy) et [POST lists/update](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-update). Si vous avez du code, des applications ou des outils qui utilisent une version antérieure de ces endpoints et envisagez de migrer vers la nouvelle API X v2, ce guide est pour vous.

Les tableaux suivants comparent les endpoints Lists de la v1.1 standard et de l’API X v2 :

**Créer une List**

| Description | v1.1 standard | API X v2 |
| :--- | :--- | :--- |
| Méthodes HTTP prises en charge | POST | POST |
| Domaine hôte | https://api.x.com | https://api.x.com |
| Chemin d’endpoint | /1.1/lists/create.json | /2/lists |
| [Authentification](/fr/resources/fundamentals/authentication) | Contexte utilisateur OAuth 1.0a | Contexte utilisateur OAuth 1.0a |
| [Limites de débit](/fr/resources/fundamentals/rate-limits) par défaut | Aucune | 300 requêtes par 15 min (par utilisateur) |

**Supprimer une List**

| Description | v1.1 standard | API X v2 |
| :--- | :--- | :--- |
| Méthodes HTTP prises en charge | POST | DELETE |
| Domaine hôte | https://api.x.com | https://api.x.com |
| Chemin d’endpoint | /1.1/lists/destroy.json | /2/lists/:id |
| [Authentification](/fr/resources/fundamentals/authentication) | Contexte utilisateur OAuth 1.0a | Contexte utilisateur OAuth 1.0a |
| [Limites de débit](/fr/resources/fundamentals/rate-limits) par défaut | Aucune | 300 requêtes par 15 min (par utilisateur) |

**Mettre à jour une List**

| Description | v1.1 standard | API X v2 |
| :--- | :--- | :--- |
| Méthodes HTTP prises en charge | POST | PUT |
| Domaine hôte | https://api.x.com | https://api.x.com |
| Chemin d’endpoint | /1.1/lists/update.json | /2/lists/:id |
| [Authentification](/fr/resources/fundamentals/authentication) | Contexte utilisateur OAuth 1.0a | Contexte utilisateur OAuth 1.0a |
| [Limites de débit](/fr/resources/fundamentals/rate-limits) par défaut | Aucune | 300 requêtes par 15 min (par utilisateur) |

Pour accéder aux endpoints de l’API X v2, vous devez [vous inscrire pour obtenir un compte développeur](https://developer.x.com/en/portal/petition/essential/basic-info). Lors de l’authentification, vous devez utiliser des clés et des jetons issus d’une [application développeur](/fr/resources/fundamentals/developer-apps) située au sein d’un [projet](/fr/resources/fundamentals/projects).

Pour en savoir plus sur l’accès aux endpoints de l’API X v2, consultez notre page [mise en route](/fr/x-api/getting-started/getting-access).
