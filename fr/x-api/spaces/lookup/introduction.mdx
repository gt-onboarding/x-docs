Les endpoints de recherche par ID renvoient des données que vous pouvez utiliser pour créer des expériences affichant des informations sur les Spaces à vos utilisateurs, ou pour obtenir des détails sur un ou plusieurs Spaces. Vous pouvez rechercher un ou plusieurs Spaces dans une même requête.

Vous pouvez également rechercher des Spaces par leur ID de créateur. Cet endpoint est utile pour récupérer les Spaces d’utilisateurs présents dans une liste, ou pour déterminer si plusieurs hôtes récurrents sont actuellement en direct ou ont un Space programmé à venir. Jusqu’à 100 utilisateurs peuvent être recherchés dans une seule requête.

**Configuration du compte**

Pour accéder à ces endpoints, vous aurez besoin de :

- D’un [compte développeur](https://developer.x.com/en/portal/petition/essential/basic-info) approuvé.
- De vous authentifier à l’aide des clés et des jetons d’une [application développeur](/fr/resources/fundamentals/developer-apps) située dans un [projet](/fr/resources/fundamentals/projects).

Pour en savoir plus sur l’accès aux endpoints de l’API X v2, consultez notre [guide de démarrage](/fr/x-api/getting-started/getting-access).
