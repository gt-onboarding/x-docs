Le point de terminaison de recherche des tendances permet aux développeurs d’obtenir les tendances pour un lieu, spécifié à l’aide de l’identifiant Where On Earth (WOEID).

<Note>
  Remarque : le WOEID est un identifiant hérité créé par Yahoo qui est désormais obsolète. L’API X utilise la valeur numérique pour identifier les emplacements de tendances aux niveaux ville et pays. Référez‑vous à notre ancien [blog post](https://blog.x.com/engineering/en_us/a/2010/woeids-in-twitters-trends.html), ou [archived data](https://archive.org/details/geoplanet_data_7.10.0.zip0.)
</Note>

Le `tweet_count` des dernières 24 heures est également renvoyé pour de nombreuses tendances lorsque cette information est disponible.

Ce point de terminaison prend en charge l’authentification app-auth et applique une limite de 75 requêtes par fenêtre de 15 minutes.

<div id="getting-started">
  ### Pour commencer
</div>

Pour utiliser cet endpoint, vous avez besoin d’un \[jeton Bearer]\(https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token) from the [developer portal](https://developer.x.com/en/portal/dashboard). Une fois que vous disposez du jeton Bearer, vous pouvez appeler l’API d’usage comme indiqué ci-dessous :

```
      curl 'https://api.x.com/2/trends/by/woeid/26062' --header 'Authorization: Bearer XXXXX'
```

Si la requête aboutit, vous devriez voir la réponse JSON suivante :

**Configuration du compte**

Pour accéder à ces endpoints, vous aurez besoin :

- D’un [compte développeur](https://developer.x.com/en/portal/petition/essential/basic-info) approuvé.
- De vous authentifier à l’aide des clés et des jetons d’une [application développeur](/fr/resources/fundamentals/developer-apps) située dans un [projet](/fr/resources/fundamentals/projects). 

Pour en savoir plus sur l’accès aux endpoints de l’API X v2, consultez notre [guide de démarrage](/fr/x-api/getting-started/getting-access).
