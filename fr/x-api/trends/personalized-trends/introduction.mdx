L’endpoint Personalized Trends permet d’obtenir des tendances personnalisées pour le compte utilisateur authentifié.

<Note>
  **Remarque :** Les tendances personnalisées ne sont renvoyées par cet endpoint que pour les comptes utilisateur abonnés à X Premium.
</Note>

<Note>
  **Configuration du compte**

  Pour accéder à cet endpoint, vous aurez besoin :

  - D’un [compte développeur](https://developer.x.com/en/portal/petition/essential/basic-info) approuvé.
  - D’authentifier en utilisant les clés et jetons d’une [application développeur](/fr/resources/fundamentals/developer-apps) située dans un [projet](/fr/resources/fundamentals/projects).

  Pour en savoir plus sur l’accès aux endpoints de l’API X v2, consultez notre [guide de démarrage](/fr/x-api/getting-started/getting-access).
</Note>

<div id="getting-started">
  ### Prise en main
</div>

<div id="authentication">
  #### Authentification
</div>

Vous pouvez authentifier cet endpoint soit avec [OAuth 1.0a en contexte utilisateur](/fr/resources/fundamentals/authentication), soit avec [OAuth 2.0 (code d’autorisation) avec PKCE](fr/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2).
 

<div id="making-a-request">
  #### Effectuer une requête
</div>

Vous pouvez appeler l’endpoint Personalized Trends comme indiqué ci-dessous :

`curl 'https://api.x.com/2/users/personalized_trends' --header 'Authorization: ••••••'`

Si la requête aboutit, vous devriez voir la réponse JSON suivante :

```
{
    "data": [
        {
            "category": "Bitcoin",
            "post_count": "42K posts",
            "trend_name": "MicroStrategy's $42 Billion Bitcoin Investment",
            "trending_since": "5 hours ago"
        },
        {
            "category": "Music",
            "post_count": "2K posts",
            "trend_name": "Drake's 'No Face' Music Video Release",
            "trending_since": "Trending now"
        },
        {
            "category": "Weather",
            "post_count": "1.2K posts",
            "trend_name": "Super Typhoon Leon Impacts Taiwan and Batanes",
            "trending_since": "Trending now"
        }
    ]
}
```

**Remarque :** Si l’utilisateur authentifié pour la requête n’est pas abonné à X Premium, il recevra la réponse suivante à la place :

```
{
    "data": [
        {
            "category": "Unknown",
            "post_count": "Unknown",
            "trending_since": ""
        }
    ]
}
```
